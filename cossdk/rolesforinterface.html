RolesForInterface collection

Contains an object for each role assigned to the interface to which the collection is related. The roles must already be assigned at the application level.

This collection supports the Add and Remove methods of the COMAdminCatalogCollection object.

Members

The RolesForInterface collection inherits from the IUnknown interface but does not have additional members.

Related Collections

You can navigate from this collection to any of the following collections:

  • ErrorInfo
  • PropertyInfo
  • RelatedCollectionInfo

You can navigate to this collection from the following collections:

  • InterfacesForComponent

Properties

The following properties are supported by the COMAdminCatalogObject object within the collection:

  • Name
