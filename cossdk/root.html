Root collection

Contains the top-level collections in the catalog. It does not contain any COMAdminCatalogObject objects or support any properties.

The Root collection does not support the Add and Remove methods of the COMAdminCatalogCollection object.

You cannot navigate to the Root collection from any collection.

Members

The Root collection inherits from the IUnknown interface but does not have additional members.

Related Collections

The following are the top-level collections in the catalog:

  • ApplicationCluster
  • ApplicationInstances
  • Applications
  • ComputerList
  • DCOMProtocols
  • EventClassesForIID
  • FilesForImport
  • InprocServers
  • LegacyServers
  • LocalComputer
  • Partitions
  • PartitionUsers
  • PropertyInfo
  • RelatedCollectionInfo
  • TransientSubscriptions
  • WOWInprocServers
  • WOWLegacyServers
