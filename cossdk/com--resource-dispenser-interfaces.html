COM+ Resource Dispenser Interfaces

Application components use resource dispensers to access shared information. The interfaces described in the following table provide detailed reference information for developers of resource dispensers.

Interfaces Description
IDispenserDriver This interface is called by the resource dispenser holder to create, enlist, evaluate, and destroy a resource.
IDispenserManager Resource dispensers use this interface to connect to the dispenser manager.
IHolder This interface is used to prepare and handle resources.
