ID3DXEffectStateManager::SetFVF method

A callback function that must be implemented by a user to set a FVF code.

Syntax

HRESULT SetFVF(
  [in]Â DWORD FVF
);

Parameters

FVF [in]

Type: DWORD

The FVF constant, that determines how to interpret vertex data. See D3DFVF.

Return value

Type: HRESULT

The user-implemented method should return S_OK. If the callback fails when setting the device state, either of the following will occur:

  • The effect will fail during ID3DXEffect::BeginPass.
  • The dynamic effect state call (such as IDirect3DDevice9::SetFVF) will fail.
