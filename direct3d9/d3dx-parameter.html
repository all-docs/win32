D3DX_PARAMETER

These flags provide additional information about effect parameters.

Effect parameter constants are used by D3DXPARAMETER_DESC.

Constant Description
D3DX_PARAMETER_ANNOTATION This parameter is marked as an annotation. See Add Information to Effect Parameters with Annotations.
D3DX_PARAMETER_LITERAL This parameter is marked as a literal value. Literal parameters cannot change after compile, allowing the compiler to optimize their usage. Shared parameters cannot be marked as a literal. See Usages and Literals (Direct3D 9).
D3DX_PARAMETER_SHARED The value of a parameter will be shared by all effects in the same namespace. Changing the value in one effect will change it in all shared effects. See Sharing Parameters. D3DX_PARAMETER_SHARED cannot be combined with D3DX_PARAMETER_LITERAL or D3DX_PARAMETER_ANNOTATION.
