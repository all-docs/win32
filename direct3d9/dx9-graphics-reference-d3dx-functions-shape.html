Shape Drawing Functions

D3DX supplies the following shape-drawing functions.

  • D3DXCreateBox
  • D3DXCreateCylinder
  • D3DXCreatePolygon
  • D3DXCreateSphere
  • D3DXCreateTeapot
  • D3DXCreateText
  • D3DXCreateTorus

All shapes are returned as mesh objects.

These functions can be used in either a left-handed or right-handed coordinate system.
