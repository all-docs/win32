ID3DXSkinInfo::Clone method

Clones a skin info object.

Syntax

HRESULT Clone(
  [in, out]Â LPD3DXSKININFO *ppSkinInfo
);

Parameters

ppSkinInfo [in, out]

Type: LPD3DXSKININFO*

Address of a pointer to an ID3DXSkinInfo object, which will contain the cloned object if the method is successful.

Return value

Type: HRESULT

If the method succeeds, the return value is D3D_OK. If the method fails, the return value can be D3DERR_INVALIDCALL.
