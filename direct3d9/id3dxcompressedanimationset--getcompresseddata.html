ID3DXCompressedAnimationSet::GetCompressedData method

Gets the data buffer that stores compressed key frame animation data.

Syntax

HRESULT GetCompressedData(
  [out]Â LPD3DXBUFFER *ppCompressedData
);

Parameters

ppCompressedData [out]

Type: LPD3DXBUFFER*

Address of a pointer to the ID3DXBuffer data buffer that receives compressed key frame animation data.

Return value

Type: HRESULT

If the method succeeds, the return value is S_OK. If the method fails, the following value will be returned: D3DERR_INVALIDCALL.
