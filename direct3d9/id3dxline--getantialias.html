ID3DXLine::GetAntialias method

Gets the line antialiasing state.

Syntax

BOOL GetAntialias();

Parameters

This method has no parameters.

Return value

Type: BOOL

Returns the antialiasing switch value. TRUE means antialiasing is on, and FALSE means antialiasing is off.
