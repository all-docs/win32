D3DSINCOSCONST1

Creates the first constant (in register src2) for the sincos - ps pixel shader instruction.

#define D3DSINCOSCONST1  -1.5500992e-006f, -2.1701389e-005f,  0.0026041667f, 0.00026041668f

Return Value

Returns the first constant needed by sincos - ps.
