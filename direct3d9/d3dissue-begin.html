D3DISSUE_BEGIN

This macro creates a value used by Issue to issue a query begin.

#define D3DISSUE_BEGIN (1 << 1)

Remarks

D3DISSUE_BEGIN is valid for the following query type.

  • D3DQUERYTYPE_OCCLUSION
