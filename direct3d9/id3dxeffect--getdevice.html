ID3DXEffect::GetDevice method

Retrieves the device associated with the effect.

Syntax

HRESULT GetDevice(
  [out]Â LPDIRECT3DDEVICE9 *ppDevice
);

Parameters

ppDevice [out]

Type: LPDIRECT3DDEVICE9*

Address of a pointer to an IDirect3DDevice9 interface, representing the device associated with the effect.

Return value

Type: HRESULT

If the method succeeds, the return value is S_OK. If the method fails, the return value can be D3DERR_INVALIDCALL.

Remarks

Calling this method will increase the internal reference count for the IDirect3DDevice9 interface. Be sure to call IUnknown::Release when you are done using the IDirect3DDevice9 interface or you will have a memory leak.
