ID3DXTextureGutterHelper::GetTexelMap method

Retrieves the (u, v) texture coordinates of each texel.

Syntax

HRESULT GetTexelMap(
  [out]Â D3DXVECTOR2 *pTexelData
);

Parameters

pTexelData [out]

Type: D3DXVECTOR2*

Pointer to the location in pixel (u, v) texture coordinates where each texel is located.

Return value

Type: HRESULT

If the method succeeds, the return value is S_OK. If the method fails, the following value will be returned.D3DERR_INVALIDCALL

Remarks

For class 2 and 4 texels, the returned (u, v) texture coordinates correspond to the closest point on the closest triangle.

The application must allocate and manage pTexelData.
