ID3DXPatchMesh::GetPatchInfo method

Gets the attributes of the patch.

Syntax

HRESULT GetPatchInfo(
  [out, retval]Â LPD3DXPATCHINFO PatchInfo
);

Parameters

PatchInfo [out, retval]

Type: LPD3DXPATCHINFO

Pointer to the structures containing the patch attributes. For more information about patch attributes, see D3DXPATCHINFO.

Return value

Type: HRESULT

If the method succeeds, the return value is D3D_OK. If the method fails, the return value can be one of the following: D3DERR_INVALIDCALL, E_OUTOFMEMORY.
