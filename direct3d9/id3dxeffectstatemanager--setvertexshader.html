ID3DXEffectStateManager::SetVertexShader method

A callback function that must be implemented by a user to set a vertex shader.

Syntax

HRESULT SetVertexShader(
  [in]Â LPDIRECT3DVERTEXSHADER9 pShader
);

Parameters

pShader [in]

Type: LPDIRECT3DVERTEXSHADER9

A pointer to a vertex shader object. See IDirect3DVertexShader9.

Return value

Type: HRESULT

The user-implemented method should return S_OK. If the callback fails when setting the device state, either of the following will occur:

  • The effect will fail during ID3DXEffect::BeginPass.
  • The dynamic effect state call (such as IDirect3DDevice9::SetVertexShader) will fail.
