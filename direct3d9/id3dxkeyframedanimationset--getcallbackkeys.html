ID3DXKeyframedAnimationSet::GetCallbackKeys method

Fills an array with callback key data used for key frame animation.

Syntax

HRESULT GetCallbackKeys(
  [out]Â LPD3DXKEY_CALLBACK pCallbackKeys
);

Parameters

pCallbackKeys [out]

Type: LPD3DXKEY_CALLBACK

Pointer to a user-allocated array of D3DXKEY_CALLBACK structures that the method is to fill with callback data.

Return value

Type: HRESULT

If the method succeeds, the return value is S_OK. If the method fails, the following value will be returned: D3DERR_INVALIDCALL.
