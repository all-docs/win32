D3DSINCOSCONST2

Creates the second constant (in register src3) for the sincos - ps pixel shader instruction.

#define D3DSINCOSCONST2 -0.020833334f, -0.12500000f, 1.0f, 0.50000000f

Return Value

Returns the second constant needed by sincos - ps.
