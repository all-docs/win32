ID3DXLine::SetPattern method

Applies a stipple pattern to the line.

Syntax

HRESULT SetPattern(
  [in]Â DWORD dwPattern
);

Parameters

dwPattern [in]

Type: DWORD

Describes the stipple pattern: 1 is opaque, 0 is transparent.

Return value

Type: HRESULT

If the method succeeds, the return value is D3D_OK. If the method fails, the return value can be one of the following: D3DERR_INVALIDCALL, D3DXERR_INVALIDDATA.
