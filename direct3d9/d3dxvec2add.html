D3DXVec2Add function

[!Note] The D3DX utility library is deprecated. We recommend that you use DirectXMath instead.

Adds two 2D vectors.

Parameters

pOut [in, out]

Type: D3DXVECTOR2*

Pointer to the D3DXVECTOR2 structure that is the result of the operation.

pV1 [in]

Type: const D3DXVECTOR2*

Pointer to a source D3DXVECTOR2 structure.

pV2 [in]

Type: const D3DXVECTOR2*

Pointer to a source D3DXVECTOR2 structure.

Return value

Type: D3DXVECTOR2*

Pointer to a D3DXVECTOR2 structure that is the sum of the two vectors.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the D3DXVec2Add function can be used as a parameter for another function.
