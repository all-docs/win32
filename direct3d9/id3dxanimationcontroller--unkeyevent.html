ID3DXAnimationController::UnkeyEvent method

Removes a specified event from an animation track, preventing the execution of the event.

Syntax

HRESULT UnkeyEvent(
  [in]Â D3DXEVENTHANDLE hEvent
);

Parameters

hEvent [in]

Type: D3DXEVENTHANDLE

Event handle to the event to be removed from the animation track.

Return value

Type: HRESULT

If the method succeeds, the return value is S_OK. If the method fails, the following value will be returned: D3DERR_INVALIDCALL.
