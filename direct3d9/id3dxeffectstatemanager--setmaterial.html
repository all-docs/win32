ID3DXEffectStateManager::SetMaterial method

A callback function that must be implemented by a user to set material state.

Syntax

HRESULT SetMaterial(
  [in]Â const D3DMATERIAL9 *pMaterial
);

Parameters

pMaterial [in]

Type: const D3DMATERIAL9*

A pointer to the material state. See D3DMATERIAL9.

Return value

Type: HRESULT

The user-implemented method should return S_OK. If the callback fails when setting the device state, either of the following will occur:

  • The effect will fail during ID3DXEffect::BeginPass.
  • The dynamic effect state call (such as IDirect3DDevice9::SetMaterial) will fail.
