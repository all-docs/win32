ID3DXPRTCompBuffer::IsTexture method

Indicates whether the buffer contains a texture.

Syntax

BOOL IsTexture();

Parameters

This method has no parameters.

Return value

Type: BOOL

Returns TRUE if the buffer contains a texture; returns FALSE otherwise.
