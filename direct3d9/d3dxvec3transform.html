D3DXVec3Transform function (D3dx9math.h)

[!Note] The D3DX utility library is deprecated. We recommend that you use DirectXMath instead.

Transforms vector (x, y, z, 1) by a given matrix.

Parameters

pOut [in, out]

Type: D3DXVECTOR4*

Pointer to the D3DXVECTOR4 structure that is the result of the operation.

pV [in]

Type: const D3DXVECTOR3*

Pointer to the source D3DXVECTOR3 structure.

pM [in]

Type: const D3DXMATRIX*

Pointer to the source D3DXMATRIX structure.

Return value

Type: D3DXVECTOR4*

Pointer to a D3DXVECTOR4 structure that is the transformed vector.

Remarks

This function transforms the vector, pV (x, y, z, 1), by the matrix pM.

The return value for this function is the same value returned in the pOut parameter. In this way, the D3DXVec3Transform function can be used as a parameter for another function.
