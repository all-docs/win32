Creates a texture shader object from the compiled shader.

ppTextureShader [out]

Type: LPD3DXTEXTURESHADER*

Returns an ID3DXTextureShader object which can be used to procedurally fill the contents of a texture using the D3DXFillTextureTX functions.

Return value

Type: HRESULT

If the function succeeds, the return value is D3D_OK. If the function fails, the return value can be one of the following: D3DERR_INVALIDCALL, D3DXERR_INVALIDDATA, E_OUTOFMEMORY.
