ID3DXTextureGutterHelper::SetFaceMap method

Sets the index of the mesh face to which each texel belongs.

Syntax

HRESULT SetFaceMap(
  [in]Â UINT *pFaceData
);

Parameters

pFaceData [in]

Type: UINT*

Pointer to the index of the mesh face to which each texel belongs.

Return value

Type: HRESULT

If the method succeeds, the return value is S_OK. If the method fails, the following value will be returned.D3DERR_INVALIDCALL

Remarks

The mesh face data input to this method is valid only for valid (non-class 0) texels. ID3DXTextureGutterHelper::GetGutterMap will return non-zero values for valid texels.
