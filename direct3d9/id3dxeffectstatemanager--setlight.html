ID3DXEffectStateManager::SetLight method

A callback function that must be implemented by a user to set a light.

Parameters

Index [in]

Type: DWORD

The zero-based index of the light. This is the same index in IDirect3DDevice9::SetLight.

pLight [in]

Type: const D3DLight9*

The light object. See D3DLIGHT9.

Return value

Type: HRESULT

The user-implemented method should return S_OK. If the callback fails when setting the device state, either of the following will occur:

  • The effect will fail during ID3DXEffect::BeginPass.
  • The dynamic effect state call (such as IDirect3DDevice9::SetLight) will fail.
