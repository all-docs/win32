Access Flags

These values indicate whether an entry in the access list describes rights that are allowed or denied.

Constant/value Description
ACTRL_ACCESS_ALLOWED 0x00000000 Indicates an access-allowed entry.
ACTRL_ACCESS_DENIED 0x10000000 Indicates an access-denied entry.

Requirement Value

Minimum supported server Windows 2000 Server [desktop apps only]
