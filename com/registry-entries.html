Registry Entries (COM)

Registry values in the following registry keys control aspects of the functionality of COM:

  • HKEY_LOCAL_MACHINE\SOFTWARE\Classes
  • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Ole
  • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion

As of Windows ServerÂ 2003, COM uses only the current process token to decide which registry hive to access, not the thread token. If COM is not able to access the user profile registry hive, it will access the HKEY_LOCAL_MACHINE\System hive.
