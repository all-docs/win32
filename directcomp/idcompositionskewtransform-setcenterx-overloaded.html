IDCompositionSkewTransform::SetCenterX methods

Changes or animates the value of the CenterX property of a 2D skew transform. The CenterX property specifies the x-coordinate of the point about which the skew is performed.

Method Description
SetCenterX(float) Changes the value of the CenterX property.
