WINBIO_NO_FORMAT Constants

If the members of a WINBIO_REGISTERED_FORMAT structure cannot be zero, you can use the following constants to check for errors.

Constant/value Description
WINBIO_NO_FORMAT_OWNER_AVAILABLE 0 No IBIA (International Biometric Industry Association) assigned owner value is available.
WINBIO_NO_FORMAT_TYPE_AVAILABLE 0 No owner assigned format type is available.
