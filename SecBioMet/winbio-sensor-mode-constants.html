WINBIO_SENSOR_MODE Constants

The following values are used in the SensorAdapterSetMode function to set the sensor adapter mode.

Constant Description
WINBIO_SENSOR_UNKNOWN_MODE The operating mode is not known.
WINBIO_SENSOR_BASIC_MODE Operate the sensor in basic mode. The sensor acts only as a capture device. Any onboard processing or storage capabilities that exist are not used.
WINBIO_SENSOR_ADVANCED_MODE Operate the sensor in advanced mode. The sensor can capture samples and perform matching and storage functions.
WINBIO_SENSOR_NAVIGATION_MODE Operate the sensor as a mouse pad. This is not currently supported.
WINBIO_SENSOR_SLEEP_MODE Operate the sensor in sleep mode.
