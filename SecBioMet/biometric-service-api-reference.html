Windows Biometric Framework API Reference

Windows Biometric Framework API reference pages contain detailed descriptions of functions, structures, and capabilities that can be used to create a client applications and plug-ins. Reference pages are divided into the following groups:

In this section

Topic Description
Client Application Reference Functions, structures, and other types that you can use to build client applications that interact with the Windows Biometric Framework.
Plug-in Reference Adapter functions, wrapper functions, and structures for creating custom plug-in adapters of three types: engine, sensor, and storage.
