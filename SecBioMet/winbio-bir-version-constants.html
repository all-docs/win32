WINBIO_BIR_VERSION Constants

The following flags are used by the HeaderVersion and PatronHeaderVersion members of the WINBIO_BIR_HEADER structure to specify the version.

Constant/value Description
WINBIO_CBEFF_HEADER_VERSION 0x11 Specifies the header version.
WINBIO_PATRON_HEADER_VERSION 0x11 Specifies the patron header version.
