WINBIO_IDENTITY_TYPE Constants

The following WINBIO_IDENTITY_TYPE constants can be used to specify the format of the identity information contained in the WINBIO_IDENTITY structure.

Constant Description
WINBIO_ID_TYPE_NULL The template has no associated ID.
WINBIO_ID_TYPE_WILDCARD The structure matches all template identities.
WINBIO_ID_TYPE_GUID A GUID identifies the template.
WINBIO_ID_TYPE_SID An account SID identifies the template.

Requirement Value
