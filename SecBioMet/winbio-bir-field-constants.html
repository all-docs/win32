WINBIO_BIR_FIELD Constants

The following constants are used to create a bitmask for the ValidFields member of the WINBIO_BIR_HEADER structure.

Constant/value Description
WINBIO_BIR_FIELD_SUBHEAD_COUNT 0x0001 Provided for conformity with NISTIR 6529-A, the Common Biometric Exchange Formats Framework (CBEFF) Patron Format A, but not used.
WINBIO_BIR_FIELD_PRODUCT_ID 0x0002 The ProductId member is valid.
WINBIO_BIR_FIELD_PATRON_ID 0x0004 Provided for conformity with NISTIR 6529-A, CBEFF Patron Format A, but not used.
WINBIO_BIR_FIELD_INDEX 0x0008 Provided for conformity with NISTIR 6529-A, CBEFF Patron Format A, but not used.
WINBIO_BIR_FIELD_CREATION_DATE 0x0010 The CreationDate member is valid.
WINBIO_BIR_FIELD_VALIDITY_PERIOD 0x0020 The ValidityPeriod member is valid.
WINBIO_BIR_FIELD_BIOMETRIC_TYPE 0x0040 The Type member is valid.
WINBIO_BIR_FIELD_BIOMETRIC_SUBTYPE 0x0080 The Subtype member is valid.
WINBIO_BIR_FIELD_CBEFF_HEADER_VERSION 0x0100 The HeaderVersion member is valid.
WINBIO_BIR_FIELD_PATRON_HEADER_VERSION 0x0200 The PatronHeaderVersion member is valid.
WINBIO_BIR_FIELD_BIOMETRIC_PURPOSE 0x0400 The Purpose member is valid.
WINBIO_BIR_FIELD_BIOMETRIC_CONDITION 0x0800 Provided for conformity with NISTIR 6529-A, CBEFF Patron Format A, but not used.
WINBIO_BIR_FIELD_QUALITY 0x1000 The DataQuality member is valid.
WINBIO_BIR_FIELD_CREATOR 0x2000 Provided for conformity with NISTIR 6529-A, CBEFF Patron Format A, but not used.
WINBIO_BIR_FIELD_CHALLENGE 0x4000 Provided for conformity with NISTIR 6529-A, CBEFF Patron Format A, but not used.
WINBIO_BIR_FIELD_PAYLOAD 0x8000 Provided for conformity with NISTIR 6529-A, CBEFF Patron Format A, but not used.
