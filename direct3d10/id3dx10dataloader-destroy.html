ID3DX10DataLoader::Destroy method

Used by a ID3DX10ThreadPump Interface to destroy the loader after a work item completes.

Syntax

HRESULT Destroy();

Parameters

This method has no parameters.

Return value

Type: HRESULT

The return value is one of the values listed in Direct3D 10 Return Codes.
