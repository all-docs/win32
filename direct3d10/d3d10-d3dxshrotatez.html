D3DXSHRotateZ function (D3DX10.h)

[!Note] The D3DX (D3DX 9, D3DX 10, and D3DX 11) utility library is deprecated and is not supported for Windows Store apps.

[!Note] Instead of using this function, we recommend that you use the Spherical Harmonics Math library function XMSHRotateZ.
