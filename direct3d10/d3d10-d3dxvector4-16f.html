D3DXVECTOR4_16F structure

[!Note] The D3DX10 utility library is deprecated. We recommend that you use DirectXMath instead.

Same as a D3DXVECTOR4, but it uses 16-bit floating point values for x, y, and z.

Syntax

typedef struct D3DXVECTOR4_16F {
  FLOAT x;
  FLOAT y;
  FLOAT z;
  FLOAT w;
} D3DXVECTOR4_16F, *LPD3DXVECTOR4_16F;

Remarks

D3DXVECTOR4_16F has the following C++ extensions.

    D3DXFLOAT16 x, y, z, w;
