IPortableDeviceValues::GetGuidValue method

The GetGuidValue method retrieves a GUID value (type VT_CLSID) specified by a key.

Syntax

HRESULT GetGuidValue(
  [in]  REFPROPERTYKEY key,
  [out] GUID           *pValue
);

Parameters

key [in]

A REFPROPERTYKEY key that specifies the item to retrieve.

pValue [out]

Pointer to the retrieved GUID value.

Return value

The method returns an HRESULT. Possible values include, but are not limited to, those in the following table.

Return code Description
S_OK The method succeeded.
DISP_E_TYPEMISMATCH The property specified by key is not a GUID type.
HRESULT_FROM_WIN32(ERROR_NOT_FOUND) The property specified by key is not in the collection.

Examples

For an example of how to use this method, see Retrieving Supported Service Methods.
