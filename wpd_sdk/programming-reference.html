WPD reference

The reference section of this documentation contains the following sections.

Section Description
Constants Constants that are used by the drivers and applications, including error codes defined by Windows Portable Devices.
Client Interfaces Interfaces that are used by a Windows Portable Device application.
Driver Interfaces Interfaces that are used by a Windows Portable Device driver.
Collection Interfaces Interfaces that are used to hold groups of values sent between the application and the driver.
Commands Commands that the application sends to the driver.
Structures and Enumeration Types Structures and enumerations that are used by drivers and applications.
