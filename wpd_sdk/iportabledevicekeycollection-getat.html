IPortableDeviceKeyCollection::GetAt method

The GetAt method retrieves a PROPERTYKEY from the collection by index.

Syntax

HRESULT GetAt(
  [in]  const DWORD       dwIndex,
  [out]       PROPERTYKEY *pKey
);

Parameters

dwIndex [in]

DWORD that contains the index of the key to be retrieved.

pKey [out]

Pointer to a PROPERTYKEY object.

Return value

The method returns an HRESULT. Possible values include, but are not limited to, those in the following table.

Return code Description
S_OK The method succeeded.
E_INVALIDARG The index passed in was out of range.
E_POINTER A required pointer argument was NULL.

Examples

For an example of how to use this method, see Retrieving Supported Service Events.
