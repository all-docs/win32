WPD_CONTENT_TYPE_CONTACT

An object that describes its type as WPD_CONTENT_TYPE_CONTACT represents personal contact data.

This type of object supports the following properties.

WPD_OBJECT_ID Required, read-only. A client cannot set this property, even at creation time.
