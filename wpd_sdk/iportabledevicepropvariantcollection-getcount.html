IPortableDevicePropVariantCollection::GetCount method

The GetCount method retrieves the number of items in this collection.

Syntax

HRESULT GetCount(
  [in] DWORD *pcElems
);

Parameters

pcElems [in]

Pointer to a DWORD that contains the number of items in the collection.

Return value

The method returns an HRESULT. Possible values include, but are not limited to, those in the following table.

Return code Description
S_OK The method succeeded.
E_POINTER A required pointer argument was NULL.

Examples

For an example of how to use this method see Retrieving the Functional Categories Supported by a Device.
