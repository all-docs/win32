IEnumPortableDeviceConnectors::Skip method

The Skip method skips the specified number of devices in the enumeration sequence.

Syntax

HRESULT Skip(
  [in] UINT32 cConnectors
);

Parameters

cConnectors [in]

The number of devices to skip.

Return value

The method returns an HRESULT. Possible values include, but are not limited to, those in the following table.

Return code Description
S_OK The method succeeded.
S_FALSE The specified number of devices could not be skipped. One possible cause: The cConnectors parameter specifies more devices than actually remain in the enumeration sequence.
