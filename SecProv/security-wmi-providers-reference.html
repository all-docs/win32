Security WMI Providers Reference

Security WMI reference pages contain detailed descriptions of the Encryptable Volume WMI Provider and the Trusted Platform Module (TPM) WMI Provider.

Security WMI Providers include the following providers.

Section Description
BitLocker Drive Encryption Provider API documentation for managing BitLocker Drive Encryption (BDE) using the Win32_EncryptableVolume class.
Trusted Platform Module Provider API documentation for managing the Trusted Platform Module (TPM) using the Win32_Tpm class.
