OP_BLOB structure

Contains an opaque byte buffer.

Syntax

typedef struct _OP_BLOB
{
    ULONG                       cbBlob;
    [size_is(cbBlob)]   PBYTE   pBlob;
} OP_BLOB, *POP_BLOB;

Members

cbBlob

Specifies the the size of pBlob in bytes.

pBlob

Points to a byte buffer.
