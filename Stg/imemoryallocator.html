IMemoryAllocator class

The IMemoryAllocator class is implemented by the memory allocator for the StgConvertPropertyToVariant function.

Methods

Name Description
Allocate Allocates memory for the StgConvertPropertyToVariant function when the function converts a SERIALIZEDPROPERTYVALUE data type to a PROPVARIANT data type.
Free Frees the memory allocated by the Allocate method.

Remarks

This class is only used by the StgConvertPropertyToVariant function.
