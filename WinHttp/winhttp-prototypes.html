WinHTTP Callbacks

WinHTTP uses the following callbacks.

Topic and description
WINHTTP_PROXY_CHANGE_CALLBACK. Represents an application-defined proxy change callback function.
WINHTTP_STATUS_CALLBACK. Prototype for an application-defined status callback function.
