WinHTTP Reference

Microsoft Windows HTTP Services (WinHTTP) reference material is available for the following languages.

Reference

  • WinHTTP C/C++ API Reference contains reference documentation for using WinHTTP from C or C++.
  • WinHttpRequest Object Reference contains reference documentation for using WinHTTP from Visual Basic and script languages through the WinHttpRequest COM object, along with additional information for Visual Basic developers.
