Windows Web Services Handles

The Windows Web Services API includes the following handles:

  • WS_CHANNEL
  • WS_ERROR
  • WS_HEAP
  • WS_LISTENER
  • WS_MESSAGE
  • WS_METADATA
  • WS_OPERATION_CONTEXT
  • WS_POLICY
  • WS_SECURITY_CONTEXT
  • WS_SECURITY_TOKEN
  • WS_SERVICE_HOST
  • WS_SERVICE_PROXY
  • WS_XML_BUFFER
  • WS_XML_READER
  • WS_XML_WRITER
