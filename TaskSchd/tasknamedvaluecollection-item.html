TaskNamedValueCollection.Item property

For scripting, gets the specified name-value pair from the collection.

Syntax

TaskNamedValueCollection.Item( _
  ByVal index, _
  ByRef pair _
) As long

Property value

A TaskNamedValuePair object that represents the requested pair.
