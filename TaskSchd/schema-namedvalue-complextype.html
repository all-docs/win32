namedValue Complex Type

Defines a name that is associated with a value in a name-value pair.

    <xs:simpleContent>
        <xs:extension
            base="nonEmptyString"

            <xs:attribute name="name"
                type="nonEmptyString"
                use="required"
             />
        </xs:extension>
    </xs:simpleContent>
</xs:complexType>
