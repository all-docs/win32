triggersType Complex Type

Defines the group (triggerGroup) for all trigger elements. The triggerGroup group contains the list of triggers that can be used in a task.

<xs:complexType name="triggersType">
    <xs:group
        minOccurs="0"
        maxOccurs="48"
        ref="triggerGroup"
     />
</xs:complexType>
