Trigger.Id property

For scripting, gets or sets the identifier for the trigger.

Syntax

Trigger.Id As String

Property value

The identifier for the trigger. This identifier is used by the Task Scheduler for logging purposes.

Remarks

When reading or writing XML for a task, the trigger identifier is specified in the Id attribute of the individual trigger elements (for example, the Id attribute of the BootTrigger element) of the Task Scheduler schema.
