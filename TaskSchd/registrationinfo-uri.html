RegistrationInfo.URI property

For scripting, gets or sets the URI of the task.

This property is read/write.

Syntax

RegistrationInfo.URI As String

Property value

The URI of the task.

Remarks

When reading or writing XML for a task, the task URI is specified using the URI element of the Task Scheduler schema.
