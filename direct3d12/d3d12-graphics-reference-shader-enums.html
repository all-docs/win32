Shader Enumerations (Direct3D 12 Graphics)

d3d12shader.h declares the following enumerations, which specify information about shaders.

In this section

Topic Description
D3D12_SHADER_VERSION_TYPE Enumerates the types of shaders that Direct3D recognizes. Used to encode the Version member of the D3D12_SHADER_DESC structure.
