Raytracing HLSL resource types (Direct3D 12)

The following HLSL resource types support the Direct3D 12 raytracing pipeline.

In this section

Topic Description
RaytracingAccelerationStructure A resource type that can be declared in HLSL and passed into TraceRay to indicate the top-level acceleration resource built using BuildRaytracingAccelerationStructure.
