Descriptor Tables

A descriptor table is logically an array of descriptors.

In this section

Topic Description
Descriptor Tables Overview Each descriptor table stores descriptors of one or more types - SRVs, UAVs, CBVs, and Samplers. A descriptor table is not an allocation of memory; it is simply an offset and length into a descriptor heap.
Using Descriptor Tables Descriptor tables, each identifying a range in a descriptor heap, are bound at slots defined by the current root signature on a command list.
