CD3DX12_SUBRESOURCE_RANGE_UINT64 structure

A helper structure to enable easy initialization of a D3D12_SUBRESOURCE_RANGE_UINT64 structure.

Syntax

struct CD3DX12_SUBRESOURCE_RANGE_UINT64  : public D3D12_SUBRESOURCE_RANGE_UINT64{
  CD3DX12_SUBRESOURCE_RANGE_UINT64 CD3DX12_SUBRESOURCE_RANGE_UINT64();
  CD3DX12_SUBRESOURCE_RANGE_UINT64 explicit CD3DX12_SUBRESOURCE_RANGE_UINT64(const D3D12_SUBRESOURCE_RANGE_UINT64 &o);
  CD3DX12_SUBRESOURCE_RANGE_UINT64 CD3DX12_SUBRESOURCE_RANGE_UINT64(UINT subresource, const D3D12_RANGE_UINT64& range);
  CD3DX12_SUBRESOURCE_RANGE_UINT64 CD3DX12_SUBRESOURCE_RANGE_UINT64(UINT subresource, UINT64 begin, UINT64 end);
                                   operator const D3D12_SUBRESOURCE_RANGE_UINT64&() const;
};

Members

CD3DX12_SUBRESOURCE_RANGE_UINT64()

Creates a new, uninitialized, instance of a CD3DX12_SUBRESOURCE_RANGE_UINT64.

explicit CD3DX12_SUBRESOURCE_RANGE_UINT64(const D3D12_SUBRESOURCE_RANGE_UINT64 &o)

Creates a new instance of a CD3DX12_SUBRESOURCE_RANGE_UINT64, initialized with values copied from a D3D12_SUBRESOURCE_RANGE_UINT64 structure.

CD3DX12_SUBRESOURCE_RANGE_UINT64(UINT subresource, const D3D12_RANGE_UINT64& range)

Creates a new instance of a CD3DX12_DEPTH_STENCIL_DESC1, initialized with the values passed in the parameter list.

CD3DX12_SUBRESOURCE_RANGE_UINT64(UINT subresource, UINT64 begin, UINT64 end)

Creates a new instance of a CD3DX12_DEPTH_STENCIL_DESC1, initialized with the values passed in the parameter list.

operator const D3D12_SUBRESOURCE_RANGE_UINT64&() const

Implicit conversion to a D3D12_SUBRESOURCE_RANGE_UINT64 structure. Because D3D12_SUBRESOURCE_RANGE_UINT64 is the underlying type of CD3DX12_SUBRESOURCE_RANGE_UINT64, the object is simply returned as a const D3D12_DEPTH_STENCIL_DESC1 reference to itself.
