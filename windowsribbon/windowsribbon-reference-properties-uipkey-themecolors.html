Remarks

UI_PKEY_ThemeColors is used by an application to query the color swatch values of a DropDownColorPicker.

Each COLORREF value corresponds to a color swatch in a DropDownColorPicker where ThemeColors is specified as the value of the ColorTemplate attribute.

The property value is an array of COLORREF values.
