Remarks

UI_PKEY_NoColorLabel is used by an application to query the value of the label associated with the No color button of a DropDownColorPicker.

The property value is a string that represents the label associated with the No color button of a DropDownColorPicker.

UI_PKEY_NoColorLabel is valid for all ColorTemplate attribute values.
