Remarks

UI_PKEY_MoreColorsLabel is used by an application to query the value of the label associated with the More colors button of a DropDownColorPicker.

UI_PKEY_MoreColorsLabel is only valid for a DropDownColorPicker where ThemeColors or StandardColors is specified as the value of the ColorTemplate attribute.
