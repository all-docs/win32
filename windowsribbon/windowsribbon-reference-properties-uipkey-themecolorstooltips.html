Remarks

UI_PKEY_ThemeColorsTooltips is used by an application to query the color swatch tooltips of a DropDownColorPicker.

The property value is an array of string values.

Each string value corresponds to the tooltip for a color swatch in a DropDownColorPicker where ThemeColors is specified as the value of the ColorTemplate attribute.
