Windows Ribbon Framework Reference

The topics contained in this section provide the Reference specifications for the Windows Ribbon framework.

In this section

Topic Description
Markup Elements Reference documentation for the Ribbon framework markup elements.
Constants and Enumerations Reference documentation for the Ribbon framework constants and enumerations.
Interfaces Reference documentation for the Ribbon framework interfaces.
Objects Reference documentation for the Ribbon framework objects.
Property Keys Reference documentation for the Ribbon framework property keys, a collection of PROPERTYKEY and PROPVARIANT key-value pairs used to describe the state of a Ribbon framework Command.
Structures The topics in this section provide the reference specifications for Windows Ribbon Framework structures.
