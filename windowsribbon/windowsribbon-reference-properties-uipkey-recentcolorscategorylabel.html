Remarks

UI_PKEY_RecentColorsCategoryLabel is used by an application to query the value of the label associated with the Recent Colors category of a DropDownColorPicker.

UI_PKEY_RecentColorsCategoryLabel is valid only for a DropDownColorPicker where ThemeColors is specified as the value of the ColorTemplate attribute (this is the only template that contains labeled categories).
