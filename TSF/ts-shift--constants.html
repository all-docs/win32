TS_SHIFT_* Constants

The TS_Shift_* constants are used by the IAnchor interface for manipulation of hidden text and character counting.

Constant/value Description
TS_SHIFT_COUNT_HIDDEN ( 0x1 ) Specifies that the anchor will be shifted to the next region boundary, including the boundary of a hidden text region. If not set, the anchor will be shifted past any adjacent hidden text until a region of visible text is found.
TS_SHIFT_HALT_HIDDEN ( 0x2 ) Not used.
TS_SHIFT_HALT_VISIBLE ( 0x4 ) Not used.
TS_SHIFT_COUNT_ONLY ( 0x8 ) The anchor is not shifted.
