ISoftKbd::GetSoftKeyboardTextFont method

The ISoftKbd::GetSoftKeyboardTextFont method retrieves the text font used by a soft keyboard.

Syntax

HRESULT GetSoftKeyboardTextFont(
  [out] LOGFONTW *pLogFont
);

Parameters

pLogFont [out]

Pointer to a buffer in which this method retrieves a LOGFONTW structure defining the text font for the soft keyboard.

Return value

This method can return one of these values.

Value Description
S_OK The method was successful.
E_INVALIDARG The pLogFont parameter is invalid.
