TS_AS_* Constants

The following flags specify the events that call the AdviseSink methods.

Constant/value Description
TS_AS_TEXT_CHANGE ( 0x1 ) Text is changed in the document.
TS_AS_SEL_CHANGE ( 0x2 ) Text is selected in the document.
TS_AS_LAYOUT_CHANGE ( 0x4 ) The layout of the document is changed.
TS_AS_ATTR_CHANGE ( 0x8 ) The attributes of the document is changed.
TS_AS_STATUS_CHANGE ( 0x10 ) The status of the document is changed.
TS_AS_ALL_SINKS ( TS_AS_TEXT_CHANGE | TS_AS_SEL_CHANGE | TS_AS_LAYOUT_CHANGE | TS_AS_ATTR_CHANGE | TS_AS_STATUS_CHANGE ) One of the previous four events occurred in the document.
