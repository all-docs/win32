TF_IAS_* Constants

The TF_IAS_* constants are used as bitfield flags in methods that insert text or embedded objects at a selection or insertion point.

Constant/value Description
TF_IAS_NOQUERY ( 0x1 ) Text is inserted and the range pointer is set to NULL upon exit. Cannot be combined with the TF_IAS_QUERYONLY flag.
TF_IAS_QUERYONLY ( 0x2 ) Do not perform the insert. Caller only requires the range pointer to be set. Cannot be combined with the TF_IAS_NOQUERY flag.
TF_IAS_NO_DEFAULT_COMPOSITION ( 0x80000000 ) TSF will not create a default composition if a composition is required. Caller must start a composition over the range.
