TS_LF_* Constants

The TS_LF_* constants specify the type of document lock.

Constant/value Description
TS_LF_SYNC ( 0x1 ) The document has a synchronous-lock if this flag is combined with other flags.
TS_LF_READ ( 0x2 ) The document has a read-only lock and cannot be modified.
TS_LF_READWRITE ( 0x6 ) The document has a read/write lock and can be modified.
