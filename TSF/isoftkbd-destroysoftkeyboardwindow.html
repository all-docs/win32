ISoftKbd::DestroySoftKeyboardWindow method

The ISoftKbd::DestroySoftKeyboardWindow method destroys a soft keyboard window.

Syntax

HRESULT DestroySoftKeyboardWindow();

Parameters

This method has no parameters.

Return value

This method can return one of these values.

Value Description
S_OK The method was successful.

Remarks

This method removes a soft keyboard window previously created with a call to ISoftKbd::CreateSoftKeyboardWindow.
