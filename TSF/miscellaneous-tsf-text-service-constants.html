Miscellaneous TSF Text Service Constants

The following values are used with text services.

Constant/value Description
TF_MENUREADY 0x00000001 Not currently used.
TF_PROPUI_STATUS_SAVETOFILE 0x00000001 The property can be serialized. If this value is not present, the property cannot be serialized. This value is used with ITfFnPropertyUIStatus::GetStatus and ITfFnPropertyUIStatus::SetStatus.

Requirements

Requirement Value
Minimum supported client Windows 2000 Professional [desktop apps only]
Minimum supported server Windows 2000 Server [desktop apps only]
Redistributable TSF 1.0 on Windows 2000 Professional
