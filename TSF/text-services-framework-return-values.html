Text Services Framework Return Values

Interface methods of the Text Services Framework produce return values in the range from 0xHHHH0200 to 0xHHHH0600, where HHHH is a hexadecimal value that depends on the severity code and facility code of a specific return value.

The return values group into the following categories.

  • Text Service Return Values
  • Manager Return Values
  • Text Store Return Values
