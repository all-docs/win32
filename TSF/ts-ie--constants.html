TS_IE_* Constants

The TS_IE_* constants describe how to insert text that can contain embedded objects used by text services.

Constant/value Description
TS_IE_CORRECTION ( 0x1 ) The text is a transform (correction) of existing content, and any special text markup information (metadata) is retained, such as .wav file data or a language identifier.
TS_IE_COMPOSITION ( 0x2 ) Not used.

Remarks

These constants are used in the dwFlags parameter of ITextStoreACP::InsertEmbedded and ITextStoreAnchor::InsertEmbedded.
