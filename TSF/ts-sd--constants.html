TS_SD_* Constants

The TS_SD_* constants describe dynamic document states used by an application at runtime.

Constant/value Description
TS_SD_READONLY ( 0x1 ) The document is read-only and cannot be modified.
TS_SD_LOADING ( 0x2 ) The document is loading and might be incomplete.
TS_SD_MASKALL ( TS_SD_READONLY | TS_SD_LOADING ) The document is both read-only and loading.

Remarks

The dwDynamicFlags member of the TS_STATUS structure uses these constants.
