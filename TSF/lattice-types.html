Lattice Types

The following constants identify specific types of lattice properties. These values are used with the ITfLMLattice::QueryType and ITfLMLattice::EnumLatticeElements methods.

GUID_LMLATTICE_VER1_0

Specifies that the lattice property is version 1.0.
