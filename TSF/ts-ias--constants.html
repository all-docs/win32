TS_IAS_* Constants

The TS_IAS_* constants are used as bitfield flags to control insertion of text or embedded objects at a selection or insertion point.

Constant/value Description
TS_IAS_NOQUERY ( 0x1 ) Text is inserted and the range pointer is set to NULL upon exit. Cannot be combined with the TS_IAS_QUERYONLY flag.
TS_IAS_QUERYONLY ( 0x2 ) Do not perform the insertion. Caller only requires the range pointer to be set. Cannot be combined with the TS_IAS_NOQUERY flag.
