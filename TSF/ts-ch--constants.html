TS_CH_* Constants

The TS_CH_* constants are used internally by TSF to indicate that text deletions have occurred.

Constant/value Description
TS_CH_PRECEDING_DEL ( 1 ) Text preceding the character position has been deleted.
TS_CH_FOLLOWING_DEL ( 2 ) Text following the character position has been deleted.
