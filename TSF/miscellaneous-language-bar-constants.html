Miscellaneous Language Bar Constants

The miscellaneous language bar constants set certain properties of language bars.

Constant/value Description
TF_DTLBI_USEPROFILEICON 0x00000001 The system language bar item should display the icon specified for the language profile. Used in the ITfSystemDeviceTypeLangBarItem::GetIconMode and ITfSystemDeviceTypeLangBarItem::SetIconMode methods.
TF_INVALIDMENUITEM (UINT)(-1) Not used.
TF_LBI_BMPF_VERTICAL 0x00000001 Not used.
TF_LBI_DESC_MAXLEN 32 Length, in WCHAR characters, of structure member TF_LANGBARITEMINFO.szDescription.
