TF_TF_* Constants

The TF_TF_* constants are used to specify options with the ITfRange::GetText method.

Constant/value Description
TF_TF_MOVESTART ( 1 ) Update start anchor.
TF_TF_IGNOREEND ( 2 ) Ignore end anchor.
