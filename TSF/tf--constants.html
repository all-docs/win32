TF_* Constants

TSF atom function calls use these constants to manage thread input processes.

Constant/value Description
TF_PROCESS_ATOM TEXT(_CTF_PROCESS_ATOM_) Used to notify atom functions that a thread input process has initiated.
TF_ENABLE_PROCESS_ATOM TEXT(_CTF_ENABLE_PROCESS_ATOM_) Used to notify TSF, through atom functions, that a thread input process has been enabled.
