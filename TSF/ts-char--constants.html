TS_CHAR_* Constants

The TS_CHAR_* constants describe the type of character.

Constant/value Description
TS_CHAR_EMBEDDED ( 0xfffc ) The character is a Unicode 2.1 object-replacement character.
TS_CHAR_REGION ( 0 ) The character is a region boundary.
TS_CHAR_REPLACEMENT ( 0xfffd ) The character is a hidden-text placeholder character or a Unicode replacement character.
