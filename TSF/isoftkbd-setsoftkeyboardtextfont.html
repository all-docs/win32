ISoftKbd::SetSoftKeyboardTextFont method

The ISoftKbd::SetSoftKeyboardTextFont method sets the text font used by a soft keyboard.

Syntax

HRESULT SetSoftKeyboardTextFont(
  [in] LOGFONTW *pLogFont
);

Parameters

pLogFont [in]

Pointer to a LOGFONTW structure defining the text font for the soft keyboard.

Return value

This method can return one of these values.

Value Description
S_OK The method was successful.
E_INVALIDARG The pLogFont parameter is invalid.
