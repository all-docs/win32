ISoftKbd::GetSoftKeyboardTypeMode method

The ISoftKbd::GetSoftKeyboardTypeMode method retrieves the type mode for a soft keyboard.

Syntax

HRESULT GetSoftKeyboardTypeMode(
  [out] TYPEMODE *lpTypeMode
);

Parameters

lpTypeMode [out]

Pointer to a buffer in which this method retrieves the type mode. Possible values are defined for the TYPEMODE enumeration.

Return value

This method can return one of these values.

Value Description
S_OK The method was successful.
E_INVALIDARG The lpTypeMode parameter is invalid.
