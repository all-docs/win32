Predefined Language Bar Items

The following GUID values identify the predefined language bar items. These values are used with GetItem to obtain a specific language bar item.

GUID_LBI_SAPILAYR_CFGMENUBUTTON

Speech tools language bar menu item.

GUID_TFCAT_TIP_HANDWRITING

Handwriting type item.

GUID_TFCAT_TIP_KEYBOARD

Keyboard type item.
