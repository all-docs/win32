QMGR Structures

[Queue Manager (QMGR) structures are available for use in the operating systems specified in a topic's Requirements section. They may be altered or unavailable in subsequent versions. Instead, use the BITS Structures and Unions.]

The Queue Manager (QMGR) interfaces use the following structures.

Structure Purpose
FILESETINFO Identifies the remote and local names of the file to download.
BACKGROUND_COPY_TIMES Not supported.
