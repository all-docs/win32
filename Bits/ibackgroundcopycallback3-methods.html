IBackgroundCopyCallback3 Methods

The IBackgroundCopyCallback3 interface exposes the following methods.

In this section

Topic Description
FileRangesTransferred method BITS calls your implementation of the FileRangesTransferred method when one or more file ranges have been downloaded. File ranges are added to the job using the IBackgroundCopyFile6::RequestFileRanges method.
