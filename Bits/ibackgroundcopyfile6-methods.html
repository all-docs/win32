IBackgroundCopyFile6 Methods

The IBackgroundCopyFile6 interface exposes the following methods.

In this section

Topic Description
UpdateDownloadPosition method Specifies a position to prioritize downloading missing data from.
RequestFileRanges method Adds a new set of file ranges to be prioritized for download.
GetFilledFileRanges method Returns the set of file ranges that have been downloaded.
