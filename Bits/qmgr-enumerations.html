QMGR Enumerations

[Queue Manager (QMGR) enumerations are available for use in the operating systems specified in a topic's Requirements section. They may be altered or unavailable in subsequent versions. Instead, use the BITS Enumerations.]

The Queue Manager (QMGR) interfaces use the following enumeration.

Enumeration Description
GROUPPROP Defines the constant values for retrieving and setting group property values.
