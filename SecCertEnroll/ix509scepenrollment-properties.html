IX509SCEPEnrollment Properties

The IX509SCEPEnrollment interface exposes the following properties.

In this section

Topic Description
Certificate property Gets the certificate for the request.
CertificateFriendlyName property Gets or sets the friendly name for the certificate.
FailInfo property Gets information when the ProcessResponseMessage method detects a failed environment.
OldCertificate property Gets or sets an old certificate that a request is intended to replace.
Request property Gets the inner PKCS10 request.
ServerCapabilities property Sets the preferred hash and encryption algorithms for the request.
SignerCertificate property Gets or sets the signer certificate for the request.
Silent property Gets or sets whether to allow UI during the request.
Status property Gets the status of the request.
TransactionId property Gets or sets the transaction id for the request.
