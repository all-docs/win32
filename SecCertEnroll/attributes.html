Attributes (Certificate Enrollment API)

Attributes can be added to a certificate request to provide a certification authority (CA) with additional information that it can use when creating and issuing a certificate. Each attribute is a Distinguished Encoding Rules (DER) encoded Abstract Syntax Notation One (ASN.1) structure that contains an object identifier (OID) and zero or more values. Attributes are defined by using interfaces included with the Certificate Enrollment API. The following topics discuss attributes in more detail:

  • Supported Attributes
  • Attribute Architecture
  • PKCS #7 Attributes
  • PKCS #10 Attributes
  • CMC Attributes
