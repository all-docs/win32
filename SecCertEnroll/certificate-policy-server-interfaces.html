Certificate Policy Server Interfaces

The following interfaces can be used to programmatically configure a certificate enrollment policy (CEP) server.

Interface Description
IX509EnrollmentPolicyServer Represents a CEP server.
IX509PolicyServerListManager Manages a collection of IX509PolicyServerUrl objects.
IX509PolicyServerUrl Specifies or retrieves property values associated with the CEP server and updates associated registry values.
