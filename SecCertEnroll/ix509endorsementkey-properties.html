IX509EndorsementKey Properties

The IX509EndorsementKey interface exposes the following properties.

In this section

Topic Description
Length property The bit length of the endorsement key. You can only access this property after the Open method has been called.
Opened property Indicates whether the Open method has been successfully called.
ProviderName property The name of the encryption provider. The default is the Microsoft Platform Crypto Provider. You must set the ProviderName property before you call the Open method. You cannot change the ProviderName property after you have called the Open method.
