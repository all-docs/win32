IObjectIds Methods

The IObjectIds interface exposes the following methods.

In this section

Topic Description
Add Method adds an IObjectId object to the collection.
AddRange Method adds a range of IObjectId objects to the collection.
Clear Method removes all IObjectId objects from the collection.
Remove Method removes an IObjectId object from the collection by index value.
