CNG Helper Functions

Cryptography API: Next Generation (CNG) defines the following helper functions that are used for performing cryptographic operations.

  • BCryptDeriveKeyCapi
  • BCryptDeriveKeyPBKDF2
  • BCRYPT_INIT_AUTH_MODE_INFO
  • ProcessPrng
  • SystemPrng
