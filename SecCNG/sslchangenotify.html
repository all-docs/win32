SslChangeNotify function

The SslChangeNotify function is not implemented and cannot be used.

Syntax

SECURITY_STATUS WINAPI SslChangeNotify(
  _In_ HANDLE hEvent,
  _In_ DWORD  dwFlags
);

Parameters

hEvent [in]

This parameter is reserved for future use.

dwFlags [in]

This parameter is reserved for future use.

Return value

Returns NTE_NOT_SUPPORTED.
