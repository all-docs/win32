SslFreeObject function

The SslFreeObject function frees a key, hash, or provider object.

Syntax

SECURITY_STATUS WINAPI SslFreeObject(
  _In_ NCRYPT_HANDLE hObject,
  _In_ DWORD         dwFlags
);

Parameters

hObject [in]

The handle of the object to free.

dwFlags [in]

This parameter is reserved for future use.

Return value

If the function succeeds, it returns zero.

If the function fails, it returns a nonzero error value.

Possible return codes include, but are not limited to, the following.

Return code/value Description
NTE_INVALID_HANDLE 0x80090026L An internal handle is not valid.
STATUS_INVALID_HANDLE 0xC0000008L The provided handle is not valid.
