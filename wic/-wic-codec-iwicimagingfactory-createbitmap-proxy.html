IWICImagingFactory_CreateBitmap_Proxy function

Proxy function for the CreateBitmap method.

Syntax

HRESULT IWICImagingFactory_CreateBitmap_Proxy(
  _In_  IWICImagingFactory         *pFactory,
  _In_  UINT                       uiWidth,
  _In_  UINT                       uiHeight,
  _In_  REFWICPixelFormatGUID      pixelFormat,
  _In_  WICBitmapCreateCacheOption option,
  _Out_ IWICBitmap                 **ppIBitmap
);

Parameters

pFactory [in]

Type: IWICImagingFactory*

uiWidth [in]

Type: UINT

The width of the new bitmap .

uiHeight [in]

Type: UINT

The height of the new bitmap.

pixelFormat [in]

Type: REFWICPixelFormatGUID

The pixel format of the new bitmap.

option [in]

Type: WICBitmapCreateCacheOption

The cache creation options of the new bitmap.

ppIBitmap [out]

Type: IWICBitmap**

A pointer that receives a pointer to the new bitmap.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
