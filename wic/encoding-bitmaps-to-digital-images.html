Encoding Image Data

This section contains conceptual and how-to topics that describe Windows Imaging Component (WIC) bitmap encoders which are used to encode digital images.

In this section

Topic Description
Image Encoding Overview An encoder writes image data to a stream. Encoders can compress, encrypt, and alter the image pixels in a number of ways prior to writing them to the stream.
