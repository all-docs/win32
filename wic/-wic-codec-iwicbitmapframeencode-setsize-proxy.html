IWICBitmapFrameEncode_SetSize_Proxy function

Proxy function for the SetSize method.

Syntax

HRESULT IWICBitmapFrameEncode_SetSize_Proxy(
  _In_ IWICBitmapFrameEncode *THIS_PTR,
  _In_ UINT                  uiWidth,
  _In_ UINT                  uiHeight
);

Parameters

THIS_PTR [in]

Type: IWICBitmapFrameEncode*

Pointer to this IWICBitmapFrameEncode object.

uiWidth [in]

Type: UINT

The width of the output image.

uiHeight [in]

Type: UINT

The height of the output image.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.

Remarks

Requirements
