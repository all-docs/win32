IWICComponentFactory_CreateQueryWriterFromBlockWriter_Proxy function

Proxy function for the CreateQueryWriterFromBlockWriter method.

  _In_  IWICComponentFactory    *THIS_PTR,
  _In_  IWICMetadataBlockWriter *pIBlockWriter,
  _Out_ IWICMetadataQueryWriter **ppIQueryWriter
);

Parameters

THIS_PTR [in]

Type: IWICComponentFactory*

Pointer to this IWICComponentFactory object.

pIBlockWriter [in]

Type: IWICMetadataBlockWriter*

ppIQueryWriter [out]

Type: IWICMetadataQueryWriter**

Return value
