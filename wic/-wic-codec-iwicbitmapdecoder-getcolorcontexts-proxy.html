IWICBitmapDecoder_GetColorContexts_Proxy function

Proxy function for the GetColorContexts method.

Syntax

HRESULT IWICBitmapDecoder_GetColorContexts_Proxy(
  _In_    IWICBitmapDecoder *THIS_PTR,
  _In_    UINT              cCount,
  _Inout_ IWICColorContext  **ppIColorContexts,
  _Out_   UINT              *pcActualCount
);

Parameters

THIS_PTR [in]

Type: IWICBitmapDecoder*

Pointer to this IWICBitmapDecoder object.

cCount [in]

Type: UINT

The number of color contexts to retrieve.

This value must be the size of, or smaller than, the size available to ppIColorContexts.

ppIColorContexts [in, out]

Type: IWICColorContext**

A pointer that receives a pointer to the IWICColorContext.

pcActualCount [out]

Type: UINT*

A pointer that receives the number of color contexts contained in the image.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
