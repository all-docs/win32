IWICMetadataBlockReader_GetCount_Proxy function

Proxy function for the GetCount method.

Syntax

HRESULT IWICMetadataBlockReader_GetCount_Proxy(
  _In_  IWICMetadataBlockReader *THIS_PTR,
  _Out_ UINT                    *pcCount
);

Parameters

THIS_PTR [in]

Type: IWICMetadataBlockReader*

Pointer to this IWICMetadataBlockReader object.

pcCount [out]

Type: UINT*

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
