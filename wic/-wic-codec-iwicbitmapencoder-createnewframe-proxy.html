IWICBitmapEncoder_CreateNewFrame_Proxy function

Proxy function for the CreateNewFrame method.

Syntax

HRESULT IWICBitmapEncoder_CreateNewFrame_Proxy(
  _In_    IWICBitmapEncoder     *THIS_PTR,
  _Out_   IWICBitmapFrameEncode **ppIFrameEncode,
  _Inout_ IPropertyBag2         **ppIEncoderOptions
);

Parameters

THIS_PTR [in]

Type: IWICBitmapEncoder*

Pointer to this IWICBitmapEncoder object.

ppIFrameEncode [out]

Type: IWICBitmapFrameEncode**

A pointer that receives a pointer to the new instance of an IWICBitmapFrameEncode.

ppIEncoderOptions [in, out]

Type: IPropertyBag2**

The named properties to use for frame initialization.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
