IWICComponentInfo_GetCLSID_Proxy function

Proxy function for the GetCLSID method.

Syntax

HRESULT IWICComponentInfo_GetCLSID_Proxy(
  _In_  IWICComponentInfo *THIS_PTR,
  _Out_ CLSID             *pclsid
);

Parameters

THIS_PTR [in]

Type: IWICComponentInfo*

Pointer to this IWICComponentInfo object.

pclsid [out]

Type: CLSID*

A pointer that receives the component's CLSID.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.

Remarks

Requirements
