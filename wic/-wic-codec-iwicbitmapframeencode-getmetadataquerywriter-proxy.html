IWICBitmapFrameEncode_GetMetadataQueryWriter_Proxy function

Proxy function for the GetMetadataQueryWriter method.

Syntax

HRESULT IWICBitmapFrameEncode_GetMetadataQueryWriter_Proxy(
  _In_  IWICBitmapFrameEncode   *THIS_PTR,
  _Out_ IWICMetadataQueryWriter **ppIMetadataQueryWriter
);

Parameters

THIS_PTR [in]

Type: IWICBitmapFrameEncode*

Pointer to this IWICBitmapFrameEncode object.

ppIMetadataQueryWriter [out]

Type: IWICMetadataQueryWriter**

A pointer that receives a metadata query writer for the frame.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.

Remarks
