IWICImagingFactory_CreateBitmapFromHBITMAP_Proxy function

Proxy function for the CreateBitmapFromHBITMAP method.

Syntax

HRESULT IWICImagingFactory_CreateBitmapFromHBITMAP_Proxy(
  _In_  IWICImagingFactory          *pFactory,
  _In_  HBITMAP                     hBitmap,
  _In_  HPALETTE                    hPalette,
  _In_  WICBitmapAlphaChannelOption options,
  _Out_ IWICBitmap                  **ppIBitmap
);

Parameters

pFactory [in]

Type: IWICImagingFactory*

hBitmap [in]

Type: HBITMAP

A bitmap handle to create the bitmap from.

hPalette [in]

Type: HPALETTE

A palette handle used to create the bitmap.

options [in]

Type: WICBitmapAlphaChannelOption

The alpha channel options to create the bitmap.

ppIBitmap [out]

Type: IWICBitmap**

A pointer that receives a pointer to the new bitmap.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
