IEnumString_Next_WIC_Proxy function

Windows Imaging Component (WIC) proxy function for IEnumString::Next.

Syntax

HRESULT IEnumString_Next_WIC_Proxy(
  _In_ IEnumString *THIS_PTR
);

Parameters

THIS_PTR [in]

Type: IEnumString*

PARAMDESC

Return value
