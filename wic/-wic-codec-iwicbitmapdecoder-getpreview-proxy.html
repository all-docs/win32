IWICBitmapDecoder_GetPreview_Proxy function

Proxy function for the GetPreview method.

Syntax

HRESULT IWICBitmapDecoder_GetPreview_Proxy(
  _In_  IWICBitmapDecoder *THIS_PTR,
  _Out_ IWICBitmapSource  **ppIBitmapSource
);

Parameters

THIS_PTR [in]

Type: IWICBitmapDecoder*

Pointer to this IWICBitmapDecoder object.

ppIBitmapSource [out]

Type: IWICBitmapSource**

A pointer that receives a pointer to the preview bitmap if supported.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.

Remarks
