System.Photo.Brightness photo metadata policy

The photo metadata policy for the System.Photo.Brightness property.

PKEY

PKEY_Photo_Brightness

Containers

JPEG, TIFF

Read-only

Yes

Input type

Double

Conflict-resolution policy

This value is generated from System.Photo.BrightnessNumerator and System.Photo.BrightnessDenominator. It can't be written directly. Values from different schemas are reconciled.
