IWICMetadataQueryWriter_SetMetadataByName_Proxy function

Proxy function for the SetMetadataByName method.

Syntax

HRESULT IWICMetadataQueryWriter_SetMetadataByName_Proxy(
  _In_       IWICMetadataQueryWriter *THIS_PTR,
  _In_       LPCWSTR                 wzName,
  _In_ const PROPVARIANT             *pvarValue
);

Parameters

THIS_PTR [in]

Type: IWICMetadataQueryWriter*

Pointer to this IWICMetadataQueryWriter object.

wzName [in]

Type: LPCWSTR

The name of the metadata item.

pvarValue [in]

Type: const PROPVARIANT*

The metadata to set.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
