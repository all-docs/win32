IWICBitmapCodecInfo_DoesSupportLossless_Proxy function

Proxy function for the DoesSupportLossless method.

Syntax

HRESULT IWICBitmapCodecInfo_DoesSupportLossless_Proxy(
  _In_  IWICBitmapCodecInfo *THIS_PTR,
  _Out_ BOOL                *pfSupportLossless
);

Parameters

THIS_PTR [in]

Type: IWICBitmapCodecInfo*

Pointer to this IWICBitmapCodecInfo object.

pfSupportLossless [out]

Type: BOOL*

A pointer that receives TRUE if the codec supports lossless formats; otherwise, FALSE.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.

Remarks
