IWICImagingFactory_CreateStream_Proxy function

Proxy function for the CreateStream method.

Syntax

HRESULT IWICImagingFactory_CreateStream_Proxy(
  _In_  IWICImagingFactory *pFactory,
  _Out_ IWICStream         **ppIWICStream
);

Parameters

pFactory [in]

Type: IWICImagingFactory*

ppIWICStream [out]

Type: IWICStream**

A pointer that receives a pointer to a new IWICStream.

Return value

Type: HRESULT

If this function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.

Remarks

Requirements
