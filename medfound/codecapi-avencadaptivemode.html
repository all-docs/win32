CODECAPI_AVEncAdaptiveMode property

Specifies the adaptive encoding mode for a video encoder.

Data type

ULONG (VT_UI4)

Property GUID

CODECAPI_AVEncAdaptiveMode

Property value

The value of this property is a bitwise OR of values from the eAVEncAdaptiveMode enumeration.

Requirements

Requirement Value
Minimum supported client Windows 8 [desktop apps | UWP apps]
Minimum supported server Windows Server 2012 [desktop apps | UWP apps]
