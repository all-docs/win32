OPM_SET_PROTECTION_LEVEL_ACCORDING_TO_CSS_DVD

Sets the HDCP protection level for DVD playback, following DVD Content Scramble System (CSS) rules.

Requirement Value
Command GUID OPM_SET_PROTECTION_LEVEL_ACCORDING_TO_CSS_DVD
Input data An OPM_SET_PROTECTION_LEVEL_PARAMETERS structure

Â 

Remarks

This command is used to set High-Bandwidth Digital Content Protection (HDCP) when playing standard DVDs. Unlike the OPM_SET_PROTECTION_LEVEL command, if HDCP cannot be set for some reason, this command does not stop playback. (DVD CSS rules contain a "best effort" provision for players.) Otherwise, this command is identical to the OPM_SET_PROTECTION_LEVEL command.
