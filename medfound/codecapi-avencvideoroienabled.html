CODECAPI_AVEncVideoROIEnabled property

Indicates whether MFSampleExtension_ROIRectangle attribute set on the input sample will be honored or not.

Data type

ULONG (VT_UI4)

Property GUID

CODECAPI_AVEncVideoROIEnabled

Remarks

Default value is 0.

If an encoder MFT accepts a non-zero value, it is expected that the encoder will honor the MFSampleExtension_ROIRectangle attribute set on the input sample.
