MF_TOPONODE_PRESENTATION_DESCRIPTOR attribute

Contains a pointer to the presentation descriptor for the media source.

Data type

IUnknown*

Remarks

This attribute applies to source nodes (MF_TOPOLOGY_SOURCESTREAM_NODE).

The value of the attribute is a pointer to the presentation descriptor's IMFPresentationDescriptor interface. This attribute is required.

The GUID constant for this attribute is exported from mfuuid.lib.
