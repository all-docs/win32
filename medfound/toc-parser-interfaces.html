Table of Contents Parser Interfaces

The following interfaces proved access to the services of Table of Contents Parser.

Interface Description
IToc Represents an individual table of contents.
ITocCollection Represents a collection of tables of contents.
ITocEntry Represents an individual entry in a table of contents.
ITocEntryList Represents a list of entries in a table of contents.
ITocParser Provides methods for embedding a tables of contents in a video file and reading tables of contents from a video file.
