MF_MT_MAX_MASTERING_LUMINANCE attribute

Specifies the maximum luminance of the display on which the content was authored, in Nits. This attribute has the same semantics as max_display_mastering_luminance as defined in the CEA-861.3 standard.

Data type

UINT32
