MFNETSOURCE_AUTORECONNECTLIMIT property

The number of times the network source tries to reconnect to the server and resume streaming if the connection is lost.

Data type

PROPVARIANT type (vt)

PROPVARIANT member

DWORD (stored as LONG)

VT_I4

lVal

Remarks

The constant MFNETSOURCE_AUTORECONNECTLIMIT defines the GUID for this property key. The property identifier (PID) is zero.

Applications can use this property to configure the network source. To set the property, pass an IPropertyStore object to the source resolver. For more information, see Configuring a Media Source.
