MESessionStarted event

Raised when the IMFMediaSession::Start method completes asynchronously.

Event values

Possible values retrieved from IMFMediaEvent::GetValue include the following.

VARTYPE Description
VT_EMPTY No event data.

Attributes

The following attributes are defined for this event.

Attribute Description
MF_EVENT_PRESENTATION_TIME_OFFSET Offset between the presentation time and the media source's time stamps.
