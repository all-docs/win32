MF_POLICY_ID attribute

An identifier that can be set on an IMFOutputPolicy.

Data type

UINT32 (treated as BOOL)

Remarks

The value can be recieved from the MEPolicySet media event. It is stored as a VT_UI4 type payload in the MEPolicySet event.
