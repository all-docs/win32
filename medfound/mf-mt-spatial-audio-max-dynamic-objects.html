MF_MT_SPATIAL_AUDIO_MAX_DYNAMIC_OBJECTS attribute

Specifies the maximum number of dynamic audio objects that can be rendered by the audio endpoint simulataneously.

Data type

UINT32

Remarks

An IMFSpatialAudioSample may contain fewer samples than the number specified by this attribute. If a sample contains more audio objects than specified by this attribute, the behavior is undefined.
