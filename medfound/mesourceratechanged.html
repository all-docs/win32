MESourceRateChanged event

Raised by a media source when the playback rate changes. This event is sent after the IMFRateControl::SetRate method completes asynchronously.

Event values

Possible values retrieved from IMFMediaEvent::GetValue include the following.

VARTYPE Description
VT_R4 The new playback rate.
