MFPKEY_ASFMEDIASINK_BASE_SENDTIME property

Base send time for the ASF media sink, in milliseconds.

Data type

PROPVARIANT type (vt)

PROPVARIANT member

ULONG

VT_UI4

ulVal

Remarks

The send time is the time at which an ASF packet is sent across the network. It does not correspond directly to the presentation time of the data in the packet.

You can use this property to configure the ASF media sink. The usage depends on which function you call to create the ASF media sink:

  • MFCreateASFMediaSink: Query the retrieved IMFMediaSink pointer for the IPropertyStore interface.

  • MFCreateASFMediaSinkActivate: Call IMFASFContentInfo::GetEncodingConfigurationPropertyStore on the IMFASFContentInfo pointer specified in the pContentInfo parameter.
