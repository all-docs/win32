MFPKEY_MediaProtectionSystemIdMapping property

Specifies a IPropertySet that defines the mapping from a property system id to a property system activation id.

PROPVARIANT type (vt)

PROPVARIANT member

IUNKNOWN

VT_UNKNOWN

Remarks

This property key is used to pass an IPropertySet that contains mappings between protection system GUIDs and runtime class strings to a media source.
