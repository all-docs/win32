Table of Contents Parser Structures

The following structures are used by Table of Contents Parser.

Structure Description
TOC_DESCRIPTOR Holds descriptive information for a table of contents.
TOC_ENTRY_DESCRIPTOR Holds descriptive information for an entry in a table of contents.
