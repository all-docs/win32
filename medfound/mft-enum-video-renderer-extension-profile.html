MFT_ENUM_VIDEO_RENDERER_EXTENSION_PROFILE attribute

Contains a list of all VideoRendererExtensionProfile entries in the VideoRendererExtensionProfiles tag of a UWP app manifest file. MFTEnumEx stores this on the attribute store of the IMFActivate object that MFTEnumEx creates for MFTs that have an associated UWP manifest containing the VideoRendererExtensionProfiles tag.
