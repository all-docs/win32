MF_CAPTURE_ENGINE_ENCODER_MFT_FIELDOFUSE_UNLOCK_Attribute attribute

Enables the capture engine to use an encoder that has field-of-use restrictions.

Data type

IUnknown*

Remarks

The value of this attribute is a pointer to the IMFFieldOfUseMFTUnlock interface, implemented by the caller. The caller's implementation of this interface is expected to perform a handshake with the encoder, as described in Field of Use Restrictions. Microsoft Media Foundation does not define the handshakeâ€”typically, it would involve some sort of cryptographic exchange.

Internally, the capture engine sets the IMFFieldOfUseMFTUnlock pointer on the encoder by setting the encoder's MFT_FIELDOFUSE_UNLOCK_Attribute attribute.
