TopoEdit Menus

The following table describes the TopoEdit menus:

Menu Description
File Menu Opens a new or existing topology.
Topology Menu Adds source, transform, and output nodes. Resolves and deletes topology nodes.
Controls Menu Plays, pauses, and stops a resolved topology.
Tools Menu Logs activities of a selected topology node.
Help Menu Provides a link to the TopoEdit help topics.
