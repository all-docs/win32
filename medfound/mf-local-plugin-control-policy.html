MF_LOCAL_PLUGIN_CONTROL_POLICY attribute

Specifies a local plugin control policy.

Data type

UINT32

Remarks

Set this attribute to one of the MF_PLUGIN_CONTROL_POLICY values.

This attributes allows the app to specify a more restrictive local policy than the process-wide policy configured by IMFPluginControl.
