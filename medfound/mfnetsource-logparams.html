MFNETSOURCE_LOGPARAMS property

Array of strings with the parameters to send to the log server.

Data type

PROPVARIANT type (vt)

PROPVARIANT member

WCHAR*

VT_LPWSTR

pwszVal

Remarks

The MFNETSOURCE_LOGPARAMS constant defines the GUID for the property key. The property identifier (PID) is zero. To set this property on the network source, pass an IPropertyStore pointer to the source resolver. For more information, see Configuring a Media Source.
