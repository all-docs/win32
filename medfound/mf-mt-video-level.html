MF_MT_VIDEO_LEVEL attribute

Specifies the MPEG-2 or H.264 level in a video media type. This is an alias of MF_MT_MPEG2_LEVEL.

Data type

UINT32

Remarks

H.264 encoders:

Supported levels are extended to include the eAVEncH264VLevel5_2.

Default : Recommended default is to select the minimum level to match video encoding configuration including resolution, frame rate etc.
