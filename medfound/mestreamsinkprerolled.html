MEStreamSinkPrerolled event

Raised by a stream sink when the stream has received enough pre-roll data to begin rendering. This event is raised by media sinks that support the IMFMediaSinkPreroll interface.

Event values

Possible values retrieved from IMFMediaEvent::GetValue include the following.

VARTYPE Description
VT_EMPTY No event data.
