MFSampleExtension_MULTIPLEXED_MANAGER attribute

Provides an instance of IMFMuxStreamSampleManager which is used to access the collection of samples from the substreams of a multiplexed media source.

Data type

IUnknown

Remarks

Pass this value into IMFAttributes::GetUnknown to get an instance of IMFMuxStreamSampleManager.
