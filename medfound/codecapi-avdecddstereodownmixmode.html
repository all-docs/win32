CODECAPI_AVDecDDStereoDownMixMode property

Specifies the stereo downmix mode for a Dolby Digital audio decoder.

Data type

UINT32

Property GUID

CODECAPI_AVDecDDStereoDownMixMode

Property value

The value of this property is a member of the eAVDecDDStereoDownMixMode enumeration.

Remarks

This attribute applies when the input to the decoder is multichannel PCM audio, and the output is stereo audio.
