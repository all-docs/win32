MF_TOPONODE_WORKQUEUE_ITEM_PRIORITY attribute

Specifies the work-item priority for a branch of the topology.

Data type

UINT32

Remarks

This attribute applies to source nodes (MF_TOPOLOGY_SOURCESTREAM_NODE). The attribute is optional.

This attribute requires the MF_TOPONODE_WORKQUEUE_ID attribute on the same node.

The GUID constant for this attribute is exported from mfuuid.lib.
