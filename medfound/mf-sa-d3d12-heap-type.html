MF_SA_D3D12_HEAP_TYPE attribute

Contains a value specifying the type of heap used for the Direct3D resources in the stream.

Data type

UINT32

Remarks

The value of this attribute is a bitwise OR combination of values from the D3D12_HEAP_TYPE enumeration.
