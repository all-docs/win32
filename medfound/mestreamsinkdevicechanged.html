MEStreamSinkDeviceChanged event

Raised by the stream sinks of the enhanced video renderer (EVR) if the video device changes. For example, the EVR raises this event when it receives an EC_DISPLAY_CHANGED event.

The Media Foundation pipeline responds to this event by resubmitting all sample requests that failed while the device was changing.

Event values

Possible values retrieved from IMFMediaEvent::GetValue include the following.

VARTYPE Description
VT_EMPTY No event data.
