MF_PD_ASF_DATA_START_OFFSET attribute

Specifies the offset, in bytes, from the start of an Advanced Systems Format (ASF) file to the start of the first data packet.

Data type

UINT64

Remarks

This attribute applies to presentation descriptors for ASF content.

The IMFASFContentInfo::GeneratePresentationDescriptor method generates this attribute from the ASF metadata.
