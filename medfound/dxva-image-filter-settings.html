DXVA Image Filter Settings

The following flags define DirectX Video Acceleration (DXVA) filter settings..

Constant Description
DXVA2_DetailFilterChromaLevel Chroma detail filter, level.
DXVA2_DetailFilterChromaRadius Chroma detail filter, radius.
DXVA2_DetailFilterChromaThreshold Chroma detail filter, threshold.
DXVA2_DetailFilterLumaLevel Luma detail filter, level.
DXVA2_DetailFilterLumaRadius Luma detail filter, radius.
DXVA2_DetailFilterLumaThreshold Luma detail filter, threshold.
DXVA2_NoiseFilterChromaLevel Chroma noise filter, level.
DXVA2_NoiseFilterChromaRadius Chroma noise filter, radius.
DXVA2_NoiseFilterChromaThreshold Chroma noise filter, threshold.
DXVA2_NoiseFilterLumaLevel Luma noise filter, level.
DXVA2_NoiseFilterLumaRadius Luma noise filter, radius.
DXVA2_NoiseFilterLumaThreshold Luma noise filter, threshold.
