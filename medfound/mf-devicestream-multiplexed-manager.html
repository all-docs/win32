MF_DEVICESTREAM_MULTIPLEXED_MANAGER attribute

Provides an instance of IMFMuxStreamAttributesManager which manages the IMFAttributes describing the substreams of a multiplexed media source.

Data type

IUnknown

Remarks

Pass this value into IMFAttributes::GetUnknown to determine if the media source provides multiplexed streams and, if so, get an instance of IMFMuxStreamAttributesManager.
