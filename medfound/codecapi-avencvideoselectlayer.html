CODECAPI_AVEncVideoSelectLayer property

Sets the video select layer for a video encoder.

Data type

ULONG (VT_UI4)

Property GUID

CODECAPI_AVEncVideoSelectLayer

Remarks

This property is also used with H.264 UVC 1.5 camera encoders.
