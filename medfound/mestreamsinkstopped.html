MEStreamSinkStopped event

Raised by a stream sink when it completes the transition to the stopped state. The transition to stopped occurs when the IMFPresentationClock::Stop method is called on the sink's presentation clock.

Event values

Possible values retrieved from IMFMediaEvent::GetValue include the following.

VARTYPE Description
VT_EMPTY No event data.
