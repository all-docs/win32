MF_CAPTURE_METADATA_PHOTO_FRAME_FLASH attribute

Indicates if a flash was triggered for the captured frame.

Data type

UINT32

Value Meaning
0 A flash was not triggered on this frame.
non-zero A flash was triggered on this frame.
1 Reserved Do not explicitly check for a value of 1. Applications should only check for !=0 to indicate if a flash was triggered.
