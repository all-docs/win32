keyword element

Specifies a key word for an MFTrace provider.

Usage

<keyword
  ID = "CDATA"/>

Attributes

Attribute Type Required Description
ID CDATA Yes The name or mask of the key word

Child elements

There are no child elements.

Parent elements

Element
mfdetours
provider

Remarks

For the mfdetours element, the valid key words are listed in the topic MFTrace Keywords.

For the provider element, the key words depend on the event provider. You can use the Wevtutil.exe utility to list the key words for a given provider.
