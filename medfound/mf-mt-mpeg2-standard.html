MF_MT_MPEG2_STANDARD attribute

For a media type that describes an MPEG-2 program stream (PS) or transport stream (TS), specifies the standard that is used to multiplex the stream.

Data type

UINT32

Value Meaning
0 Standard MPEG-2 PS or TS.
1 Advanced Television Systems Committee (ATSC) standard.
2 Digital Video Broadcasting (DVB) standard.
3 Association of Radio Industries and Businesses (ARIB) standard.
