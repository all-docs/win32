MF_BYTESTREAM_LAST_MODIFIED_TIME attribute

Specifies when a byte stream was last modified.

Data type

Byte array

Remarks

This attribute is optional. The value of the attribute is a FILETIME structure.

The GUID constant for this attribute is exported from mfuuid.lib.
