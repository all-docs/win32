MFPKEY_WMADRC_AVGREF Property

Specifies the average volume level of audio content.

Constant for IPropertyBag

g_wszWMADRCAverageReference

Data Type

VT_I4

Remarks

You can get this value from the encoder after the content is processed. This value can also be set on the decoder for the purpose of dynamic range control, but it will have an effect only if the MFPKEY_WMADEC_DRCMODE property is set.

For more information on dynamic range control see the web article Windows Media Audio Professional Codec Features.
