MF_MEDIA_ENGINE_AUDIO_CATEGORY attribute

Specifies the category of the audio stream.

Data type

AUDIO_STREAM_CATEGORY

Remarks

The value of this attribute is a member of the AUDIO_STREAM_CATEGORY enumeration.

This attribute is used with the IMFMediaEngineClassFactory::CreateInstance method to initialize the Media Engine. The attribute is optional.
