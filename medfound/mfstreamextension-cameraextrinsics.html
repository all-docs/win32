MFStreamExtension_CameraExtrinsics attribute

Contains the camera extrinsics for the stream.

Data type

Get/set

To get this attribute, call IMFMediaSourceEx::GetStreamAttributes.

Remarks

The value of the attribute is a MFCameraExtrinsics.
