MF_MT_D3D12_TEXTURE_LAYOUT attribute

Indicates the texture layout options that were used to create the associated Direct3D resources.

Data type

UINT32

Remarks

The value of this attribute corresponds to a member of the D3D12_TEXTURE_LAYOUT enumeration.
