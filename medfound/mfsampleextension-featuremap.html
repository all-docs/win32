MFSampleExtension_FeatureMap attribute

Contains one MACROBLOCK_DATA structure for each macroblock in the input frame.

Data type

BLOB

Remarks

Applications can set this value on an IMFSample by invoking the IMFAttributes::SetBlob method.

Requirements

Requirement Value
Minimum supported client Windows 11, version 24H2
Minimum supported server None.
