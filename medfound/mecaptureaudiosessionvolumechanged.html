MECaptureAudioSessionVolumeChanged event

Sent by an audio capture source when the volume changes.

Event values

Possible values retrieved from IMFMediaEvent::GetValue include the following.

VARTYPE Description
VT_EMPTY No event data.

Remarks

This event is sent by the media stream of the audio capture source.

The audio capture source sends this event if an external action changes the volumeâ€”for example, if the user changes the volume through the Control Panel. The capture source does not send the event if the application changes the volume directly on the source.
