MF_MEDIATYPE_MULTIPLEXED_MANAGER attribute

Provides an instance of IMFMuxStreamMediaTypeManager which can be used to get the media types of the substreams of a multiplexed media source as well as control the combination of substreams that are multiplexed by the source.

Data type

IUnknown

Remarks

Pass this value into IMFAttributes::GetUnknown to get an instance of IMFMuxStreamMediaTypeManager.
