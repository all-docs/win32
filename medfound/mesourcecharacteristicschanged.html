MESourceCharacteristicsChanged event

Raised by a media source when the source's characteristics change.

Event values

Possible values retrieved from IMFMediaEvent::GetValue include the following.

VARTYPE Description
VT_EMPTY No event data.

Attributes

The following attributes are defined for this event.

Attribute Description
MF_EVENT_SOURCE_CHARACTERISTICS New characteristics of the media source.
MF_EVENT_SOURCE_CHARACTERISTICS_OLD Previous characteristics of the media source.
