MF_DEVICESTREAM_REQUIRED_CAPABILITIES attribute

Specifies a list of unicode strings representing the device capabilities required by the sensor transform.

Data type

WCHAR*

Remarks

This attribute is optional and only required if the sensor transform accesses a protected resource. The value must be a semicolon delimited list of string tokens defined in DeviceCapability.
