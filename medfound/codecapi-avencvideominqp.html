CODECAPI_AVEncVideoMinQP property

Specifies the minimum quantization parameter (QP) for video encoding.

Data type

ULONG (VT_UI4)

Property GUID

CODECAPI_AVEncVideoMinQP

Property value

The value is the minimum QP that is used in variable-QP encoding.
