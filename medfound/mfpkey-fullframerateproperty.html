MFPKEY_FULLFRAMERATE Property

Specifies whether the endoder is allowed to drop frames.

Constant for IPropertyBag

g_wszWMVCFullFrameRate

Data Type

VT_I4

Remarks

If you set this property to 1, the encoder will not drop frames. If you set this property to 0, the encoder might drop frames.
