CODECAPI_AVDecSoftwareDynamicFormatChange property

Sets whether to use software dynamic format change for internal resizing.

Data type

BOOL

Property GUID

CODECAPI_AVDecSoftwareDynamicFormatChange

Property value

If set to TRUE, software dynamic format change is used for internal resizing.

Requirements

Requirement Value
Minimum supported client Windows 8 [desktop apps | UWP apps]
