File Menu

The File menu provides the following commands:

Command Description
New... Clears the existing topology and removes topology nodes from the Topology Pane.If a topology is currently open, prompts the user to save the topology. For information about saving the topology, see Saving and Loading Topologies.Creates an empty topology. The user can add topology nodes to the new topology. For more information, see Building Topologies by Using TopoEdit.
Render File... Creates a playback topology for a media file. TopoEdit automatically creates and connects the nodes in the topology. For more information, see Building a Playback Topology with TopoEdit.
Render Transcode... Creates a transcoding topology. For more information, see Building a Transcode Topology with TopoEdit. Note: This feature requires WindowsÂ 7.
Open... Loads an existing topology saved as an XML file. See Saving and Loading Topologies.
Save... Saves the currently open topology as an XML file. See Saving and Loading Topologies.
Exit Closes the TopoEdit application.
