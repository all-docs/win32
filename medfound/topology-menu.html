Topology Menu

The Topology menu provides the following commands:

Command Description
Add Source... Adds a source node for the specified media file. See Adding Source Nodes with TopoEdit.
Add SAR Adds an output node for the Streaming Audio Renderer (SAR). See Adding Output Nodes with TopoEdit.
Add EVR Adds an output node for the enhanced video renderer (EVR). See Adding Output Nodes with TopoEdit.
Add Transform... Adds the specified transform node. See Adding Transform Nodes with TopoEdit.
Add Tee Adds a tee node for sending the output to multiple output nodes.
Add Custom MFT... Adds a custom Media Foundation Transform (MFT). See Adding Transform Nodes with TopoEdit.
Add Custom Sink... Adds an output node for a custom stream sink. See Adding Output Nodes with TopoEdit.
Add Video Capture Source... Adds a source node for a video capture device. Note: This feature requires WindowsÂ 7.
Add Audio Capture Source... Adds a source node for an audio capture device. Note: This feature requires WindowsÂ 7.
Resolve Topology Creates a full topology for playback. See Resolving a Topology with TopoEdit.
Delete Removes an item, topology nodes or node connections from the Topology Pane.
