MF_MT_D3D12_RESOURCE_FLAG_DENY_SHADER_RESOURCE attribute

Indicates whether shader resource view creation is disallowed for the Direct3D resources in the stream associated with the media type.

Data type

UINT32

Remarks

The value of this attribute corresponds to the D3D12_RESOURCE_FLAG_DENY_SHADER_RESOURCE member of the D3D12_RESOURCE_FLAGS enumeration.
