MF_ASFPROFILE_MAXPACKETSIZE attribute

Specifies the maximum packet size for an ASF file, in bytes.

Data type

UINT32

Remarks

This attribute applies to ASF profile objects. To set this attribute, use the IMFASFProfile interface.

The GUID constant for this attribute is exported from mfuuid.lib.
