CODECAPI_AVEncVideoContentType property

Indicates the type of video content for a video encoder.

Data type

ULONG

Property GUID

CODECAPI_AVEncVideoContentType

Property value

The value of this property is a member of the eAVEncVideoContentType enumeration.

Remarks

This property is a hint to the encoder to optimize its encoding settings.
