MF_MT_D3D_RESOURCE_VERSION attribute

Specifies the Direct3D version of the resources stored in the data stream associated with the media type.

Data type

UINT32

Remarks

The value of this attribute corresponds to a member of the MF_MT_D3D_RESOURCE_VERSION_ENUM enumeration.
