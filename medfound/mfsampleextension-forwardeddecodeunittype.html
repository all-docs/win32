MFSampleExtension_ForwardedDecodeUnitType attribute

Specifies the type, NALU or SEI, of a unit attached to an IMFSample in a MFSampleExtension_ForwardedDecodeUnits collection.

Data type

UINT32

Remarks

The value of this attribute is a value from the MF_CUSTOM_DECODE_UNIT_TYPE enumeration.
