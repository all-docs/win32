EAP Structures

Vendors should use the following structure types for authentication protocols and authentication providers:

  • PPP_EAP_INFO
  • PPP_EAP_INPUT
  • PPP_EAP_OUTPUT
  • PPP_EAP_PACKET
  • RAS_AUTH_ATTRIBUTE
