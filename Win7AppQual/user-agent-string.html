User Agent String

The User Agent String contains information about a browser's identity. The User Agent String is reported to the web server as an HTTP header every time that a browser makes a request to a web server. You can also access it through a client-side script. For example, you can display the User Agent String by typing the following URL into the Windows Internet ExplorerÂ 8 address bar: "javascript:alert(navigator.userAgent)". Typically, the User Agent String is parsed specifically for the MSIE substring. Based on the reported version of the browser, the client-side or server-side programming logic takes a different action.
