IDStorageQueue::CancelRequestsWithTag method (dstorage.h)

Attempts to cancel a group of previously enqueued read requests. All previously enqueued requests whose DSTORAGE_REQUEST::CancellationTag matches the formula (CancellationTag & mask) == value will be cancelled.

A cancelled request might or might not complete its original read request.

A cancelled request is not counted as a failure in either IDStorageStatusArray or DSTORAGE_ERROR_RECORD.

Syntax

void CancelRequestsWithTag(
  UINT64 mask,
  UINT64 value
);

Parameters

mask

The mask for the cancellation formula.

value

The value for the cancellation formula.

Return value

None
