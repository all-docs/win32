IDStorageQueue2 interface (dstorage.h)

Represents a DirectStorage queue to perform read operations, allowing enqueuing an operation to set a specific event object to a signaled state. Also allows you to retrieve info related to a specified compression format.

Inheritance

The IDStorageQueue2 interface inherits from the IDStorageQueue1 interface.

Methods

The IDStorageQueue2 interface has these methods.

 
IDStorageQueue2::GetCompressionSupport Retrieves support information about the queue for a specified compression format.
