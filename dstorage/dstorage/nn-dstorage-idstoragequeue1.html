IDStorageQueue1 interface (dstorage.h)

Represents a DirectStorage queue to perform read operations, allowing enqueuing an operation to set a specific event object to a signaled state.

Inheritance

The IDStorageQueue1 interface inherits from the IDStorageQueue interface.

Methods

The IDStorageQueue1 interface has these methods.

 
IDStorageQueue1::EnqeueSetEvent Enqueues an operation to the queue to set the specified event object to a signaled state.
