DSTORAGE_COMPRESSION enumeration (dstorage.h)

Defines constants that specify DirectStorage compression codec behavior.

Syntax

enum DSTORAGE_COMPRESSION : INT32 {
  DSTORAGE_COMPRESSION_FASTEST = -1,
  DSTORAGE_COMPRESSION_DEFAULT = 0,
  DSTORAGE_COMPRESSION_BEST_RATIO = 1
} ;

Constants

 
DSTORAGE_COMPRESSION_FASTEST Specifies compressing data at a fast rate that might not yield the best compression ratio.
DSTORAGE_COMPRESSION_DEFAULT Specifies compressing data at an average rate with a good compression ratio.
DSTORAGE_COMPRESSION_BEST_RATIO Specifies compressing data at a slow rate with the best compression ratio.
