IDStorageQueue::Query method (dstorage.h)

Obtains information about the queue. It includes the DSTORAGE_QUEUE_DESC structure used for the queue's creation, as well as the number of empty slots and number of entries that need to be enqueued to trigger automatic submission.

Syntax

void Query(
  DSTORAGE_QUEUE_INFO *info
);

Parameters

info

Receives the queue information.

Return value

None
