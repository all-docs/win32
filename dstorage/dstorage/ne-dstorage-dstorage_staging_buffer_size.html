DSTORAGE_STAGING_BUFFER_SIZE enumeration (dstorage.h)

Defines constants that specify common staging buffer sizes.

Syntax

enum DSTORAGE_STAGING_BUFFER_SIZE {
  DSTORAGE_STAGING_BUFFER_SIZE_0,
  DSTORAGE_STAGING_BUFFER_SIZE_32MB
} ;

Constants

 
DSTORAGE_STAGING_BUFFER_SIZE_0 Specifies that there is no staging buffer. Use this value to force DirectStorage to deallocate any memory it has allocated for staging buffers.
DSTORAGE_STAGING_BUFFER_SIZE_32MB Specifies the default staging buffer size of 32MB.
