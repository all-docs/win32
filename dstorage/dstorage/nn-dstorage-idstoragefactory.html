IDStorageFactory interface (dstorage.h)

Represents the static DirectStorage object used to create DirectStorage queues, open files for DirectStorage access, and other global operations.

Inheritance

The IDStorageFactory interface inherits from the IUnknown interface.

Methods

The IDStorageFactory interface has these methods.

 
IDStorageFactory::CreateQueue Creates a DirectStorage queue object.
IDStorageFactory::CreateStatusArray Creates a DirectStorage status array object.
IDStorageFactory::OpenFile Opens a file for DirectStorage access.
IDStorageFactory::SetDebugFlags Sets flags used to control the debug layer.
IDStorageFactory::SetStagingBufferSize Sets the size of staging buffer(s) used to temporarily store content loaded from the storage device before they are decompressed.
