DSTORAGE_PRIORITY enumeration (dstorage.h)

Defines constants that specify the priority of a DirectStorage queue.

Syntax

enum DSTORAGE_PRIORITY {
  DSTORAGE_PRIORITY_LOW,
  DSTORAGE_PRIORITY_NORMAL,
  DSTORAGE_PRIORITY_HIGH,
  DSTORAGE_PRIORITY_REALTIME,
  DSTORAGE_PRIORITY_FIRST,
  DSTORAGE_PRIORITY_LAST,
  DSTORAGE_PRIORITY_COUNT
} ;

Constants

 
DSTORAGE_PRIORITY_LOW Specifies low priority.
DSTORAGE_PRIORITY_NORMAL Specifies normal priority.
DSTORAGE_PRIORITY_HIGH Specifies high priority.
DSTORAGE_PRIORITY_REALTIME Specifies real-time priority.
DSTORAGE_PRIORITY_FIRST These values can be used for iterating over all priority levels.
DSTORAGE_PRIORITY_LAST These values can be used for iterating over all priority levels.
DSTORAGE_PRIORITY_COUNT These values can be used for iterating over all priority levels.
