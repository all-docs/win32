DSTORAGE_REQUEST_OPTIONS structure (dstorage.h)

Options for a DirectStorage request.

Syntax

struct DSTORAGE_REQUEST_OPTIONS {
  DSTORAGE_COMPRESSION_FORMAT       CompressionFormat : 8;
  DSTORAGE_REQUEST_SOURCE_TYPE      SourceType : 1;
  DSTORAGE_REQUEST_DESTINATION_TYPE DestinationType : 7;
  UINT64                            Reserved : 48;
};

Members

CompressionFormat

A DSTORAGE_COMPRESSION_FORMAT enum value indicating how the data is compressed.

Reserved1

Type: UINT8[7]

Reserved fields. Must be 0.

SourceType

A DSTORAGE_REQUEST_SOURCE_TYPE enum value indicating whether the source of the request is a file or a block of memory.

DestinationType

A DSTORAGE_REQUEST_DESTINATION_TYPE enum value indicating the destination of the request. Block of memory, resource.

Reserved

Reserved field. Must be 0.
