DSTORAGE_SOURCE union (dstorage.h)

Describes the source specified for a DirectStorage request. For a request, the value of request.Options.SourceType determines which of these union fields is active. See IDStorageQueue::EnqueueRequest.

Syntax

union DSTORAGE_SOURCE {
  DSTORAGE_SOURCE_MEMORY Memory;
  DSTORAGE_SOURCE_FILE   File;
};
