IDStorageQueue::GetErrorEvent method (dstorage.h)

Obtains an event to wait on. When there is any error happening for read requests in this queue, the event will be signaled, and you may call RetrieveErrorRecord to retrieve diagnostic information.

Syntax

HANDLE GetErrorEvent();

Return value

A HANDLE to an event.
