IDStorageQueue interface (dstorage.h)

Represents a DirectStorage queue to perform read operations.

Inheritance

The IDStorageQueue interface inherits from the IUnknown interface.

Methods

The IDStorageQueue interface has these methods.

 
IDStorageQueue::CancelRequestsWithTag Attempts to cancel a group of previously enqueued read requests.
IDStorageQueue::Close Closes the DirectStorage queue, regardless of the reference count on this object.
IDStorageQueue::EnqueueRequest Enqueues a read request to the queue.
IDStorageQueue::EnqueueSignal Enqueues fence write.
IDStorageQueue::EnqueueStatus Enqueues a status write.
IDStorageQueue::GetErrorEvent Obtains an event to wait on.
IDStorageQueue::Query Obtains information about the queue.
IDStorageQueue::RetrieveErrorRecord When the error event is signaled, this function can be called to retrieve a DSTORAGE_ERROR_RECORD.
IDStorageQueue::Submit Submits all requests enqueued so far to DirectStorage to be executed.
