Side-by-Side Assembly Structures

The side-by-side assembly API uses the following structures.

Structure Description
ASSEMBLY_INFO Contains information about an assembly in the system's side-by-side assembly store.
FUSION_INSTALL_REFERENCE Contains information about the application which references the side-by-side assembly.
