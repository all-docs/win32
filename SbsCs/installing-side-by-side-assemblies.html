Installing Side-by-side Assemblies

You may install side-by-side assemblies as shared assemblies or as private assemblies. For more information, see Installing Side-by-Side Assemblies as Private Assemblies and Installing Side-by-side Assemblies as Shared Assemblies.

Note that:

  • Assemblies installed to the global assembly cache by an installation using the per-user installation context are not protected by Windows File Protection.
  • Assemblies that are installed to the global assembly cache by an installation using the per-machine installation context are protected by Windows File Protection.
