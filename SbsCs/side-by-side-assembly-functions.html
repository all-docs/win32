Side-by-Side Assembly Functions

The side-by-side assembly API uses the following functions to access the Side-by-Side Assembly interfaces.

Function Description
CreateAssemblyCache Obtains an instance of the IAssemblyCache interface.
CreateAssemblyNameObject Obtains an instance of the IAssemblyName interface.
