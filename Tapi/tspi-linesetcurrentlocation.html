TSPI_lineSetCurrentLocation

This TSPI_lineSetCurrentLocation function is obsolete. TAPI called this function when the user (using the Dialing Properties dialog box) or an application (using the lineSetCurrentLocation function) changed the address translation location.

Currently, a change in translation location results in a LINE_DEVSTATE message with dwParam1 set to LINEDEVSTATE_TRANSLATECHANGE.
