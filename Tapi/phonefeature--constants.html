PHONEFEATURE_ Constants

The PHONEFEATURE_ constants list the operations that can be invoked on a phone using this API. Each of the PHONEFEATURE_ values (except PHONEFEATURE_GENERICPHONE) corresponds to a TAPI function with an identical or similar name.
