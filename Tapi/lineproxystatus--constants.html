LINEPROXYSTATUS_ Constants

The LINEPROXYSTATUS_ constants indicate the status of the proxy on a line that the application currently has open.

See LINEPROXYREQUEST_ Constants for a list and description of all possible proxy request values.

LINEPROXYSTATUS_ALLOPENFORACD

The line now has proxies open for all the proxy request types required for ACD operations by TAPI version 3.0 and higher.

LINEPROXYSTATUS_CLOSE

A proxy has closed.

LINEPROXYSTATUS_OPEN

A new proxy has opened.
