Redirect Identification

When an application redirects a session, TAPI retains identification information about the location that redirected the session and the location to which it was redirected.

"Redirecting" refers to the location that invoked the redirect operation. "Redirection" identifies the new destination for the session.

Not all service providers support use of this information.
