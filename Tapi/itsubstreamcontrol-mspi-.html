ITSubStreamControl (MSPI)

The ITSubStreamControl interface is exposed on the Stream Object if there is an associated MSP and it implements substreams. The methods of this interface allow an application to create and remove substreams.

The ITSubStreamControl interface is implemented by an MSP. Please see ITSubStreamControl in the MSP Interface section for details on this interface.
