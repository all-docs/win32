TAPI Quick Function Reference

The following hyperlinked summary of TAPI functions is organized by category of operation. For additional information, see the overviews for Assisted Telephony and Service Levels.

Reference section Description
Basic Telephony Services Reference Functions associated with POTS-level telephony tasks, such as lineMakeCall.
Supplementary Telephony Services Reference Functions with more advanced features, such as call redirect.
Extended Telephony Services Reference Functions that enable access to service provider-specific functions not directly defined by TAPI.
Assisted Telephony Services Reference Functions that establish voice calls and media calls, such as in using a phone dialer.
