What's New for TSPI Version 3.0

TSPI 3.0 adds functions and messages to support communication with a Media Service Provider (MSP). Functions have also been added which enhance the ease of tracking conferences using the Call Hub concept.

Functions and messages that were new or changed for TSPI version 3.0 are as follows:

  • TSPI_lineCloseMSPInstance
  • TSPI_lineCreateMSPInstance
  • TSPI_lineGetCallHubTracking
  • TSPI_lineGetCallIDs
  • TSPI_lineMSPIdentify
  • TSPI_lineReceiveMSPData
  • TSPI_lineSetCallHubTracking
  • LINE_QOSINFO
  • LINE_SENDMSPDATA
  • LINE_DEVSPECIFICEX
  • LINEADDRCAPFLAGS_NOPSTNADDRESSTRANSLATION flag
  • LINEERR_INVALADDRESSTYPE flags
