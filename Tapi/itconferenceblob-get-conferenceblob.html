ITConferenceBlob::get_ConferenceBlob method

[ Rendezvous IP Telephony Conferencing controls and interfaces are not available for use in Windows Vista, Windows Server 2008, and subsequent versions of the operating system. The RTC Client API provides similar functionality.]

The get_ConferenceBlob method gets a pointer to the textual conference blob currently stored in the conference blob object.

Syntax

HRESULT get_ConferenceBlob(
  [out] BSTR *ppBlob
);

Parameters

ppBlob [out]

Pointer to a BSTR containing the conference blob.

Return value

This method can return one of these values.

Return code Description
S_OK Method succeeded.
E_POINTER The ppBlob parameter is not a valid pointer.
E_OUTOFMEMORY Insufficient memory exists to perform the operation.
E_FAIL Unspecified error.
E_NOTIMPL This method is not yet implemented.

Remarks

The application must use SysFreeString to free the memory allocated for the ppBlob parameter.

Requirements

Requirement Value
TAPI version Requires TAPI 3.0 or later

Library Uuid.lib
DLL Sdpblb.dll
