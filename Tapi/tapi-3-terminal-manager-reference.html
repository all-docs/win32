TAPI 3 Terminal Manager Reference

Terminal Manager interfaces Description
ITTerminalControl Provides methods to connect and disconnect a terminal from a filter graph, and run or stop a renderer.
ITTerminalManager Provides methods to create dynamic terminals.
ITTerminalManager2 Provides methods to retrieve information about pluggable terminal classes registered in the current system; derives from the ITTerminalManager interface.
