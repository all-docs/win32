ITLocalParticipant interface

The ITLocalParticipant interface is exposed on the stream object when the IPConf MSP supports the call. The MSP implements methods that allow an application to get and set participant information. The ITLocalParticipant interface is created by calling QueryInterface on ITStream.

Members

The ITLocalParticipant interface inherits from the IUnknown interface. ITLocalParticipant also has these types of members:

  • Methods

Methods

The ITLocalParticipant interface has these methods.

Method Description
get_LocalParticipantTypedInfo Gets participant information, such as e-mail name or geographical location.
put_LocalParticipantTypedInfo Sets participant information.
