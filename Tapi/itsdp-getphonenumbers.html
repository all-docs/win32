ITSdp::GetPhoneNumbers method

[ Rendezvous IP Telephony Conferencing controls and interfaces are not available for use in Windows Vista, Windows Server 2008, and subsequent versions of the operating system. The RTC Client API provides similar functionality.]

The GetPhoneNumbers method gets an array of phone numbers associated with a conference blob.

Syntax

HRESULT GetPhoneNumbers(
  [out] VARIANT *pNumbers,
  [out] VARIANT *pNames
);

Parameters

pNumbers [out]

VARIANT pointer to a SAFEARRAY of BSTRs listing phone numbers.

pNames [out]

VARIANT pointer to a SAFEARRAY of BSTRs listing names.

Return value

This method can return one of these values.

Return code Description
S_OK Method succeeded.
E_POINTER The pNumbers or pNames parameter is not a valid pointer.
E_OUTOFMEMORY Insufficient memory exists to perform the operation.
E_FAIL Unspecified error.
E_NOTIMPL This method is not yet implemented.

Remarks

The lists that pNumbers and pNames point to are the same length.

Requirements
