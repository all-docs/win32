Summaries of Call Center Programming Elements

Call center functionality is a feature of the TAPI 3.x. The programming elements appearing in the TAPI 2.2 data set support a TAPI 3.x call center. For more information about call centers, see About Call Center Controls.

The TAPI 2.2 call center programming element summaries include:

  • Call Center Functions
  • Call Center Structures
  • Call Center Messages
  • Call Center Constants
