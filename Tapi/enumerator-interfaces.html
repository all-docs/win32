Enumerator Interfaces

The following interfaces enumerate TAPI 3 elements according to COM standards. These interfaces constitute stand-alone objects, and are also summarized with their related objects.

Interface Item enumerated
IEnumAddress ITAddress
IEnumBstr BSTR strings, such as those returned by ITAddressCapabilities::EnumerateDeviceClasses.
IEnumCall ITCallInfo
IEnumCallHub ITCallHub
IEnumCallingCard ITCallingCard
IEnumLocation ITLocationInfo
IEnumParticipant ITParticipant
IEnumPhone ITPhone
IEnumPluggableSuperclassInfo ITPluggableTerminalSuperclassInfo
IEnumPluggableTerminalClassInfo ITPluggableTerminalClassInfo
IEnumQueue ITQueue
IEnumTerminal ITTerminal
IEnumTerminalClass Terminal Class
