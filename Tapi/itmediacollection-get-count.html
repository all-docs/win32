ITMediaCollection::get_Count method

[ Rendezvous IP Telephony Conferencing controls and interfaces are not available for use in Windows Vista, Windows Server 2008, and subsequent versions of the operating system. The RTC Client API provides similar functionality.]

The get_Count method gets the number of media in the session.

Syntax

HRESULT get_Count(
  [out] LONG *pVal
);

Parameters

pVal [out]

Count of media in the session.

Return value

This method can return one of these values.

Return code Description
S_OK Method succeeded.
E_POINTER The pVal parameter is not a valid pointer.
E_OUTOFMEMORY Insufficient memory exists to perform the operation.
E_FAIL Unspecified error.
E_NOTIMPL This method is not yet implemented.

Requirements

Requirement Value
TAPI version Requires TAPI 3.0 or later
Header Sdpblb.h
Library Uuid.lib
DLL Sdpblb.dll
