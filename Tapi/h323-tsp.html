The H323 TSP (H323.tsp), when paired with the H323 MSP (H323msp.dll), allows an application to make and receive video and audio calls using the H.323 protocol for IP telephony.

This TSP supports address type values of LINEADDRESSTYPE_PHONENUMBER, LINEADDRESSTYPE_IPADDRESS, and LINEADDRESSTYPE_DOMAINNAME. The LINEADDRESSTYPE_DOMAINNAME refers to the domain name of the machine.
