LINEQUEUESTATUS_ Constants

The LINEQUEUESTATUS_ constants indicate the change in status of an ACD queue on an agent handler.

LINEQUEUESTATUS_NEWQUEUE

A queue has been added to those available.

LINEQUEUESTATUS_QUEUEREMOVED

This queue has been removed from those available.

LINEQUEUESTATUS_UPDATEINFO

Update of information about the ACD queue.
