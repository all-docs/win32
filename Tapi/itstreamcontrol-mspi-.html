ITStreamControl (MSPI)

The ITStreamControl interface is exposed on the Call Object if there is an associated MSP. The methods of this interface allow an application to create and remove streams.

The ITStreamControl interface is implemented by an MSP. Please see ITStreamControl in the MSP Interface section for details on this interface.
