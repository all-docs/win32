CAudioCaptureTerminal

The CAudioCaptureTerminal terminal is derived from CSingleFilterTerminal and implements a static audio capture terminal for wave devices using the DirectShow wavein filter. Most MSPs will not need to override the implementation of this terminal.

Defined in MSPtmac.h.

Remarks

The put_Balance and get_Balance methods of CAudioCaptureTerminal are not implemented and will return E_NOTIMPL.
