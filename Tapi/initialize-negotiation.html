INITIALIZE_NEGOTIATION

The INITIALIZE_NEGOTIATION datatype is a special value of type DWORD. It can be passed as the dwDeviceID parameter in the TSPI_lineNegotiateTSPIVersion and TSPI_phoneNegotiateTSPIVersion functions. Doing so indicates that TAPI can negotiate a TSPI interface version number independent of any specific devices.
