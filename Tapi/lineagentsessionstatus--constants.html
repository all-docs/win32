LINEAGENTSESSIONSTATUS_ Constants

The LINEAGENTSESSIONSTATUS_ constants describe various agent session states.

LINEAGENTSESSIONSTATUS_NEWSESSION

A new agent session has been created.

LINEAGENTSESSIONSTATUS_STATE

Status of the current agent session.

LINEAGENTSESSIONSTATUS_UPDATEINFO

Update of the current agent session statistics
