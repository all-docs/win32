ITConnection::GetEncryptionKey method

[ Rendezvous IP Telephony Conferencing controls and interfaces are not available for use in Windows Vista, Windows Server 2008, and subsequent versions of the operating system. The RTC Client API provides similar functionality.]

The GetEncryptionKey method gets the encryption key.

Syntax

HRESULT GetEncryptionKey(
  [out] BSTR         *ppKeyType,
  [out] VARIANT_BOOL *pfValidKeyData,
  [out] BSTR         *ppKeyData
);

Parameters

ppKeyType [out]

Pointer to a BSTR containing the type of encryption key.

pfValidKeyData [out]

Indicates validity of encryption key data.

ppKeyData [out]

Pointer to a BSTR containing the encryption key data.

Return value

This method can return one of these values.

Return code Description
S_OK Method succeeded.
E_POINTER The ppKeyType, pfValidKeyData, or ppKeyData parameter is not a valid pointer.
E_OUTOFMEMORY Insufficient memory exists to perform the operation.
E_FAIL Unspecified error.
E_NOTIMPL This method is not yet implemented.

Remarks

The application must use SysFreeString to free the memory allocated for the ppKeyType and ppKeyData parameters.
