ITConnection::SetAddressInfo method

[ Rendezvous IP Telephony Conferencing controls and interfaces are not available for use in Windows Vista, Windows Server 2008, and subsequent versions of the operating system. The RTC Client API provides similar functionality.]

The SetAddressInfo method sets the address information.

Syntax

HRESULT SetAddressInfo(
  [in] BSTR          pStartAddress,
  [in] LONG          NumAddresses,
  [in] unsigned char Ttl
);

Parameters

pStartAddress [in]

Pointer to a BSTR containing the start address.

NumAddresses [in]

Number of addresses to be used for the session.

Ttl [in]

time to live (TTL) scope for transmissions on the addresses.

Return value

This method can return one of these values.

Value Meaning
S_OK Method succeeded.
E_INVALIDARG The pStartAddress, NumAddresses, or Ttl parameter is not valid.
E_OUTOFMEMORY Insufficient memory exists to perform the operation.
E_FAIL Unspecified error.
E_NOTIMPL This method is not yet implemented.

Remarks

The application must use SysAllocString to allocate memory for the pStartAddress parameter and use SysFreeString to free the memory when the variable is no longer needed.
