ITParticipantSubStreamControl interface

[ITParticipantSubStreamControl is not available for use in Windows Vista, Windows Server 2008, and subsequent versions of the operating system. The RTC Client API provides similar functionality.]

The ITParticipantSubStreamControl interface is implemented by the IPConf MSP. This interface is exposed on the call object. This interface provides methods that allow an application to either discover or control the matching of substream to participant. The ITParticipantSubStreamControl interface is created by calling QueryInterface on ITCallInfo.

Members

The ITParticipantSubStreamControl interface inherits from the IUnknown interface. ITParticipantSubStreamControl also has these types of members:

  • Methods

Methods

The ITParticipantSubStreamControl interface has these methods.

Method Description
get_ParticipantFromSubStream Gets participants associated with a given substream.
get_SubStreamFromParticipant Gets substreams associated with a given participant.
SwitchTerminalToSubStream Sets a participant on a substream.
