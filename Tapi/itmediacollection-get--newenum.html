ITMediaCollection::get__NewEnum method

[ Rendezvous IP Telephony Conferencing controls and interfaces are not available for use in Windows Vista, Windows Server 2008, and subsequent versions of the operating system. The RTC Client API provides similar functionality.]

The get__NewEnum method returns an enumerator for the collection.

Syntax

HRESULT get__NewEnum(
  [out] IUnknown **pVal
);

Parameters

pVal [out]

Pointer to an IUnknown interface on an enumerator object for the collection.

Call the QueryInterface method on the returned IUnknown interface to obtain a pointer to an IEnumVARIANT enumeration interface on the collection. IEnumVARIANT provides a number of methods that you can use to iterate through the collection.

For more information, see the following Remarks section.

Return value

This method can return one of these values.

Return code Description
S_OK Method succeeded.
E_POINTER The pVal parameter is not a valid pointer.
E_OUTOFMEMORY Insufficient memory exists to perform the operation.
E_FAIL Unspecified error.
E_NOTIMPL This method is not yet implemented.

Remarks

This method is interchangeable with get_EnumerationIf except that it returns IUnknown instead of IEnumMedia.
