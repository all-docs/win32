LINEAGENTSTATUSEX_ Constants

The LINEAGENTSTATUSEX_ constants describe the status of an agent.

LINEAGENTSTATUSEX_NEWAGENT

An agent has been added.

LINEAGENTSTATUSEX_STATE

The current agent status.

LINEAGENTSTATUSEX_UPDATEINFO

The agent status has updated information.
