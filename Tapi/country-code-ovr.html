Country or Region Code

The country or region code is relevant only when the address type is LINEADDRESSTYPE_PHONENUMBER. It identifies an area of the world.

All service providers that support use of telephone numbers must support use of this information.

TAPI 2.x: See lineGetCallInfo (dwCountryCode member of lpCallInfo).

TAPI 3.x: See ITCallInfo::get_CallInfoLong (CIL_COUNTRYCODE member of CALLINFO_LONG).
