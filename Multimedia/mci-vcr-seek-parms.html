MCI_VCR_SEEK_PARMS structure

The MCI_VCR_SEEK_PARMS structure contains parameters for the MCI_SEEK command for video-cassette recorders.

Syntax

typedef struct tagMCI_VCR_SEEK_PARMS {
  DWORD_PTR dwCallback;
  DWORD     dwTo;
  DWORD     dwMark;
  DWORD     dwAt;
} MCI_VCR_SEEK_PARMS;

Members

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

dwTo

Position to seek to.

dwMark

Numbered mark to seek for.

dwAt

Time when seek begins.

Remarks

Positions are specified in the current time format.

When assigning data to the members of this structure, set the corresponding flags in the fdwCommand parameter of the mciSendCommand function to validate the members.
