MM_MIM_LONGDATA message

The MM_MIM_LONGDATA message is sent to a window when either a complete MIDI system-exclusive message is received or when a buffer has been filled with system-exclusive data.

MM_MIM_LONGDATA 
wParam = (WPARAM) hInput 
lParam = (LPARAM) lpMidiHdr

Parameters

hInput

Handle to the MIDI input device that received the data.

lpMidiHdr

Pointer to a MIDIHDR structure identifying the buffer.

Return Value

This message does not return a value.

Remarks

The returned buffer might not be full. To determine the number of bytes recorded into the returned buffer, use the dwBytesRecorded member of the MIDIHDR structure pointed to by lpMidiHdr.

No time stamp is available with this message. For time-stamped input data, you must use the messages that are sent to callback functions.
