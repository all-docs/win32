MCI_STATUS_PARMS structure

The MCI_STATUS_PARMS structure contains information for the MCI_STATUS command.

Syntax

typedef struct {
  DWORD_PTR dwCallback;
  DWORD_PTR dwReturn;
  DWORD     dwItem;
  DWORD     dwTrack;
} MCI_STATUS_PARMS;

Members

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

dwReturn

Contains information on return.

dwItem

Capability being queried.

dwTrack

Length or number of tracks.

Remarks

The MCI_STATUS_ITEM flag must be set in the fdwCommand parameter of the mciSendCommand function to validate the dwItem member, which should contain one of the constants indicating what status information is being requested.
