ICM_ABOUT message

The ICM_ABOUT message notifies a video compression driver to display its About dialog box or queries a video compression driver to determine if it has an About dialog box. You can send this message explicitly or by using the ICAbout macro.

ICM_ABOUT 
wParam = (DWORD_PTR) (UINT_PTR) hwnd; 
lParam = 0;

Parameters

hwnd

Handle to the parent window of the displayed dialog box. You can also determine if a driver has an About dialog box by specifying -1 in this parameter, as in the ICQueryAbout macro. The driver returns ICERR_OK if it has an About dialog box or ICERR_UNSUPPORTED otherwise.

Return Value

Returns ICERR_OK if the driver supports this message or ICERR_UNSUPPORTED otherwise.
