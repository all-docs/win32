Multimedia Playback

[The feature associated with this page, MCIWnd Window Class, is a legacy feature. It has been superseded by MediaPlayer. MediaPlayer has been optimized for Windows 10 and Windows 11. Microsoft strongly recommends that new code use MediaPlayer instead of MCIWnd Window Class, when possible. Microsoft suggests that existing code that uses the legacy APIs be rewritten to use the new APIs if possible.]

The MCIWndCreate function provides the means for controlling an MCIWnd window and the device associated with it. In general, this function registers the MCIWnd window class and creates an MCIWnd window for using MCI services. This section describes how to use MCIWndCreate to perform the following tasks:

  • Adding User-Controlled Playback
  • Automating Playback
  • Using Window Styles to Change the MCIWnd Window
  • Allowing the User to Specify Devices and Files
