MCIWNDM_PUT_DEST message

The MCIWNDM_PUT_DEST message redefines the coordinates of the destination rectangle used for zooming or stretching the images of an AVI file during playback. You can send this message explicitly or by using the MCIWndPutDest macro.

MCIWNDM_PUT_DEST 
wParam = 0; 
lParam = (LPARAM) (LPRECT) prc;

Parameters

prc

Pointer to a RECT structure containing the coordinates of the destination rectangle.

Return Value

Returns zero if successful or an error otherwise.

Requirements

Requirement Value
Minimum supported client Windows 2000 Professional [desktop apps only]
Minimum supported server Windows 2000 Server [desktop apps only]
