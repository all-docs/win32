ICM_DRAW_STOP message

The ICM_DRAW_STOP message notifies a rendering driver to stop its internal clock for the timing of drawing frames. You can send this message explicitly or by using the ICDrawStop macro.

ICM_DRAW_STOP 
wParam = 0; 
lParam = 0;

Return Value

This message does not return a value.

Remarks

This message is used by hardware that performs its own asynchronous decompression, timing, and drawing.
