MCIWNDM_CAN_SAVE message

The MCIWNDM_CAN_SAVE message determines if an MCI device can save data. You can send this message explicitly or by using the MCIWndCanSave macro.

MCIWNDM_CAN_SAVE 
wParam = 0; 
lParam = 0;

Return Value

Returns TRUE if the device supports saving or FALSE otherwise.

Requirements

Requirement Value
Minimum supported client Windows 2000 Professional [desktop apps only]
Minimum supported server Windows 2000 Server [desktop apps only]
Header Vfw.h
