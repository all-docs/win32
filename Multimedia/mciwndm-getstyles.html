MCIWNDM_GETSTYLES message

The MCIWNDM_GETSTYLES message retrieves the flags specifying the current MCIWnd window styles used by a window. You can send this message explicitly or by using the MCIWndGetStyles macro.

MCIWNDM_GETSTYLES 
wParam = 0; 
lParam = 0;

Return Value

Returns a value representing the current styles of the MCIWnd window. The return value is the bitwise OR operator of the MCIWnd window styles (MCIWNDF flags).
