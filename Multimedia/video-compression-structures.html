Video Compression Structures

[The feature associated with this page, Video Compression Manager, is a legacy feature. Microsoft strongly recommends that new code does not use this feature.]

The following structures are used with video compression.

  • COMPVARS
  • ICCOMPRESS
  • ICCOMPRESSFRAMES
  • ICDECOMPRESS
  • ICDECOMPRESSEX
  • ICDRAW
  • ICDRAWBEGIN
  • ICDRAWSUGGEST
  • ICINFO
  • ICOPEN
  • ICSETSTATUSPROC
