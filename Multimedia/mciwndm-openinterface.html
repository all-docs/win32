MCIWNDM_OPENINTERFACE message

The MCIWNDM_OPENINTERFACE message attaches the data stream or file associated with the specified interface to an MCIWnd window. You can send this message explicitly or by using the MCIWndOpenInterface macro.

MCIWNDM_OPENINTERFACE 
wParam = 0; 
lParam = (LPARAM) (LPUNKNOWN) pUnk;

Parameters

pUnk

Pointer to an IAVI interface that points to a file or a data stream in a file.

Return Value

Returns zero if successful or an error otherwise.

Requirements

Requirement Value
Minimum supported client Windows 2000 Professional [desktop apps only]
Minimum supported server Windows 2000 Server [desktop apps only]
