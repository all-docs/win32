ICM_DRAW_END message

The ICM_DRAW_END message notifies a rendering driver to decompress the current image to the screen and to release resources allocated for decompression and drawing. You can send this message explicitly or by using the ICDrawEnd macro.

ICM_DRAW_END 
wParam = 0; 
lParam = 0;

Return Value

Returns ICERR_OK if successful or an error otherwise.

Remarks

The ICM_DRAW_BEGIN and ICM_DRAW_END messages do not nest. If your driver receives ICM_DRAW_BEGIN before decompression is stopped with ICM_DRAW_END, it should restart decompression with new parameters.
