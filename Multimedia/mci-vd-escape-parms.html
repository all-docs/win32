MCI_VD_ESCAPE_PARMS structure

The MCI_VD_ESCAPE_PARMS structure contains the command sent to a device for the MCI_ESCAPE command for videodisc devices.

Syntax

typedef struct {
  DWORD_PTR dwCallback;
  LPCTSTR   lpstrCommand;
} MCI_VD_ESCAPE_PARMS;

Members

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

lpstrCommand

Command to send to device.

Remarks

When assigning data to the members of this structure, set the corresponding flags in the fdwCommand parameter of the mciSendCommand function to validate the members.
