Core Audio Functions

This section describes the functions that are used by the Core Audio APIs beginning in Windows 8.

Function Description
ActivateAudioInterfaceAsync Allows Windows Store apps to access preexisting Component Object Model (COM) interfaces in the WASAPI family. Applies beginning with Windows 8.
