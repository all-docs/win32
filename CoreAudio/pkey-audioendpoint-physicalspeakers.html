PKEY_AudioEndpoint_PhysicalSpeakers

The PKEY_AudioEndpoint_PhysicalSpeakers property specifies the channel-configuration mask for the audio endpoint device. The mask indicates the physical configuration of a set of speakers and specifies the assignment of channels to speakers. For more information about channel-configuration masks, see KSPROPERTY_AUDIO_CHANNEL_CONFIG.

The vt member of the PROPVARIANT structure is set to VT_UI4.

The uintVal member of the PROPVARIANT structure contains a channel-configuration mask that is cast to type UINT.
