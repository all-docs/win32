HWConfig_NIC class

The HWConfig_NIC class is the event type class for network interface card configuration events.

The following syntax is simplified from MOF code.

Syntax

[EventType(13), EventTypeName("NIC")]
class HWConfig_NIC : HWConfig
{
  string NICName;
};
