ALPC_Unwait class

This class is the event type class for ALPC end wait events.

The following syntax is simplified from MOF code.

Syntax

[EventType(37), EventTypeName("ALPC-Unwait")]
class ALPC_Unwait : ALPC
{
  uint32 Status;
};
