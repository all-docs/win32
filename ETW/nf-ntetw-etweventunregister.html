EtwEventUnregister function

Description

Unregisters an event.

Syntax

ULONG EVNTAPI EtwEventUnregister(
    [in] REGHANDLE RegHandle
);

Parameters

RegHandle [in]

Handle to an event.

Returns

Returns an HRESULT.

Remarks

Providers can only call this function from their ControlCallback function.
