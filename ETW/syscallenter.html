SysCallEnter class

This class is the event type class for system call enter events.

The following syntax is simplified from MOF code.

Syntax

[EventType{51}, EventTypeName{"SysClEnter"}]
class SysCallEnter : PerfInfo
{
  uint32 SysCallAddress;
};
