Event Tracing Constants

The following constants are used in event tracing.

Constant Description
NT Kernel Logger Constants Constants defined for system events traced by the NT Kernel Logger event tracing session.
Logging Mode Constants Options used by the LogFileMode members of EVENT_TRACE_LOGFILE, EVENT_TRACE_PROPERTIES and TRACE_LOGFILE_HEADER structures.
