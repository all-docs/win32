SysCallExit class

This class is the event type class for system call exit events.

The following syntax is simplified from MOF code.

Syntax

[EventType{52}, EventTypeName{"SysClExit"}]
class SysCallExit : PerfInfo
{
  uint32 SysCallNtStatus;
};
