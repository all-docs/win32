Messages in Active Directory Domain Services

Messages in Active Directory Domain Services are functional in WindowsÂ 2000 and WindowsÂ NTÂ 4.0 with Service PackÂ 6a (SP6a) and later with DSClient. They are also functional in WindowsÂ 98/95 workstations with IE4.01 and later and DSClient. However, if multiselect property pages are launched from the shell, then the WM_ADSPROP_NOTIFY_ERROR message and its corresponding helper functions, ADsPropSendErrorMessage and ADsPropShowErrorDialog are not functional and will not be displayed. If multiselect property pages are launched within an admin tool, for example, AD Users and Computers, then all messages are functional and available to be displayed within the multiselect property pages.

Active Directory Domain Services use the following messages:

  • WM_ADSPROP_NOTIFY_APPLY
  • WM_ADSPROP_NOTIFY_CHANGE
  • WM_ADSPROP_NOTIFY_ERROR
  • WM_ADSPROP_NOTIFY_EXIT
  • WM_ADSPROP_NOTIFY_FOREGROUND
  • WM_ADSPROP_NOTIFY_PAGEHWND
  • WM_ADSPROP_NOTIFY_PAGEINIT
  • WM_ADSPROP_NOTIFY_SETFOCUS
  • WM_ADSPROP_SHEET_CREATE
  • WM_DSA_SHEET_CLOSE_NOTIFY
  • WM_DSA_SHEET_CREATE_NOTIFY
