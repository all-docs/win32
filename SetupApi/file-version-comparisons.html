File Version Comparisons

If the SP_COPY_NEWER flag is specified during a file copy operation, the setup functions check for an existing copy of the file in the target directory. If an existing copy is found, the functions compare the versions of the target and source file to determine which is newer.

The file version information used during version checks is that specified in the dwFileVersionMS and dwFileVersionLS members of a VS_FIXEDFILEINFO structure, used by the version functions. If one of the files does not have version resources specified or if they have the same version information, the source file is treated as the newer file.
