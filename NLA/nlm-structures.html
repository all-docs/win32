NLM structures

The following structures support the Network List Manager API:

Structure Description
NLM_SIMULATED_PROFILE_INFO Represents profile information to be used during cost simulation.
NLM_SOCKADDR Represents the IPv4/IPv6 destination address.
NLM_USAGE_DATA Represents information that indicates the data usage of a data plan.
NLM_DATA_PLAN_STATUS Represents the current status of a data plan.
