Restart Manager Callback Functions

The Restart Manager API uses the callback functions in the following table.

Callback Function Description
RM_WRITE_STATUS_CALLBACK The application that started the Restart Manager session can pass a pointer to this function to the Restart Manager functions to receive a percentage of completeness.
