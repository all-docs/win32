CIM_HostedFileSystem class

The CIM_HostedFileSystem association represents a link between the computer system and the file system hosted on the computer system.

[!IMPORTANT] The DMTF (Distributed Management Task Force) CIM (Common Information Model) classes are the parent classes upon which WMI classes are built. WMI currently supports only the CIM 2.x version schemas.

The following syntax is simplified from Managed Object Format (MOF) code and includes all inherited properties. Properties are listed in alphabetic order, not MOF order.

Syntax

[Abstract, UUID("{A2EFC898-E3D3-11d2-8601-0000F8102E5F}"), AMENDMENT]
class CIM_HostedFileSystem : CIM_SystemComponent
{
  CIM_FileSystem     REF PartComponent;
  CIM_ComputerSystem REF GroupComponent;
};

Properties

The CIM_HostedFileSystem class has these properties.

GroupComponent

Data type: CIM_ComputerSystem

Access type: Read-only

Qualifiers: Max (1), Min (1), Override ("GroupComponent")

A CIM_ComputerSystem that describes the computer system.

PartComponent

Data type: CIM_FileSystem

Access type: Read-only

Qualifiers: Override ("PartComponent"), Weak

A CIM_FileSystem that describes the file system owned by the computer system.

Remarks

The CIM_HostedFileSystem class is derived from CIM_SystemComponent.

WMI does not implement this class.
