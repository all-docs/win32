CIM_AllocatedResource class

The CIM_AllocatedResource class represents an association between logical devices and system resources and indicates that the resource is assigned to the device.

[!IMPORTANT] The DMTF (Distributed Management Task Force) CIM (Common Information Model) classes are the parent classes upon which WMI classes are built. WMI currently supports only the CIM 2.x version schemas.

The following syntax is simplified from MOF code and includes all inherited properties.

Syntax

[Abstract, UUID("{8502C579-5FBB-11D2-AAC1-006008C78BC7}"), AMENDMENT]
class CIM_AllocatedResource : CIM_Dependency
{
  CIM_LogicalDevice  REF Dependent;
  CIM_SystemResource REF Antecedent;
};

Properties

The CIM_AllocatedResource class has these properties.

Antecedent

Data type: CIM_SystemResource

Access type: Read-only

Qualifiers: Override ("Antecedent")

A CIM_SystemResource that describes the resource.

Dependent

Data type: CIM_LogicalDevice

Access type: Read-only

Qualifiers: Override ("Dependent")

A CIM_LogicalDevice that contains the logical device to which the resource is assigned.

Remarks

The CIM_AllocatedResource class is derived from CIM_Dependency.

WMI does not implement this class. For more information about classes derived from CIM_AllocatedResource, see Win32 Classes.
