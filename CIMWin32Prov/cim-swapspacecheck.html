CIM_SwapSpaceCheck class

The CIM_SwapSpaceCheck class specifies the amount of swap space that must be available on the system. The amount is specified in the SwapSpaceSize property. Details of this check are compared with the corresponding details found in a CIM_OperatingSystem object referenced by the CIM_InstalledOS association for the CIM_ComputerSystem object that describes the environment. When the value of the TotalSwapSpaceSize property is greater than or equal to the value specified in the SwapSpaceSize property, the condition is satisfied.

[!IMPORTANT] The DMTF (Distributed Management Task Force) CIM (Common Information Model) classes are the parent classes upon which WMI classes are built. WMI currently supports only the CIM 2.x version schemas.
