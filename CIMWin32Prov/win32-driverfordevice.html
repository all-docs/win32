Win32_DriverForDevice class

The Win32_DriverForDevice association WMI class relates a printer instance to a printer driver instance.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties. Properties are listed in alphabetic order, not MOF order.

Syntax

class Win32_DriverForDevice : CIM_Dependency
{
  Win32_Printer       REF Antecedent;
  Win32_PrinterDriver REF Dependent;
};

Properties

The Win32_DriverForDevice class has these properties.

Antecedent

Data type: Win32_Printer

Access type: Read/write

Reference to the Win32_Printer instance that represents the printer.

This property is inherited from CIM_Dependency.

Dependent

Data type: Win32_PrinterDriver

Access type: Read-only

Reference to the Win32_PrinterDriver instance that represents the printer driver for the printer.

This property is inherited from CIM_Dependency.

Remarks

The Win32_DriverForDevice class is derived from CIM_Dependency.
