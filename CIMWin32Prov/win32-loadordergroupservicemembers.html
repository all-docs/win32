Win32_LoadOrderGroupServiceMembers class

The Win32_LoadOrderGroupServiceMembers association WMI class relates a load order group and a base service.

[!Note]
Win32_SystemDriver objects are members of that load order group. Not all services are members of groups, and not all groups have services within them.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties. Properties are listed in alphabetic order, not MOF order.

Syntax

[Dynamic, Provider("CIMWin32"), UUID("{8502C4EF-5FBB-11D2-AAC1-006008C78BC7}"), AMENDMENT]
class Win32_LoadOrderGroupServiceMembers : CIM_Component
{
  Win32_LoadOrderGroup REF GroupComponent;
  Win32_BaseService    REF PartComponent;
};

Properties

The Win32_LoadOrderGroupServiceMembers class has these properties.

GroupComponent

Data type: Win32_LoadOrderGroup

Access type: Read-only

Qualifiers: key, Override ("GroupComponent"), MappingStrings ("WMI|Win32_LoadOrderGroup")

Reference to the instance representing the load order group properties associated with the base service.

PartComponent

Data type: Win32_BaseService

Access type: Read-only

Qualifiers: key, Override ("PartComponent"), MappingStrings ("WMI|Win32_BaseService")

Reference to the instance representing the service that is a member of a load order group.

Remarks

The Win32_LoadOrderGroupServiceMembers class is derived from CIM_Component.
