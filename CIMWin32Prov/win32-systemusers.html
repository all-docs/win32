Win32_SystemUsers class

The Win32_SystemUsers association WMI class relates a computer system and a user account on that system.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties. Properties and methods are in alphabetic order, not MOF order.

GroupComponent

Data type: Win32_ComputerSystem

Access type: Read-only

Qualifiers: key, Override ("GroupComponent"), MappingStrings ("WMI|Win32_ComputerSystem")

Reference to the instance representing the computer system containing the user account.

PartComponent

Data type: Win32_UserAccount

Access type: Read-only

Qualifiers: key, Override ("PartComponent"), MappingStrings ("WMI|Win32_UserAccount")

Reference to the instance representing the user account on the computer system.

Remarks

The Win32_SystemUsers class is derived from CIM_SystemComponent.
