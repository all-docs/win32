CIM_AssociatedSensor class

The CIM_AssociatedSensor class associates an installed sensor with a logical device. The sensor measures critical input and output properties, and can be included with the device or installed nearby.

[!IMPORTANT] The DMTF (Distributed Management Task Force) CIM (Common Information Model) classes are the parent classes upon which WMI classes are built. WMI currently supports only the CIM 2.x version schemas.

The following syntax is simplified from Managed Object Format (MOF) code and includes all inherited properties. Properties are listed in alphabetic order, not MOF order.

Syntax

[Abstract, UUID("{956597A0-7D80-11D2-AAD3-006008C78BC7}"), AMENDMENT]
class CIM_AssociatedSensor : CIM_Dependency
{
  CIM_LogicalDevice REF Dependent;
  CIM_Sensor        REF Antecedent;
};

Antecedent

Data type: CIM_Sensor

Qualifiers: Override ("Antecedent")

A CIM_Sensor that describes the sensor.

Dependent

Data type: CIM_LogicalDevice

Access type: Read-only

Qualifiers: Override ("Dependent")

A CIM_LogicalDevice that describes the logical device for which information is measured by the sensor.

Remarks

The CIM_AssociatedSensor class is derived from CIM_Dependency.

WMI does not implement this class. For more information about classes derived from CIM_AssociatedSensor, see Win32 Classes.
