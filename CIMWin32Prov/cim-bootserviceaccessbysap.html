CIM_BootServiceAccessBySAP class

The CIM_BootServiceAccessBySAP class associates a boot service and its access points.

[!IMPORTANT] The DMTF (Distributed Management Task Force) CIM (Common Information Model) classes are the parent classes upon which WMI classes are built. WMI currently supports only the CIM 2.x version schemas.

The following syntax is simplified from Managed Object Format (MOF) code and includes all inherited properties. Properties are listed in alphabetic order, not MOF order.

Syntax

[UUID("{6EDF1DD2-DB35-11d2-85FC-0000F8102E5F}"), Abstract, AMENDMENT]
class CIM_BootServiceAccessBySAP : CIM_ServiceAccessBySAP
{
  CIM_BootSAP     REF Dependent;
  CIM_BootService REF Antecedent;
};

Antecedent

Data type: CIM_BootService

Qualifiers: Override ("Antecedent")

A CIM_BootService that describes the boot service.

Dependent

Data type: CIM_BootSAP

Access type: Read-only

Qualifiers: Override ("Dependent")

A CIM_BootSAP that describes an access point for the boot service.

Remarks

The CIM_BootServiceAccessBySAP class is derived from CIM_ServiceAccessBySAP.

WMI does not implement this class.
