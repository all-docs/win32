Win32_PageFileElementSetting class

The Win32_PageFileElementSetting association WMI class relates the initial settings of a page file and the state of those settings during normal use.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties. Properties and methods are in alphabetic order, not MOF order.

Properties

The Win32_PageFileElementSetting class has these properties.

Element

Data type: Win32_PageFileUsage

Access type: Read-only

Qualifiers: key, Override ("Element"), MappingStrings ("WMI|Win32_PageFileUsage")

Reference to the instance representing the properties of a page file while the Win32 system is in use.

Setting

Data type: Win32_PageFileSetting

Access type: Read-only

Qualifiers: key, Override ("Setting"), MappingStrings ("WMI|Win32_PageFileSetting")

Reference to the instance representing the initial settings of a page file when the Win32 system is starting up.

Remarks

The Win32_PageFileElementSetting class is derived from CIM_ElementSetting.
