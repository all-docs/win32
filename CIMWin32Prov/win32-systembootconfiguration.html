Win32_SystemBootConfiguration class

The Win32_SystemBootConfiguration association WMI class relates a computer system and its boot configuration.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties. Properties and methods are in alphabetic order, not MOF order.

Setting

Data type: Win32_BootConfiguration

Access type: Read-only

Qualifiers: Override ("Setting"), MappingStrings ("WMI|Win32_BootConfiguration")

Reference to the instance representing the boot configuration for the computer system.

Remarks

The Win32_SystemBootConfiguration class is derived from Win32_SystemSetting.
