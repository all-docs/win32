D2D1_MATRIX_5X4_F

Represents a 5-by-4 matrix.

typedef D2D_MATRIX_5X4_F D2D1_MATRIX_5X4_F;

Remarks

D2D1_MATRIX_5X4 is a new name for the D2D_MATRIX_5X4_F structure. For a list of fields provided by the matrix, see D2D_MATRIX_5X4_F.
