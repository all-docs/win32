Direct2D helper classes

The D2D1 namespace defines the following classes.

In this section

Topic Description
ColorF Defines a set of constants that represents known colors and provides convenience methods for instantiating different colors.
Matrix3x2F The Matrix3x2F class represents a 3-by-2 matrix and provides convenience methods for creating matrices.
Matrix4x3F The Matrix4x3F class represents a 4-by-3 matrix and provides convenience methods for creating matrices.
Matrix4x4F The Matrix4x4F class represents a 4-by-4 matrix and provides convenience methods for creating matrices.
Matrix5x4F The Matrix5x4F class represents a 5-by-4 matrix and provides convenience methods for creating matrices.
