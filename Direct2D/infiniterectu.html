InfiniteRectU function

Creates a rectangle that has its upper-left corner set to (0u, 0u) and its lower-right corner set to (infinity, infinity).

Syntax

D2D1_RECT_U InfiniteRectU(void);

Parameters

This function has no parameters.

Return value

Type: D2D1_RECT_U

The new rectangle.
