Tint effect

This effect tints the source image by multiplying the source image by the specified color. It has a single input.

The CLSID for this effect is CLSID_D2D1Tint.

Effect properties

Display name and index enumeration Type and default value Description
ColorD2D1_TINT_PROP_COLOR D2D1_VECTOR_4F(1.0f, 1.0f, 1.0f, 1.0f) Colors from the source image are multiplied by this value.
ClampOutputD2D1_TINT_PROP_CLAMP_OUTPUT BOOLFALSE Whether or not to clamp the output values to the range [0, 1].
