D2D1_MATRIX_4X3_F

Represents a 4-by-3 matrix.

typedef D2D_MATRIX_4X3_F D2D1_MATRIX_4X3_F;

Remarks

D2D1_MATRIX_4X3 is a new name for the D2D_MATRIX_4X3_F structure. For a list of fields provided by the matrix, see D2D_MATRIX_4X3_F.
