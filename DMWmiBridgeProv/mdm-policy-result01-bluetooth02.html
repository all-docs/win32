MDM_Policy_Result01_Bluetooth02 class

The MDM_Policy_Result01_Bluetooth02 class represents the Bluetooth management policies available.

The following syntax is simplified from MOF code and includes all inherited properties.

[InPartition("local-system"), dynamic, provider("DMWmiBridgeProv")]
class MDM_Policy_Result01_Bluetooth02
{
  string InstanceID;
  string ParentID;
  sint32 AllowDiscoverableMode;
  string LocalDeviceName;
  sint32 AllowAdvertising;
  string ServicesAllowedList;
  sint32 AllowPrepairing;
};
