MDM_ActiveSync_User_Accounts01_01 class

The MDM_ActiveSync_User_Accounts01_01 class defines all available ActiveSync accounts.

The following syntax is simplified from MOF code and includes all inherited properties.

Syntax

[InPartition("local-user"), dynamic, provider("DMWmiBridgeProv")]
class MDM_ActiveSync_User_Accounts01_01
{
  string InstanceID;
  string ParentID;
  string EmailAddress;
  string Domain;
  string AccountIcon;
  string AccountType;
  string AccountName;
  string Password;
  string ServerName;
  string UserName;
};
