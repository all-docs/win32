MDM_EnterpriseModernAppManagement_StoreLicenses02_01 class

The MDM_EnterpriseModernAppManagement_StoreLicenses02_01 class is used to manage licenses for store apps.

The following syntax is simplified from MOF code and includes all inherited properties.

Syntax

[InPartition("local-system", "local-user"), dynamic, provider("DMWmiBridgeProv")]
class MDM_EnterpriseModernAppManagement_StoreLicenses02_01
{
  string InstanceID;
  string ParentID;
  string LicenseCategory;
  string LicenseUsage;
  string RequesterID;
};

Properties

The MDM_EnterpriseModernAppManagement_StoreLicenses02_01 class has these properties.

InstanceID

Data type: string

Access type: Read-only

Qualifiers: key

Optional node. License ID for a store installed app. The license ID is generally the PFN of the app.

Supported operations are Add, Get, and Delete.
