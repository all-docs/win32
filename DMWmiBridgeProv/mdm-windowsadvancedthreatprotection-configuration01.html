MDM_WindowsAdvancedThreatProtection_Configuration01 class

The MDM_WindowsAdvancedThreatProtection_Configuration01 class is used to determine the configuration of Windows Defender Advanced Threat Protection (WDATP) endpoints.

The following syntax is simplified from MOF code and includes all inherited properties.

Syntax

[InPartition("local-system"), dynamic, provider("DMWmiBridgeProv1")]
class MDM_WindowsAdvancedThreatProtection_Configuration01
{
  string InstanceID;
  string ParentID;
  sint32 SampleSharing;
  string GroupIds;
  sint32 TelemetryReportingFrequency;
};
