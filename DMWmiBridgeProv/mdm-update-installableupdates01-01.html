MDM_Update_InstallableUpdates01_01 class

The MDM_Update_InstallableUpdates01_01 class is used to manage and control the rollout of approved updates.

The following syntax is simplified from MOF code and includes all inherited properties.

Syntax

[InPartition("local-system"), dynamic, provider("DMWmiBridgeProv1")]
class MDM_Update_InstallableUpdates01_01
{
  string InstanceID;
  string ParentID;
  sint32 Type;
  sint32 RevisionNumber;
};

InstanceID

Data type: string

Access type: Read-only

Qualifiers: key

Identifies the name of the parent node. For this class, the string is the GUID of the approved update.

ParentID

Data type: string

Access type: Read-only
