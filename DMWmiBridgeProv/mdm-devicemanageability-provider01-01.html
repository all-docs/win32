MDM_DeviceManageability_Provider01_01 class

The MDM_DeviceManageability_Provider01_01 class is used retrieve the general information about MDM configuration capabilities on the device.

The following syntax is simplified from MOF code and includes all inherited properties.

Syntax

[InPartition("local-system"), dynamic, provider("DMWmiBridgeProv1")]
class MDM_DeviceManageability_Provider01_01
{
  string InstanceID;
  string ParentID;
  string ConfigInfo;
  string EnrollmentInfo;
};
