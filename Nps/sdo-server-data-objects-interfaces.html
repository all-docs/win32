Server Data Objects Interfaces

The Server Data Objects (SDO) API exposes the following interfaces.

Interface Purpose
ISdo Manipulate an SDO object.
ISdoCollection Manipulate a collection of SDO objects.
ISdoDictionaryOld Manipulate the attribute dictionary.
ISdoMachine Manage the SDO computer, and retrieve SDO objects.
ISdoServiceControl Manage the service being administered through SDO, that is, either Internet Authentication Service (IAS) or Remote Access Server (RAS). Note: Internet Authentication Service (IAS) was renamed Network Policy Server (NPS) starting with Windows Server 2008.
