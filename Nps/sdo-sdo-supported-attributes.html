[!Note]

An NPS Server Data Object has a Profiles property which provides access to a collection of attributes. The supported attributes are as follows.

  • Acct-Provider-Name
  • Acct-Provider-Type
  • Auth-Provider-Name
  • Auth-Provider-Type
  • Ignore-User-Dialin-Properties
  • Manipulation-Rule
  • Manipulation-Target
  • NP-Allowed-EAP-Type
  • NP-Allowed-Port-Types
  • NP-Authentication-Type
  • NP-Called-Station-ID
