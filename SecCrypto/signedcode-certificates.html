SignedCode.Certificates property

[The Certificates property is available for use in the operating systems specified in the Requirements section. Instead, use Platform Invocation Services (PInvoke) to call the Win32 API SignerSignEx, SignerTimeStampEx, and WinVerifyTrust functions to sign content with an Authenticode digital signature. For information about PInvoke, see Platform Invoke Tutorial. The .NET and CryptoAPI via P/Invoke: Part 1 and .NET and CryptoAPI via P/Invoke: Part 2 subsections of Extending .NET Cryptography with CAPICOM and P/Invoke may also be helpful.]

The Certificates property retrieves the collection of certificates in the signed executable file.

Syntax

SignedCode.Certificates As Certificates

Property value

The Certificates collection that contains all the certificates in the signed executable file.
