EKUs.Count property

[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows ServerÂ 2008, WindowsÂ Vista, and WindowsÂ XP. Instead, use the X509ExtensionCollection Class in the System.Security.Cryptography.X509Certificates namespace.]

The Count property retrieves the number of EKU objects in the collection.

This property is read-only.

Syntax

EKUs.Count As Long

Property value

The number of EKU objects in the collection. Each EKU object represents a single extended key usage (EKU) property of a certificate.

Remarks

The Count property can be used to specify the last EKU object in a collection when retrieving a specific EKU object using the EKUs.Item property.
