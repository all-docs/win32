Template.MajorVersion property

[The MajorVersion property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Template to retrieve the certificate extension template.]

The MajorVersion property retrieves the major version number of the template.

Syntax

Template.MajorVersion As Long

Property value

The major version number of the template.
