SHA/MD5 RSA Signature Type

CSPs for PROV_RSA_SCHANNEL must support the CALG_SSL3_SHAMD5 hash that is compatible with the Microsoft Base Cryptographic Provider used in SSL 3.0 and TLS 1.0 client authentication. This hash consists of a concatenation of an MD5 hash and a SHA hash signed with an RSA or Diffie-Hellman private key. A handle to a hash value of this type is created by using the CryptCreateHash or CPCreateHash function with CALG_SSL3_SHAMD5 as the Algid parameter. Examples of using hash functions can be seen in Example C Program: Creating and Hashing a Session Key and Example C Program: Signing a Hash and Verifying the Hash Signature.
