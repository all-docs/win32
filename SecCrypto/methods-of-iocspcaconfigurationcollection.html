Methods of IOCSPCAConfigurationCollection

The following methods are defined by the IOCSPCAConfigurationCollection interface. The property access methods are not shown here. To see the properties for IOCSPCAConfigurationCollection, see Properties of IOCSPCAConfigurationCollection

Method Description
CreateCAConfiguration Creates a new certification authority (CA) configuration and adds it to the configuration set.
DeleteCAConfiguration Removes a named CA configuration from the configuration set.
