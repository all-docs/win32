IChain2::Certificates property

[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows ServerÂ 2008, WindowsÂ Vista, and WindowsÂ XP. Instead, use the X509Chain Class in the System.Security.Cryptography.X509Certificates namespace.]

The Certificates property retrieves a Certificates collection that represents the certificates in the chain. This is the default property.

This property is read-only.

Syntax

Chain.Certificates As Certificates

Property value

A Certificates collection that is used to retrieve information about each certificate in the chain. The first certificate in the returned collection, Certificates.Item(1), is the end certificate of the chain. The last certificate in the collection, Certificates.Item(Certificates.Count), is the root certificate of the chain.
