PolicyInformation.OID property

[The OID property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Policies to process policy information in the Certificate policies extension.]

The OID property retrieves the policy's OID. This is the default property.

Syntax

PolicyInformation.OID As OID

Property value

The policy's object identifier as an OID object.
