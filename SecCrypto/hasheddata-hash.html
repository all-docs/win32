HashedData.Hash method

[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows ServerÂ 2008, WindowsÂ Vista, and WindowsÂ XP. Instead, use the HashAlgorithm Class in the System.Security.Cryptography namespace.]

The Hash method creates a hash of the specified string.

Syntax

HashedData.Hash( _
  ByVal newVal _
)

Parameters

newVal

String that contains the value to hash.

Return value

This method does not return a value.

Remarks

To create the hash of a large amount of data, call the Hash method for each piece of data. The hash of each piece of data is concatenated to the Value property until the property is read. The contents of the Value property are reset when the property is read.
