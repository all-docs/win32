[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows ServerÂ 2008, WindowsÂ Vista, and WindowsÂ XP. Instead, use the X509ExtensionCollection Class in the System.Security.Cryptography.X509Certificates namespace.]

The Extensions object represents a collection of Extension objects. Each Extension object represents a single certificate extension.

When to use

The Extensions object is used to perform the following tasks:

  • Retrieve the number of certificate extensions in the collection.
  • Retrieve a specific Extension object from the collection.
  • Iterate through the collection.
