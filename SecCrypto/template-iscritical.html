Template.IsCritical property

[The IsCritical property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Template to retrieve the certificate extension template.]

The IsCritical property retrieves a Boolean value that indicates whether the template extension is marked critical.

Syntax

Template.IsCritical As Boolean

Property value

If true, the template extension is marked critical.
