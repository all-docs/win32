OIDs.Clear method

[The Clear method is available for use in the operating systems specified in the Requirements section. Instead, use the OidCollection Class in the System.Security.Cryptography namespace.]

The Clear method clears all OID objects from the collection.

Syntax

OIDs.Clear()

Parameters

This method has no parameters.

Return value

This method does not return a value.
