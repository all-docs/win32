Qualifier.NoticeNumbers property

[The NoticeNumbers property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Policies to process qualifiers that are part of the policy information in the Certificate Policies extension.]

The NoticeNumbers property retrieves the collection of user notice numbers. This property may be empty.

Syntax

Qualifier.NoticeNumbers As NoticeNumbers

Property value

The collection of user notice numbers.
