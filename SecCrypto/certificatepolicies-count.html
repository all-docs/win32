The Count property retrieves the number of PolicyInformation objects in the collection.

This property is read-only.

Syntax

CertificatePolicies.Count As Long

Property value

The number of PolicyInformation objects in the collection. Each PolicyInformation object represents a single certificate policy in the collection.

Remarks

The Count property can be used to specify the last PolicyInformation object in the collection when retrieving a specific PolicyInformation object using the CertificatePolicies.Item property.
