PolicyInformation.Qualifiers property

[The Qualifiers property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Policies to process policy information in the Certificate policies extension.]

The Qualifiers property retrieves a collection of the policy's qualifiers.

Syntax

PolicyInformation.Qualifiers As Qualifiers

Property value

The policy's Certification Practice Statement (CPS) pointer or user notice qualifiers, as a Qualifiers object.
