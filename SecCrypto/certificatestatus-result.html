The Result property retrieves a value that indicates whether the certificate is valid. This is the default property.

Syntax

CertificateStatus.Result As Boolean

Property value

If true, the certificate is valid. The certificate's validity is checked using the current setting of the CheckFlag property and the various policy settings.
