CAPICOM_EKU enumeration

The CAPICOM_EKU enumeration type defines the extended key usage names based on where they can be used.

Members

Member Description Value
CAPICOM_EKU_OTHER Certificate has uses defined in local policy. This is used if the EKU needed is not predefined and the OID value must be set by the application. 0
CAPICOM_EKU_SERVER_AUTH Certificate can be used to authenticate a server. 1
CAPICOM_EKU_CLIENT_AUTH Certificate can be used to authenticate a client. 2
CAPICOM_EKU_CODE_SIGNING Certificate can be used to create a digital signature. 3
CAPICOM_EKU_EMAIL_PROTECTION Certificate can be used for email protection. 4
CAPICOM_EKU_SMARTCARD_LOGON Certificate can be used for smart card logon. Introduced in CAPICOM 2.0. 5
CAPICOM_EKU_ENCRYPTING_FILE_SYSTEM Certificate can be used for EFS. Introduced in CAPICOM 2.0. 6

Remarks

The CAPICOM_EKU enumeration type is used by the EKU.Name property.
