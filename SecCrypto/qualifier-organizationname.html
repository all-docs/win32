Qualifier.OrganizationName property

[The OrganizationName property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Policies to process qualifiers that are part of the policy information in the Certificate Policies extension.]

The OrganizationName property retrieves the name of the organization associated with the qualifier. This property may be empty.

Syntax

Qualifier.OrganizationName As String

Property value

The name of the organization associated with the qualifier.
