CAPICOM_SIGNED_DATA_VERIFY_FLAG enumeration

The CAPICOM_SIGNED_DATA_VERIFY_FLAG enumeration indicates what is checked when a digital signature is verified.

Members

Member Description Value
CAPICOM_VERIFY_SIGNATURE_ONLY Only the signature is checked. 0
CAPICOM_VERIFY_SIGNATURE_AND_CERTIFICATE Both the signature and the validity of the certificate used to create the signature are checked. 1
