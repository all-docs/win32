PublicKey.Algorithm property

[The Algorithm property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Certificate2.PublicKey Property in the System.Security.Cryptography.X509Certificates namespace.]

The Algorithm property retrieves the OID object that identifies the algorithm used by the public key. This is the default property.

Syntax

PublicKey.Algorithm As OID

Property value

The OID object that identifies the algorithm used by the public key.
