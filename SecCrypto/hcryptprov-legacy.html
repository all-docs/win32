HCRYPTPROV_LEGACY

The HCRYPTPROV_LEGACY data type is used to replace the HCRYPTPROV data type where the HCRYPTPROV data type is no longer used. Function parameters and data structures that are of type HCRYPTPROV_LEGACY should be set to NULL.

typedef ULONG_PTR HCRYPTPROV_LEGACY;
