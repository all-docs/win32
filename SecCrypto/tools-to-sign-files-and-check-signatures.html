Tools to Sign Files and Check Signatures

CryptoAPI Tools are the following tools that you can use to sign files digitally, to confirm signatures on files, and to create catalog files.

Tool Remarks
MakeCat Makes an unsigned catalog file. This catalog file is subsequently used to compare the hashes within the catalog file to the hashes of individual files, which helps prove that the original files are free of tampering.
SetReg Sets registry keys that control the certificate verification process.
SignTool Digitally signs files and confirms digital signatures of files.
