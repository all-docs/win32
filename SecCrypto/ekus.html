EKUs object

[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows ServerÂ 2008, WindowsÂ Vista, and WindowsÂ XP. Instead, use the X509ExtensionCollection Class in the System.Security.Cryptography.X509Certificates namespace.]

The EKUs object represents a collection of EKU objects. Each EKU object represents a single extended key usage (EKU) property of a certificate.

When to use

The EKUs collection is used to perform the following tasks:

  • Retrieve the number of EKU properties in the collection.
  • Retrieve a specific EKU object from the collection.
  • Iterate through the collection.
