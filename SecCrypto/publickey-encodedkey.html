PublicKey.EncodedKey property

[The EncodedKey property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Certificate2.PublicKey Property in the System.Security.Cryptography.X509Certificates namespace.]

The EncodedKey property retrieves the value of the public key.

Syntax

PublicKey.EncodedKey As EncodedData

Property value

An EncodedData object that provides access to the value of the public key.
