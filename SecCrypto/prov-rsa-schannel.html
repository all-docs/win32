PROV_RSA_SCHANNEL

The PROV_RSA_SCHANNEL provider type supports both RSA and Schannel protocols.

Algorithms Supported

For descriptions of each of these algorithms, see the glossary.

Purpose Supported algorithms
Key Exchange RSA
Signature RSA
Encryption Any PROV_RSA_SCHANNEL CSP must implement one or more of the following algorithms: RC4 DES Triple DES
Hashing MD5SHA
