CAPICOM_KEY_STORAGE_FLAG enumeration

The CAPICOM_KEY_STORAGE_FLAG enumeration defines key storage flags.

Members

Member Description Value
CAPICOM_KEY_STORAGE_DEFAULT Default key storage. 0
CAPICOM_KEY_STORAGE_EXPORTABLE The key is exportable. 1
CAPICOM_KEY_STORAGE_USER_PROTECTED The key is user protected. 2

Remarks

This enumeration is used by the following method:

  • Certificate.Load
  • Store.Load
