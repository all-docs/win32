The CertificatePolicies object represents a collection of PolicyInformation objects. Each PolicyInformation object represents a single certificate policy.

When to use

The CertificatePolicies object is used to perform the following tasks:

  • Retrieve the number of certificate policies in the collection.
  • Retrieve a specific PolicyInformation object from the collection.
  • Iterate through the collection.
