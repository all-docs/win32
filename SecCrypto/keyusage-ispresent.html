KeyUsage.IsPresent property

[The IsPresent property is available for use in the operating systems specified in the Requirements section. Instead, use the X509EnhancedKeyUsageExtension Class in the System.Security.Cryptography.X509Certificates namespace.]

The IsPresent property retrieves a Boolean value that indicates whether the KeyUsage extension is present.

This property is the default property of the KeyUsage object.

Syntax

KeyUsage.IsPresent As Boolean

Property value

If true, the KeyUsage extension is present.
