PrivateKey.IsRemovable method

[The IsRemovable method is available for use in the operating systems specified in the Requirements section. Instead, use the X509Certificate2.PrivateKey Property in the System.Security.Cryptography.X509Certificates namespace.]

The IsRemovable method returns a Boolean value that indicates whether the private key is in removable storage.

Syntax

PrivateKey.IsRemovable()

Parameters

This method has no parameters.

Return value

If true, the private key is in removable storage.

Remarks

The return value of this method is dependent on the cryptographic service provider (CSP) used. This method will return a trustworthy value if a Microsoft CSP is used. If the CSP is not a Microsoft CSP, the return value cannot be trusted to be accurate.
