KeyUsage.IsDigitalSignatureEnabled property

[The IsDigitalSignatureEnabled property is available for use in the operating systems specified in the Requirements section. Instead, use the X509EnhancedKeyUsageExtension Class in the System.Security.Cryptography.X509Certificates namespace.]

The IsDigitalSignatureEnabled property retrieves a Boolean value that indicates whether the digitalSignature bit is set.

Syntax

KeyUsage.IsDigitalSignatureEnabled As Boolean

Property value

If true, the digitalSignature bit is set.
