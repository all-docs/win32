KeyUsage.IsNonRepudiationEnabled property

[The IsNonRepudiationEnabled property is available for use in the operating systems specified in the Requirements section. Instead, use the X509EnhancedKeyUsageExtension Class in the System.Security.Cryptography.X509Certificates namespace.]

The IsNonRepudiationEnabled property retrieves a Boolean value that indicates whether the nonRepudiationEnabled bit is set.

Syntax

KeyUsage.IsNonRepudiationEnabled As Boolean

Property value

If true, the nonRepudiationEnabled bit is set.
