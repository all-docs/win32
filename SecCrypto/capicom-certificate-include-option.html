CAPICOM_CERTIFICATE_INCLUDE_OPTION enumeration

The CAPICOM_CERTIFICATE_INCLUDE_OPTION enumeration type defines which certificates in a chain are saved. This enumeration type was introduced in CAPICOM 2.0.

Members

Member Description Value
CAPICOM_CERTIFICATE_INCLUDE_CHAIN_EXCEPT_ROOT Saves all certificates in the chain with the exception of the root entity. 0
CAPICOM_CERTIFICATE_INCLUDE_WHOLE_CHAIN Saves the complete certificate chain. 1
CAPICOM_CERTIFICATE_INCLUDE_END_ENTITY_ONLY Saves only the end entity certificate. 2

Remarks

The CAPICOM_CERTIFICATE_INCLUDE_OPTION enumeration type is used by the Certificate.Save method.
