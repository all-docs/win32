Properties of IOCSPCAConfigurationCollection

The following properties are defined by the IOCSPCAConfigurationCollection interface.

Property Description
_NewEnum Gets an enumerator for the configuration set.
Count Gets the number of certification authority (CA) configurations in the configuration set.
Item Gets a CA configuration identified by index in the configuration set.
ItemByName Gets a CA configuration identified by name in the configuration set.
