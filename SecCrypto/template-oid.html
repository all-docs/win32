Template.OID property

[The OID property is available for use in the operating systems specified in the Requirements section. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace by calling the constructor that takes an OID as a parameter, and then use the OID for Certificate Template to retrieve the certificate extension template.]

The OID property retrieves an OID object that identifies the Template object.

This property is read-only.

Syntax

Template.OID As OID

Property value

An OID object that identifies the Template object.
