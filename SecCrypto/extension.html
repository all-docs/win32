Extension object

[CAPICOM is a 32-bit only component that is available for use in the following operating systems: Windows ServerÂ 2008, WindowsÂ Vista, and WindowsÂ XP. Instead, use the X509Extension Class in the System.Security.Cryptography.X509Certificates namespace.]

The Extension object represents a single certificate extension.

When to use

The Extension object is used to perform the following tasks:

  • Retrieve the OID of the certificate extension.
  • Retrieve the certificate extension data represented by an EncodedData object.
  • Determine whether the certificate extension is marked as critical.

Members

The Extension object has these types of members:

  • Properties

Properties

The Extension object has these properties.

Property Access type Description
EncodedData Read-only Retrieves the encoded data for the extension.
IsCritical Read-only Retrieves a Boolean value that indicates whether the extension is marked critical.
OID Read-only Retrieves the object identifier for the extension. This is the default property.

Remarks

The Extension object cannot be created.

The Extension object is used by the Extensions collection object.
