Properties of ICertSrvSetup

The following properties are defined by the ICertSrvSetup interface.

Property Description
CAErrorId Gets the ID for additional error information related to a failed certification authority (CA) specification.
CAErrorString Gets the string data for additional error information related to a failed CA specification.
