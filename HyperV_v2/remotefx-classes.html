Remote Desktop classes

The Remote Desktop virtualization WMI classes are used to manage Remote Desktop and RemoteFX resources and virtual machines.

In this section

Topic Description
Msvm_InteractiveSessionACE Represents an access control entry (ACE) that determines access to the interactive session of a virtual machine.
Msvm_Physical3dGraphicsProcessor Describes the physical 3-D graphics processing unit (GPU).
Msvm_PhysicalGPUInfo Contains information about a RemoteFX physical graphics processing unit (GPU).
Msvm_RdvComponent Represents the state of the RDV component, which is responsible for providing a transport for the parent to the guest for configuration purposes.
Msvm_RdvComponentSettingData Represents the configured state of the Remote Desktop Virtualization (RDV) component. The default state is Enabled.
Msvm_Synth3dVideoPool Contains information about the synthetic 3-D video graphics processing units (GPUs) available on the host system.
Msvm_Synthetic3DDisplayController Represents the synthetic 3-D display controller that is assigned to a virtual machine.
Msvm_Synthetic3DDisplayControllerSettingData Represents settings for a synthetic 3-D display controller for a virtual machine.
Msvm_Synthetic3DService Describes the synthetic 3-D GPU service.
Msvm_Synthetic3DServiceSettingData Represents the settings for the synthetic 3-D service present on a single host system.
Msvm_TerminalService Manages all remote terminal connections to a particular host.
Msvm_TerminalServiceSettingData Represents the settings for the virtual computer terminal services on a host.
