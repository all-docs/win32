Hyper-V application health monitoring interfaces

The Hyper-V application health monitoring API defines the following COM interfaces.

In this section

Topic Description
IVmApplicationHealthMonitor Reports the health status of an application that is running in a virtual machine to the Hyper-V integration components running in the same virtual machine.
