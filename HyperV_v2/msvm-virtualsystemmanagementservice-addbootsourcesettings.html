AddBootSourceSettings method of the Msvm_VirtualSystemManagementService class

Adds boot sources to a virtual system configuration when applied to a "state" virtual system configuration.

Syntax

uint32 AddBootSourceSettings(
  [in]  CIM_VirtualSystemSettingData REF AffectedConfiguration,
  [in]  string                           BootSourceSettings[],
  [out] CIM_SettingData              REF ResultingBootSourceSettings[],
  [out] CIM_ConcreteJob              REF Job
);

Parameters

AffectedConfiguration [in]

A CIM_VirtualSystemSettingData containing the affected configuration.

BootSourceSettings [in]

An array containing the boot source settings.

ResultingBootSourceSettings [out]

On success, returns a CIM_SettingData with the resulting boot source settings.

Job [out]

If the operation is performed asynchronously, this method will return 4096, and this parameter will contain a reference to an object derived from CIM_ConcreteJob.

Return value

On success, returns 0 or 4096; otherwise, returns an error.
