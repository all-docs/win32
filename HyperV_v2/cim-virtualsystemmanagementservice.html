CIM_VirtualSystemManagementService class

Represents a service that manages virtual systems.

Syntax

[Abstract, Version("2.22.0"), UMLPackagePath("CIM::Core::Virtualization"), AMENDMENT]
class CIM_VirtualSystemManagementService : CIM_Service
{
};

Members

The CIM_VirtualSystemManagementService class has these types of members:

  • Methods

Methods

The CIM_VirtualSystemManagementService class has these methods.

Method Description
AddResourceSettings Adds resources to a virtual system configuration.
DefineSystem Defines a virtual system.
DestroySystem Deletes a virtual system.
ModifyResourceSettings Modifies the virtual resource settings for a virtual system configuration.
ModifySystemSettings Modifies virtual system settings.
RemoveResourceSettings Removes virtual resource settings from a virtual system configuration.
