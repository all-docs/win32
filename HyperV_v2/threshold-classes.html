Threshold classes

Contains the classes introduced in Windows 10.

In this section

Topic Description
Msvm_CollectedCollections Associates the Msvm_VirtualSystemCollection to the contained Msvm_ComputerSystem objects.
Msvm_CollectedReferencePoints Associates the Msvm_ReferencePointCollection to the contained Msvm_VirtualSystemReferencePoint objects.
Msvm_CollectedSnapshots Associates the Msvm_SnapshotCollection to the contained Msvm_VirtualSystemSettingData objects.
Msvm_CollectedVirtualSystems Associates the Msvm_VirtualSystemCollection to the contained Msvm_ComputerSystem objects.
Msvm_CollectionManagementService Manages the collections on the Hyper-V host.
Msvm_CollectionReferencePointExportSettingData Export setting data to be passed in to the ExportReferencePoint method of the Msvm_CollectionReferencePointService class.
Msvm_CollectionReferencePointService Service to create, destroy and export reference points
Msvm_CollectionReferencePointSettingData Provides additional information to be used with the CreateReferencePoint method of the Msvm_CollectionReferencePointService class.
Msvm_CollectionSettingData The class that represents configured settings for an Msvm_VirtualSystemCollection.
Msvm_CollectionSnapshotExportSettingData Export setting data to be passed in to the ExportSnapshot method of Msvm_CollectionSnapshotService class.
Msvm_CollectionSnapshotService Service to create, destroy and export snapshot collection of virtual systems.

Msvm_ElementConformsToProfile Defines the registered profiles to which the referenced system conforms.
Msvm_EthernetSwitchPortExtendedAclSettingData Represents the extended port ACL settings.
Msvm_EthernetSwitchPortIsolationSettingData Represents the Isolation setting data.
Msvm_EthernetSwitchPortRoutingDomainSettingData Represents the routing domain setting data.
Msvm_EthernetSwitchPortVfpSettingData Use to set VFP as mandatory on a port.
Msvm_GuestClusterInformation Used in the QueryGuestClusterInformation method in the Msvm_VssService class to retrieve information about the guest cluster the VM is part of.
Msvm_GuestCommunicationService Represents the guest communication service. It is used for communication with the guest, from the Hyper-V host.
Msvm_GuestCommunicationServiceSettingData Represents the settings of the guest communication service.
Msvm_GuestServiceInterfaceSettingDataComponent Association class between a guest service interface component and the guest service resource.
Msvm_ManagementCollection Represents a collection of other collections.
Msvm_MoveUnmanagedVhd Moves a virtual hard drive from the source to the destination path. The following syntax is simplified from MOF code and includes all inherited properties.
Msvm_ReferencePointCollection Represents a collection of virtual system reference points.
Msvm_ReferencePointOfVirtualSystem Associates the Msvm_VirtualSystemReferencePoint to the corresponding Msvm_VirtualSystem objects.
Msvm_ReferencePointOfVirtualSystemCollection Associates the Msvm_ReferencePointCollection to the corresponding Msvm_VirtualSystemCollection objects.
Msvm_ResourceDependentOnResource Establishes that a CIM_ResourceAllocationSettingData instance representing a resource allocation depends on another resource allocation.
Msvm_SerialPortSettingData Describes the setting data for a virtual serial port. The following syntax is simplified from MOF code and includes all inherited properties.
Msvm_ServiceOfVssComponent An association between an instance of Msvm_VssComponent and an instance of Msvm_VssService that represents a service for performing operations on the VSS component.
Msvm_SnapshotCollection Represents a collection of virtual system snapshots.
Msvm_SnapshotOfVirtualSystemCollection Associates the Msvm_SnapshotCollection to the corresponding Msvm_VirtualSystemCollection objects.
Msvm_StandaloneV2ElementConformsToProfile Defines the registered profiles to which the referenced standalone system conforms. The following syntax is simplified from MOF code and includes all inherited properties.
Msvm_SyntheticDisplayControllerSettingData Describes the setting data for a virtual synthetic display controller. The following syntax is simplified from MOF code and includes all inherited properties.
Msvm_SyntheticKeyboard Represents a synthetic keyboard device.
Msvm_TPM Represents the state of the TPM device.
Msvm_TPMSettingData Represents the configured state of the TPM device.
Msvm_VHDSetInformation Provides information about a VHD Set file.
Msvm_VHDSnapshotInformation Provides information about a snapshot within a VHD Set file
Msvm_VirtualEthernetSwitchNicTeamingMember Represents the association between a team ExternalEthernetPort and a member ExternalEthernetPort.
Msvm_VirtualEthernetSwitchNicTeamingSettingData Represents the switch nic teaming settings.
Msvm_VirtualMachineToDisks Setting data to be passed as an array to the Msvm_CollectionReferencePointExportSettingData class.
Msvm_VirtualSystemCollection Represents a collection of virtual systems.
Msvm_VirtualSystemReferencePoint Represents a reference point for a virtual system.
Msvm_VirtualSystemReferencePointExportSettingData Provides additional information to be used with the ExportReferencePoint method of the Msvm_VirtualSystemReferencePointService class.
Msvm_VirtualSystemReferencePointService Describes a virtual system reference point service. The following syntax is simplified from MOF code and includes all inherited properties.
Msvm_VirtualSystemReferencePointSettingData Provides additional information to be used with the CreateReferencePoint method of the Msvm_VirtualSystemReferencePointService class.
Msvm_VirtualSystemSnapshotSettingData Provides additional information to be used with the CreateSnapshot method of the Msvm_VirtualSystemSnapshotService class.
Msvm_VssService Represents the guest VSS service. It is used for querying guest cluster information from the Hyper-V host.
