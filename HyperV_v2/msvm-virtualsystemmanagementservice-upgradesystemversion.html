UpgradeSystemVersion method of the Msvm_VirtualSystemManagementService class

Upgrades the virtual system.

When applied to the system settings of a "current" virtual system configuration

Syntax

uint32 UpgradeSystemVersion(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 UpgradeSettingData,
  [out] CIM_ConcreteJob    REF Job
);

Parameters

ComputerSystem [in]

A reference to a CIM_ComputerSystem that represents the virtual computer system to be upgraded.

UpgradeSettingData [in]

The upgrade setting data.

Job [out]

If the operation is performed asynchronously, this method will return 4096, and this parameter will contain a reference to an object derived from CIM_ConcreteJob.

Return value

On success, returns a 0 or 4096; otherwise, returns an error.
