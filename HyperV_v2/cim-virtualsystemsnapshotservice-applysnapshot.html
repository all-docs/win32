ApplySnapshot method of the CIM_VirtualSystemSnapshotService class

Applies a virtual system snapshot to the virtual system that it was created from.

Syntax

uint32 ApplySnapshot(
  [in]  CIM_VirtualSystemSettingData REF Snapshot,
  [out] CIM_ConcreteJob              REF Job
);

Parameters

Snapshot [in]

A CIM_VirtualSystemSettingData reference to the virtual system snapshot.

Job [out]

If the operation is long running, then optionally a CIM_ConcreteJob representing the job may be returned.

[!Note]
This parameter was read/write in Windows 8.1.

Return value

Returns a 0 on success; otherwise, returns an error.
