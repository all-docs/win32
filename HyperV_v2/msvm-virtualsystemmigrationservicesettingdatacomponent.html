Msvm_VirtualSystemMigrationServiceSettingDataComponent class

An association used to represent virtual system migration network settings of the virtual system migration service.

The following syntax is simplified Managed Object Format (MOF) code, and it includes all of the inherited properties.

Syntax

[Association, Aggregation, Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_VirtualSystemMigrationServiceSettingDataComponent : CIM_Component
{
  Msvm_VirtualSystemMigrationServiceSettingData REF GroupComponent;
  Msvm_VirtualSystemMigrationNetworkSettingData REF PartComponent;
};

Properties

The Msvm_VirtualSystemMigrationServiceSettingDataComponent class has these properties.

GroupComponent

Data type: Msvm_VirtualSystemMigrationServiceSettingData

Access type: Read-only

Qualifiers: Key, Override ("GroupComponent"), Aggregate

A reference to an instance of the Msvm_VirtualSystemMigrationServiceSettingData class that represents the migration service settings.

PartComponent

Data type: Msvm_VirtualSystemMigrationNetworkSettingData

Access type: Read-only

Qualifiers: Key, Override ("PartComponent")

A reference to an instance of the Msvm_VirtualSystemMigrationNetworkSettingData class that represents the migration network settings.
