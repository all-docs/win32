CIM_View class

The CIM_View class is an superclass for classes that provide de-normalized, aggregate representations of managed resources.

Syntax

[Abstract, Version("2.26.0"), UMLPackagePath("CIM::Core::CoreElements"), AMENDMENT]
class CIM_View : CIM_ManagedElement
{
};

Members

The CIM_View class does not define any members.
