CIM_ResourcePoolConfigurationService class

Manages the configuration of resource pools by using jobs.

Syntax

[Abstract, Version("2.22.0"), UMLPackagePath("CIM::Core::Resource")]
class CIM_ResourcePoolConfigurationService : CIM_Service
{
};

Members

The CIM_ResourcePoolConfigurationService class has these types of members:

  • Methods

Methods

The CIM_ResourcePoolConfigurationService class has these methods.

Method Description
AddResourcesToResourcePool Starts a job to add resources to a resource pool.
ChangeParentResourcePool Start a job to change the parent resource pool of a resource pool.
CreateChildResourcePool Start a job to create a resource pool from a parent resource pool.
CreateResourcePool Starts a job to create a root resource pool.
DeleteResourcePool Start a job to delete a resource pool.
RemoveResourcesFromResourcePool Starts a job to remove resources from a resource pool.
