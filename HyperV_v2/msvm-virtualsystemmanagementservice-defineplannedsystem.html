DefinePlannedSystem method of the Msvm_VirtualSystemManagementService class

Defines a planned virtual system.

Input that is not completely specified may be filled out with default values.

Syntax

uint32 DefinePlannedSystem(
  [in]  string                           SystemSettings,
  [in]  string                           ResourceSettings[],
  [in]  CIM_VirtualSystemSettingData REF ReferenceConfiguration,
  [out] CIM_ComputerSystem           REF ResultingSystem,
  [out] CIM_ConcreteJob              REF Job
);

Parameters

SystemSettings [in]

The system settings for the virtual system.

ResourceSettings [in]

The resource settings for the virtual system.

ReferenceConfiguration [in]

A CIM_VirtualSystemSettingData containing the reference configuration.

ResultingSystem [out]

A CIM_ComputerSystem containing the resulting system.

Job [out]

If the operation is performed asynchronously, this method will return 4096, and this parameter will contain a reference to an object derived from CIM_ConcreteJob.

Return value

On success, returns 0 or 4096; otherwise, returns an error.
