Hyper-V VM placement API

The Hyper-V virtual machine (VM) placement API contains the compatibility info for the VMs or hosting computer system.

In this section

Topic Description
Msvm_CompatibilityVector References the compatibility info for a virtual machine (VM) (when run on a VM computer system) or a host (when run on a host computer system).
