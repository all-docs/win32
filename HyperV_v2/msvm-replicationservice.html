Msvm_ReplicationService class

Manages the replication for a virtual machine.

The following syntax is simplified Managed Object Format (MOF) code, and it includes all of the inherited properties.

Syntax

[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_ReplicationService : CIM_Service
{
  string   InstanceID;
  string   Caption = "Hyper-V Replica Service";
  string   Description = "Replication Service";
  string   ElementName;
  datetime InstallDate;
  string   Name = "replicasvc";
  uint16   OperationalStatus[];
  string   StatusDescriptions[];
  string   Status = "OK";
  uint16   HealthState = 5;
  uint16   CommunicationStatus;
  uint16   DetailedStatus;
  uint16   OperatingStatus;
  uint16   PrimaryStatus;
  uint16   EnabledState = 2;
  string   OtherEnabledState;
  uint16   RequestedState = 12;
  uint16   EnabledDefault = 2;
  datetime TimeOfLastStateChange;
  uint16   AvailableRequestedStates[];
  uint16   TransitioningToState;
  string   SystemCreationClassName = "Msvm_ComputerSystem";
  string   SystemName;
  string   CreationClassName = "Msvm_ReplicationService";
  string   PrimaryOwnerName;
  string   PrimaryOwnerContact;
  string   StartMode;
  boolean  Started = True;
};

Methods

The Msvm_ReplicationService class has these methods.

Method Description
AddAuthorizationEntry Adds an authorization entry to a server.
ChangeReplicationModeToPrimary Changes the extended replication relationship to the primary relationship for a replica virtual machine. The replica virtual machine must be in a failover committed state. Windows 8.1: This method is not supported until Windows 8.1 and Windows Server 2012 R2.
CommitFailover Commits the recovery snapshot that the InitiateFailover method has used for a failover.
CreateReplicationRelationship Creates a new replication relationship for a virtual machine.
GetReplicationStatistics Retrieves replication statistics for a virtual machine.
GetReplicationStatisticsEx Retrieves the replication statistics that are associated with specified replication relationship of the virtual machine. Windows 8.1: This method is not supported until Windows 8.1 and Windows Server 2012 R2.
GetSystemCertificates Retrieves the system certificates on a host system.
ImportInitialReplica Imports the initial replication for a virtual machine.
InitiateFailback Initiates the failback for a recovery virtual machine. That is, sets the failover for the virtual machine to an app or crash consistent image. Windows 8.1: This method is not supported until Windows 8.1 and Windows Server 2012 R2.
InitiateFailover Initiates a failover for a virtual machine to an application or standard replication point image.
ModifyAuthorizationEntry Modifies an authorization entry on a server.
ModifyReplicationSettings Modifies the replication settings for a virtual machine.
ModifyServiceSettings Modifies the settings for the Hyper-V Replica service.
RemoveAuthorizationEntry Removes the authorization entry from a server.
RemoveReplicationRelationship Removes a virtual machine replication relationship.
RemoveReplicationRelationshipEx Removes the specified virtual machine replication relationship. For a replica virtual machine, primary replication can't be removed if extended replication is enabled. Windows 8.1: This method is not supported until Windows 8.1 and Windows Server 2012 R2.
RequestStateChange Requests a state change.
ResetReplicationStatistics Resets the replication statistics for a virtual machine.
ResetReplicationStatisticsEx Resets replication statistics that are associated with the specified replication relationship of a virtual machine. Windows 8.1: This method is not supported until Windows 8.1 and Windows Server 2012 R2.
Resynchronize Performs a resynchronization operation on the specified virtual machine.
ReverseReplicationRelationship Replicates a failed-over virtual machine back to the primary server.
RevertFailover Reverts the current failover for a virtual machine by discarding the current failover disk.
SetAuthorizationEntry Sets the replication authorization entry for a virtual machine.
SetFailoverNetworkAdapterSettings Configures the network adapter's IP settings to be applied to a virtual machine after a failover.
StartReplication Starts the replication of a virtual machine.
StartService Starts the service.
StopService Stops the service.
TestReplicaSystem Creates a new replica of a virtual machine with the specified snapshot for testing purposes.
TestReplicationConnection Verifies if the replication can be enabled from the current host system to the specified recovery system.

Data type: uint16

The last requested or desired state for the element. The actual state of the element is represented by EnabledState. If this occurs, the value 12 ("Not Applicable") is used. This property is inherited from CIM_EnabledLogicalElement, and it is always set to 12 (Not Applicable).
