CIM_ProcessIndication class

CIM_ProcessIndication is an abstract superclass for specialized indication classes, that address specific changes and alerts published by providers and instrumentation.

Syntax

[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};

Members

The CIM_ProcessIndication class does not define any members.
