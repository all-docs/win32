Redstone Classes

Contains the new classes for Windows 10, version 1703.

In this section

Topic Description
Msvm_AssignableDeviceDismountSettingData Represents the settings of a virtual system to import. Used by the Dismount method of the Msvm_AssignableDeviceService class.
Msvm_AssignableDeviceService Manages the assignable devices on a host computer system.
Msvm_CollectionReferencePointExportJob This class represents a collection reference point export operation job.
Msvm_EthernetSwitchHardwareOffloadSettingData Represents the switch offload settings.
Msvm_EthernetSwitchPortMigrationQosSettingData Represents the VFP QOS settings.
Msvm_EthernetSwitchPortRdmaSettingData Represents the port RDMA feature setting data.
Msvm_EthernetSwitchPortTeamMappingSettingData Represents the port team mapping feature setting data.
Msvm_GpuPartition Represents the state of the GPU partition.
Msvm_GpuPartitionSettingData Represents the configured state of a GPU partition device.
Msvm_NetworkConnectionDiagnosticInformation Provides information about the network connectivity for a virtual machine.
Msvm_NetworkConnectionDiagnosticSettingData Represents the settings used to test the network connectivity of a virtual machine.
Msvm_PartitionableGpu Represents a partitionable GPU. Each GPU can be sliced into a number of GPU partitions, which can be assigned to a virtual machine as a vGPU.
Msvm_PciExpress Represents the state of the PCI Express port.
Msvm_PciExpressSettingData Represents the configured state of a PCI Express port.
Msvm_SecurityElement Represents the runtime security settings of a CIM_ComputerSystem.
Msvm_SecurityService Represents the security service. It is used for configuring virtual system security settings.
Msvm_SecuritySettingData Represents the configured state of the security settings for
Msvm_StorageSettingData Represents the storage-specific settings for a virtual system.
Msvm_SummaryInformationBase Used in the GetSummaryInformation method in the Msvm_VirtualSystemManagementService class to quickly retrieve common information related to a virtual system or snapshot.
Msvm_SystemComponentSettingData A generic base class for setting data classes representing components of a virtual system.
Msvm_VirtualSystemReferencePointExportJob This class represents a virtual system reference point export operation job.
