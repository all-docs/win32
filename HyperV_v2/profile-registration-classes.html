Profile registration classes

The following are virtualization WMI classes related to profile registration.

In this section

Topic Description
Msvm_ElementConformsToProfile Defines the registered profiles to which the referenced system conforms.
Msvm_HostedResourcePool Represents a specialization of the system component association that establishes that the resource pool is defined in the context of the system.
Msvm_ReferencedProfile Describes a profile that is referenced by another registered profile.
Msvm_RegisteredProfile Describes a set of classes with required properties and methods, necessary to manage a real-world entity or to support a usage scenario, in an interoperable fashion.
Msvm_ResourcePoolComponent Represents a resource pool element of the Microsoft Windows Hyper-V platform.
Msvm_ResourcePoolRegistration Registers a service that provides global resource pool-related objects.
Msvm_ResourceTypeDefinition Defines a mapping of a resource type to its implementation classes.
Msvm_VirtualizationComponent Represents a service of the Microsoft Windows Hyper-V platform.
Msvm_VirtualizationComponentRegistration Represents the registration of a service in the Microsoft Hyper-V platform.
Msvm_VirtualSystemResourceComponent Represents a virtual device service of the Microsoft Windows Hyper-V platform.
Msvm_VirtualSystemResourceRegistration Registers a service that provides virtual machine-specific resource-related objects.
