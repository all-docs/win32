FindMountedStorageImageInstance method of the Msvm_ImageManagementService class

Finds an Msvm_MountedStorageImage object for a given disk image path.

Syntax

uint32 FindMountedStorageImageInstance(
  [in]  string                       SelectionCriterion,
  [in]  uint16                       CriterionType,
  [out] Msvm_MountedStorageImage REF Image
);

Parameters

SelectionCriterion [in]

A fully-qualified path that specifies the location of a disk image file.

CriterionType [in]

The type of criterion to look for when finding the disk image.

unknown (0)

Path (2)

Image [out]

A reference to the Msvm_MountedStorageImage (can be null if the image is not mounted).

Return value

This method returns one of the following values:

Completed with No Error (0)

Failed (32768)

Access Denied (32769)

Not Supported (32770)

Status is unknown (32771)

Timeout (32772)

Invalid parameter (32773)

System is in use (32774)

Invalid state for this operation (32775)

Incorrect data type (32776)

System is not available (32777)

Out of memory (32778)

File not found (32779)

Object not found (32789)
