Memory classes

The memory classes model the memory for a virtual machine.

The following are virtualization WMI classes related to memory.

In this section

Topic Description
Msvm_ElementAllocatedFromNumaNode Associates an instance of an allocated resource with the physical NUMA node from which it was allocated.
Msvm_Memory Represents the memory currently allocated to a virtual machine.
Msvm_MemorySettingData Represents the configured state of the memory for a virtual machine.
Msvm_NumaNode Represents a Non-Uniform Memory Access (NUMA) node of a virtual system.
