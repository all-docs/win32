ModifyFeatureSettings method of the Msvm_VirtualEthernetSwitchManagementService class

Modifies the feature settings of an Ethernet switch port.

Syntax

uint32 ModifyFeatureSettings(
  [in]  string                      FeatureSettings[],
  [out] Msvm_FeatureSettingData REF ResultingFeatureSettings[],
  [out] CIM_ConcreteJob         REF Job
);

Parameters

FeatureSettings [in]

An array of strings that contain an embedded instance of a class derived from the Msvm_FeatureSettingData class, that describes the switch port feature settings to be modified.

ResultingFeatureSettings [out]

An array of references to instances of the Msvm_FeatureSettingData class that represent the modified feature settings.

Job [out]

If the operation is performed asynchronously, this method will return 4096, and this parameter will contain a reference to an object derived from CIM_ConcreteJob.

Return value

This method returns one of the following values.

Completed with No Error (0)

Not Supported (1)

Failed (2)

Timeout (3)

Invalid Parameter (4)

Invalid State (5)

Incompatible Parameters (6)

DMTF Reserved (..)

Method Parameters Checked - Job Started (4096)

Method Reserved (4097..32767)

Vendor Specific (32768..65535)
