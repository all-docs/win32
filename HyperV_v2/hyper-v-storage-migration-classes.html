Hyper-V migration API reference

The Hyper-V migration API defines the following programming elements.

In this section

Topic Description
Msvm_MigrationJob This class represents a migration operation job created for storage or virtual system migration by the virtual system migration service.
Msvm_VirtualSystemMigrationCapabilities Defines the means by which a client can discover the methods provided by the migration service, and valid range of virtual system migration setting data.
Msvm_VirtualSystemMigrationNetworkSettingData Represents the network on which the virtual system migration service is listening for incoming virtual system migration.
Msvm_VirtualSystemMigrationService Represents the virtual system migration service. It is used for migrating a virtual system or for migrating the storage of a virtual system from one virtualization platform to another.
Msvm_VirtualSystemMigrationServiceSettingData Represents the settings for the virtual system migration service on a host.
Msvm_VirtualSystemMigrationServiceSettingDataComponent An association used to represent virtual system migration network settings of the virtual system migration service.
Msvm_VirtualSystemMigrationSettingData Represents the migration settings for migrating a virtual system and the storage attached to a virtual system.
