RequestStateChange method of the Msvm_PlannedComputerSystem class

Requests that the state of the planned system be changed to the value specified.

Syntax

uint32 RequestStateChange(
  [in]  uint16              RequestedState,
  [out] CIM_ConcreteJob REF Job,
  [in]  datetime            TimeoutPeriod
);

Parameters

RequestedState [in]

The requested state for the planned system.

Enabled (2)

Disabled (3)

Shut Down (4)

Offline (6)

Test (7)

Defer (8)

Quiesce (9)

Reboot (10)

Reset (11)

DMTF Reserved (..)

Vendor Reserved (32768..65535)

Job [out]

This parameter is not used and should be Null.

TimeoutPeriod [in]

This parameter is not used.

Return value

This method returns one of the following values.

Return code/value Description
0 Success
4096
32768
32769 Access denied.
32770
32771
32772
32773
32774
32775 The value specified in the RequestedState parameter is not supported.
32776
32777
32778
