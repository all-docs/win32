ModifySystemComponentSettings method of the Msvm_VirtualSystemManagementService class

Modifies generic system component settings.

Syntax

uint32 ModifySystemComponentSettings(
  [in]  string                              ComponentSettings[],
  [out] Msvm_SystemComponentSettingData REF ResultingComponentSettings[],
  [out] CIM_ConcreteJob                 REF Job
);

Parameters

ComponentSettings [in]

An array of component settings to update.

ResultingComponentSettings [out]

On success, returns an array of Msvm_SystemComponentSettingData that reference the resulting component settings.

Job [out]

If the operation is performed asynchronously, this method will return 4096, and this parameter will contain a reference to an object derived from CIM_ConcreteJob.

Return value

On success, returns a 0 or 4096; otherwise, returns an error.
