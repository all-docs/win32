Hyper-V virtual Fibre Channel API reference

The Hyper-V virtual Fibre Channel API defines the following programming elements.

In this section

Topic Description
Msvm_ExternalFcPort Represents an external Fibre Channel port.
Msvm_FcActiveConnection Connects a switch port to the Fibre Channel endpoint to which the port is connected.
Msvm_FcDeviceSAPImplementation Represents an association between a service access point and the logical device that implements it.
Msvm_FcEndpoint Represents the logical connection point for a Fibre Channel port.
Msvm_FcPortAllocationSettingData Represents the configured state of a synthetic Fibre Channel port or a Fibre Channel switch port.
Msvm_FcSwitchPort Represents a port on the virtual Fibre Channel switch.
Msvm_SyntheticFcPort Represents a synthetic Fibre Channel port.
Msvm_SyntheticFcPortSettingData Represents the configured state of a synthetic Fibre Channel port.
Msvm_VirtualFcSwitch Represents a virtual Fibre Channel switch.
Msvm_VirtualFcSwitchSettingData Represents the configuration of a virtual Fibre Channel switch.
