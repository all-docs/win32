LockMedia method of the Msvm_DVDDrive class

Locks or releases the media.

Syntax

uint32 LockMedia(
  [in] boolean Lock
);

Parameters

Lock [in]

true to lock the media; false to release the media.

Return value

This method returns one of the following values:

Completed with No Error (0)

Not supported (1)
