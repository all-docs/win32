Msvm_EthernetPortFailoverSettingDataComponent class

An association used to establish relationships between one instance of an Msvm_EmulatedEthernetPortSettingData and one or more instances of an Msvm_EthernetSwitchFeatureSettingData.

The following syntax is simplified Managed Object Format (MOF) code, and it includes all of the inherited properties.

Syntax

[Association, Aggregation, Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_EthernetPortFailoverSettingDataComponent : CIM_Component
{
  CIM_ResourceAllocationSettingData      REF GroupComponent;
  Msvm_FailoverNetworkAdapterSettingData REF PartComponent;
};

Properties

The Msvm_EthernetPortFailoverSettingDataComponent class has these properties.

GroupComponent

Data type: CIM_ResourceAllocationSettingData

Access type: Read-only

Qualifiers: Key, Override ("GroupComponent"), Aggregate

A reference to an instance of the Msvm_EmulatedEthernetPortSettingData or Msvm_SyntheticEthernetPortSettingData class that represents the Ethernet port.

PartComponent

Data type: Msvm_FailoverNetworkAdapterSettingData

Access type: Read-only

Qualifiers: Key, Override ("PartComponent")

A reference to an instance of the Msvm_FailoverNetworkAdapterSettingData class that represents the guest network adapter configuration.
