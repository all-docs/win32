DestroySnapshot method of the CIM_VirtualSystemSnapshotService class

Destroy an existing virtual system snapshot. This method may as a side effect destroy other snapshots that are dependent on the affected snapshot.

Syntax

uint32 DestroySnapshot(
  [in]  CIM_VirtualSystemSettingData REF AffectedSnapshot,
  [out] CIM_ConcreteJob              REF Job
);

Parameters

AffectedSnapshot [in]

A CIM_VirtualSystemSettingData reference to the affected virtual system snapshot.

Job [out]

If the operation is long running, then optionally a CIM_ConcreteJob representing the job may be returned.

[!Note]
This parameter was read/write in Windows 8.1.

Return value

Returns a 0 on success; otherwise, returns an error.
