Msvm_SecurityService class

Represents the security service. It is used for configuring virtual system security settings.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties.

Syntax

[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_SecurityService : CIM_Service
{
};

Members

The Msvm_SecurityService class has these types of members:

  • Methods

Methods

The Msvm_SecurityService class has these methods.

Method Description
GetKeyProtector Method to retrieve the key protector for a virtual system.
ModifySecuritySettings Modifies the current security settings of a virtual machine.
RestoreLastKnownGoodKeyProtector Method to restore back to the last known good key protector.
SetKeyProtector Method to set the key protector for a virtual system.
SetSecurityPolicy Method to set the key protector for a virtual system.
StartService Starts the service.
StopService Stops the service.
