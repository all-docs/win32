Processor classes

The processor classes model a processor for a virtual machine.

The following are virtualization WMI classes related to the processor.

In this section

Topic Description
Msvm_Processor Represents the virtual processor in a virtual machine.
Msvm_ProcessorPool Aggregates the processor resources that may be allocated to a virtual machine.
Msvm_ProcessorSettingData Represents the virtual processor settings for a virtual machine.
