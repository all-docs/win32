UI_ANIMATION_SECONDS_INFINITE

Indicates that there are no scheduled events.

Constant/value Description
UI_ANIMATION_SECONDS_INFINITE -1.0 UI_ANIMATION_SECONDS

Remarks

This value can be returned in a call to the IUIAnimationManager2::EstimateNextEventTime method.
