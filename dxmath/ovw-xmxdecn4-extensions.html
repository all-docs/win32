XMXDECN4 Extensions

XMXDECN4 define a 4D vector for storing signed, normalized values as 10 bit signed x-,y-, and z- components and unsigned, normalized values as a 2 bit unsigned w-component.

[!Note]
The features listed here are only available when developing with C++.
