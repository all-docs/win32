AccessibleObjectFromPointReturnedNot_S_OK

Text

AccessibleObjectFromPoint({0}, {1}) returned {2} and expected S_OK

Type

Warning

Description

AccessibleObjectFromPoint did not return S_OK as expected for the given coordinates.

Possible causes

  • User interaction during verification, such as moving focus to a non-target HWND, interfered with the verification process.
  • An incorrect or invalid MSAA implementation.
