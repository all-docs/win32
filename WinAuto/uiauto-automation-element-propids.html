Automation Element Property Identifiers

This topic describes the named constants that identify the properties of Microsoft UI Automation elements.

Note: As an option, the user agent can also offer a localized description of the W3C ARIA role in the LocalizedControlType property. When the localized string is not specified, the system will provide the default LocalizedControlType string for the element.

Variant type: VT_BSTR Default value: empty string

Note: This property is NULL if the item is not currently displaying a UI.

Note: Use the default value only if the automation element represents a completely new type of control.

Note: For an element to appear in the content view, both the IsContentElement property and the IsControlElement property must be TRUE.

Variant type: VT_BOOL Default value: TRUE
