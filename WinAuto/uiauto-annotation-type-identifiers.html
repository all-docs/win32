Annotation Type Identifiers

This topic describes the named constants that are used to identify types of annotations in a document.

Constant/value Description
AnnotationType_AdvancedProofingIssue 60020 An advanced proofing issue.
AnnotationType_Author 60019 The author of the document.
AnnotationType_CircularReferenceError 60022 A circular reference error that occurred.
AnnotationType_Comment 60003 A comment. Comments can take different forms depending on the application.
AnnotationType_ConflictingChange 60018 A conflicting change that was made to the document.
AnnotationType_DataValidationError 60021 A data validation error that occurred.
AnnotationType_DeletionChange 60012 A deletion change that was made to the document.
AnnotationType_EditingLockedChange 60016 An editing locked change that was made to the document.
AnnotationType_Endnote 60009 The endnote for a document.
AnnotationType_ExternalChange 60017 An external change that was made to the document.
AnnotationType_Footer 60007 The footer for a page in a document.
AnnotationType_Footnote 60010 The footnote for a page in a document.
AnnotationType_FormatChange 60014 A format change that was made.
AnnotationType_FormulaError 60004 An error in a formula. Formula errors typically include red text and exclamation marks.
AnnotationType_GrammarError 60002 A grammatical error, often denoted by a green squiggly line.
AnnotationType_Header 60006 The header for a page in a document.
AnnotationType_Highlighted 60008 Highlighted content, typically denoted by a contrasting background color.
AnnotationType_InsertionChange 60011 An insertion change that was made to the document.
AnnotationType_Mathematics 60023 A text range containing mathematics.
AnnotationType_MoveChange 60013 A move change that was made to the document.
AnnotationType_SpellingError 60001 A spelling error, often denoted by a red squiggly line.
AnnotationType_TrackChanges 60005 A change that was made to the document.
AnnotationType_Unknown 60000 The annotation type is unknown.
AnnotationType_UnsyncedChange 60015 An unsynced change that was made to the document.
