Deprecated GUIDs

This topic describes the GUIDs that are used with the UiaLookupId function.

[!Note]
The UiaLookupId function and the GUIDs described in this topic are deprecated. Instead, applications should use the identifiers described in the following sections:

Â 

  • Control Pattern Identifiers
  • Control Type Identifiers
  • Event Identifiers
  • Property Identifiers
  • Text Attribute Identifiers

This section lists the following GUIDs:

  • PATTERNID GUIDs
  • CONTROLTYPEID GUIDs
  • EVENTID GUIDs
  • PROPERTYID GUIDs
  • TEXTATTRIBUTEID GUIDs
