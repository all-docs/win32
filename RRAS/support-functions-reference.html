Support Functions Reference

The following functions are provided to routing protocols by the router manager. When the router manager calls the StartProtocol function (implemented by the routing protocol), the router manager passes the routing protocol a SUPPORT_FUNCTIONS structure containing pointers to these functions:

DemandDialRequest

MIBEntryCreate

MIBEntryDelete

MIBEntryGet

MIBEntryGetFirst

MIBEntryGetNext

MIBEntrySet

SetInterfaceReceiveType

ValidateRoute
