IPX_NEXT_HOP_ADDRESS structure

[This API has been superseded by the Routing Table Manager Version 2 API and will not be available beyond Windows Server 2003. Applications should use the Routing Table Manager Version 2 API.]

The IPX_NEXT_HOP_ADDRESS structure contains the address for the next-hop router for an IPX route.

Syntax

typedef struct _IPX_NEXT_HOP_ADDRESS {
  BYTE NHA_Mac[6];
} IPX_NEXT_HOP_ADDRESS, *PIPX_NEXT_HOP_ADDRESS;

Members

NHA_Mac

Specifies the MAC address of next-hop router.

Requirements

Requirement Value
Minimum supported client None supported
Minimum supported server Windows 2000 Server [desktop apps only]
End of server support Windows Server 2003
