The Peer Infrastructure and Teredo

A user can check the status of the Teredo interface from the command prompt by typing netsh interface teredo show state and examining the output. If the state is "offline", Teredo is not active, which prevents the user from connecting to other users via their Teredo addresses. It is possible Teredo may be offline because your firewall is disabled or does not support IPv6.
