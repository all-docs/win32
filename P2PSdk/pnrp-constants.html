PNRP Constants

The PNRP Namespace Provider API uses the following constants.

Constant Description
PNRP_CLOUD_FLAGS Specifies the validity of a cloud name.
PNRP_CLOUD_STATE Specifies the different states a PNRP cloud can be in.
PNRP_RESOLVE_CRITERIA Specifies the criteria that PNRP uses to resolve searches.
