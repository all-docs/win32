Graphing API Callbacks

The Graphing API uses the following callbacks:

Callback Description
PFNPEER_FREE_SECURITY_DATA Specifies the function that the Peer Graphing Infrastructure calls to free data returned by PFNPEER_SECURE_RECORD and PFNPEER_VALIDATE_RECORD callbacks.
PFNPEER_SECURE_RECORD Specifies the function that the Peer Graphing Infrastructure calls to secure records.
PFNPEER_VALIDATE_RECORD Specifies the function that the Peer Graphing Infrastructure calls to validate records.
