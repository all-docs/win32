Peer Distribution API Enumerations

The Microsoft Peer Distribution service supports the following enumerations.

Enumeration Description
PEERDIST_CLIENT_INFO_BY_HANDLE_CLASS Defines the possible client information values.
PEERDIST_STATUS Defines the possible Status values of the Peer Distribution service.
