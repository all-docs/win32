Grouping API Structures

The Grouping API uses the following structures.

Structure Description
PEER_CREDENTIAL_INFO Defines information used to obtain and issue a peer's security credentials.
PEER_EVENT_MEMBER_CHANGE_DATA Contains data that describes a change in the status of a peer group member.
PEER_GROUP_EVENT_DATA Contains information about a specific peer group event that has occurred.
PEER_GROUP_EVENT_REGISTRATION Defines the particular peer group event a member can register for.
PEER_GROUP_PROPERTIES Contains data about the membership policy of a peer group.
PEER_INVITATION_INFO Defines information about an invitation to join a peer group. Invitations are represented as Unicode strings.
PEER_MEMBER Contains information that describes a member of a peer group.
