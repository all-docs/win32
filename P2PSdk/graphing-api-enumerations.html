Graphing API Constants

The Graphing API uses the following constants:

Constant Description
PEER_GRAPH_EVENT_TYPE Specifies peer event types the application is to be notified for.
PEER_GRAPH_PROPERTY_FLAGS Specifies properties of a peer graph.
PEER_GRAPH_SCOPE Specifies the network scope of a peer graph.
PEER_GRAPH_STATUS_FLAGS Specifies the current status of a node within the peer graph.
PEER_NODE_CHANGE_TYPE Specifies the types of peer node graph status changes.
