DirectWrite functions

DirectWrite provides the following function. All other operations are performed by using DirectWrite interface objects that are created by the IDWriteFactory object that is created with the function below.

In this section

Topic Description
DWriteCreateFactory Creates a DirectWrite factory object that is used for subsequent creation of individual DirectWrite objects.
DWriteCoreCreateFactory (DWriteCore) Creates a DirectWrite factory object that is used for subsequent creation of individual DWriteCore objects.
