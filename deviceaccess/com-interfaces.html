COM Interfaces - Device Access

Component Object Model (COM)interfaces in the Device Access API.

In this section

Topic Description
ICreateDeviceAccessAsync The ICreateDeviceAccessAsync interface is returned from a call to CreateDeviceAccessInstance. It enables the caller to control the operation of binding to an instance of a device in order to retrieve another interface that can be used to interact with that device.
IDeviceIoControl Sends a control code to a device driver.This action causes the device to perform the corresponding operation.
IDeviceRequestCompletionCallback Provides a method to handle the completion of calls to the DeviceIoControlAsyncmethod.
