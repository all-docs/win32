IMN_CHANGECANDIDATE notification code

Notifies the application when an IME is about to change the content of the candidate window. The application receives this command through the WM_IME_NOTIFY message with parameter settings as shown below.

IMN_CHANGECANDIDATE

Parameters

wParam

Set to IMN_CHANGECANDIDATE.

lParam

Candidate list flag. Each bit corresponds to a candidate list: bit 0 to the first list, bit 1 to the second list, and so on. If a specified bit is 1, the corresponding candidate window is about to be changed.

Return Value

This command has no return value.

Remarks

An application should process this command if it displays candidates itself.

The IME window changes the appearance of the candidate window when it processes this command. An application can get information about the system window with the ImmGetCandidateListCount and ImmGetCandidateList
