LOCALE_ICALENDARTYPE

Current calendar type. Possible values are defined in Calendar Type Information. The calendar type can only represent a calendar that is available for the corresponding locale. To retrieve all calendars available for a locale, the application can use the following NLS functions:

  • EnumCalendarInfo
  • EnumCalendarInfoEx
  • EnumCalendarInfoExEx
