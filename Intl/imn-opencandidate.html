IMN_OPENCANDIDATE event

Notifies an application when an IME is about to open the candidate window. The application receives this command through the WM_IME_NOTIFY message with parameter settings as shown below.

IMN_OPENCANDIDATE

Parameters

wParam

Set to IMN_OPENCANDIDATE.

lParam

Candidate list flag. Each bit corresponds to a candidate list: bit 0 to the first list, bit 1 to the second, and so on. If a specified bit is 1, the corresponding candidate window is about to be opened.

Return Value

This command has no return value.

Remarks

An application should process this command if it displays candidates itself. The application can retrieve a list of candidates to display by using the ImmGetCandidateList function.

By default, the IME window creates a candidate window when it processes this command.
