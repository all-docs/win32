IUpdateOrchestratorUpdateProvider interface

This interface is implemented by the caller and is created using the CLSID provided to IUpdateOrchestrator::Register.

Methods

Method Description
::CancelScan Cancel a scan that is in progress.
::Deserialize Deserialize the bytes into an update.
::get_ScanAttemptFrequencyInMinutes Get the scan attempt frequency.
::ScanForPendingUpdates Scan for pending updates.
::get_ScanSlaInMinutes Gets the Default Scan Sla.

Remarks

USO (Windows Update Session Orchestrator) will call the provider daily in order to get the collection of updates the Updater wishes to perform.

Requirements

Requirement Version
Minimum supported client Windows 10 1903
