IUpdateOrchestratorAction interface

Tells the UpdateOrchestrator about the next action to be performed on an update.

Methods

Method Description
::get_ActionKind Returns the action kind.
::get_IsExclusive Determines if no two actions of its kind can be performed at the same time.
::get_IsPausable Determines if an action can be paused.
::get_Name Returns the action name.
::get_RequiresNetwork Determines if an action needs a network connection.

Requirements

Requirement Version
Minimum supported client Windows 10 1903
