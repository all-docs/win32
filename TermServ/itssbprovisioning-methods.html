ITsSbProvisioning Methods

The ITsSbProvisioning interfaces exposes the following methods.

In this section

CancelJob method

Cancels a provisioning job.

CreateVirtualMachines method

Creates a virtual machine asynchronously.

DeleteVirtualMachines method

Deletes a virtual machine asynchronously.

PatchVirtualMachines method

Patches a virtual machine asynchronously.
