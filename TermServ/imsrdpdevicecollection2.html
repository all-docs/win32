IMsRdpDeviceCollection2 interface

Represents a collection of device objects. This is an enhancement to the IMsRdpDeviceCollection interface.

Members

The IMsRdpDeviceCollection2 interface inherits from IMsRdpDeviceCollection. IMsRdpDeviceCollection2 also has these types of members:

  • Methods

Methods

The IMsRdpDeviceCollection2 interface has these methods.

Method Description
AddDeviceByInstanceId Adds an unlisted device to the device collection.
RedirectNow Forces devices that were selected or unselected from the collection to be redirected or removed.
