What's New in Windows 7

Windows 7 introduces the following new programming elements for Remote Desktop Services.

Requirement Value
Remote Desktop ActiveX Control Interfaces The following interfaces (and their associated properties) were added to the Remote Desktop ActiveX Control Interface.IMsRdpClientAdvancedSettings7 Interface
Remote Desktop Services Web Access Control API Remote Desktop Services Web Access Control exposes interfaces that support the publishing of user-specific resources in RemoteApp and Desktop Connection.Remote Desktop Services Web Access Control API
RemoteApp and Desktop Connection Runtime API RemoteApp and Desktop Connection Runtime API supports the development of custom clients in RemoteApp and Desktop Connection.RemoteApp and Desktop Connection Runtime API
