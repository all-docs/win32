IMsRdpDeviceCollection2::AddDeviceByInstanceId method

Adds an unlisted device to the device collection.

Syntax

HRESULT AddDeviceByInstanceId(
  [in] RedirectDeviceType Type,
  [in] BSTR               InstanceId
);

Parameters

Type [in]

Type: RedirectDeviceType

A value of the RedirectDeviceType enumeration that specifies the type of device being added.

InstanceId [in]

Type: BSTR

The instance identifier of the device to add.

Return value

Type: HRESULT

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
