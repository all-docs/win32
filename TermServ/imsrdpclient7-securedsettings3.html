IMsRdpClient7::SecuredSettings3 property

Retrieves an object that supports the IMsRdpClientSecuredSettings2 interface.

This property is read-only.

Syntax

HRESULT get_SecuredSettings3(
  [out] IMsRdpClientSecuredSettings2 **ppSecuredSettings
);

Property value

The address of an IMsRdpClientSecuredSettings2 interface pointer that receives the object.
