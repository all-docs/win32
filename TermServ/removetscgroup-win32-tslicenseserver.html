RemoveTSCGroup method of the Win32_TSLicenseServer class

[RemoveTSCGroup is no longer available for use as of Windows Server 2012.]

This method is not supported.

Windows Server 2008 R2 and Windows Server 2008: Removes the Terminal Server Computers local group from the Remote Desktop license server.

Syntax

uint32 RemoveTSCGroup();

Parameters

This method has no parameters.

Return value

Returns WBEM_E_NOT_SUPPORTED.

Windows Server 2008 R2 and Windows Server 2008: If the method succeeds, it returns zero. If the method is unsuccessful, it returns a nonzero value. For a list of error codes, see Remote Desktop Services WMI Provider Error Codes.

Remarks

You must be a member of the Administrators group to call this method.

Managed Object Format (MOF) files contain the definitions for Windows Management Instrumentation (WMI) classes. MOF files are not installed as part of the Microsoft Windows Software Development Kit (SDK). They are installed on the server when you add the associated role by using the Server Manager. For more information about MOF files, see Managed Object Format (MOF).
