Remote Desktop Gateway Interfaces

The Remote Desktop Gateway (RDÂ Gateway) API supports the following interfaces. You can use these interfaces to implement plug-ins that provide customized authentication and authorization. There is no dependency between the authentication plug-in and the authorization plug-in, and you can implement only a single plug-in if you choose.

The authentication plug-ins are ITSGAuthenticateUserSink and ITSGAuthenticationEngine.

The authorization plug-ins are ITSGAccountingEngine, ITSGAuthorizeConnectionSink, ITSGAuthorizeResourceSink, and ITSGPolicyEngine.
