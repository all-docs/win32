IMsRdpClient6 interface

Provides the methods and properties needed to configure and use the client control. Derives from the IMsRdpClient5 interface.

Members

The IMsRdpClient6 interface inherits from IMsRdpClient5. IMsRdpClient6 also has these types of members:

  • Properties

Properties

The IMsRdpClient6 interface has these properties.

Property Access type Description
AdvancedSettings7 Read-only The interface to IMsRdpClientAdvancedSettings6.
TransportSettings2 Read-only The interface to IMsRdpClientTransportSettings2.

Remarks

The IMsRdpClient6 interface has been extended by the following interfaces, with each new interface inheriting all the methods and properties of the previous interfaces:

  • IMsRdpClient7
  • IMsRdpClient8
  • IMsRdpClient9
  • IMsRdpClient10

For more information about Remote Desktop Web Connection, see Requirements for Remote Desktop Web Connection.
