IMsRdpDrive::Name property

Retrieves the name of the drive.

This property is read-only.

Syntax

HRESULT get_Name(
  [out] BSTR *pName
);

Property value

The drive name.

Error codes

If the method succeeds, S_OK is returned. Any other HRESULT value indicates that the call failed.
