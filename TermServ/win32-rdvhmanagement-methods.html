Win32_RdvhManagement Methods

The Win32_RdvhManagement Methods class exposes the following methods.

In this section

RdvMigrateVmToDifferentHost method

Initiates a live migration of a virtual machine to a specified host.

RdvCopyBaseVmLocally method

Copies a local base virtual machine to a specified remote desktop virtualization (RDV) host server.

RdvSetupVMPermissions method

Sets the permissions on a virtual machine for the current user.

RdvUndoVMPermissions method

Reverts permissions set by RdvSetupVMPermissions on the specified virtual machine.

RdvCreateUserDiskTemplate method

Creates a user disk template, with the specified maximum size, at the specified location.
