ITsSbClientConnection::Domain property

Retrieves a value that indicates the domain name of the Remote Desktop Connection (RDC) client.

This property is read-only.

Syntax

HRESULT get_Domain(
  [out, retval] BSTR *pVal
);

Property value

A pointer to a BSTR variable that contains the domain name of the RDC client. When you have finished using the string, free it by calling the SysFreeString function.
