IMsRdpDeviceV2 interface

Contains information about a device object. This is an enhancement of the IMsRdpDevice interface.

Members

The IMsRdpDeviceV2 interface inherits from IMsRdpDevice. IMsRdpDeviceV2 also has these types of members:

  • Properties

Properties

The IMsRdpDeviceV2 interface has these properties.

Property Access type Description
CmClassGuid Read-only Contains the configuration manager setup class GUID for the device.
CmDeviceInstance Read-only Contains the configuration manager device instance of the device.
DeviceText Read-only Contains the device text.
DriveLetterBitmap Read-only Contains a bitfield that represents a map of drive letters contained within the device.
IsCompositeDevice Read-only Specifies if the device is a composite device.
IsOptionalDevice Read-only Specifies if the device is optional for USB redirection.
IsUSBDevice Read-only Specifies if the device is for USB redirection.
