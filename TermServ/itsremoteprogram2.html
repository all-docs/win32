ITSRemoteProgram2 interface

Defines properties for use with a RemoteApp.

Members

The ITSRemoteProgram2 interface inherits from ITSRemoteProgram. ITSRemoteProgram2 also has these types of members:

  • Properties

Properties

The ITSRemoteProgram2 interface has these properties.

Property Access type Description
RemoteApplicationArgs Write-only The command-line arguments for the RemoteApp.
RemoteApplicationName Write-only The name of the RemoteApp.
RemoteApplicationProgram Write-only The path and file name of the RemoteApp program.
