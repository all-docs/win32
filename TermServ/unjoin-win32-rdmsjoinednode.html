Unjoin method of the Win32_RDMSJoinedNode class

Removes a node from Remote Desktop Management Services (RDMS).

Syntax

uint32 Unjoin();

Parameters

This method has no parameters.

Return value

Returns 0 on success, otherwise returns a WMI error code.
