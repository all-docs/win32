IMsRdpDeviceCollection2::RedirectNow method

Forces devices that were selected or unselected from the collection to be redirected or removed.

Syntax

HRESULT RedirectNow(
  [in] RedirectDeviceType Type
);

Parameters

Type [in]

Type: RedirectDeviceType

A value of the RedirectDeviceType enumeration that specifies the type of device to be redirected.

Return value

Type: HRESULT

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
