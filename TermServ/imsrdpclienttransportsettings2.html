IMsRdpClientTransportSettings2 interface

Manages client transport settings for the Remote Desktop Gateway (RD Gateway) server.

Members

The IMsRdpClientTransportSettings2 interface inherits from IMsRdpClientTransportSettings. IMsRdpClientTransportSettings2 also has these types of members:

  • Properties

Properties

The IMsRdpClientTransportSettings2 interface has these properties.

Property Access type Description
GatewayCredSharing Read/write Indicates whether the single sign-on feature for RD Gateway is enabled.
GatewayDomain Read/write The domain name that a user provides to connect to the RD Gateway server.
GatewayEncryptedOtpCookie Read/write The encrypted OTP cookie.
GatewayEncryptedOtpCookieSize Read/write The size, in bytes, of the encrypted OTP cookie.
GatewayPassword Write-only The password that a user provides to connect to the RD Gateway server.
GatewayPreAuthRequirement Read/write Indicates whether the RD Gateway one-time password (OTP) feature is enabled.
GatewayPreAuthServerAddr Read/write The web address of the pre-authentication server.
GatewaySupportUrl Read/write The web address of the site that provides technical support for the RD Gateway server.
GatewayUsername Read/write The user name that a user provides to connect to the RD Gateway server.
