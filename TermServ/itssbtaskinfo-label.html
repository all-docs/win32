ITsSbTaskInfo::Label property

Retrieves the label that describes the purpose of the task.

This property is read-only.

Syntax

HRESULT get_Label(
  [out, retval] BSTR *pLabel
);

Property value

A pointer to a BSTR value that contains the label.
