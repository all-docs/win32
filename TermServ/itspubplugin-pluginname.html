ItsPubPlugin::pluginName property

Retrieves the name of the plug-in.

This property is read-only.

Syntax

HRESULT get_pluginName(
  [out, retval] BSTR *pVal
);

Property value

A pointer to a BSTR variable to receive the name of the plug-in.
