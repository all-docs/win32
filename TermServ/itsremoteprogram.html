ITSRemoteProgram interface

Includes methods to set and retrieve the RemoteApp mode and the start-up parameters for a RemoteApp program, such as the path of the executable file and the working directory.

Members

The ITSRemoteProgram interface inherits from the IDispatch interface. ITSRemoteProgram also has these types of members:

  • Methods
  • Properties

Methods

The ITSRemoteProgram interface has these methods.

Method Description
ServerStartProgram Specifies a RemoteApp program to start in the remote session.

Properties

The ITSRemoteProgram interface has these properties.

Property Access type Description
RemoteProgramMode Read/write The RemoteApp mode.
