IMsRdpClientNonScriptable2 interface

Provides access to the nonscriptable properties of a client's remote session on the Remote Desktop ActiveX control. Derives from the IMsRdpClientNonScriptable interface. Methods of this interface can only be accessed through the vtable; they are not available for use to scriptable clients.

Members

The IMsRdpClientNonScriptable2 interface inherits from IMsRdpClientNonScriptable. IMsRdpClientNonScriptable2 also has these types of members:

  • Properties

Properties

The IMsRdpClientNonScriptable2 interface has these properties.

Property Access type Description
UIParentWindowHandle Read/write The window handle to be the parent window for the control. This allows any windows displayed by the control to be properly modal with respect to any windows displayed by the parent application.

Remarks

This interface has been extended by the following interfaces, with each new interface inheriting all the methods and properties of the previous interfaces:

  • IMsRdpClientNonScriptable3
  • IMsRdpClientNonScriptable4
  • IMsRdpClientNonScriptable5

For more information about Remote Desktop Web Connection, see Requirements for Remote Desktop Web Connection.
