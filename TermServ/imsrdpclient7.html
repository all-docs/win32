IMsRdpClient7 interface

Provides the methods and properties needed to configure and use the client control. Derives from the IMsRdpClient6 interface.

Members

The IMsRdpClient7 interface inherits from IMsRdpClient6. IMsRdpClient7 also has these types of members:

  • Methods
  • Properties

Methods

The IMsRdpClient7 interface has these methods.

Method Description
GetStatusText Retrieves the status text for the specified status code.

Properties

The IMsRdpClient7 interface has these properties.

Property Access type Description
AdvancedSettings8 Read-only An object that supports the IMsRdpClientAdvancedSettings7 interface.
RemoteProgram2 Read-only An object that supports the ITSRemoteProgram2 interface.
SecuredSettings3 Read-only An object that supports the IMsRdpClientSecuredSettings2 interface.
TransportSettings3 Read-only An object that supports the IMsRdpClientTransportSettings3 interface.

Remarks

The IMsRdpClient7 interface has been extended by the following interfaces, with each new interface inheriting all the methods and properties of the previous interfaces:

  • IMsRdpClient8
  • IMsRdpClient9
  • IMsRdpClient10

For more information about Remote Desktop Web Connection, see Requirements for Remote Desktop Web Connection.
