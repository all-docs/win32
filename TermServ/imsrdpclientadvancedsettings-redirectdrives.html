IMsRdpClientAdvancedSettings::RedirectDrives property

Specifies if redirection of disk drives is allowed.

This property is read/write.

Syntax

HRESULT put_RedirectDrives(
  [in]  VARIANT_BOOL redirectDrives
);

HRESULT get_RedirectDrives(
  [out] VARIANT_BOOL *predirectDrives
);

Property value

Set this parameter to VARIANT_TRUE to allow redirection or VARIANT_FALSE otherwise. VARIANT_TRUE prompts the user to confirm allowing redirection at connection time, for security purposes.

Error codes

Returns S_OK if successful.

Remarks

For more information about Remote Desktop Web Connection, see Requirements for Remote Desktop Web Connection.
