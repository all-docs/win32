IMsRdpDrive interface

Contains information about a drive object.

Members

The IMsRdpDrive interface inherits from the IUnknown interface. IMsRdpDrive also has these types of members:

  • Properties

Properties

The IMsRdpDrive interface has these properties.

Property Access type Description
Name Read-only Retrieves the name of the drive.
RedirectionState Read/write Indicates the redirection state of the drive.
