Remote Desktop Services AudioEndpoint Enumeration Types

The following enumeration types are used with the AudioEndpoint API.

In this section

AE_POSITION_FLAGS

Defines constants for the AE_CURRENT_POSITION structure. These constants describe the degree of validity of the current position.

APO_BUFFER_FLAGS

Defines the buffer validation flags for the APO_CONNECTION_PROPERTY structure associated with each APO connection.

Remarks

The Remote Desktop Services AudioEndpoint API is for use in Remote Desktop scenarios; it is not for client applications.
