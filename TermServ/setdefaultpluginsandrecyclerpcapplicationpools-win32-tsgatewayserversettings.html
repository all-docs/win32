SetDefaultPluginsAndRecycleRpcApplicationPools method of the Win32_TSGatewayServerSettings class

Sets the current authentication and authorization plug-ins for the Remote Desktop Gateway (RD Gateway) server and recycles the RPC application pools in IIS.

Syntax

uint32 SetDefaultPluginsAndRecycleRpcApplicationPools();

Parameters

This method has no parameters.

Return value

Type: uint32

If the method succeeds, it returns zero. If the method is unsuccessful, it returns a nonzero value. For a list of error codes, see Remote Desktop Services WMI Provider Error Codes.

Remarks

Calling this method results in all existing connections being disconnected.

You must be a member of the Administrators group to call this method.

Managed Object Format (MOF) files contain the definitions for Windows Management Instrumentation (WMI) classes. MOF files are not installed as part of the Microsoft Windows Software Development Kit (SDK). They are installed on the server when you add the associated role by using the Server Manager.
