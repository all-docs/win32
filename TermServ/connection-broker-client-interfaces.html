Remote Desktop Connection Broker client interfaces

The Remote Desktop Connection Broker client API defines the following interfaces.

In this section

IConnectionBrokerClient

Provides the methods needed to use the Remote Desktop Connection Broker client.

IConnectionBrokerRequest

Provides the methods needed to obtain the results of the asynchronous IConnectionBrokerClient::GetTargetInfo method.
