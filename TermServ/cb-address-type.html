CB_ADDRESS_TYPE enumeration

Specifies the address type.

Syntax

typedef enum _CB_ADDRESS_TYPE { 
  CB_ADDR_UNDEFINED  = 0,
  CB_ADDR_IPv4       = 4,
  CB_ADDR_IPv6       = 6
} CB_ADDRESS_TYPE;

Constants

CB_ADDR_UNDEFINED

The address type is undefined.

CB_ADDR_IPv4

The address is an IPv4 address.

CB_ADDR_IPv6

The address is an IPv6 address.
