SetAuthenticationPluginAndRecycleRpcApplicationPools method of the Win32_TSGatewayServerSettings class

Sets the current authentication plug-in for the Remote Desktop Gateway (RD Gateway) server and recycles the RPC application pools in IIS.

This method is equivalent to calling the SetAuthenticationPlugin and RecycleRpcApplicationPools methods in sequence.

Syntax

uint32 SetAuthenticationPluginAndRecycleRpcApplicationPools(
  [in] string PluginName
);

Parameters

PluginName [in]

Type: string

Name of the authentication plug-in

Return value

Type: uint32

If the method succeeds, it returns zero. If the method is unsuccessful, it returns a nonzero value. For a list of error codes, see Remote Desktop Services WMI Provider Error Codes.

Remarks

Calling this method results in all existing connections being disconnected.

You must be a member of the Administrators group to call this method.
