IMsRdpClientNonScriptable5::DisableConnectionBar property

Specifies whether the Remote Desktop ActiveX control should disable the connection bar. If this property contains VARIANT_TRUE, the connection bar should be disabled. If this property contains VARIANT_FALSE, the connection bar should be enabled.

This property is write-only.

Syntax

HRESULT put_DisableConnectionBar(
  [in] VARIANT_BOOL fDisableConnectionBar
);

Property value

Specifies the new property value.
