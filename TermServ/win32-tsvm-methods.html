Win32_TSVm Methods

The Win32_TSVm Methods class exposes the following methods.

In this section

QueryOfflineInformation method

Retrieves the properties of the current virtual desktop server (VDS), but only when the server is offline.

DumpVmInfo method

This member is for internal testing purposes, and should not be used in your code.

Export method

Retrieves the virtual machine session monitoring information.
