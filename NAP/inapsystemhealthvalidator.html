INapSystemHealthValidator interface

[!Note]
The Network Access Protection platform is not available starting with Windows 10

The INapSystemHealthValidator provides methods that must be implemented by SHV developers to enable the NAP system to communicate with an SHV.

Members

The INapSystemHealthValidator interface inherits from the IUnknown interface. INapSystemHealthValidator also has these types of members:

  • Methods

Methods

The INapSystemHealthValidator interface has these methods.

Method Description
INapSystemHealthValidator::Validate The NAP system calls this application defined method to validate the SoHRequest received from a client.
