INapSystemHealthValidationRequest::SetSoHResponse method

[!Note]
The Network Access Protection platform is not available starting with Windows 10

The INapSystemHealthValidationRequest::SetSoHResponse method allows System Health Validators (SHVs) to set the SoHResponse to be sent to its System Health Agent (SHA) counterpart on the client-side.

Syntax

HRESULT SetSoHResponse(
  [in] const SoHResponse *sohResponse
);

Parameters

sohResponse [in]

A pointer to a SoHResponse structure.

Return value

Other COM-specific error codes also may be returned.

Return code Description
S_OK Operation succeeded.
E_ACCESSDENIED Permissions error, access denied.
E_OUTOFMEMORY System resource limit, could not perform the operation.
