INapEnforcementClientConnection::Initialize method

[!Note]
The Network Access Protection platform is not available starting with Windows 10

The INapEnforcementClientConnection::Initialize method initializes the client connection.

Syntax

HRESULT Initialize(
  [in] EnforcementEntityId id
);

Parameters

id [in]

A pointer to an EnforementEntityId that identifies the enforcement client requesting the connection. This value is set by the NapAgent during connection creation.

Return value

Other COM-specific error codes also may be returned.

Return code Description
S_OK Operation succeeded.
E_ACCESSDENIED Permissions error, access denied.
E_OUTOFMEMORY System resource limit, could not perform the operation.
