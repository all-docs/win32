INapEnforcementClientConnection::GetIsolationInfo method

[!Note]
The Network Access Protection platform is not available starting with Windows 10

The INapEnforcementClientConnection::GetIsolationInfo method is used get the isolation information of the client.

Syntax

HRESULT GetIsolationInfo(
  [out] IsolationInfo **isolationInfo
);

Parameters

isolationInfo [out]

A pointer to a pointer to an IsolationInfo structure that contains the connectivity state of the client.

Return value

Other COM-specific error codes also may be returned.

Return code Description
S_OK Operation succeeded.
E_ACCESSDENIED Permissions error, access denied.
E_OUTOFMEMORY System resource limit, could not perform the operation.

Remarks

This information is set by the NapAgent after processing a SoHResponse and must not be set by the enforcer.
