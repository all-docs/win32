INapSystemHealthAgentRequest::GetCorrelationId method

[!Note]
The Network Access Protection platform is not available starting with Windows 10

The INapSystemHealthAgentRequest::GetCorrelationId method is used by system health agents to correlate SoH's and SoH-Responses.

Syntax

HRESULT GetCorrelationId(
  [out] CorrelationId *correlationId
);

Parameters

correlationId [out]

A pointer to a unique CorrelationId for the SoH exchange.

Return value

Other COM-specific error codes also may be returned.

Return code Description
S_OK Operation succeeded.
E_ACCESSDENIED Permissions error, access denied.
E_OUTOFMEMORY System resource limit, could not perform the operation.
