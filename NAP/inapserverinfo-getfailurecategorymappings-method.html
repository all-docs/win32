INapServerInfo::GetFailureCategoryMappings method

[!Note]
The Network Access Protection platform is not available starting with Windows 10

The INapServerInfo::GetFailureCategoryMappings method retrieves the failure category mappings for a specified SHA or SHV.

Syntax

HRESULT GetFailureCategoryMappings(
  [in]  SystemHealthEntityId   id,
  [out] FailureCategoryMapping *mapping
);

Parameters

id [in]

A SystemHealthEntityId that contains the unique identifier of the SHA or the SHV.

mapping [out]

A pointer to a FailureCategoryMapping that contains the mapping data.

Return value

Other COM-specific error codes also may be returned.

Return code Description
S_OK Operation succeeded.
E_ACCESSDENIED Permissions error, access denied.
E_OUTOFMEMORY System resource limit, could not perform the operation.
