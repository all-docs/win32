INapSoHConstructor interface

[!Note]
The Network Access Protection platform is not available starting with Windows 10

The INapSoHConstructor provides methods that are used by SHAs to construct SoHRequests and by SHVs to construct SoHResponses.

Members

The INapSoHConstructor interface inherits from the IUnknown interface. INapSoHConstructor also has these types of members:

  • Methods

Methods

The INapSoHConstructor interface has these methods.

Method Description
INapSoHConstructor::AppendAttribute Adds a TLV to the end of the SoH buffer.
INapSoHConstructor::GetSoH Retrieves the constructed SoH packet.
INapSoHConstructor::Initialize Initializes the SoH packet.
INapSoHConstructor::Validate Checks the validity of the SoH packet.
