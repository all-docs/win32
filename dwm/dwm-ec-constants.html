DWM Enable Composition Constants

Flags used by the DwmEnableComposition function to change the state of Desktop Window Manager (DWM) composition.

DWM_EC_DISABLECOMPOSITION

0

Disable composition.

DWM_EC_ENABLECOMPOSITION

1

Enable composition.
