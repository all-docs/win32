Trust-Attributes attribute

This attribute stores the trust attributes for a trusted domain. Possible attribute values are as follows:

  • TRUST_ATTRIBUTE_NON_TRANSITIVE Disable transitivity.
  • TRUST_ATTRIBUTE_TREE_PARENT Trust is set to the organization tree parent.
  • TRUST_ATTRIBUTE_TREE_ROOT Trust set to another tree root in the forest.
  • TRUST_ATTRIBUTE_UPLEVEL_ONLY Trusted link valid only for uplevel client.
