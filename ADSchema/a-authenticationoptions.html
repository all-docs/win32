Authentication-Options attribute

The authentication options used in ADSI to bind to directory services objects.

Entry Value

Ldap-Display-Name authenticationOptions
Size 4 bytes. Values defined in IADS.h: ADS_SECURE_AUTHENTICATION 0x1, ADS_USE_ENCRYPTION 0x2, ADS_USE_SSL 0x2, ADS_READONLY_SERVER 0x4, ADS_PROMPT_CREDENTIALS 0x8, ADS_NO_AUTHENTICATION 0x10, ADS_FAST_BIND 0x20, ADS_USE_SIGNING 0x40, ADS_USE_SEALING 0x80
