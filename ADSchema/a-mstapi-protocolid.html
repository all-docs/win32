ms-TAPI-Protocol-Id attribute

This attribute indicates the TAPI conference type. This attribute is used to determine how the binary BLOB in the ms-TAPI-Conference-Blob attribute is to be interpreted. This attribute is an arbitrary string, typically less than 100 characters in length.

Entry Value
CN ms-TAPI-Protocol-Id
Ldap-Display-Name msTAPI-ProtocolId
Size -
Update Privilege No special privileges required.
Update Frequency Set once at the time of creating the Rt-Conference object.
