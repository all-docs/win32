Object-Class-Category attribute

This attribute contains the class type, such as abstract, auxiliary, or structured.

Entry Value
CN Object-Class-Category
Ldap-Display-Name objectClassCategory
Size 4 bytes. Structural 1, abstract 2, auxiliary 3. Class 88, 0 should not be used.
Update Privilege The designer of the object would set this value.
Update Frequency This value should never change.
