User-Shared-Folder-Other attribute

Specifies a UNC path to the user's additional shared documents folder. The path must be a network UNC path of the form **\\Server\Share\**Directory. This value can be a null string.

Entry Value
CN User-Shared-Folder-Other
Ldap-Display-Name userSharedFolderOther
Size -
Update Privilege Domain administrator or account owner.
Update Frequency When the user's record is created and whenever the shared folder needs to change.
