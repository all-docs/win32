Interval syntax

Represents a time interval value. The actual time units depend on which attribute is using this syntax. This syntax is identical to the LargeInteger syntax, except the high and low values are unsigned integers.

Entry Value
Name Interval
Syntax ID 2.5.5.16
OM ID 65
MAPI Type -
ADS Type ADSTYPE_LARGE_INTEGER
Variant Type VT_DISPATCH
SDS Type A COM object that can be cast to an IADsLargeInteger.
