IP_DSCP_TRAFFIC_TYPE

The following table describes IP_DSCP_TRAFFIC_TYPE Socket Options.

Option Get Set Optval Type Description
DSCP_TRAFFIC_TYPE Yes Yes DWORD By setting this value to values defined in DSCP_TRAFFIC_TYPE, an application will be able to ask its network packets to be treated according to the type of service being requested. Similarly calls to getsockopt can be used to obtain the current setting for the type of traffic requested on the given socket
