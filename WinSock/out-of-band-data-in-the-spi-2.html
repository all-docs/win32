Out-of-Band Data in the SPI

The service providers which support the out-of-band data (OOB) abstraction for the stream-style sockets must adhere to the semantics in this section. We will describe OOB data handling in a protocol-independent manner. Please refer to the Winsock Annexes for a discussion of OOB data implemented using urgent data in TCP/IP service providers. In the following, the use of WSPRecv also applies to WSPRecvFrom.
