LIBMAIN.CPP

In the example provider component, a code example of the standard DLL entry point for Adssmp.dll is in Libmain.cpp. Supported routines are listed in the following table.

Item Description
DllGetClassObject Standard DLL entry point for locating class factories.
DllCanUnloadNow Standard DLL entry point to determine if DLL can be unloaded.
LibMain Standard DLL initialization entry point.
IsCompatibleOleVersion Verify compatibility with the OLE run time.
DllMain Entry point for most Windows 2000 or Windows NT versions.
