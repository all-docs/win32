Provider Overview for ADSI

Besides the Provider Minimum Requirements, other use information includes:

  • ADSI Component Interaction describes the context in which the provider component runs.
  • Schema Interfaces discuss the schema management interfaces from the provider's point of view.
  • Enumerating Container Objects describes the requirements to support enumerations.
  • Provider Registry Information lists the registry entries an ADSI provider uses.
  • Support for Queries describes the requirements to support searching of the directory.
  • Implementation Issues for ADSI Providers includes notes to providers about various software issues.
