ADSI Utility Interfaces

ADSI provides several utility interfaces for ADSI objects that can help in accessing and maintaining the Active Directory. These interfaces are:

  • IADsDeleteOps
  • IADsObjectOptions
  • IADsPathname
  • IADsNameTranslate
