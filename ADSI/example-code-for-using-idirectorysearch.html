Example Code for Using IDirectorySearch

The Platform Software Development Kit (SDK) includes a code example topic, DsSrch, that provides a simple command-prompt application that uses the IDirectorySearch interface to query a directory service. The complete code example is included in the Platform SDK in the Samples\NetDs\ADSI\Samples\General\DsSrch directory.
