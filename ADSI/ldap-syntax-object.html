LDAP Syntax Object

The LDAP provider uses the following mapping between the LDAP syntax and ADSI syntax.

Boolean VT_BOOL
Counter VT_I4
EmailAddress VT_BSTR

Integer VT_I4
Interval VT_I4
List VT_VARIANT (VT_BSTR | VT_ARRAY)
NetAddress VT_BSTR
OctetString VT_VARIANT
Path VT_BSTR
PhoneNumber VT_BSTR
PostalAddress VT_BSTR
SmallInterval VT_I4
String VT_BSTR
Time VT_DATE
