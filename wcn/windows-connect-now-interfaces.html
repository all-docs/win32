Windows Connect Now Interfaces

The Windows Connect Now API (WCNAPI) defines and utilizes the following interfaces.

Interface Description
IWCNConnectNotify Receive a success or failure notification when a Windows Connect Now connect session completes.
IWCNDevice Configures the device and initiates the session.
