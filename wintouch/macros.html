Macros (Windows Touch Gestures)

This section contains macros for Windows Touch gestures.

The following macros are specified in this section.

Macro Description
GID_ROTATE_ANGLE_FROM_ARGUMENT Interprets the binary radian when receiving the value in the WM_GESTUREGID_COMMAND_ROTATEullArgument field.
GID_ROTATE_ANGLE_TO_ARGUMENT Takes a radian value and converts it into a binary radian value.
