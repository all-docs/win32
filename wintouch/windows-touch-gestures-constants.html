Windows Touch Gestures Constants

This section lists the constants used for Windows Touch gestures.

GESTURECONFIGMAXCOUNT

256

Defines the maximum number of gesture configurations that can be included in a single call to SetGestureConfig or GetGestureConfig.

Requirements

Requirement Value
Minimum supported client Windows 7 [desktop apps only]
Minimum supported server Windows Server 2008 R2 [desktop apps only]
Header Winuser.h (include Windows.h)
