Functions (Boundary Feedback)

This section describes the boundary feedback functions.

Function Description
BeginPanningFeedback This function is invoked by the user before using boundary feedback.
EndPanningFeedback This function is called upon completion of boundary feedback and optionally triggers an animation.
UpdatePanningFeedback This function is called during boundary feedback and optionally triggers an animation.
