Classes and Interfaces

This section contains class and interface references for inertia and manipulations.

The following classes and interfaces are used for inertia and manipulations.

Class/Interface Description
_IManipulationEvents Handles manipulation events.
IInertiaProcessor Handles calculations regarding object motion for Windows Touch.
IManipulationProcessor Provides functionality for monitoring and responding to Windows Touch input.
