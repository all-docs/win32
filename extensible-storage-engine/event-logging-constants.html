Event Logging Constants

The following constants indicate the detail level for event log messages for the JET_ParamEventLoggingLevel system parameter.

Constant/value Description
JET_EventLoggingDisable0 Disables event logging.
JET_EventLoggingLevelMax100 Sets the maximum level to use for event logs, which is currently 100 characters.
