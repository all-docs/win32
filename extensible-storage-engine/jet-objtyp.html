JET_OBJTYP

The JET_OBJTYP group of constants represent the type of a database object. Currently, only tables are supported.

Constant/value Description
JET_objtypNil0 Represents all types of objects. Currently, only tables are supported.
JET_objtypTable1 Represents a table.
