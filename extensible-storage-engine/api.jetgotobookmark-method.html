Api.JetGotoBookmark method

Positions a cursor to an index entry for the record that is associated with the specified bookmark. The bookmark can be used with any index defined over a table. The bookmark for a record can be retrieved using JetGetBookmark(JET_SESID, JET_TABLEID, [], Int32, Int32).
