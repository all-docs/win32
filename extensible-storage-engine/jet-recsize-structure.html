JET_RECSIZE Structure

The JET_RECSIZE structure is used by JetGetRecordSize to return information about a record's usage requirements in user data space, number of set columns, number of values, and ESE record structure overhead space.

WindowsÂ Vista: The JET_RECSIZE structure is introduced in WindowsÂ Vista.

      unsigned __int64 cNonTaggedColumns;
      unsigned __int64 cTaggedColumns;
      unsigned __int64 cLongValues;
      unsigned __int64 cMultiValues;
    } JET_RECSIZE;
