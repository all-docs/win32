JET_RSTMAP Structure

The JET_RSTMAP structure enables the remapping of database file paths that are stored in the transaction logs during recovery, when used by the JetInit and JetExternalRestore functions. This enables the databases to be moved when offline or when restored from backup.

    typedef struct {
      xRPC_STRING tchar* szDatabaseName;
      xRPC_STRING tchar* szNewDatabaseName;
    } JET_RSTMAP;

Members

szDatabaseName

The current absolute path of a database that is associated with the transaction logs that are replayed during recovery.

szNewDatabaseName

The new absolute path for the database.
