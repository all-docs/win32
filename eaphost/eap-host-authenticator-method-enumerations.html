EAPHost Authenticator Method Enumerations

The EAPHost Authenticator Method enumerations are as follows.

Enumeration Description
EAP_AUTHENTICATOR_SEND_TIMEOUT Indicates to the information system (IS) the amount of time to wait for user input after the packet is sent. The timeout value can be set to none.
EAP_METHOD_AUTHENTICATOR_RESPONSE_ACTION Defines the response instructions sent by the authenticator to the supplicant or EAP peer method.
