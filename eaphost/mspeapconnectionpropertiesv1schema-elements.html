mspeapconnectionpropertiesv1 Schema Elements

The mspeapconnectionpropertiesv1 schema consists of the following schema elements.

  • AcceptServerName (PeapExtensionsType)
  • DisableUserPromptForServerValidation (ServerValidationParameters)
  • EapType
  • EnableQuarantineChecks (EapType)
  • FastReconnect (EapType)
  • IdentityPrivacy (PeapExtensionsType)
  • InnerEapOptional (EapType)
  • PeapExtensions (EapType)
  • PeapExtensionsV2 (PeapExtensionsType)
  • PerformServerValidation (PeapExtensionsType)
  • RequireCryptoBinding (EapType)
  • ServerNames(ServerValidationParameters)
  • ServerValidation (EapType)
  • TrustedRootCA (ServerValidationParameters)
