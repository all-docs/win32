EAPHost Peer Method Enumerations

The EAPHost Peer Method API enumerations are as follows.

Enumeration Description
EapCredentialType Defines the EAP credential types and the appropriate credentials.
EapPeerMethodResponseAction Defines the set of actions an EAP authenticator can indicate to a supplicant or EAP peer method during authentication.
EapPeerMethodResultReason Defines the set of results of an EAP authentication session returned by an EAP authenticator method to an EAP peer method.
