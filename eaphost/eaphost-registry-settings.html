EAPHost Registry Settings

Registry values in the following registry keys control aspects of the functionality of EAPHost.

Subkey Description
BypassNegotiation Determines if capabilities negotiation between the RAS server and client occurs.
AssumePhase2Fragmentation Determines if the RAS server and client assume phase 2 fragmentation.
OverrideEAPCertificateSelection Determines whether the client will override the default smart card certificate selection behavior and provide the user with more certificates to choose from.
PeapServerUseAllPurposeCert Determines if all-purpose certificates are used for PEAP authentication.
PeapServerAcceptAllPurposeCert Determines if all-purpose certificates are accepted for PEAP authentication.
TlsServerUseAllPurposeCert Determines if all-purpose certificates are used for EAP-TLS authentication.
TlsServerAcceptAllPurposeCert Determines if all-purpose certificates are accepted for EAP-TLS authentication.
