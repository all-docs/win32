InnerEapOptional (EapType) Element

The InnerEapOptional (EapType) element indicates whether to perform GUEST access.

<xs:element name="InnerEapOptional"
    type="boolean"
 />

The InnerEapOptional element is defined by the EapType element.

Remarks

If the InnerEapOptional element is TRUE, then the Eap element must be present and describe the inner method and it's configuration; if FALSE, then PEAP will perform GUEST access. The InnerEapOptional element is optional.
