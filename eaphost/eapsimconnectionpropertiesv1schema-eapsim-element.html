EapSim element (eapsimconnectionpropertiesv1schema)

The EapSim element is a derived type of the EapType element from the BaseEapConnectionProperties schema.

This derived EapSim element contains the following elements:

  • UseStrongCipherKeys
  • DontRevealPermanentID
  • ProviderName
  • Realm

                    <xs:element name="Realm">
                    <xs:complexType>

                    </xs:element>

                    <xs:any processContents="lax" minOccurs="0" maxOccurs="unbounded" namespace="##other"/>
                </xs:sequence>
    </xs:complexType>
