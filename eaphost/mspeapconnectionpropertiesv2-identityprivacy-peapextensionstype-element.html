The IdentityPrivacy (PeapExtensionsType) Element

The IdentityPrivacy (PeapExtensionsType) element indicates whether a user's true identity or an anonymous identity is sent.

<xs:element name="IdentityPrivacy"
    type="IdentityPrivacyParameters"
 />

The IdentityPrivacy element is defined by the PeapExtensionsType element.

Remarks

The IdentityPrivacy element is optional.
