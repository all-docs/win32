eaptlsconnectionpropertiesv1 Schema Elements

The eaptlsconnectionpropertiesv1 schema is made up of the following schema elements.

  • AcceptServerName (EapType)
  • CertificateStore (CredentialsSourceParameters)
  • CredentialsSource (EapType)
  • DifferentUsername (EapType)
  • DisableUserPromptForServerValidation (ServerValidationParameters)
  • EapType
  • PerformServerValidation (EapType)
  • ServerNames (ServerValidationParameters)
  • ServerValidation (EapType)
  • SimpleCertSelection (CertSelection)
  • SmartCard (CredentialsSourceParameters)
  • TLSExtensions (EapType)
  • TrustedRootCA (ServerValidationParameters)
