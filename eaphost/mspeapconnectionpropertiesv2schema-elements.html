mspeapconnectionpropertiesv2 Schema Elements

The mspeapconnectionpropertiesv2 schema consists of the following schema elements.

  • AcceptServerName (PeapExtensionsType)
  • AnonymousUserName (IdentityPrivacyParameters)
  • EnableIdentityPrivacy (IdentityPrivacyParameters)
  • IdentityPrivacy (PeapExtensionsType)
  • PeapExtensionsV2 (PeapExtensionsType)
  • PerformServerValidation (PeapExtensionsType)
