EAPHost Authenticator Method Structures

The EAPHost Authenticator Method structures are as follows.

Structure Description
EAP_AUTHENTICATOR_METHOD_ROUTINES Contains a set of function pointers to the EAPHost Authenticator Method APIs.
EAP_METHOD_AUTHENTICATOR_RESULT Contains the EAP attributes provided as the result of an EAP authentication session.
