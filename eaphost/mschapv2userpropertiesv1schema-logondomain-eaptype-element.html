LogonDomain (EapType) Element

TheLogonDomain (EapType) element identifies the domain of the user or machine being authenticated.

<xs:element name="LogonDomain
          
        "
    type="string"
 />

The LogonDomain element is defined by the EapType element.

Remarks

If the LogonDomain (EapType) element is not present, the domain from winlogon is used. This element is optional.
