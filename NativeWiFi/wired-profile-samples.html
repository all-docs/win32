Wired Profile Samples

The following topics show sample XML profiles for various wired network configurations. These XML profiles use LAN_profile schema elements and OneX schema elements.

You can add a profile to the profile store using the netsh command-line utility. For more information, see Netsh Commands for Wired Local Area Network (LAN).
