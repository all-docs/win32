OneX Schema

The OneX Schema defines 802.1X configuration elements. All OneX schema elements apply to both wired and wireless profiles. For a list of defined elements, see OneX Schema Elements.

The root element of an 802.1X profile is the OneX element. Each profile will have exactly one root element. The OneX element is in the https://www.microsoft.com/networking/OneX/v1 namespace.
