LAN_profile Schema

The LAN_profile schema defines a wired (LAN) network profile. A LAN profile contains network settings for an Ethernet interface.

The root element of a LAN profile is the LANProfile element. Each profile will have exactly one root element. All LAN_profile elements are in the namespace https://www.microsoft.com/networking/LAN/profile/v1.

To view sample LAN profiles, see Wired profile samples.
