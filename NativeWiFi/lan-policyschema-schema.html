LAN_policy Schema

The LAN_policy schema defines a wired (LAN) policy profile.

The root element of a LAN policy profile is the LANPolicy element. Each profile has exactly one root element. All LAN_policy elements are in the namespace https://www.microsoft.com/networking/LAN/policy/v1.
