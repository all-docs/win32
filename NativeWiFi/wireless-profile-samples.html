Wireless profile samples

The following topics show sample XML profiles for various wireless configurations. These XML profiles use WLAN_profile schema elements and OneX schema elements.

You can add a wireless profile to the profile store programmatically by calling WlanSetProfile. You can also add a profile to the profile store using the netsh command-line utility. For more information, see Netsh Commands for Wireless Local Area Network (wlan).
