XPS OM Trunk Interfaces

The trunk-level interfaces of the XPS Document API describe the structure of an XPS document as the XPS OM represents it.

XPS OM Package InterfacesThe package interfaces describe the features and operations that act on the top level of an XPS OM.
XPS OM Document InterfacesThe document interfaces describe the organization and sequence of the package's FixedDocuments and the documents' FixedPages.
XPS OM Part InterfacesThe part interfaces describe the resources that are used by the document and its components.
