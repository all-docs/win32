DocumentStaple

This topic is not current. For the most current information, see the Print Schema Specification.

Describes the stapling characteristics of the output. Each document is stapled separately. The JobStapleAllDocuments and DocumentStaple keywords are mutually exclusive. It is up to the driver to determine constraint handling between these keywords.

  • Element Information
  • Structural Content
  • Extensible Markup Language (XML) Content

Element Information

Name Value
Element Type Feature
Scoping Prefix Document
Notes Top, Bottom, Left, and Right are relative to the PageImageableSize.

Structural Content

The XML structure of this element is:

<psf:Feature name="psk:DocumentStaple">
  <psf:Property name="psf:SelectionType">
    <psf:Value xsi:type="xs:string">psk:PickOne</psf:Value>
  </psf:Property>
  <psf:Option name="psk:_OptionName_">
    <psf:Property name="psf:IdentityOption">
      <psf:Value xsi:type="xs:string">_IdentityOptionValue_</psf:Value>
    </psf:Property>
    <psf:ScoredProperty name="psk:Angle" >
      <psf:Value xsi:type="xs:integer">_AngleValue_</psf:Value>
    </psf:ScoredProperty>
    <psf:ScoredProperty name="psk:SheetCapacity" >
      <psf:Value xsi:type="xs:integer">_SheetCapacityValue_</psf:Value>
    </psf:ScoredProperty>
  </psf:Option>
</psf:Feature>

Structure Variables

The following table outlines the characteristics of the variables defined in the XML structure.

Name Data type Unit Supported values Summary
_OptionName_ string characters Valid fully qualified name as defined by Namespaces in XML. If no namespace is specified, default namespace is assumed. The name of the option.
_IdentityOptionValue_ string n/a True, False. Defines an Option which when selected would disable this feature.
_AngleValue_ integer degrees Greater than 0. Specifies the staple angle, relative to the X direction of the PageImageableSize. The staple angle is measured in a counter-clockwise direction.
_SheetCapacityValue_ integer sheets of media Greater than 0. Specifies number of sheets supported by the stapling option for the currently selected MediaType.
