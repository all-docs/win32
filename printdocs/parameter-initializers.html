Parameter Initializers

This topic is not current. For the most current information, see the Print Schema Specification.

For a summary of the ParameterInit element type, please refer to ParameterInit section.

A ParameterInit element type assigns a value to a ParameterDef. The ParameterDef is therefore initialized in the PrintTicket. To see an example of a ParameterInit element type, refer to the Print Schema PrintTicket Example.

For more information on the interaction between ParameterDef elements and ParameterInit elements, refer to ParameterDef and ParameterInit Elements section.
