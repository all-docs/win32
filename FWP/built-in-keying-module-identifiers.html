Built-in Keying Module Identifiers

The identifiers for the keying modules that are built in to the Windows Filtering Platform (WFP) are each represented by a GUID.

These identifiers are defined as follows.

FWPM_KEYING_MODULE_IKE

Internet Key Exchange (IKE) keying module.

FWPM_KEYING_MODULE_IKEV2

Internet Key Exchange version 2 (IKEv2) keying module.

[!Note]
Available only on Windows Server 2008 R2 and Windows 7.

FWPM_KEYING_MODULE_AUTHIP

AuthIP keying module.

Requirement Value
