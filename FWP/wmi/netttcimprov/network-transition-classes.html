NetTtCim Provider

This section provides reference information for NetTtCim provider classes defined in NetTtCim.mof and implemented in NetTtCim.dll.

In this section

Topic Description
MSFT_Net6to4Configuration Represents the global 6to4 configuration shared across all 6to4 interfaces.
MSFT_Net6to4State Represents per-interface 6to4 Configuration settings.
MSFT_NetDnsTransitionConfiguration Provides configuration settings for DNS64.
MSFT_NetDnsTransitionInterfaceAssociation Describes the association between DNS64 settings and accept/send interfaces.
MSFT_NetDnsTransitionMonitoring Provides counters for DNS64 queries.
MSFT_NetIPHttpsConfiguration Provides configuration settings for the IP over HTTPS (IP-HTTPs) Tunneling Protocol.
MSFT_NetIpHTTPsState Represents per-interface IP-HTTPs configuration settings.
MSFT_NetISATAPConfiguration Represents global ISATAP Configuration shared across all ISATAP interfaces.
MSFT_NetISATAPState Represents per-interface ISATAP Configuration settings.
MSFT_NetNatTransitionConfiguration Provides configuration settings for Network Address Translation (NAT).
MSFT_NetNatTransitionInterfaceAssociation Describes the association between NAT64 settings and inbound/outbound interfaces.
MSFT_NetNatTransitionMonitoring Describes current active sessions for NAT.
MSFT_NetSettingData Defines a base class for classes that contain networks settings data, such as MSFT_Net6to4Configuration, MSFT_NetDnsTransitionConfiguration, MSFT_NetDnsTransitionMonitoring, MSFT_NetIPHttpsConfiguration, MSFT_NetISATAPConfiguration, MSFT_NetNatTransitionConfiguration, MSFT_NetNatTransitionMonitoring, and MSFT_NetTeredoConfiguration.
MSFT_NetTeredoConfiguration Represents global Teredo configuration shared across all Teredo interfaces.
MSFT_NetTeredoState Represents per-interface Teredo Configuration settings.
