MSFT_NetAdapterPowerManagement_Offload class

MSFT_NetAdapterPowerManagement_Offload is the base class for various network adapter offloads.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties.

Syntax

[Dynamic, Provider("NetAdapterCim")]
class MSFT_NetAdapterPowerManagement_Offload
{
  uint32 OffloadType;
  uint32 ID;
  uint32 Priority;
  string FriendlyName;
};

Properties

The MSFT_NetAdapterPowerManagement_Offload class has these properties.

FriendlyName

Data type: string

Access type: Read-only

Supplies the friendly name of the offload.

ID

Data type: uint32

Access type: Read-only

Supplies the identifier ID of this offload, assigned by the system.

OffloadType

Data type: uint32

Access type: Read-only

Supplies the type of offload.

Unspecified (0)

IPv4Arp (1)

IPv6NS (2)

RsnRekey (3 )

Priority

Data type: uint32

Access type: Read-only

Supplies the priority of this offload. A lower value implies a higher priority offload.
