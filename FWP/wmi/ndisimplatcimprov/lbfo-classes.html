MsNetImPlatform Provider

This section provides reference information for MsNetImPlatform provider classes implemented in NdisIMPlatCim.dll.

In this section

Topic Description
MSFT_NetImPlatAdapter Represents Network Adapters used in IM Platform, ie. the members and team NICs.
MSFT_NetImPlatProvider Represents an IMPlatform Provider.
MSFT_NetImPlatTeam Represents a team in the IM Platform.
MSFT_NetLbfoProvider Represents an Lbfo Provider.
MSFT_NetLbfoTeam Represents an Lbfo team.
MSFT_NetLbfoTeam_Provider Represents an association between an Lbfo team and its providers.
MSFT_NetLbfoTeam_TeamMember Represents an association between an Lbfo team and its members.
MSFT_NetLbfoTeam_TeamNic Represents an association between an Lbfo Team and its TeamNics.
MSFT_NetLbfoTeamMember Represents members of an Lbfo team.
MSFT_NetLbfoTeamNic Represents a NetAdapter that is a Team NIC for an Lbfo Team.
