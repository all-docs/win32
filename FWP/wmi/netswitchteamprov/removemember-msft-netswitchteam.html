RemoveMember method of the MSFT_NetSwitchTeam class

Removes an existing member from a switch team.

Syntax

uint32 RemoveMember(
  [in] string Name,
  [in] string Team
);

Parameters

Name [in]

The name of the member to remove from the switch team.

Team [in]

The switch team in which to remove a member.
