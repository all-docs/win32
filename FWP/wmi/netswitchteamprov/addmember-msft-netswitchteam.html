AddMember method of the MSFT_NetSwitchTeam class

Adds a new member to an existing switch team.

Syntax

uint32 AddMember(
  [in] string Name,
  [in] string Team
);

Parameters

Name [in]

The name of the member to add.

Team [in]

The switch team in which to add the member.
