MSFT_NetSwitchTeam class

Represents a switch team.

The following syntax is simplified from Managed Object Format (MOF) code and includes all of the inherited properties.

Syntax

[Dynamic, Provider("NetSwitchTeam")]
class MSFT_NetSwitchTeam : MSFT_NetImPlatTeam
{
};

Members

The MSFT_NetSwitchTeam class has these types of members:

  • Methods

Methods

The MSFT_NetSwitchTeam class has these methods.

Method Description
AddMember Adds a new member to an existing switch team.
Create Creates a new switch team with initial members.
RemoveMember Removes an existing member from a switch team.
Rename Renames a switch team.
