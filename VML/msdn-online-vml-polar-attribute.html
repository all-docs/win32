Specifies the center position for polar handles. Read/write. VgVector2D.

Applies To

H (subelement of Handles)

Tag Syntax

Remarks

The default value is a null string. If values are specified, this indicates that the measurement is polar and that the Position attribute defines radius and angle values of the handle instead of x and y values.
