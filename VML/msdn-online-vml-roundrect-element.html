Predefined round rectangle shape.

RoundRect uses the ArcSize attribute.

Remarks

The following is the minimum code needed to produce a round rect.

   <v:roundrect
   fillcolor="green"
   style="top:1;left:1;width:50;height:50">
   </v:roundrect>
