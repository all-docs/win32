Defines the brush color that strokes the path of a shape. Read/write. IVgColor.

Applies To

Shape

Tag Syntax

Script Syntax

element .strokecolor="expression"

expression=element.strokecolor

Remarks

The value is duplicated from the Color attribute of the Stroke element.

VML Standard Attribute

Example

The color of the stroke of the rectangle is red.

   <v:shape id="rect01"
   strokecolor="red" fillcolor="white"
   coordorigin="0 0" coordsize="200 200"
   style="position:relative;top:1;left:1;width:20;height:20"
   path="m 1,1 l 1,200, 200,200, 200,1 x e">
   </v:shape>
