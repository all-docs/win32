ShellFolderViewOC.SetFolderView method

Forwards the events of the specified ShellFolderView object to the corresponding ShellFolderViewOC event handler.

Syntax

iRetVal = ShellFolderViewOC.SetFolderView(
  oShellFolderView
)

Parameters

oShellFolderView [in]

Type: IDispatch*

A ShellFolderView object. Its EnumDone and SelectionChanged events will be forwarded to the corresponding ShellFolderViewOC event handler.
