Accessing the Control Panel in Safe Mode

By default, as of WindowsÂ Vista Control Panel items are not shown when Windows is run in safe mode. To allow a new Control Panel item to be viewed in safe mode, registry values appropriate to the item type can be set. The values are interpreted as follows:

Value Meaning
1 The item should appear in minimal safe mode only.
2 The item should appear in safe mode only if networking is enabled.
3 The item should always appear in any form of safe mode.
