SMC_DISPLAYCHEVRONTIP message

Notifies you that an infotip is about to be displayed for the chevron associated with the item specified by the accompanying SMDATA structure.

SMC_DISPLAYCHEVRONTIP

Parameters

This message has no parameters.

Return value

Return S_OK to display the infotip. Return S_FALSE to prevent the infotip from being displayed.

Remarks

This notification is received by the IShellMenuCallback::CallbackSM method.
