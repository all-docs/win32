FolderItem.Verbs method

Retrieves the item's FolderItemVerbs object. This object is the collection of verbs that can be executed on the item.

Syntax

retVal = FolderItem.Verbs()

Parameters

This method has no parameters.

Return value

Type: FolderItemVerbs**

An object reference to the FolderItemVerbs object.

Examples

The following example uses Verbs to retrieve the FolderItemVerbs object representing the set of verbs that can be executed on the Windows folder. Proper usage is shown for JScript, VBScript, and Visual Basic.

JScript:

<script language="JScript">
    function fnFolderItemVerbsJ()
    {
        var objShell = new ActiveXObject("shell.application");
        var objFolder2;
        var ssfWINDOWS = 36;
        
        objFolder2 = objShell.NameSpace(ssfWINDOWS);
        if (objFolder2 != null)
        {
            var objFolderItem;
            
            objFolderItem = objFolder2.Self;
            if (objFolderItem != null)
            {
                var objItemVerbs;
                
                objItemVerbs = objFolderItem.Verbs();
                if (objItemVerbs != null)
                {
                    // Add code here
                }
            }
        }
    }
</script>
