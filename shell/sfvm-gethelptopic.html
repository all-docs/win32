SFVM_GETHELPTOPIC message

Allows the callback object to specify an HTML Help file and a topic within it. Used by IShellFolderViewCB::MessageSFVCB.

SFVM_GETHELPTOPIC 

    lParam = (LPARAM)(SFVM_HELPTOPIC_DATA*) phtd;

Parameters

phtd [out]

The address of a SFVM_HELPTOPIC_DATA structure that specifies the HTML Help file and topic.
