SMC_REFRESH message

Sends notification that the menus have completely refreshed and you can reset your state.

SMC_REFRESH

Parameters

This message has no parameters.

Return value

Return S_OK.

Remarks

This notification is received by the IShellMenuCallback::CallbackSM method.
