How to Create Cascading Menus with the IExplorerCommand Interface

Another option for adding verbs to a cascading menu is through IExplorerCommand::EnumSubCommands. This method enables data sources that provide their command module commands through the IExplorerCommandProvider interface to use those commands as verbs on a shortcut menu. In WindowsÂ 7 and later, you can provide the same verb implementation using the IExplorerCommand interface as you can with the IContextMenu interface.
