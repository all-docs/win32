CFolderItemsFDF class

CFolderItemsFDF is a collection of FolderItems. It implements the IInsertItem interface.

Members

The CFolderItemsFDF class inherits from CFolderItems but does not have additional members.

Remarks

FDF stands for From Different Folders.

The CFolderItems class enumerates items from multiple folders and uses the IInsertItem interface to insert the items into the collection.
