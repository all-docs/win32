SFVM_BACKGROUNDENUM message

Allows the callback object to request enumeration on a background thread. Used by IShellFolderViewCB::MessageSFVCB.


                SFVM_BACKGROUNDENUM

Parameters

This message has no parameters.

Remarks

In response to this notification, return S_OK to enable background enumeration. By default, the system folder object will display the "flashlight" animation while the enumeration is taking place. To specify a custom animation, handle SFVM_GETANIMATION.
