SMC_SHCHANGENOTIFY message

Notifies you that a change has taken place.

SMC_SHCHANGENOTIFY
    lParam = (LPARAM) (PSMCSCHANGENOTIFYSTRUCT) psmc

Parameters

psmc

A pointer to an SMCSHCHANGENOTIFYSTRUCT structure with information on the change.

Return value

Return S_OK.

Remarks

This notification is received by the IShellMenuCallback::CallbackSM method.
