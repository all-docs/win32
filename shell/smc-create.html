SMC_CREATE message

Notifies you that a menu band has been created.

SMC_CREATE 
    lParam = (LPARAM) (LPSMDATA) psmd

Parameters

psmd

A pointer to the pvUserData member of an SMDATA structure.

Return value

Return S_OK.

Remarks

This notification is received by the IShellMenuCallback::CallbackSM method.
