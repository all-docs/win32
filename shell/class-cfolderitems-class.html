CFolderItems class

CFolderItems is a collection of FolderItems. It implements the following interfaces: IPersistFolder, IObjectWithSite, FolderItems3, IObjectSafety, IDispatch.

Remarks

The CFolderItems class only enumerates items from a specific folder.
