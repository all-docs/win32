Provider (DataRoamingPartners) Element

The Provider (DataRoamingPartners) element identifies the name and provider ID for a cellular network.

This element has the following child elements.

  • ProviderID
  • ProviderName

This element can be defined multiple times within the DataRoamingPartners element. It must be defined at least once.

<xs:element name="Provider"
    type="providerType"
 />

The Provider element is defined by the DataRoamingPartners element.
