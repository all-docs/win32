providerNameType Simple Type

The providerNameType simple type defines a string type for the ProviderName element in the Mobile Broadband profile. This string is at least one character long and at most 20 characters long.

<xs:simpleType name="providerNameType">
    <xs:restriction
        base="normalizedString"
    >
        <xs:minLength
            value="1"
         />
        <xs:maxLength
            value="20"
         />
    </xs:restriction>
</xs:simpleType>
