Mobile Broadband API Structures

The Mobile Broadband API defines and uses the following structures.

  • MBN_CONTEXT
  • MBN_DEVICE_SERVICE
  • MBN_INTERFACE_CAPS
  • MBN_PIN_INFO
  • MBN_PROVIDER
  • MBN_PROVIDER2
  • MBN_SMS_FILTER
  • MBN_SMS_STATUS_INFO
