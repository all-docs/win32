Attributes and Elements

Attributes

Child Elements

Child Element Description
AdminEnable Specifies whether the profile is enabled administratively.This is a new element for v4.
AdminRoamControl Specifies whether the profile is administratively roam controlled. This element is new for v4. The value of this element is a roamControlType value. This is an optional element; if no value is specified, then AllRoamAllowed is the default.
ApnID An APN ID associated with this profile.This element is new in v4, and it is optional.
Context Specifies the parameters required to establish a data connection.
Name The name of the profile. For more information, see the documentation for the v1 Name element.
OemConnectionId The OEM connection ID for the modem DM configuration.
ProfileCreationType (in ModemDMConfigProfile) Specifies how this modem DM profile was created.This value is used to decide whether a user can delete the profile. Users can only delete UserProvisioned profiles.
RoamApplicability Specifies that this profile is active only when the current roaming condition is the one specified. Otherwise, the profile is not applicable, and cannot be used to activate a Packet Data Protocol (PDP) context. The value of this element must be a valid roamApplicabilityType value.
SimIccID The SIM Identifcation number for GSM devices. For more details , see the documentation for the v1 SimIccID element.
