Mobile Broadband Profile Schema Reference

The Mobile Broadband Profile Schema defines a Mobile Broadband Profile.

Profiles store Mobile Broadband connection related user and network configuration parameters. They also store the user preferences for a connection.

The root element of a Mobile Broadband profile is the MBNProfile element. Each profile has exactly one root element. The Mobile Broadband Profile Schema strictly enforces the order of the nodes. Nodes specified in a profile must adhere to the Mobile Broadband Profile Schema and appear in the order prescribed in the schema definition. For example, UserLogonCred must be specified immediately after AccessString.
