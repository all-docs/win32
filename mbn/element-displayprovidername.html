DisplayProviderName

The DisplayProviderName element is an optional providerNameType that contains the network connection name to display in the Windows Connection Manager. This name will only be displayed if the subscriber is on a home network and not roaming. The roaming network name is displayed based on information from the mobile broadband device.

Element hierarchy

<DisplayProviderName>

Syntax

<DisplayProviderName>

  providerNameType

</DisplayProviderName>

Attributes and Elements

None.

Child Elements

None.

Parent Elements

This outermost (document) element may not be contained by any other elements.
