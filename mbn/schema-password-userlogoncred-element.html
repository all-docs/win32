Password (UserLogonCred) Element

The Password (UserLogonCred) element specifies the password used to authenticate a user.

The element is a string of up to 255 characters.

This element is optional.

<xs:element name="Password"
    type="string"
 />

The Password element is defined by the UserLogonCred element.
