Mobile Broadband Profile Schema v1 Elements

A Mobile Broadband v1 Profile is comprised of the following elements. All of the named elements are in the namespace https://www.microsoft.com/networking/WWAN/profile/v1.

  • MBNProfile
    • AutoConnectOnInternet (MBNProfile)
    • ConnectionMode (MBNProfile)
    • Context (MBNProfile)
      • AccessString (contextType)
      • AuthProtocol (contextType)
      • Compression (contextType)
      • UserLogonCred (contextType)
        • IgnorePassword (UserLogonCred)
        • Password (UserLogonCred)
        • UserName (UserLogonCred)
    • DataRoamingPartners (MBNProfile)
      • Provider (DataRoamingPartners)
        • ProviderID (providerType)
        • ProviderName (providerType)
    • Description (MBNProfile)
    • HomeProviderName (MBNProfile)
    • ICONFilePath (MBNProfile)
    • IsDefault (MBNProfile)
    • Name (MBNProfile)
    • ProfileCreationType (MBNProfile)
    • SimIccID (MBNProfile)
    • SubscriberID (MBNProfile)
