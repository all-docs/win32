FeatureInfo object

The FeatureInfo object contains information regarding the targeted feature and is created from the Session object using the FeatureInfo Method.

The FeatureInfo object has these types of members:

  • Properties

Properties

The FeatureInfo object has these properties.

Property Access type Description
Attributes Read/write Returns the value for the feature in the Attributes column of the Feature table.
Description Returns the description of the feature.
Title Read-only Returns the title of the feature.
