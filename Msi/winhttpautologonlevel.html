WinHttpAutoLogonLevel

The automatic logon (auto-logon) policy determines when it is acceptable for WinHTTP to include the default credentials in a request to the server.

You can set this value to L for WINHTTP_AUTOLOGON_SECURITY_LEVEL_LOW, set to M for WINHTTP_AUTOLOGON_SECURITY_LEVEL_MEDIUM, and set to H for WINHTTP_AUTOLOGON_SECURITY_LEVEL_HIGH. The default level is WINHTTP_AUTOLOGON_SECURITY_LEVEL_MEDIUM. For more information about automatic logon policy, see the section for Authentication in WinHTTP.
