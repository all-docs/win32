MSIUSEREALADMINDETECTION property

Set the MSIUSEREALADMINDETECTION property to 1 to request that the installer use actual user information when setting the AdminUser property. When running on WindowsÂ Vista, the Privileged and AdminUser are the same. Authors should use the Privileged property in new packages. Legacy packages that require distinct Privileged and AdminUser properties can restore the difference by setting the MSIUSEREALADMINDETECTION property.
