Patch.MediaDisks property

The MediaDisks property enumerates all the media disks for this product instance. This property calls the MsiSourceListEnumMediaDisks. Returns the disk information as array of Record objects.

This property is read-only.

Syntax

propVal = Patch.MediaDisks

Property value

Remarks

In each record the first field contains the disk Id, the second field contains the volume label and the third field contains the disk prompt registered for the disk.
