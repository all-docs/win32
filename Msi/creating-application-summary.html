Creating Application Summary property

The Creating Application Summary property conveys which application created the installer database. In general, the value for this summary property is the name of the software that is used to author this database.

Requirements

Requirement Value
Version Windows Installer 5.0 on Windows Server 2012, Windows 8, Windows Server 2008 R2 or Windows 7. Windows Installer 4.0 or Windows Installer 4.5 on Windows Server 2008 or Windows Vista. Windows Installer on Windows Server 2003 or Windows XP
