CCP_DRIVE property

The CCP_DRIVE property is set to the root path of the removable volume that is to be searched by RMCCPSearch. The RMCCPSearch action uses file signatures to validate that qualifying products are installed on a system before performing an upgrade installation.

This property is typically set via the user interface or a custom action. This property should be set before the RMCCPSearch action is run.

Requirements

Requirement Value
Version Windows Installer 5.0 on Windows Server 2012, Windows 8, Windows Server 2008 R2 or Windows 7. Windows Installer 4.0 or Windows Installer 4.5 on Windows Server 2008 or Windows Vista. Windows Installer on Windows Server 2003 or Windows XP. See the Windows Installer Run-Time Requirements for information about the minimum Windows service pack that is required by a Windows Installer version.
