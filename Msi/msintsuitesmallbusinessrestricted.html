MsiNTSuiteSmallBusinessRestricted property

On WindowsÂ 2000 and later operating systems, the installer sets the MsiNTSuiteSmallBusinessRestricted property to 1 if Microsoft Small Business Server is installed with the restrictive client license in force. The installer sets this property to 1 only if the VER_SUITE_SMALLBUSINESS_RESTRICTED flag is set in the OSVERSIONINFOEX structure. Otherwise the installer does not set this property.
