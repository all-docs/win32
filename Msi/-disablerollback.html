DISABLEROLLBACK property

Setting the DISABLEROLLBACK property disables the rollback option for the current configuration. Set this property to 1 to prevent the installer from generating a rollback script and saving copies of deleted files during the installation.

Remarks

By default, rollback is enabled.

Requirements

Requirement Value
Version Windows Installer 5.0 on Windows Server 2012, Windows 8, Windows Server 2008 R2 or Windows 7. Windows Installer 4.0 or Windows Installer 4.5 on Windows Server 2008 or Windows Vista. Windows Installer on Windows Server 2003, Windows XP, and Windows 2000. See the Windows Installer Run-Time Requirements for information about the minimum Windows service pack that is required by a Windows Installer version.
