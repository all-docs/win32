ConfigurableItem.HelpLocation property

The HelpLocation property returns the value from the HelpLocation column of the ModuleConfiguration table.

This property is read-only.

Syntax

propVal = ConfigurableItem.HelpLocation

Property value

C++

See get_HelpLocation function.
