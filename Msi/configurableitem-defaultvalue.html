ConfigurableItem.DefaultValue property

The DefaultValue property of the ConfigurableItem object returns the value from the DefaultValue column of the ModuleConfiguration table.

This property is read-only.

Syntax

propVal = ConfigurableItem.DefaultValue

Property value

C++

See get_DefaultValue function
