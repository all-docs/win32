InstallSFPCatalogFile Action

The InstallSFPCatalogFile action installs the catalogs used by WindowsÂ Me for Windows File Protection. InstallSFPCatalogFile queries the Component table, File table, FileSFPCatalog table and SFPCatalog table. Catalogs are installed if they are associated with a file in a component that is set for local installation or if they are associated with a file being repaired in a locally installed component.

Sequence Restrictions

The InstallSFPCatalogFile action must be sequenced before InstallFiles and after CostFinalize.

ActionData Messages

Field Description of action data
[1] Name of the catalog being installed.
[2] Name of the catalog this catalog's installation depends upon.
