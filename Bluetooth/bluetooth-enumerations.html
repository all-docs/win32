Bluetooth Enumerations

The following enumerations support the Microsoft Bluetooth implementation.

Section Content
BLUETOOTH_AUTHENTICATION_REQUIREMENTS Defines the 'Man in the Middle' protection methods for authentication.
BLUETOOTH_AUTHENTICATION_METHOD Defines the supported authentication types during device pairing.
BLUETOOTH_IO_CAPABILITY Defines the input/output capabilities of a Bluetooth Device.
