Bluetooth and WSAQUERYSET

The WSAQUERYSET structure is used in operations including device inquiry, service inquiry, and setting the service.

The following topics describe the use of the WSAQUERYSET structure in each of these operations:

  • Bluetooth and WSAQUERYSET for Device Inquiry
  • Bluetooth and WSAQUERYSET for Service Inquiry
  • Bluetooth and WSAQUERYSET for Set Service

For more information and a complete WSAQUERYSET reference or for other Windows Sockets elements, see the Windows Sockets section of the Platform Software Development Kit (SDK).
