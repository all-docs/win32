/winmd switch

The /winmd switch specifies the name of the output metadata (.winmd) file.

midlrt /winmd metadata_filename

Switch Options

metadata_filename

Specifies the name of the metadata file.

Remarks

Examples

midlrt /winmd file.winmd

Requirements

Requirement Value
Client Windows 8
Server Windows Server 2012
