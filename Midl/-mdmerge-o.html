/o switch (MDMERGE)

The /o switch specifies the output directory for composed metadata files.

mdmerge /o output_directory

Switch Options

output_directory

Specifies the directory that contains the merged metadata files.

Remarks

Examples

mdmerge /o output_directory

Requirements

Requirement Value
Client Windows 8
Server Windows Server 2012
