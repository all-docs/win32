Compiler Errors

The following error messages are generated during MIDL compilation:

Note: The Mktyplib.exe tool is obsolete. Use the MIDL compiler instead.

Note: The Mktyplib.exe tool is obsolete. Use the MIDL compiler instead.

Note: The Mktyplib.exe tool is obsolete. Use the MIDL compiler instead.

Note: Note that as of Windows 2000 release, 64-bit TLB files are supported by OLE Automation by converting 32-bit TLB information at run time. Therefore, only 32-bit TLB generation is supported by MIDL.

If MIDL is being used just to generate a header file, the /notlb switch will suppress generation of the TLB file.
