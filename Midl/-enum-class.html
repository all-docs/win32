/enum_class switch

The /enum_class switch enables use of the C++ enum class construct.

midlrt /enum_class

Switch Options

This switch has no parameters.

Remarks

Examples

midlrt /enum_class

Requirements

Requirement Value
Client Windows 8
Server Windows Server 2012
