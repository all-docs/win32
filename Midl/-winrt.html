/winrt switch

The /winrt switch enables Windows Runtime semantics.

midlrt /winrt

Switch Options

This switch has no parameters.

Remarks

Examples

midlrt /winrt

Requirements

Requirement Value
Client Windows 8
Server Windows Server 2012
