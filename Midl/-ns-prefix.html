/ns_prefix switch

The /ns_prefix switch prepends the "ABI" prefix to all types.

midlrt /ns_prefix

Switch Options

This switch has no parameters.

Remarks

Examples

midlrt /ns_prefix

Requirements

Requirement Value
Client Windows 8
Server Windows Server 2012
