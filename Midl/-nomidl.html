/nomidl switch

The /nomidl switch suppresses running MIDL.EXE after processing the Windows Runtime IDL file.

midlrt /nomidl

Switch Options

This switch has no parameters.

Remarks

Examples

midlrt /nomidl

Requirements

Requirement Value
Client Windows 8
Server Windows Server 2012
