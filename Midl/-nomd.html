/nomd switch

The /nomd switch specifies that no metadata (.winmd) file is generated.

midlrt /nomd

Switch Options

This switch has no parameters.

Remarks

Examples

midlrt /nomd

Requirements

Requirement Value
Client Windows 8
Server Windows Server 2012
