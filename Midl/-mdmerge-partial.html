/partial switch

The /partial switch specifies that unresolved types are resolved against metadata files that are listed by the /metadata_dir switch.

mdmerge /partial

Switch Options

This switch has no parameters.

Remarks

Examples

mdmerge /partial

Requirement Value
Client Windows 8
Server Windows Server 2012
