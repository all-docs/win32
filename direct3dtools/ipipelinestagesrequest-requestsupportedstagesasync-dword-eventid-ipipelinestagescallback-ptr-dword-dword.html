IPipeLineStagesRequest::RequestSupportedStagesAsync method

An asynchronous request to get a list of stages used for the specified frame and event.

Syntax

HRESULT RequestSupportedStagesAsync(
   DWORD                     frameNumber,
   EventID                   eventID,
   IPipeLineStagesCallback * requestCallback,
   DWORD                     requestCookie,
   DWORD                     progressIntervalMsecs
);

Parameters

frameNumber
The specified frame.

eventID
The specified event.

requestCallback
The address of callback used to notify the host of results.

requestCookie
A cookie that uniquely identifies the request, and can be used to signal for it to be cancelled.

progressIntervalMsecs
Not used.

Return value

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
