G (Network Monitor)

generic formatter

A property data formatter that Network Monitor provides. The generic formatter can format the data that is displayed in the details pane of the Network Monitor UI. When you want Network Monitor to use the generic formatter, you must specify the FormatPropertyInstance function in InstanceData member of the PROPERTYINFO structure of a property.
