IESP::Resume method

The Resume method restarts a paused capture.

Syntax

HRESULT STDMETHODCALLTYPE Resume();

Parameters

This method has no parameters.

Return value

If the method is successful, the return value is NMERR_SUCCESS.

If the method is unsuccessful, the return value is one of the following error codes:

Return code Description
NMERR_CAPTURE_NOT_PAUSED The capture is not paused. Call IESP::Pause to pause the capture.
NMERR_NOT_CONNECTED The NPP is not connected to the network. Call IESP::Connect to connect to the network.
NMERR_NOT_ESP The NPP is connected to the network, but not with the IESP::Connect method.

Remarks

While the capture is in a paused state, new data is not added to the current capture file until IESP::Resume is called to restart the capture. When Pause and Resume are used to stop and restart the capture, all captured information is put in the same capture file.

When using Pause and Resume to control the capture, Network Monitor continues to add conversation statistics to the existing statistics for the current capture.

To stop the capture, call IESP::Stop.
