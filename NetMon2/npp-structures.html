NPP Structures

This section describes the NPP structures used by NPP methods. These structures are used to retrieve statistics, provide system status and statistical information, and indicate which computers are running Network Monitor. The following tables describes these structures.

NPP Structures Description
SESSIONSTATS Provides session information when conversation statistics are retrieved.
STATIONSTATS Provides statistics about a specific station.
STATISTICS Provides network statistics when total statistics are retrieved and when the current capture has paused or stopped.
QUERYTABLE Indicates which computers are using Network Monitor.
STATIONQUERY Provides information about a computer that is using Network Monitor. This structure is used by the NPP QUERYTABLE structure.
NETWORKSTATUS Provides current network status in terms of system state.
