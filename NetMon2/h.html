H (Network Monitor)

handoff set

A protocol set that follows a protocol. A parser uses the handoff set of a protocol if the parser can detect the next protocol from the data in the capture. See also follow set

handoff table

A table that contains the parser handoff set. You can use a handoff table to access the handoff set information â€” without accessing the parser-defined INI file. The handoff set information is stored in the INI file of each parser. See also handoff set

hex pane

The pane in the Network Monitor frame viewer window that shows the hexadecimal bytes transmitted in frames that are included in a capture. By default, the hex pane appears at the bottom of the window.
