IRTC::Resume method

The Resume method restarts a paused capture.

Syntax

HRESULT STDMETHODCALLTYPE Resume();

Parameters

This method has no parameters.

Return value

If the method is successful the return value is NMERR_SUCCESS.

If the method is unsuccessful, the return value is one of the following error codes:

Return code Description
NMERR_CAPTURE_NOT_PAUSED The capture is not paused. Call IRTC::Pause to pause the capture.
NMERR_NOT_CONNECTED The NPP is not connected to the network. Call IRTC::Connect to connect the NPP to the network.
NMERR_NOT_REALTIME The NPP is connected to the network but not with the IRTC::Connect method.

Remarks

While the capture is in a paused state, new data is not captured until a call to the IRTC::Resume method restarts the capture.

When using the Pause and Resume methods to control the capture, Network Monitor continues to add conversation statistics to the existing statistics for the current capture.

To stop the capture, call the IRTC::Stop method.
