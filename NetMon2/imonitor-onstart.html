IMonitor::OnStart method

The OnStart method is an optional method that can be implemented by the monitor. The MCSVC calls this method to request that the monitor start the capture.

Syntax

HRESULT OnStart(
  [in] IUnknown *pUnkNPP
);

Parameters

pUnkNPP [in]

An IUnknown pointer to the IRTC interface. The monitor can use this interface to obtain statistics that can be used to determine if the capture should be started.

Return value

If the method is successful, the return value is S_OK (which is the same as NOERROR). When S_OK is returned, the MCSVC calls the IRTC::Start method.

If the method is unsuccessful, the return value is an error code. The MCSVC will not start the capture if any error code is returned.

Remarks

The MCSVC calls this method before the NPP's IRTC::Start method is called to start the capture.
