Monitor Interfaces

The following COM interfaces are used to develop monitors.

Interface Description
IMonitor Provides methods that control monitor operation. This interface must be implemented by the monitor.
IMonitorEventer Provides methods used to send events. This interface is provided by the Monitor Control Service (MCSVC).
