Writing a Protocol Parser

This topic contains information about writing a protocol parser, and includes examples of the export functions that the parser DLL must implement.

The following topics are included in this section.

Term Description
Programming Considerations Identifies the basic programming tips to help you write a parser DLL.
Implementing Parser Export Functions Provides examples of implementing export functions.
Formatting Displayed Data Identifies the process of formatting data that is displayed in the details pane of the Network Monitor UI.
Specifying a Follow Set Identifies the process of specifying a follow set, and shows you how Network Monitor accesses the follow set of a parser to determine the next protocol.
Specifying a Handoff Set Identifies the process of specifying a handoff set, and shows you how the parser accesses its handoff set to get a handle to the next protocol.
