IRTC::Disconnect method

The Disconnect method disconnects the NPP from the network.

Syntax

HRESULT STDMETHODCALLTYPE Disconnect();

Parameters

This method has no parameters.

Return value

If the method is successful, the return value is NMERR_SUCCESS.

If the method is unsuccessful, the return value is one of the following error codes:

Return code Description
NMERR_CAPTURING The NPP is capturing data. You cannot disconnect from the network while the data capture is in progress.
NMERR_NOT_CONNECTED The NPP is not connected to the network.
NMERR_NOT_REALTIME The NPP is connected to the network but not with the IRTC::Connect method.

Remarks

This method cannot be called when the NPP is capturing data. You must call the IRTC::Stop method before calling IRTC::Disconnect.
