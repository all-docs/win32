IStats::Pause method

The Pause method temporarily stops the current capture.

Syntax

HRESULT STDMETHODCALLTYPE Pause();

Parameters

This method has no parameters.

Return value

If the method is successful, the return value is NMERR_SUCCESS.

If the method is unsuccessful, the return value is one of the following error codes:

Return code Description
NMERR_CAPTURE_PAUSED The capture is already paused.
NMERR_NOT_CAPTURING The NPP is not capturing data. Call the IStats::Start method to start the capture.
NMERR_NOT_CONNECTED The NPP is not connected to the network. Call the IStats::Connect method to connect the NPP to the network.
NMERR_NOT_STATS_ONLY The NPP is connected to the network but not with the IStats::Connect method.

Remarks

While the capture is paused, new frames are not captured until a call to the IStats::Resume method restarts the capture.

When you use the IStats::Pause and IStats::Resume methods to control the capture, Network Monitor continues to add conversation statistics whenever the capture is running.

To restart the capture call IStats::Resume. To stop the capture, call IStats::Stop.
