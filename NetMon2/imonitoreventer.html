IMonitorEventer interface

The IMonitorEventer interface provides methods for submitting events and allocating and freeing monitor resources.

Members

The IMonitorEventer interface inherits from the IUnknown interface. IMonitorEventer also has these types of members:

  • Methods

Methods

The IMonitorEventer interface has these methods.

Method Description
FreeEventData Frees space allocated for monitor data.
GetEventData Allocates space for monitor data.
SendNMEvent Submits events to WMI.
