BLOB Structures

The BLOB Helper API includes the following structures.

Structure Description
ADDRESS Contains a single, supported address.
ADDRESSPAIR Constructs a capture filter.
ADDRESSTABLE Specifies a table of address pairs.
ANDEXP Specifies an array of pattern matches.
BLOB_TABLE Specifies an array of BLOBs.
CAPTUREFILE_HEADER_VALUES Defines the structure of the Network Monitor header file.
CAPTUREFILTER Specifies capture filter data.
EXPRESSION Specifies an expression of peers structured in logical AND statements.
FRAME Specifies a frame.
FRAME_DESCRIPTOR Specifies details of raw frames.
FRAMETABLE Specifies a set of frames.
GENERIC_PORT Specifies a port number used in IP or IPX protocols.
IPX_ADDRESS Specifies an IPX address.
NETWORKINFO Specifies a NIC.
PATTERNMATCH Specifies pattern elements used to evaluate frames.
TRIGGER Specifies that an event occur at a specific time.
UPDATE_EVENT Updates events.
VINES_IP_ADDRESS Specifies an address on a Vines network.
