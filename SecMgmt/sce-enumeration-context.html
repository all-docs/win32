SCE_ENUMERATION_CONTEXT

The SCE_ENUMERATION_CONTEXT data type is an opaque handle provided by the Security Configuration tool set. It is used by the PFSCE_QUERY_INFO function to navigate through the security database.

typedef ULONG SCE_ENUMERATION_CONTEXT, *PSCE_ENUMERATION_CONTEXT;
