SCESVC_HANDLE

The SCESVC_HANDLE data type is an opaque handle provided by the Security Configuration tool set. It is used by methods of the ISceSvcAttachmentData and ISceSvcAttachmentPersistInfo interfaces to pass information between the Security Configuration snap-in and the snap-in extension.

typedef PVOID SCESVC_HANDLE;
