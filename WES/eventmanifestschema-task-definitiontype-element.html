task (DefinitionType) Element

[Beginning with the message compiler that ships with the Windows 7 version of the Windows SDK, this element is no longer available.]

Defines a task-specific event that your provider can log.

    type="TaskEventDefinitionType"
 />

The task element is defined by the DefinitionType complex type.
