Query Schema Complex Types

The following are the complex types that the Query schema defines.

Complex type Description
QueryListType Defines a list of queries that can contain a set of selectors and suppressors that are used to include events in or exclude events from the result set.
QueryType Defines a set of selector and suppressor queries that are used to include events in or exclude events from the result set.
