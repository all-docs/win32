Event Schema Simple Types

The following are the simple types that the Event schema defines.

Simple type Description
GUIDType Defines a globally unique identifier type, in Registry format.
HexInt32Type Defines a 1-byte hexadecimal type.
HexInt64Type Defines an 8-byte hexadecimal type.
