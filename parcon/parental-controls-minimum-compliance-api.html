Parental Controls Minimum Compliance API

The minimum compliance API implements simple interface methods for applications to obtain basic state information on a given parentally controlled user account, its web restrictions settings, and games restrictions settings.

  • IWindowsParentalControls
  • IWPCGamesSettings
  • IWPCSettings
  • IWPCWebSettings
