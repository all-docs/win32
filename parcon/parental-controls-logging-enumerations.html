Parental Controls Logging Enumerations

Parental Controls provides the following logging enumerations.

Flag enumerations:

  • WPCFLAG_IM_FEATURE
  • WPCFLAG_IM_LEAVE_FLAG
  • WPCFLAG_ISBLOCKED
  • WPCFLAG_LOGOFF_TYPE

Enumerations that are defined in Wpcevent.h for parameters to be passed to the Windows Event Log publishing API for each event type:

  • WPC_ARGS_CONVERSATIONINITEVENT
  • WPC_ARGS_CONVERSATIONJOINEVENT
  • WPC_ARGS_CONVERSATIONLEAVEEVENT
  • WPC_ARGS_CUSTOMEVENT
  • WPC_ARGS_EMAILCONTACTEVENT
  • WPC_ARGS_EMAILRECEIVEDEVENT
  • WPC_ARGS_EMAILSENTEVENT
  • WPC_ARGS_FILEDOWNLOADEVENT
  • WPC_ARGS_GAMESTARTEVENT
  • WPC_ARGS_IMCONTACTEVENT
  • WPC_ARGS_IMFEATUREEVENT
  • WPC_ARGS_MEDIADOWNLOADEVENT
  • WPC_ARGS_MEDIAPLAYBACKEVENT
  • WPC_ARGS_SAFERAPPBLOCKED
  • WPC_ARGS_SETTINGSCHANGEEVENT
  • WPC_ARGS_URLVISITEVENT
  • WPC_MEDIA_EXPLICIT_TYPE
  • WPC_MEDIA_TYPE
