ID3DX11EffectPass::GetAnnotationByIndex method

Get an annotation by index.

Syntax

ID3DX11EffectVariable* GetAnnotationByIndex(
   UINT Index
);

Parameters

Index

Type: UINT

A zero-based index.

Return value

Type: ID3DX11EffectVariable*

A pointer to an ID3DX11EffectVariable.

Remarks
