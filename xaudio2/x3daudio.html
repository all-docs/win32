X3DAudio

X3DAudio is an API used in conjunction with XAudio2 to create the illusion of a sound coming from a point in 3D space.

In this section

Topic Description
X3DAudio Overview X3DAudio is an API used with XAudio2 to position sound in 3D space to create the illusion of sound coming from a point in space relative to the position of the camera.
