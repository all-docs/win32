FXEQ Constants

Defines values for use with the FXEQ_PARAMETERS structure.

Constant Description
FXEQ_MIN_FRAMERATE Minimum frame rate of FXEQ input.
FXEQ_MAX_FRAMERATE Maximum frame rate of FXEQ input.
FXEQ_MIN_FREQUENCY_CENTER Minimum value for the FrequencyCenter members.
FXEQ_MAX_FREQUENCY_CENTER Maximum value for the FrequencyCentermembers.
FXEQ_DEFAULT_FREQUENCY_CENTER_0 Default value for the FrequencyCenter0 member.
FXEQ_DEFAULT_FREQUENCY_CENTER_1 Default value for the FrequencyCenter1 member.
FXEQ_DEFAULT_FREQUENCY_CENTER_2 Default value for the FrequencyCenter2 member.
FXEQ_DEFAULT_FREQUENCY_CENTER_3 Default value for the FrequencyCenter3 member.
FXEQ_MIN_GAIN Minimum value for the Gain members.
FXEQ_MAX_GAIN Maximum value for the Gain members.
FXEQ_DEFAULT_GAIN Default value for the Gain members.
FXEQ_MIN_BANDWIDTH Minimum value for the Bandwidth members.
FXEQ_MAX_BANDWIDTH Maximum value for the Bandwidth members.
FXEQ_DEFAULT_BANDWIDTH Default value for the Bandwidth members.
