Authorization Data Types

The following data types are used with authorization applications.

In this section

Topic Description
ACCESS_MASK Defines standard, specific, and generic rights. These rights are used in access control entries (ACEs) and are the primary means of specifying the requested or granted access to an object.
SECURITY_DESCRIPTOR_CONTROL Set of bit flags that qualify the meaning of a security descriptor or its components.
SECURITY_INFORMATION Identifies the object-related security information being set or queried.
