Collections (Authorization)

The Authorization Manager API provides interfaces that represent collections of other objects. For example, the IAzRoles interface represents a collection of IAzRole objects.

The other interfaces that represent collections are IAzApplications, IAzApplicationGroups, IAzOperations, IAzScopes, and IAzTasks. Each of these interfaces provides properties that facilitate enumerating objects in a collection.
