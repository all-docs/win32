MSFT Classes

MSFT classes are specific to the Windows implementation of the Common Information Model (CIM).

Objects can be divided into several main categories:

  • Policy provider classes
  • WMI troubleshooting classes

The following MSFT classes are defined in System.mof but are not implemented by WMI.

  • MSFT_WmiCoreEvent
  • MSFT_WmiCoreLogoffEvent
  • MSFT_WmiCoreLogonEvent
  • MSFT_WmiCoreObject
  • MSFT_WmiCoreStatus
  • MSFT_WmiCoreUser
  • MSFT_WmiTaskFailure
