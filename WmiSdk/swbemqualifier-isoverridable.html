SWbemQualifier.IsOverridable property

The IsOverridable property of the SWbemQualifier object is a Boolean value that indicates if this qualifier can be overridden when propagated.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read/write.

Syntax

SWbemQualifier.IsOverridable As Boolean

Property value
