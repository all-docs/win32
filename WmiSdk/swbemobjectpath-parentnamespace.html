SWbemObjectPath.ParentNamespace property

The ParentNamespace property of the SWbemObjectPath object contains the name of the parent namespace that is part of the object path. This property is read-only.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read-only.

SWbemObjectPath.ParentNamespace As String
