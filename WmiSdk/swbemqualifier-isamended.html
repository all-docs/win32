SWbemQualifier.IsAmended property

The IsAmended property of the SWbemQualifier object is a Boolean value that indicates if this qualifier has been localized using a merge operation.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read-only.

Syntax

SWbemQualifier.IsAmended As Boolean
