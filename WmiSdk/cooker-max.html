COOKER_MAX

The COOKER_MAX counter type formula identifies the largest value from a set of observations of a property in a Win32_PerfRawData instance. This calculated maximum value is placed in a property in a Win32_PerfFormattedData instance. This counter type is defined only within WMI, and it is not available to the performance monitoring technologies, such as Performance Counters Version 6.0.

For more information about the counter type formula, see Counter Types.

For more information about high-performance providers and scripting, see WMI Performance Counter Types.
