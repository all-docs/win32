SWbemQualifier.Value property

The Value property of the SWbemQualifier object contains the variant value of this qualifier. This is the default property of this object.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read/write.

Syntax

SWbemQualifier.Value As Variant

Property value
