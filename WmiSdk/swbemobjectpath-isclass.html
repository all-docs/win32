SWbemObjectPath.IsClass property

The IsClass property of the SWbemObjectPath object is a Boolean value that indicates if this path represents a class. This property is read-only.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read-only.

SWbemObjectPath.IsClass As Boolean
