temporary consumer

A consumer that receives event notifications while the consumer is active. See also, permanent consumers.

timer event

An event that either occurs one time at a specific time of day (absolute timer event), or repeatedly at regular intervals (interval timer event). A timer event differs from other events, because it requires explicit action by the consumer for it to occur. Notifications of timer events are sent by the CIM Object Manager. A timer event is represented by an instance of the system class __TimerEvent.
