COOKER_RANGE

The COOKER_RANGE counter type formula shows the difference between the Min and Max values for a set of raw observations of a property in a Win32_PerfRawData instance. This counter type is defined only within WMI, and it is not available to the performance monitoring technologies, such as Performance Counters Version 6.0.

For more information about the counter type formula, see Counter Types.

For more information about high-performance providers and scripting, see WMI Performance Counter Types.
