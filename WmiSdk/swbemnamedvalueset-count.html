SWbemNamedValueSet.Count property

Use the Count property of the SWbemNamedValueSet object to determine how many items are in the collection. This property is read-only.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read-only.

SWbemNamedValueSet.Count As Integer
