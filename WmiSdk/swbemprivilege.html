SWbemPrivilege object

The SWbemPrivilege object represents a single privilege. This object cannot be created by the VBScript CreateObject call.

Members

The SWbemPrivilege object has these types of members:

  • Properties

Properties

The SWbemPrivilege object has these properties.

Property Access type Description
Displayname Read-only Display name of this privilege.
Identifier Read/write Integer that represents the privilege that is being set or retrieved.
IsEnabled Read/write Boolean value that indicates if this privilege is enabled.
Name Read-only Name of this privilege.
