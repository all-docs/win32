CHString::CHString constructors

[The CHString class is part of the WMI Provider Framework which is now considered in final state, and no further development, enhancements, or updates will be available for non-security related issues affecting these libraries. The MI APIs should be used for all new development.]

Each of the following constructors initializes a new CHString object with the specified data.

Overload list

Constructor Description
CHString() Creates an empty CHString object.
CHString(LPCSTR) Initializes with the LPCSTR value.
CHString(LPCWSTR) Initializes with the LPCWSTR value.
CHString(WCHAR,int) Initializes with int copies of the WCHAR value.
CHString(LPCWSTR,int) Initializes with int copies of the LPCWSTR value.
CHString(const CHString&) Replicates the CHString parameter.
CHString(const unsigned char*) Initializes with the value pointed to by char*.
