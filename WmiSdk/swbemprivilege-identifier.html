SWbemPrivilege.Identifier property

The Identifier property of an SWbemPrivilege object is an WbemPrivilegeEnum integer that represents the privilege that is being set or retrieved. This property is read-only.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read/write.

Syntax

SWbemPrivilege.Identifier As Integer
