SWbemNamedValue.Name property

The Name property of the SWbemNamedValue object is a unique case-insensitive string that is used when getting an SWbemNamedValue item. This property is read-only.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read-only.

Syntax

SWbemNamedValue.Name As String
