SWbemPrivilegeSet.Count property

The Count property of the SWbemPrivilege object contains the number of items in an SWbemPrivilegeSet collection. This property is read-only.

For an explanation of this syntax, see Document Conventions for the Scripting API.

This property is read-only.

Syntax

SWbemPrivilegeSet.Count As Integer
