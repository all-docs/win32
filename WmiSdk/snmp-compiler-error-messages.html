SNMP Compiler Error Messages

This section describes the various error conditions that can result when the Simple Network Management Protocol (SNMP) information module compiler parses a Management Information Base (MIB) module.

The error messages have three severity levels:

  • Fatal errors
  • Warnings
  • Information messages

The severity level is shown in angle brackets at the beginning of the error message description. The following is a list of WMI SNMP provider errors:

  • Errors 101 through 110
  • Errors 201 through 210
  • Errors 211 through 220
  • Errors 221 through 230
  • Errors 231 through 240
  • Errors 281 through 290
  • Errors 1001 through 1010
  • Errors 1011 through 1020
  • Errors 1021 through 1030
  • Errors 1031 through 1040
  • Errors 1041 through 1050
  • Errors 1051 through 1060
  • Errors 1061 through 1070
  • Errors 1071 through 1080
  • Errors 1081 through 1090
  • Errors 1091 through 1100
  • Errors 1101 through 1110
  • Errors 1451 through 1460
  • Errors 1501 through 1510
