NDF Structures

The following data structures support the Network Diagnostics Framework (NDF) API, which allows Microsoft's extensible Helper Classes to be extended by other developers.

  • DIAG_SOCKADDR
  • DiagnosticsInfo
  • HELPER_ATTRIBUTE
  • HelperAttributeInfo
  • HYPOTHESIS
  • HypothesisResult
  • LIFE_TIME
  • OCTET_STRING
  • RepairInfo
  • RepairInfoEx
  • RootCauseInfo
  • ShellCommandInfo
  • UiInfo
