NDF Enumerated Types

The following enumerated types support Network Diagnostics Framework (NDF):

  • ATTRIBUTE_TYPE
  • DIAGNOSIS_STATUS
  • PROBLEM_TYPE
  • REPAIR_RISK
  • REPAIR_STATUS
  • REPAIR_SCOPE
  • UI_INFO_TYPE
