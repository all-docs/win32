IContextNode::IsStringSupported method

Indicates whether recognized string of this IContextNode came from the system dictionary, user dictionary, or word list.

Syntax

HRESULT IsStringSupported(
  [out, retval] VARIANT_BOOL *pfIsSupported
);

Parameters

pfIsSupported [out, retval]

VARIANT_TRUE if the recognized string value of this IContextNode is supported by the IInkAnalysisRecognizer with any corresponding hint nodes applied; otherwise, VARIANT_FALSE.

Return value

For a description of the return values, see Classes and Interfaces - Ink Analysis.
