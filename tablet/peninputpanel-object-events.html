PenInputPanel Object Events

The following table describes which threads the PenInputPanel object events can fire on.

Event Threads
InputFailed Fires on the thread in which the PenInputPanel is created.
PanelChanged Fires on the thread in which the PenInputPanel is created.
PanelMoving Fires on the thread in which the PenInputPanel is created.
VisibleChanged Fires on the thread in which the PenInputPanel is created.
