InkAnalyzer class

Implements the IInkAnalyzer interface.

Remarks

This class implements the IInkAnalyzer COM interface.

_IAnalysisEvents is the default source of events and provides standard events for the IInkAnalyzer.

_IAnalysisProxyEvents provides the data proxy events for the IInkAnalyzer. For more information, see Data Proxy with Ink Analysis.
