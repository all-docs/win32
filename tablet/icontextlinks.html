IContextLinks interface

Contains a collection of objects that implement the IContextLink interface.

Members

The IContextLinks interface inherits from the IUnknown interface. IContextLinks also has these types of members:

  • Methods

Methods

The IContextLinks interface has these methods.

Method Description
GetContextLink Retrieves the IContextLink at the specified index.
GetCount Retrieves the number of IContextLink objects in this collection.

Remarks

This is usually accessed through the IContextNode::GetContextLinks method.
