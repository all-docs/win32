Recognizer Enumerations

This section describes the recognizer enumerations.

In This Section

Enumeration Description
ALT_BREAKS Specifies how to create alternates from a best result string.
CONFIDENCE_LEVEL Indicates the level of confidence the recognizer has in the recognition result.
LINE_METRICS Represents the lines found in a segment.
PROPERTY_UNITS Specifies the unit of measurement of a property.
