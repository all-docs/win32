InkEdit Enumerations

This section contains Enumerations belonging to the InkEdit Control.

Enumeration Description
AppearanceConstants Specifies whether the control appears flat or 3-D.
BorderStyleConstants Specifies whether the control has a border.
InkApplicationGesture Specifies the interest in a set of application-specific gestures.
InkDisplayMode Specifies whether a selection appears as ink or text.
InkEditStatus Specifies whether the InkEdit control is idle, collecting ink, or recognizing ink.
InkInsertMode Specifies how ink is inserted onto the InkEdit control.
InkMode Specifies the collection mode settings for drawn ink-whether ink collection is disabled, ink is collected, or ink and gestures are collected.
InkMouseButton Specifies which mouse button was pressed.
InkMousePointer Specifies the type of mouse pointer that appears.
MouseButton Specifies which mouse button was pressed.
ScrollBarsConstants Specifies how the scroll bars of an InkEdit control appear on the screen.
SelAlignmentConstants Specifies the alignment of the paragraph relative to the margins of the InkEdit control.
