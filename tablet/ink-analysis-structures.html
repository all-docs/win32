Ink Analysis Structures

This section contains information about the structures used in ink analysis.

Structures

Structure Description
InkAnalysisRecognizerGuide Represents the area that the IInkAnalysisRecognizer object uses and in which ink can be drawn. The area is known as the recognition guide.
