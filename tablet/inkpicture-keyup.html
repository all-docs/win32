InkPicture.KeyUp event

Occurs when a key is released while the InkPicture control has focus.

Syntax

void KeyUp(
  [in, out] short *KeyCode,
  [in, out] short *Shift
);

Parameters

KeyCode [in, out]

The ASCII value of the key that is being pressed.

Shift [in, out]

The state of the SHIFT key.

Return value

This event does not return a value.

Remarks

This event method is defined in the _IInkPictureEvents interface. The _IInkPictureEvents interface implements the IDispatch interface with an identifier of DISPID_IPEKeyUp.

Requirements

Requirement Value
Minimum supported client Windows XP Tablet PC Edition [desktop apps only]
