InkEdit Methods

This section contains Methods belonging to the InkEdit Control.

Method Description
GetGestureStatus Retrieves the interest of the InkEdit control in a known set of gestures.
Recognize Specifies that recognition should occur.
Refresh Causes the control to redraw.
SetGestureStatus Modifies the interest of the InkEdit control in a known set of gestures.
