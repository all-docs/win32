Using InkPicture on Earlier Versions of Windows

You can use the InkPicture Control to render ink on Microsoft Windows 2000, Windows Server 2003, and non-Tablet PC editions of Windows XP. However, you cannot use the control to input ink on these operating systems.

The control's InkEnabled property is set to FALSE, and attempting to change the property has no effect. You can assign values to other properties and programmatically manipulate ink, but you cannot use the control to collect ink.
