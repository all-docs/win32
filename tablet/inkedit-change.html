InkEdit.Change event

Occurs when the content of the InkEdit control changes.

Syntax

void Change();

Parameters

This event has no parameters.

Return value

This event does not return a value.

Remarks

This event occurs whenever any properties that affect the contents of the InkEdit control change.

This event method is defined in the _IInkEditEvents interface. The _IInkEditEvents interface implements the IDispatch interface with an identifier of DISPID_IeeChange.
