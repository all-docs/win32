Text Input Panel Enumerations

This section contains the enumerations used by the ITextInputPanel Interface.

In This Section

Enumeration Description
CorrectionMode Enumeration Defines values that represent the correction modes of the Tablet PC Input Panel.
CorrectionPosition Enumeration Defines values that represent the post insertion correction positions of the Input Panel.
EventMask Specifies the events that you can set attention for.
InPlaceDirection Enumeration Defines values that represent the positions where the Input Panel can be placed, relative to the current text insertion point.
InPlaceState Enumeration Defines values that represent how the Input Panel is displayed onscreen.
InteractionMode Enumeration Defines values that dictate the positioning of the Input Panel.
PanelInputArea Enumeration Defines values that represent the input areas of the Input Panel.
