Flicks Messages

This section contains details about Flicks messages.

Message Description
WM_TABLET_FLICK The message that occurs when a flick gesture occurs.
WM_TABLET_QUERYSYSTEMGESTURESTATUS The message that occurs when the system asks a window which system gestures it would like to receive.
