Ink Analysis Constants

This section contains information about the constants used in ink analysis.

Constants

Constants Description
Analysis Hint Properties Defines globally unique identifiers (GUIDs) for properties of an analysis hint node (see IContextNode::GetType).
Context Node Properties Defines globally unique identifiers (GUIDs) for properties of an IContextNode.
Context Node Types Defines values that specify the type of IContextNode objects.
