Journal Reader Namespace Types

This section documents the XML types used by the Journal Reader component.

In This Section

Type Description
AlternatesListType complexType Defines the type that contains the list of recognition alternates for an ink word.
BackgroundImageType complexType Defines the type that holds information about the background image used by the Journal note, if any.
BackgroundType complexType Defines the type that contains the background information for the Journal note.
BaselineType complexType Defines the type that contains information about the baseline of a paragraph.
BoundsType attributeGroup Defines an attribute group that is used by a variety of elements in a Journal XML file.
ColorType simpleType Defines the type that is used to specify valid values for the color of certain elements in a Journal XML file.
ContentGroup group Defines the type that contains a set of grouped content in a Journal note.
ContentType complexType Defines the type that contains some form of content within a Journal XML file.
ContentTypeType simpleType Defines the type that defines valid values for the Type attribute of the Content element [Journal Reader].
DrawingType complexType Defines the type that contains ink that has been classified by the Journal analyzing engine as a Drawing element (as opposed to an InkWord element).
FlagType complexType Defines the type that contains the encoded binary image and position information for a flag embedded in a Journal document.
GroupNodeType complexType Defines the type that contains a set of grouped content in a Journal note.
HorizontalType complexType Defines the type that contains information about horizontal lines used by the stationery in a Journal note.
ImageType complexType Defines the type that contains the binary information for an image in a Journal note. Also see the BackgroundImageType complexType.
InkWordType complexType Defines the type that contains the ink data, alternates, and confidence for ink content that is an InkWord element (as opposed to a Drawing element).
JournalPageType complexType Defines the type that contains an individual page in a Journal note.
LineLayoutType complexType Defines the type that contains information about the lines that are used in a given Journal note's stationery.
LineLayoutStyleType simpleType Defines the valid values for the Style attribute of the LineLayout element.
LineType complexType Defines the type that contains a line of paragraph.
MarginType complexType Defines the type that contains the details about any margins in the Journal note, such as style, color, and position.
MarginTypeType simpleType Defines the type that contains valid values for the Type attribute for the Margin element.
ParagraphType complexType Defines the type that holds a paragraph in a Journal note.
ParagraphLineMetricsType complexType Defines the type that contains information about the line metrics of a paragraph, such as the baseline.
ReflowType complexType Defines the type that indicates the group has been reflowed.
ScalarTransformType complexType Defines the type that holds the information about any transformation applied to ink.
StationeryType complexType Defines the type that contains the stationery used by the Journal note.
TextType complexType Defines the type that contains the text value in a Content element [Journal Reader].
TitleAreaType complexType Defines the type that contains position and bounds information for the title in a Journal note.
TitleInfoType complexType Defines the type that contains information about the title in a Journal note.
TitleType complexType Defines the type that contains title information for the Journal note.
VerticalType complexType Defines the type that contains details about the vertical lines used in a Journal note's stationery.
