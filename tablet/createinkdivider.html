CreateInkDivider function

Instantiates an implementation of the InkDivider interface and returns its handle.

This function is not intended to be used by application code.

Syntax

INT_PTR CreateInkDivider(void);

Parameters

This function has no parameters.

Return value

Returns a handle to the created object.

Remarks

This class inherits from the InkDivider base class and implements virtual functions.
