Confidence Element

Contains the confidence rating returned by the Journal ink analyzer for the InkWord.

Definition

<xs:element name="Confidence" type="xs:string" minOccurs="0" />

Parent Elements

InkWord

Values

Valid values for this field include "0", "1", and "2". 0 indicates Strong confidence, 1 indicates Intermediate confidence, and 2 indicates Poor confidence.

Child Elements

None.

Attributes

None.

Element Information

Value
Element type xs:string
Namespace urn:schemas-microsoft-com:tabletpc:richink
Schema name Journal Reader
