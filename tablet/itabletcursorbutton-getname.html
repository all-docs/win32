ITabletCursorButton::GetName method

Retrieves the name of the stylus button.

Syntax

HRESULT GetName(
  [out] LPWSTR *ppwszName
);

Parameters

ppwszName [out]

The name of the stylus button.

Return value

This method can return one of these values.

Return code Description
S_OK Success.
E_FAIL An unspecified error occurred.

Remarks

It is the caller's responsibility to free the memory returned from this method by using CoTaskMemFree.

Requirements

Requirement Value
Minimum supported client Windows XP Tablet PC Edition [desktop apps only]
Minimum supported server None supported
