IInkAnalysisRecognizer::GetCapabilities method

Retrieves the capabilities of the recognizer.

Syntax

HRESULT GetCapabilities(
  [out] InkAnalysisRecognizerCapabilities *pCapabilities
);

Parameters

pCapabilities [out]

A bitwise combination of RecognizerCapabilities values.

Return value

For a description of the return values, see Classes and Interfaces - Ink Analysis.

Remarks

This value is constant for each IInkAnalysisRecognizer
