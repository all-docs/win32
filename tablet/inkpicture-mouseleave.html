InkPicture.MouseLeave event

Occurs when the mouse pointer leaves the InkPicture control.

Syntax

void MouseLeave();

Parameters

This event has no parameters.

Return value

This event does not return a value.

Remarks

This event method is defined in the _IInkPictureEvents interface. The _IInkPictureEvents interface implements the IDispatch interface with an identifier of DISPID_IPEMouseLeave.

Requirements

Requirement Value
Minimum supported client Windows XP Tablet PC Edition [desktop apps only]
Minimum supported server None supported
Header Msinkaut.h (also requires Msinkaut_i.c)
