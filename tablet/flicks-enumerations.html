Flicks Enumerations

This section contains the enumerations used in the WM_TABLET_FLICK Message.

Enumeration Description
FLICKDIRECTION Represents the direction of the flick.
FLICKMODE Specifies where flick gestures are visible.
FLICKACTION_COMMANDCODE Specifies the command code filed in the FLICK_DATA Structure and represents what kind of flick action is to be carried out.
SCROLLDIRECTION Specifies what direction the scroll box should move.
KEYMODIFIER Specifies which, if any, modifier keys were pressed when the flick gesture occurred.
