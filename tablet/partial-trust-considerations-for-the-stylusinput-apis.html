Partial Trust Considerations for the StylusInput API

The RealTimeStylus that takes the handle parameter requires the UIPermissionWindow.AllWindows and SecurityPermissionFlag.UnmanagedCode permissions, in addition to the permissions required by the constructor that takes the attachedControl parameter.

For more information about security and trust issues, see Security and Trust.
