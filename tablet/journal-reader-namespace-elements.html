Journal Reader Namespace Elements

This section documents the XML namespace elements used by the Journal Reader component.

In This Section

Element Description
Alternate element Contains a recognition alternate for an InkWord element.
AlternateList element Contains the set of Alternate elements that hold the alternates returned by the recognizer for the parent InkWord element.
Background element Contains the background for a JournalDocument element or JournalPage element.
Baseline element Provides (x, y) information for the starting and ending points of the baseline of a paragraph in the Journal document. The coordinate space used for these values is HIMETRIC.
Bitmap element Contains the Base64 encoded data for a bitmap contained in the Journal file as a background image.
CanReClassify element When true, indicates that the ink can be re-analyzed. This element is false if the user has explicitly set the type of the ink by using the Convert Handwriting to Text or the Change Shape To features of Journal.
Confidence element Contains the confidence rating returned by the Journal ink analyzer for the InkWord element.
Content element [Journal Reader] Contains the content for a Journal page.
Date element Contains the date the Journal note was created.
DocImage element If the Journal note being read was created by using the Journal Note Writer printer driver, then the background document image is held in this element.
Drawing element Contains content that has been classified by the analyzer or the user as a drawing.
Flag element Contains a base64 encoded bitmap of a flag associated with section of a Journal note.
GroupNode element Contains a set of elements-Paragraph element, InkWord element, Drawing element, Text element, Image element, or Flag element-that are grouped together in the Journal note.
Horizontal element Contains formatting information for the horizontal lines used for the stationery in a Journal note.
Image element Contains encoded binary data for an image in the Journal document, if present.
InkClass element Represents the collected strokes of ink within an ink space.
InkObject element Contains the Base64 encoded binary data of the Ink object for an InkWord element or Drawing element.
InkWord element Contains information about a given ink word in the Journal note, including position, alternates, and the actual ink data.
JournalDocument element The top-level element in a Journal note XML file.
JournalPage element Contains the details about an individual page in a Journal note.
Line element Contains a line within a Paragraph element.
LineLayout element Contains information about the layout of lines used in the stationery for the Journal note.
Margin element Defines the margin lines drawn on the page.
Paragraph element Contains a paragraph.
ParagraphLineMetrics element Contains information about the lines of a paragraph.
Path element The path to the image used for the background of a Journal note.
Reflow element Indicates the group has been reflowed.
ScalarTransform element Scalar transform used by the drawing or InkWord.
Stationery element Contains information about the stationery used in the Journal note, including the background color or image, the title information, and the line layout information, such as narrow rules and margin rules.
Text element Contains text information from the Journal note.
Title element Contains title information about the Journal note.
TitleArea element Contains location and bounding information for the note Title element, if present.
TitleInfo element Contains the title of the note and the date in the title, if present.
Vertical element Contains the information that describes the vertical component of the lines in the stationery used by the Journal note.
