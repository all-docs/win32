IInkAnalysisRecognizer::SetAsHighestPriorityInkAnalysisRecognizer method

Modifies this IInkAnalysisRecognizer to the highest priority in the IInkAnalyzer internal ordered list of recognizers.

Syntax

HRESULT SetAsHighestPriorityInkAnalysisRecognizer();

Parameters

This method has no parameters.

Return value

For a description of the return values, see Classes and Interfaces - Ink Analysis.
