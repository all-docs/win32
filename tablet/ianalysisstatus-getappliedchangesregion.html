IAnalysisStatus::GetAppliedChangesRegion method

Retrieves the region of the document that corresponds to changes that were made in the IInkAnalyzer object's context node tree as a result of ink analysis.

Syntax

HRESULT GetAppliedChangesRegion(
  [out] IAnalysisRegion **pAppliedChangesRegion
);

Parameters

pAppliedChangesRegion [out]

A pointer to the IAnalysisRegion of the document where changes were made.

Return value

For a description of the return values, see Classes and Interfaces - Ink Analysis.

Remarks

[!Caution]
To avoid a memory leak, call IUnknown::Release on *pAppliedChangesRegion when you no longer need to use the analysis region.

This method is most frequently used when the application receives information for debugging purposes and needs to invalidate the area where changes might occur so that the debugging information is painted.
