Flicks Constants

The following are the Flicks constants.

Constant/value Description
FLICK_WM_HANDLED_MASK 0x1 The value to return after handling the WM_TABLET_FLICK Message message. If FLICK_WM_HANDLED_MASK is returned, no further action occurs. Otherwise, Windows sends follow-up notifications, such as WM_APPCOMMAND, WM_VSCROLL, or WM_KEYDOWN, depending on which action is associated with the pen flick.
NUM_FLICK_DIRECTIONS 8 The number of directions defined in the FLICKDIRECTION enumeration.
