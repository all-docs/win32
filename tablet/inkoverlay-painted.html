InkOverlay.Painted event

Occurs when the InkOverlay object or InkPicture control has completed redrawing itself.

Syntax

void Painted(
  [in] long         hDC,
  [in] InkRectangle *Rect
);

Parameters

hDC [in]

The device context on which the event occurred.

Rect [in]

The InkRectangle that was repainted.

Return value

This event does not return a value.

Remarks

This event method is defined in the _IInkOverlayEvents and _IInkPictureEvents dispatch-only interfaces (dispinterfaces) with an ID of DISPID_IOEPainted.

Requirements

Requirement Value
Minimum supported client Windows XP Tablet PC Edition [desktop apps only]
Minimum supported server None supported
