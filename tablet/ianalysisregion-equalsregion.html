IAnalysisRegion::EqualsRegion method

Determines whether the specified IAnalysisRegion contains the same value as the current IAnalysisRegion object.

Syntax

HRESULT EqualsRegion(
  [in]  IAnalysisRegion *pOtherRegion,
  [out] VARIANT_BOOL    *pfRegionsEqual
);

Parameters

pOtherRegion [in]

The IAnalysisRegion object to compare with the current IAnalysisRegion.

pfRegionsEqual [out]

VARIANT_TRUE if the specified IAnalysisRegion contains the same value as the current IAnalysisRegion; otherwise, VARIANT_FALSE.

Return value

For a description of the return values, see Classes and Interfaces - Ink Analysis.
