_IAnalysisProxyEvents::ContextNodeLinkDeleting event

Occurs before the IInkAnalyzer deletes a IContextLink object between two IContextNode objects.

Syntax

HRESULT ContextNodeLinkDeleting(
  [in] IInkAnalyzer *pInkAnalyzer,
  [in] IContextLink *pContextLinkToBeDeleted
);

Parameters

pInkAnalyzer [in]

The IInkAnalyzer deleting the link.

pContextLinkToBeDeleted [in]

The IContextLink object to be deleted.

Return value

For a description of the return values, see Classes and Interfaces - Ink Analysis.

Remarks

Use this event when your application maintains its own data structure, which is synchronized with that of the IInkAnalyzer. This event occurs during the reconcile phase of ink analysis, or in response to an IInkAnalyzer method that removes an IContextLink from an IContextNode.

For more information about synchronizing your application data with the IInkAnalyzer, see Data Proxy with Ink Analysis.
