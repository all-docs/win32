DeleteInkDivider function

Deletes an InkDivider object and releases associated resources.

This function is not intended to be used by application code.

Syntax

HRESULT WINAPI DeleteInkDivider(
  _In_ INT_PTR hDivider
);

Parameters

hDivider [in]

A handle to the InkDivider object.

Return value

This function can return one of these values.

Return code Description
S_OK The method succeeded.
E_INVALIDARG The hDivider parameter is invalid.

Requirements

Requirement Value
Minimum supported client Windows XP Tablet PC Edition [desktop apps only]
Minimum supported server None supported
