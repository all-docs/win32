ASFLeakyBucketPairs

[The feature associated with this page, Windows Media Format 11 SDK, is a legacy feature. It has been superseded by Source Reader and Sink Writer. The ASFLeakyBucketPairs attribute is an optional attribute that describes the buffering requirements for a variable bit rate file.

Global Constant

g_wszASFLeakyBucketPairs

Data Type

WMT_TYPE_BINARY

Remarks

This attribute has the following format:

struct
{
    WORD wReserved;
    WM_LEAKY_BUCKET_PAIR bucket[2];
};

Where wReserved must equal zero, and bucket is an array of WM_LEAKY_BUCKET_PAIR structures. The array must contain at least two entries, but can be larger. The reader object uses this attribute to determine the amount of content to buffer before playback.
