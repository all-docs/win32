SNMP Application Syntax Values

[SNMP is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use Windows Remote Management, which is the Microsoft implementation of WS-Man.]

The SNMP Application Syntax Values are used by management applications that employ the Microsoft SNMP Management API.

Constant Description
ASN_IPADDRESS Indicates an IP address variable.
ASN_COUNTER32 Indicates a counter variable.
ASN_GAUGE32 Indicates a gauge variable. This variable describes an Unsigned32 type in conformance with RFC 1902.
ASN_TIMETICKS Indicates a timeticks variable.
ASN_OPAQUE Indicates an opaque variable.
ASN_COUNTER64 Indicates a counter variable that increases until it reaches a maximum value of (2^64) - 1.
ASN_UINTEGER32 Indicates a 32-bit unsigned integer variable. This variable describes a UInteger32 type in conformance with RFC 1442.
ASN_RFC2578_UNSIGNED32 See ASN_GAUGE32.
