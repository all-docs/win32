AVDecVideoAcceleration_H264 property

[The feature associated with this page, DirectShow, is a legacy feature. It has been superseded by MediaPlayer, IMFMediaEngine, and Audio/Video Capture in Media Foundation. Those features have been optimized for Windows 10 and Windows 11. Microsoft strongly recommends that new code use MediaPlayer, IMFMediaEngine and Audio/Video Capture in Media Foundation instead of DirectShow, when possible. Microsoft suggests that existing code that uses the legacy APIs be rewritten to use the new APIs if possible.]

Enables or disables hardware acceleration for H.264 video decoding.

This property is read/write.

Data type

UINT32 (VT_UI4)

Property GUID

CODECAPI_AVDecVideoAcceleration_H264

Remarks

If the value is zero, the decoder does not use DirectX Video Acceleration (DXVA) for H.264 video decoding. For DirectShow filters, set this property before the decoder's output pin is connected.

[!NOTE] When using Media Foundation (or the IMFTransform interface), this property has no effect. See Supporting Direct3D 11 video decoding in Media Foundation for how to use hardware acceleration with Media Foundation components.
