[The feature associated with this page, DirectShow, is a legacy feature. It has been superseded by MediaPlayer, IMFMediaEngine, and Audio/Video Capture in Media Foundation. Those features have been optimized for Windows 10 and Windows 11. Microsoft strongly recommends that new code use MediaPlayer, IMFMediaEngine and Audio/Video Capture in Media Foundation instead of DirectShow, when possible. The ToolTipMaxWidth property sets or retrieves the maximum width for the ToolTip associated with the MSWebDVD object.

[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth

Return Value

Returns the maximum width for the application's ToolTip as an Integer.

Remarks

This property is read/write with a default value of 200 pixels.
