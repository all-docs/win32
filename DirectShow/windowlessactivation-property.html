The WindowlessActivation property initializes the MSWebDVD object at design time for either windowed or windowless mode.

[ bWindowless = ] MSWebDVD.WindowlessActivation

Return Value

Returns whether the object is in windowed mode as a Boolean.

Remarks

This property is read/write with a default value of true. Therefore, you only need to set this property if you are running the MSWebDVD object in a windowed container. When contained in a Web page in MicrosoftÂ® Internet Explorer, MSWebDVD is always in windowless mode, and you don't need to set this property.
