Data Copy

The data copy feature enables you to send data from one application to another.

In This Section

Name Description
Using Data Copy Provides an example that demonstrates how to send information between two applications.
Data Copy Reference Contains the API reference.

Data Copy Messages

Name Description
WM_COPYDATA Sent to pass data to another application.

Data Copy Structures

Name Description
COPYDATASTRUCT Contains data to be passed to another application by the WM_COPYDATA message.
