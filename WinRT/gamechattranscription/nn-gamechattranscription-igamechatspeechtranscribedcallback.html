IGameChatSpeechTranscribedCallback interface

[!IMPORTANT] This API is not available to all apps. Unless your developer account is specially provisioned by Microsoft, calls to these APIs will fail at runtime.

Your application should implement this interface in order to receive notifications about the transcription results.

Inheritance

The IGameChatSpeechTranscribedCallback interface derives from the IUnknown interface.

Methods

The IGameChatSpeechTranscribedCallback interface has these methods.

 
IGameChatSpeechTranscribedCallback::SpeechTranscribed GameChatTranscription calls your implementation of this function when transcription results are available.
