IGameChatTranscriber interface

[!IMPORTANT] This API is not available to all apps. Unless your developer account is specially provisioned by Microsoft, calls to these APIs will fail at runtime.

Maps one-to-one to an audio stream that needs to be transcribed. Can accept encoded audio sequentially for a single audio stream.

Inheritance

The IGameChatTranscriber interface derives from the IUnknown interface.

Methods

The IGameChatTranscriber interface has these methods.

 
IGameChatTranscriber::ProcessEncodedAudio Your application should call ProcessEncodedAudio with each sequential audio buffer to be processed for transcription.
IGameChatTranscriber::EndSpokenPhrase Your application should call EndSpokenPhrase when it detects a period of silence in the audio stream.
