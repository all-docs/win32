IGameChatTranscriber::EndSpokenPhrase method

[!IMPORTANT] This API is not available to all apps. Unless your developer account is specially provisioned by Microsoft, calls to these APIs will fail at runtime.

Your application should call EndSpokenPhrase when it detects a period of silence in the audio stream.

Syntax

HRESULT EndSpokenPhrase();

Return value

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
