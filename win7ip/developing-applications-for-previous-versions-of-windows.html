Set the Build Target to Windows 7

All applications that use libraries in the Platform Update for WindowsÂ Vista must be built against the WindowsÂ 7 target platform.

You can set the target platform to WindowsÂ 7 either in your source code or by using the /D option with the Visual Studio compiler.

No Redistributable Components

Your application does not need to install redistributable components, such as DLLs or other run-time files.

Requires Updated End-User Computer

Because Platform Update for WindowsÂ Vista and Platform Update for Windows ServerÂ 2008 are hosted by Windows Update, end-users with automatic updates enabled are highly likely to already have these updates as well as the required service packs.

If the end-user's computer does not have Platform Update for WindowsÂ Vista or Platform Update for Windows ServerÂ 2008 installed and your application requires APIs that are supported with these updates, your application may not be able to run on the end-user's computer or may encounter errors during execution.

To avoid the problems that could be caused by your user's computer being out-of-date, you want to verify that your user's computer has the Platform Update for WindowsÂ Vista or the Platform Update for Windows ServerÂ 2008 update during the installation of your application. You can use the Windows Update Agent API to check your end-user's computer for installed updates. You can also use the Windows Update Agent API to download and install required updates during application installation if the end-user has not already installed the updates.

For an example of an installer that demonstrates how to use the Windows Update Agent API, see Direct3D 11 Deployment for Game Developers in the DirectX SDK (https://msdn.microsoft.com/directx/aa937788.aspx).

Although the D3D11InstallHelper installer sample that is discussed in Direct3D 11 Deployment for Game Developers, was written for applications that use Direct3D 11, it provides a good example of how to interact with the Windows Update Agent API to initiate and track the download and installation of updates that are hosted by Windows Update.
