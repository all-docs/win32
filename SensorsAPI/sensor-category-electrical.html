SENSOR_CATEGORY_ELECTRICAL

The SENSOR_CATEGORY_ELECTRICAL category contains sensors that provide information about electrical systems.

Platform-Defined Sensor Types

This category includes the following platform-defined sensor types.

Sensor type Description
SENSOR_TYPE_CAPACITANCE {CA2FFB1C-2317-49C0-A0B4-B63CE63461A0} Capacitance sensors.
SENSOR_TYPE_CURRENT {5ADC9FCE-15A0-4BBE-A1AD-2D38A9AE831C} Electrical current sensors.
SENSOR_TYPE_ELECTRICAL_POWER {212F10F5-14AB-4376-9A43-A7794098C2FE} Electrical power sensors.
SENSOR_TYPE_FREQUENCY {8CD2CBB6-73E6-4640-A709-72AE8FB60D7F} Electrical frequency sensor.
SENSOR_TYPE_INDUCTANCE {DC1D933F-C435-4C7D-A2FE-607192A524D3} Inductance sensors.
SENSOR_TYPE_POTENTIOMETER {2B3681A9-CADC-45AA-A6FF-54957C8BB440} Potentiometers.
SENSOR_TYPE_RESISTANCE {9993D2C8-C157-4A52-A7B5-195C76037231} Resistance sensors.
SENSOR_TYPE_VOLTAGE {C5484637-4FB7-4953-98B8-A56D8AA1FB1E} Voltage sensors.

Platform-Defined Data Fields

Platform-defined property keys for this category are based on SENSOR_DATA_TYPE_ELECTRICAL_GUID:

{BBB246D1-E242-4780-A2D3-CDED84F35842}

This category includes the following platform-defined data fields.

Data field name and PID Description
SENSOR_DATA_TYPE_CAPACITANCE_FARAD (PID = 4) VT_R8 Capacitance in farads.
SENSOR_DATA_TYPE_CURRENT_AMPS (PID = 3) VT_R8 Current in amperes.
SENSOR_DATA_TYPE_ELECTRICAL_FREQUENCY_HERTZ (PID = 9) VT_R8 Electrical frequency in hertz.
SENSOR_DATA_TYPE_ELECTRICAL_PERCENT_OF_RANGE (PID = 8) VT_R8 Potentiometer reading as a percentage of its possible range.
SENSOR_DATA_TYPE_ELECTRICAL_POWER_WATTS (PID = 7) VT_R8 Electrical power in watts.
SENSOR_DATA_TYPE_INDUCTANCE_HENRY (PID = 6) VT_R8 Inductance in henries.
SENSOR_DATA_TYPE_RESISTANCE_OHMS (PID = 5) VT_R8 Resistance in ohms.
SENSOR_DATA_TYPE_VOLTAGE_VOLTS (PID = 2) VT_R8 Electrical potential in volts.
