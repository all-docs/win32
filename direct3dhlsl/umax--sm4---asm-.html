umax (sm4 - asm)

Component-wise unsigned integer maximum.

umax dest[.mask], src0[.swizzle], src1[.swizzle],
Item Description
dest [in] The address of the result of the operation. dest = src0 > src1 ? src0 : src1
src0 [in] The value to compare to src1.
src1 [in] The value to compare to src0.

Remarks

This instruction applies to the following shader stages:

x x x
