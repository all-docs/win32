mad (sm4 - asm)

Component-wise multiply & add.

mad[_sat] dest[.mask], [-]src0[_abs][.swizzle], [-]src1[_abs][.swizzle], [-]src2[_abs][.swizzle]
Item Description
dest [in] The result of the operation. dest = src0 * src1 + src2
src0 [in] The multiplicand.
src1 [in] The multiplier.
src2 [in] The addend.

Remarks

This instruction applies to the following shader stages:

x x x
