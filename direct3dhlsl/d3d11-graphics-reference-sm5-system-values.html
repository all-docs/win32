Shader Model 5 System Values

Shader Model 5 implements the system values from Shader Model 4 and earlier, as well as the following new system values:

  • SV_DispatchThreadID
  • SV_DomainLocation
  • SV_GroupID
  • SV_GroupIndex
  • SV_GroupThreadID
  • SV_GSInstanceID
  • SV_InsideTessFactor
  • SV_OutputControlPointID
  • SV_TessFactor
