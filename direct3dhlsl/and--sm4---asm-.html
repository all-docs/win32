and (sm4 - asm)

Bitwise AND.

and dest[.mask], src0[.swizzle], src1[.swizzle]
Item Description
dest [in] The address of the result of the operation.
src0 [in] The 32-bit value to AND with src1.
src1 [in] The 32-bit value to AND with src0.

Remarks

Component-wise logical AND of each pair of 32-bit values from src0 and src1. The 32-bit results are placed in dest.

This instruction applies to the following shader stages:

Vertex Shader Geometry Shader Pixel Shader
x x x
