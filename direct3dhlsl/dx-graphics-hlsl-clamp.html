clamp

Clamps the specified value to the specified minimum and maximum range.

ret clamp(x, min, max)

Parameters

Item Description
x [in] A value to clamp.
min [in] The specified minimum range.
max [in] The specified maximum range.

Return Value

The clamped value for the x parameter.

Remarks

For values of -INF or INF, clamp will behave as expected. However for values of NaN, the results are undefined.
