def pragma Directive

Pragma directive that manually allocates a floating-point shader register.

#pragma def( target, register, val1, val2,val3, val4 )

Parameters

Item Description
target Target that contains the register to allocate.
register Floating-point shader register to allocate.
val0 First byte of the value to allocate to the specified register.
val1 Second byte of the value to allocate to the specified register.
val2 Third byte of the value to allocate to the specified register.
val3 Fourth byte of the value to allocate to the specified register.

Remarks

The def pragma allows a developer to prefill a floating-point shader register with the specified value. This pragma is infrequently used.
