What's New in Services for Windows 8

Windows 8 and Windows Server 2012 include the following new capabilities for services.

New Service Functions

Function Description
QueryServiceDynamicInformation Retrieves dynamic information related to the current service start.

Updated API Elements

API Element Description
HandlerEx Added SERVICE_CONTROL_USERMODEREBOOT.
SERVICE_STATUS Added SERVICE_ACCEPT_USERMODEREBOOT.
SERVICE_TRIGGER_SPECIFIC_DATA_ITEM Added SERVICE_TRIGGER_DATA_TYPE_LEVEL, SERVICE_TRIGGER_DATA_TYPE_KEYWORD_ANY, and SERVICE_TRIGGER_DATA_TYPE_KEYWORD_ALL.
