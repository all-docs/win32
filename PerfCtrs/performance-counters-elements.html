Performance Counters Elements

The Performance Counters schema defines the following elements:

  • counter (counterSet)
  • counterAttribute (counterAttributes)
  • counterAttributes (counter)
  • counters
  • counterSet (provider)
  • provider (counters)
  • struct (structs)
  • structs (counterSet)

This section contains the names of the elements that you use in your manifest; however, to get the details for each element, see the complex type that contains the element. The root element of the performance counters section of the manifest is the counters element.
