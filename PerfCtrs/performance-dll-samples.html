Performance DLL Samples

The Windows SDK (WSDK) contains three complete sample performance DLLs. These samples are located in the Samples\WinBase\WinNT\PerfTool\PerfDlls directory. The root of this path is the base installation directory of the WSDK. You can download the WSDK from the Microsoft Windows Software Development Kit (SDK) (search for Windows SDK and select the download for the latest released operating system).
