FWinIMEColorStyle function

Specifies whether the specified color is a Windows color.

Syntax

BOOL __cdecl FWinIMEColorStyle(
  _In_Â const IMECOLORSTY *pcolorstyle
);

Parameters

pcolorstyle [in]

An IMECOLORSTY structure returned from a PColorStyleBackFromIMEStyle or PColorStyleTextFromIMEStyle function.

Return value

Returns TRUE when the color is a Windows color.

Remarks

This function has no associated import library or header file; you must call it using the LoadLibrary and GetProcAddress functions.
