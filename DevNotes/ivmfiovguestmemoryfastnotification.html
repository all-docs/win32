IVmFiovGuestMemoryFastNotification interface

IVmFiovGuestMemoryFastNotification provides fast, event-based notifications of guest memory accesses.

IVmFiovGuestMemoryFastNotification provides a service that allows the user mode emulation driver to perform work when the guest touches specific physical pages. This is useful for protocols that have a doorbell mechanism to signal when work is ready, if that doorbell can be placed on a separate physical page.
