cbSize

The size of this structure.

dwPromptFlags

This flag is ignored.

Value Meaning
PST_PF_ALWAYS_SHOW 0x00000001 Requests that the provider show the prompt dialog to the user even if it is not required for this access.
PST_PF_NEVER_SHOW 0x00000002 Do not show the prompt dialog to the user.

hwndApp

The handle to the parent window for the user interface. The hwndApp member determines where the user interface appears. If NULL is passed, the desktop is considered to be the parent window.

The prompt string.
