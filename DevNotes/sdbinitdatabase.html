SdbInitDatabase function

Opens the shim database.

Syntax

HSDB WINAPI SdbInitDatabase(
  _In_Â DWORD Â Â dwFlags,
  _In_Â LPCTSTR pszDatabasePath

Parameters

dwFlags [in]

This parameter specifies the format of the path in the pszDatabasePath parameter. It can be one of the following values.

Value Meaning
HID_DOS_PATHS 0x00000001 An MS-DOS style path.
HID_DATABASE_FULLPATH 0x00000002 A full path.
HID_NO_DATABASE 0x00000004 The pszDatabasePath parameter is ignored and no database is opened.
HID_DATABASE_TYPE_MASK 0xF00F0000 This parameter specifies a predefined database. The pszDatabasePath parameter is ignored.

Â 

If dwFlags contains HID_DATA_TYPE_MASK, this parameter can also include one of the following values.

Value Meaning
SDB_DATABASE_MAIN_SHIM 0x80030000 Application shim database.
SDB_DATABASE_MAIN_MSI 0x80020000 MSI database.
SDB_DATABASE_MAIN_DRIVERS 0x80040000 Database of drivers to be blocked.

Â 

pszDatabasePath [in]

The path to the database. This parameter can be NULL if the dwFlags parameter specifies a predefined database.

Return value

The function returns a handle to the opened database.
