Shim Database Types

Represent the types for main and custom shim databases.

Constant/value Description
SDB_DATABASE_MAIN 0x80000000 The main database. If this flag is not present, the database is a custom database.
SDB_DATABASE_SHIM 0x00010000 The database contains application entries to be shimmed.
SDB_DATABASE_MSI 0x00020000 The database contains MSI entries.
SDB_DATABASE_DRIVERS 0x00040000 The database contains driver block entries.
SDB_DATABASE_DETAILS 0x00080000 The database contains Apphelp details.
SDB_DATABASE_SP_DETAILS 0x00100000 The database contains SP Apphelp details.
SDB_DATABASE_RESOURCE 0x00200000 The resource DLL contains Apphelp details.
SDB_DATABASE_TYPE_MASK 0xF02F0000 A mask used to extract main database information.
SDB_DATABASE_MAIN_SHIM SDB_DATABASE_SHIM | SDB_DATABASE_MSI | SDB_DATABASE_MAIN
SDB_DATABASE_MAIN_MSI SDB_DATABASE_MSI | SDB_DATABASE_MAIN
SDB_DATABASE_MAIN_DRIVERS SDB_DATABASE_DRIVERS | SDB_DATABASE_MAIN
SDB_DATABASE_MAIN_DETAILS SDB_DATABASE_DETAILS | SDB_DATABASE_MAIN
SDB_DATABASE_MAIN_SP_DETAILS SDB_DATABASE_SP_DETAILS | SDB_DATABASE_MAIN
SDB_DATABASE_MAIN_RESOURCE SDB_DATABASE_RESOURCE | SDB_DATABASE_MAIN
