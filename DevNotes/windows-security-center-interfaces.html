Windows Security app interfaces

The following interfaces are available for use with the Windows Security app.

In this section

Topic Description
IWscProduct Provides methods for getting product information for an individual provider to interact with the Windows Security app.
IWSCProductList Provides methods to collect product information for the selected type of providers installed on the computer.
