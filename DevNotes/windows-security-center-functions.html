Windows Security app functions

The following functions are used with the Windows Security app.

In this section

Topic Description
WscGetSecurityProviderHealth Gets the aggregate health state of the security provider categories represented by the specified WSC_SECURITY_PROVIDER enumeration values.
WscRegisterForChanges Registers a callback function to be run when the Windows Security app detects a change that could affect the health of one of the security providers.
WscUnRegisterChanges Cancels a callback registration that was made by a call to the WscRegisterForChanges function.
