IDXCoreAdapterList::IsAdapterPreferenceSupported method

Description

Determines whether a specified DXCoreAdapterPreference value is understood by the current operating system (OS). You can call IsAdapterPreferenceSupported before calling IDXCoreAdapterList::Sort.

Syntax

bool IsAdapterPreferenceSupported(
  DXCoreAdapterPreference preference
);

Parameters

preference

Type: DXCoreAdapterPreference

A DXCoreAdapterPreference value that will be checked to see whether it's supported by the OS.

Returns

Type: bool

Returns true if the sort type is understood by the current OS. Otherwise, returns false.
