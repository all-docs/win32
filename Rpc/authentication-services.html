Authentication Services

The Security Support Provider Interface (SSPI) provides the underlying authentication services for RPC. Therefore, when your application specifies an authentication service, it selects an SSP. For a list of the SSPs that SPPI currently supports, see Authentication-Service Constants. For more information, see Security Support Provider Interface (SSPI).

[!Note]
The terms Security Provider and Authentication Service are interchangeable.
