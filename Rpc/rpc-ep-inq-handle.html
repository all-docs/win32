RPC_EP_INQ_HANDLE

The RPC_EP_INQ_HANDLE data type declares a handle for an inquiry context. RPC applications use it to view server address information stored in the endpoint map.

typedef I_RPC_HANDLE* RPC_EP_INQ_HANDLE;
