IDeliveryOptimizationJob interface

Use the IDeliveryOptimizationJob interface to download ranges of a file.

Members

The IDeliveryOptimizationJob interface inherits from the IUnknown interface. IDeliveryOptimizationJob also has these types of members:

  • Methods

Methods

The IDeliveryOptimizationJob interface has these methods.

Method Description
AddFileWithRanges Adds a file to a download job and specifies the ranges of the file you want to download.
