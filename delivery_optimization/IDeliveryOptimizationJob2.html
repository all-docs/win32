IDeliveryOptimizationJob2 interface

The IDeliveryOptimizationJob2 enables adding a single file to the download job, and retrieving the file immediately. This interface also supports setting and getting optional job properties.

Members

The IDeliveryOptimizationJob2 interface inherits from the IUnknown interface. IDeliveryOptimizationJob2 also has these types of members:

  • Methods

Methods

The IDeliveryOptimizationJob2 interface has these methods.

Method Description
AddFile The AddFile method adds a single file to an existing Delivery Optimization job.
GetProperty The AddFile method adds a single file to an existing Delivery Optimization job.
SetProperty This method is unused.
