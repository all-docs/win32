The Teredo Security Model

The Teredo technology, when coupled with the Windows Security app and Windows Firewall, provides enhanced security to the end user. For Windows Firewall, IPv6 traffic sent and received over the Teredo interface is processed the same as native IPv6 traffic. The Teredo interface is recognized as another mini-port in the system.

For details on usage, see Implementing the Teredo Security Model.
