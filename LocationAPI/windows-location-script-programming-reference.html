Location API Object Model Reference

[The Location API object model is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, to access location from a website, use the W3C Geolocation API. To access location from a desktop application, use the Windows.Devices.Geolocation API.]

The Location API provides objects that you can use in scripting languages, such as Microsoft JScript. You can use these objects to add location functionality to your webpages. These objects are documented in the following sections.

  • LocationDisp.DispCivicAddressReport
  • LocationDisp.CivicAddressReportFactory
  • LocationDisp.DispLatLongReport
  • LocationDisp.LatLongReportFactory
  • LocationDisp Events
