IVMDVDDrive::AttachImage method

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Attaches an ISO image to the DVD drive within the virtual machine (VM).

Syntax

HRESULT AttachImage(
  [in] BSTR imagePath
);

Parameters

imagePath [in]

The full path to the ISO image that is to be attached.

Return value

This method can return one of these values.

Return code/value Description
S_OK 0 The operation was successful.
E_INVALIDARG 0x80000003 The imagePath parameter is a directory.
E_POINTER 0x80004003 The imagePath parameter is NULL.
VM_E_VM_UNKNOWN 0xA0040207 The VM could not be found.
VM_E_DRIVE_IN_USE 0xA0040727 A host DVD drive or ISO image is already attached to this drive.
VM_E_DRIVE_INVALID 0xA0040502 The bus location for this drive is invalid, or the drive does not appear to be a valid DVD drive.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
