IVMDVDDrive::Attachment property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the type of media attached to the DVD drive within the virtual machine.

This property is read-only.

Syntax

HRESULT get_Attachment(
  [out, retval] VMDVDDriveAttachmentType *driveAttachment
);

Property value

The attached media type. For a list of values, see VMDVDDriveAttachmentType.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
E_FAIL 0x80004005 An unexpected error has occurred.
VM_E_VM_UNKNOWN 0xA0040207 The virtual machine could not be found.
VM_E_DRIVE_INVALID 0xA0040502 The bus location for this drive is invalid.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
