IVMFloppyDrive::AttachHostDrive method

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Attaches a physical drive on the host to the floppy drive in the virtual machine.

Syntax

HRESULT AttachHostDrive(
  [in] BSTR driveLetter
);

Parameters

driveLetter [in]

The drive letter of the physical floppy drive on the host machine to is to be attached.

Return value

This method can return one of these values.

Return code/value Description
S_OK 0 The operation was successful.
E_INVALIDARG 0x80000003 The driveLetter parameter is NULL, is not a valid floppy drive, or the given path is empty.
VM_E_VM_UNKNOWN 0xA0040207 The configuration for this virtual machine is not valid or cannot be found.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
