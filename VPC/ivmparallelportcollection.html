IVMParallelPortCollection interface

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Defines the collection of parallel ports within the virtual machine. To obtain an IVMParallelPortCollection object, use the IVMVirtualMachine::ParallelPorts property.

Members

The IVMParallelPortCollection interface inherits from the IDispatch interface. IVMParallelPortCollection also has these types of members:

  • Properties

Properties

The IVMParallelPortCollection interface has these properties.

Property Access type Description
_NewEnum Read-only An enumerator for the collection.
Count Read-only The number of parallel ports in this collection.
Item Read-only The parallel port object that corresponds to the specified index.
