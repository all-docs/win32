IVMVirtualPC::MaximumSerialPortsPerVM property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the maximum number of serial ports per virtual machine.

This property is read-only.

Syntax

HRESULT get_MaximumSerialPortsPerVM(
  [out, retval] long *maxPorts
);

Property value

The maximum number of serial ports per virtual machine.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
VM_E_HARDWARE_VIRTUALIZATION_DISABLED 0xA0040951 The processor does not support Hardware Accelerated Virtualization (HAV) extensions.
