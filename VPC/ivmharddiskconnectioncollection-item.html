IVMHardDiskConnectionCollection::Item property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the hard disk connection object that corresponds to the specified index.

This property is read-only.

Syntax

HRESULT get_Item(
  [in]          long                  index,
  [out, retval] IVMHardDiskConnection **hardDiskConnection
);

Property value

The IVMHardDiskConnection object.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The hardDiskConnection parameter is NULL.
DISP_E_BADINDEX 0x8002000B The index of the requested item does not correspond to an item in this collection.
VM_E_VM_UNKNOWN 0xA0040207 The configuration is unknown.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
