IVMGuestOS::IsUserLoggedOn method

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Determines whether the requested session is present.

Syntax

HRESULT IsUserLoggedOn(
  [in]          long         inRailSession,
  [out, retval] VARIANT_BOOL *outSessionPresent
);

Parameters

inRailSession [in]

Set to 0 for a Rail session or 1 for an RDP session.

outSessionPresent [out, retval]

Set to VARIANT_TRUE if the session is present and VARIANT_FALSE otherwise.

Return value

This method can return one of these values.

Return code/value Description
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
E_INVALIDARG 0x80000003 The outSessionPresent parameter is not valid or is NULL.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
HRESULT_FROM_WIN32(ERROR_OUTOFMEMORY) 0x8007000e There is not enough memory available to complete this request.
VM_E_TIMED_OUT 0xA0040202 The operation did not complete in a timely manner.
VM_E_VM_NOT_RUNNING 0xA0040206 The virtual machine (VM) must be running for this operation.
VM_E_ADDITIONS_FEATURE_NOT_AVAIL 0xA0040505 The integration components feature is not installed in this VM.
VM_E_VM_PAUSED 0xA00400507 The VM is paused.
