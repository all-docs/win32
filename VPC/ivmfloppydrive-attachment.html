IVMFloppyDrive::Attachment property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the type of media attached to the floppy drive in the virtual machine (VM).

This property is read-only.

Syntax

HRESULT get_Attachment(
  [out, retval] VMFloppyDriveAttachmentType *driveAttachment
);

Property value

The type of media. For a list of values, see VMFloppyDriveAttachmentType.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
VM_E_VM_UNKNOWN 0xA0040207 The configuration for this VM is not valid or cannot be found.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
