IVMParallelPortCollection::Count property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the number of parallel ports in this collection.

This property is read-only.

Syntax

HRESULT get_Count(
  [out, retval] long *count
);

Property value

The number of parallel ports.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
