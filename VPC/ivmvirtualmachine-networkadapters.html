IVMVirtualMachine::NetworkAdapters property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves an enumerable collection of NICs attached to the virtual machine.

This property is read-only.

Syntax

HRESULT get_NetworkAdapters(
  [out, retval] IVMNetworkAdapterCollection **networkAdapterCollection
);

Property value

An IVMNetworkAdapterCollection object.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
VM_E_VM_UNKNOWN 0xA0040207 The configuration is unknown.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
