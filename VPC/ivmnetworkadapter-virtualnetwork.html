IVMNetworkAdapter::VirtualNetwork property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the virtual network to which the network interface is attached.

This property is read-only.

Syntax

HRESULT get_VirtualNetwork(
  [out, retval] IVMVirtualNetwork **virtualNetwork
);

Property value

An IVMVirtualNetwork interface that represents the virtual network to which this virtual network interface is attached.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
S_FALSE 1 This network interface is unplugged and is not connected to a virtual network.
VM_E_INVALID_VIRTUAL_NETWORK_ID 0xA00400702 This interface is connected to a virtual network with an invalid ID.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
