IVMDVDDriveEvents interface

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Defines the outgoing event interface for the IVMDVDDrive interface.

Members

The IVMDVDDriveEvents interface inherits from the IDispatch interface. IVMDVDDriveEvents also has these types of members:

  • Methods

Methods

The IVMDVDDriveEvents interface has these methods.

Method Description
OnMediaEject Receives notification that media has been ejected from the drive.
OnMediaInsert Receives notification that media has been inserted into the drive.
