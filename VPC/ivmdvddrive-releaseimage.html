IVMDVDDrive::ReleaseImage method

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Releases a captured ISO image from the DVD drive.

Syntax

HRESULT ReleaseImage();

Parameters

This method has no parameters.

Return value

This method can return one of these values.

Return code/value Description
S_OK 0 The operation was successful.
VM_E_VM_UNKNOWN 0xA0040207 The virtual machine could not be found.
VM_E_MEDIA_WRONG_TYPE 0xA00400728 The currently captured media is not a host disk drive.
VM_E_DRIVE_INVALID 0xA0040502 The drive could not be initialized due to an invalid bus location.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
