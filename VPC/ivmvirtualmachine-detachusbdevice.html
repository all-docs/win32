IVMVirtualMachine::DetachUSBDevice method

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Releases a USB device from a virtual machine.

Syntax

HRESULT DetachUSBDevice(
  [in] IVMUSBDevice *inUSBDevice
);

Parameters

inUSBDevice [in]

A IVMUSBDevice pointer that represents the USB device to be detached from the virtual machine.

Return value

This method can return one of these values.

Return code/value Description
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
VM_E_VM_NOT_RUNNING 0xA0040206 The virtual machine is not running.
VM_E_USB_DETACH_FAILED 0xA00400927 The detach operation failed.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
