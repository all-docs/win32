IVMGuestOS::CanShutdown property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Indicates whether the guest operating system can be cleanly shut down (requires Integration Components).

This property is read-only.

Syntax

HRESULT get_CanShutdown(
  [out, retval] VARIANT_BOOL *canShutdown
);

Property value

VARIANT_TRUE if the operating system supports the shutdown operation and VARIANT_FALSE otherwise.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
S_FALSE 1 The virtual machine is not turned on.
E_POINTER 0x80004003 The parameter is NULL.
VM_E_VM_UNKNOWN 0xA0040207 The virtual machine could not be found.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
