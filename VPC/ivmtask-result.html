IVMTask::Result property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the result of the task.

This property is read-only.

Syntax

HRESULT get_Result(
  [out, retval] VMTaskResult *result
);

Property value

The result of the task. For a list of values, see VMTaskResult.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter value is NULL.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
