IVMGuestOS::ProductType property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the product type of the guest operating system running in the virtual machine.

This property is read-only.

Syntax

HRESULT get_ProductType(
  [out, retval] BSTR *productType
);

Property value

The product type. The following string values are supported.

Value Meaning
"0x0000002" The system is a domain controller and the operating system is Windows Server 2008 R2, Windows Server 2008, Windows Server 2003 R2, Windows Server 2003, or Windows 2000 Server.
"0x0000003" The operating system is Windows Server 2008 R2, Windows Server 2008, Windows Server 2003 R2, Windows Server 2003, or Windows 2000 Server. Note that a server that is also a domain controller is reported as VER_NT_DOMAIN_CONTROLLER, not VER_NT_SERVER.
"0x0000001" The operating system is Windows 7, Windows Vista, Windows XP, or Windows 2000 Professional.
