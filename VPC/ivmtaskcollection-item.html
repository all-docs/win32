IVMTaskCollection::Item property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the task object that corresponds to the specified index.

This property is read-only.

Syntax

HRESULT get_Item(
  [in]          long    index,
  [out, retval] IVMTask **task
);

Property value

The IVMTask object.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The task parameter is NULL.
DISP_E_BADINDEX 0x8002000B The index of the requested item does not correspond to an item in this collection.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
