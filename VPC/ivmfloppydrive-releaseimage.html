IVMFloppyDrive::ReleaseImage method

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Releases a floppy media image on the host from the floppy drive.

Syntax

HRESULT ReleaseImage();

Parameters

This method has no parameters.

Return value

This method can return one of these values.

Return code/value Description
S_OK 0 The operation was successful.
VM_E_VM_UNKNOWN 0xA0040207 The configuration for this virtual machine is not valid or cannot be found.
VM_E_MEDIA_WRONG_TYPE 0xA00400728 The media attached to this floppy drive is not a floppy disk image.
VM_E_NO_MEDIA_CAPTURED 0xA00400652 There is no media attached to this floppy drive.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
