IVMGuestOS::TerminalServerPort property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Port used by Remote Desktop Services (formerly known as Terminal Services) in the guest operating system.

This property is read-only.

Syntax

HRESULT get_TerminalServerPort(
  [out, retval] long *tsPort = 3389
);

Property value

Returns the port used by Remote Desktop Services in the guest operating system.

Value Meaning
3389 Default value
1 65535 Valid range
0 Valid port value is not available.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
S_FALSE 1 Remote Desktop Services is not yet initialized in the guest operating system.
E_POINTER 0x80004003 The tsPort parameter is NULL.
VM_E_VM_NOT_RUNNING 0xA0040206 The virtual machine is not running.
VM_E_ADDITIONS_FEATURE_NOT_AVAIL 0xA0040505 Integration components are not installed in this virtual machine.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.

Remarks

The value of this property is not valid unless the TerminalServicesInitialized property is VARIANT_TRUE.

If the TerminalServicesInitialized property is VARIANT_FALSE due to a connection error on the port, then the value returned by the TerminalServerPort property contains the error value.
