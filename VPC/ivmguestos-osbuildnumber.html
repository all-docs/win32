IVMGuestOS::OSBuildNumber property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the build number of the guest operating system running in the virtual machine.

This property is read-only.

Syntax

HRESULT get_OSBuildNumber(
  [out, retval] BSTR *buildNumber
);

Property value

The build number.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
VM_E_VM_NOT_RUNNING 0xA0040206 The VM is not running.
VM_E_ADDITIONS_FEATURE_NOT_AVAIL 0xA0040505 Integration components are not installed in this VM.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
