IVMSerialPortCollection interface

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Defines the collection of serial ports within the virtual machine. To obtain an IVMSerialPortCollection object, use the IVMVirtualMachine::SerialPorts property.

Members

The IVMSerialPortCollection interface inherits from the IDispatch interface. IVMSerialPortCollection also has these types of members:

  • Properties

Properties

The IVMSerialPortCollection interface has these properties.

Property Access type Description
_NewEnum Read-only An enumerator for the collection.
Count Read-only The number of serial ports in this collection.
Item Read-only The serial port object that corresponds to the specified index.
