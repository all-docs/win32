IVMHardDisk::Type property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the type of the virtual hard disk.

This property is read-only.

Syntax

HRESULT get_Type(
  [out, retval] VMHardDiskType *type
);

Property value

The type of the current hard disk image.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
HRESULT_FROM_WIN32(ERROR_FILE_NOT_FOUND) 0x80070002 The current hard disk image file could not be found.
VM_E_DRIVE_INVALID 0xA0040502 The type of the current hard disk image is invalid.
VM_E_HD_IMAGE_OPEN_FAIL 0xA004067C An error occurred while attempting to open the current hard disk image file.
VM_E_HD_IMAGE_ACCESS 0xA0040681 An error occurred while attempting to access the current hard disk image file.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
