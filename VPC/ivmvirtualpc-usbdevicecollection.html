IVMVirtualPC::USBDeviceCollection property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves an enumerable collection of all USB devices connected to the host.

This property is read-only.

Syntax

HRESULT get_USBDeviceCollection(
  [out, retval] IVMUSBDeviceCollection **usbDeviceCollection
);

Property value

A reference to an IVMUSBDeviceCollection pointer that represents a collection of IVMUSBDevice objects.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
VM_E_USB_ENUMERATION_FAILED_MORE_DEVICES 0xA0040930 More than 16 USB devices are connected to the host.
VM_E_HARDWARE_VIRTUALIZATION_DISABLED 0xA0040951 The processor does not support Hardware Accelerated Virtualization (HAV) extensions.
