IVMVirtualNetwork::Name property

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Retrieves the unique name of the virtual network instance.

This property is read-only.

Syntax

HRESULT get_Name(
  [out, retval] BSTR *virtualNetworkName
);

Property value

The name of the virtual network.

Error codes

Name/value Meaning
S_OK 0 The operation was successful.
E_POINTER 0x80004003 The parameter is NULL.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred or the virtual network instance is unknown.

Remarks

Virtual network names are case-insensitive, for example, "MyNetwork" and "mynetwork" refer to the same virtual network.
