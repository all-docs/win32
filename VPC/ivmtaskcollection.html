IVMTaskCollection interface

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Defines the collection of task objects. To obtain an IVMTaskCollection object, use the IVMVirtualPC::Tasks property.

Members

The IVMTaskCollection interface inherits from the IDispatch interface. IVMTaskCollection also has these types of members:

  • Properties

Properties

The IVMTaskCollection interface has these properties.

Property Access type Description
_NewEnum Read-only An enumerator for the collection.
Count Read-only The number of tasks in this collection.
Item Read-only The task object that corresponds to the specified index.
