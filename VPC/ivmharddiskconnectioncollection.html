IVMHardDiskConnectionCollection interface

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Defines the collection of hard disk connections within the virtual machine. To obtain an IVMHardDiskConnectionCollection object, use the IVMVirtualMachine::HardDiskConnections property.

Members

The IVMHardDiskConnectionCollection interface inherits from the IDispatch interface. IVMHardDiskConnectionCollection also has these types of members:

  • Properties

Properties

The IVMHardDiskConnectionCollection interface has these properties.

Property Access type Description
_NewEnum Read-only An enumerator for the collection.
Count Read-only The number of hard disk connections in this collection.
Item Read-only The hard disk connection object that corresponds to the specified index.
