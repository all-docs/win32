IVMDVDDrive::AttachHostDrive method

[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the Hyper-V WMI provider (V2).]

Attaches a host drive to the DVD drive within the virtual machine.

Syntax

HRESULT AttachHostDrive(
  [in] BSTR driveLetter
);

Parameters

driveLetter [in]

The letter of the host drive that is to be attached.

Return value

This method can return one of these values.

Return code/value Description
S_OK 0 The operation was successful.
E_INVALIDARG 0x80000003 No drive letter was specified, or the specified drive letter is invalid.
E_FAIL 0x80004005 An unexpected error has occurred.
VM_E_VM_UNKNOWN 0xA0040207 The virtual machine could not be found.
VM_E_DRIVE_IN_USE 0xA0040727 A host DVD drive or ISO image is already attached to this drive within the virtual machine, or the specified host drive is already in use within another virtual machine.
VM_E_DRIVE_INVALID 0xA0040502 The bus location for this drive is invalid, or the drive does not appear to be a valid DVD drive.
DISP_E_EXCEPTION 0x80020009 An unexpected error has occurred.
