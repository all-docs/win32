Finding What You Need in the DNS WMI Provider

The Domain Name System (DNS) Windows Management Instrumentation (WMI) Provider documentation is presented in the following four sections.

Section Content
About the DNS WMI Provider Introduction to DNS and the DNS WMI Provider, describes the structure of the DNS WMI Provider class hierarchy.
Using the DNS WMI Provider Guidance on how to use the DNS WMI Provider, with links to samples that demonstrate its use.
DNS WMI Provider Reference Complete reference of all DNS WMI classes, properties, and methods.
DNS WMI Provider Scripting Examples Extensive sample scripts demonstrating how to use the DNS WMI Provider.
