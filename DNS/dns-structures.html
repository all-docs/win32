DNS structures

The following structures are defined for use with DNS.

  • DNS_ADDR
  • DNS_ADDR_ARRAY
  • DNS_APPLICATION_SETTINGS
  • DNS_CUSTOM_SERVER
  • DNS_HEADER
  • DNS_MESSAGE_BUFFER
  • DNS_PROXY_INFORMATION
  • DNS_QUERY_CANCEL
  • DNS_QUERY_RAW_CANCEL
  • DNS_QUERY_RAW_REQUEST
  • DNS_QUERY_RAW_RESULT
  • DNS_QUERY_REQUEST
  • DNS_QUERY_REQUEST3
  • DNS_QUERY_RESULT
  • DNS_RECORD
  • DNS_RECORD_FLAGS
  • DNS_SERVICE_BROWSE_REQUEST
  • DNS_SERVICE_CANCEL
  • DNS_SERVICE_INSTANCE
  • DNS_SERVICE_REGISTER_REQUEST
  • DNS_SERVICE_RESOLVE_REQUEST
  • IP4_ARRAY
  • IP6_ADDRESS
  • MDNS_QUERY_HANDLE
  • MDNS_QUERY_REQUEST

The following Resource Record (RR) structures are also included in the DNS API. These structures are used with the DNS_RECORD structure to programmatically manage DNS resource records.

  • DNS_A_DATA
  • DNS_AAAA_DATA
  • DNS_ATMA_DATA
  • DNS_DHCID_DATA
  • DNS_DNSKEY_DATA
  • DNS_DS_DATA
  • DNS_KEY_DATA
  • DNS_LOC_DATA
  • DNS_MINFO_DATA
  • DNS_MX_DATA
  • DNS_NAPTR_DATA
  • DNS_NSEC_DATA
  • DNS_NULL_DATA
  • DNS_NXT_DATA
  • DNS_OPT_DATA
  • DNS_PTR_DATA
  • DNS_RRSET
  • DNS_RRSIG_DATA
  • DNS_SIG_DATA
  • DNS_SOA_DATA
  • DNS_SRV_DATA
  • DNS_TKEY_DATA
  • DNS_TSIG_DATA
  • DNS_TXT_DATA
  • DNS_WINS_DATA
  • DNS_WINSR_DATA
  • DNS_WIRE_QUESTION
  • DNS_WIRE_RECORD
  • DNS_WKS_DATA
