MicrosoftDNS_ISDNType class

The subclass of MicrosoftDNS_ResourceRecord that represents an ISDN record.

The following syntax is simplified from MOF code.

Syntax

class MicrosoftDNS_ISDNType : MicrosoftDNS_ResourceRecord
{
  string ISDNNumber;
  string SubAddress;
};

  • Methods

Methods

The MicrosoftDNS_ISDNType class has these methods.

Method Description
CreateInstanceFromPropertyData Instantiates an ISDN Resource Record based on the data in the method's input parameters: the record's DNS Server Name, Container Name, Owner Name, class (default = IN), time-to-live value, and the ISDN number and subaddress of the owner. It returns a reference to the new object as an output parameter. Qualifiers: Implemented, static
Modify Updates the TTL, ISDN Number, and Subaddress to the values specified as the input parameters of this method. If a new value for a parameter is not specified, then the current value for the parameter is not changed. The method returns a reference to the modified object as an output parameter. Qualifiers: Implemented

SubAddress

Data type: string
