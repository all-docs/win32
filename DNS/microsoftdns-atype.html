MicrosoftDNS_AType class

Subclass of MicrosoftDNS_ResourceRecord that represents a host Address (A) record.

The following syntax is simplified from MOF code.

Syntax

class MicrosoftDNS_AType : MicrosoftDNS_ResourceRecord
{
  string IPAddress;
};

  • Methods

Methods

The MicrosoftDNS_AType class has these methods.

Method Description
CreateInstanceFromPropertyData Instantiates a Host Address (A) RR based on the data in the method's input parameters: the record's DNS Server Name, Container Name, Owner Name, class (default = IN), time-to-live value and the IP address of the host. It returns a reference to the new object as an output parameter. Qualifiers: Implemented, static
Modify Updates the TTL and IP address to the values specified as the input parameters of this method. If a new value for a parameter is not specified, then the current value for the parameter is not changed. The method returns a reference to the modified object as an output parameter. Qualifiers: Implemented

Data type: string

Access type: Read-only

String representing the IPv4 address of the host.
