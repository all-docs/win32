MicrosoftDNS_MBType class

The subclass of MicrosoftDNS_ResourceRecord that represents a Mailbox (MB) resource record.

The following syntax is simplified from MOF code.

Syntax

class MicrosoftDNS_MBType : MicrosoftDNS_ResourceRecord
{
  string MBHost;
};

  • Methods
  • Properties

Methods

The MicrosoftDNS_MBType class has these methods.

Method Description
CreateInstanceFromPropertyData Instantiates an MB RR based on the data in the method's input parameters: the record's DNS Server Name, Container Name, Owner Name of the mailbox, class (default = IN), time-to-live value and the mailbox host. It returns a reference to the new object as an output parameter. Qualifiers: Implemented, static
Modify Updates the TTL and MB host to the values specified as the input parameters of this method. If a new value for a parameter is not specified, then the current value for the parameter is not changed. The method returns a reference to the modified object as an output parameter. Qualifiers: Implemented

Properties

The MicrosoftDNS_MBType class has these properties.

MBHost

Data type: string

Access type: Read-only

FQDN that specifies a host of the mailbox specified in the record's Owner Name.
