DXGI Constants

This section contains info about the constants provided by DXGI.

In this section

Topic Description
DXGI_DEBUG_ID Globally unique identifier (GUID) values that identify producers of debug messages.
DXGI_ENUM_MODES Options for enumerating display modes.
DXGI_ERROR Error codes that can be returned by a DXGI function.
DXGI_PRESENT The DXGI_PRESENT constants specify options for presenting frames to the output.
DXGI_SHARED_RESOURCE_RW Shared resource constants.
DXGI_STATUS Status codes that can be returned by DXGI functions.
DXGI_USAGE Flags for surface and resource creation options.
