IAppxStreamingDataSource4 interface

Provides access to a data source. Deployment will query for this interface on the data source returned by the data source factory. Upon success, deployment will request the MSIXVC (Microsoft Installer for Xbox Virtual Console) manifest. An MSIXVC data source is expected to implement this interface. It's expected that appxsvc can access all of the files referenced in the manifest (for example, visual assets) during the appropriate steps of the install process (for example, tile registration).

Inheritance

The IAppxStreamingDataSource4 interface derives from the IUnknown interface.

Methods

The IAppxStreamingDataSource4 interface has these methods.

 
Apply Applies an update from an outdated package to an updating package.
GetDiskSpace Retrieves a value representing the total disk space used by the package.
GetManifestStream Retrieves the appx manifest stream of an MSIXVC package.
GetSignatureFileStream Retrieves the signature file stream of an MSIXVC package.
GetTotalSize Returns the total download size of the MSIXVC package.
Move Moves the package from the current volume to another.
Pause Terminates the request, causing the corresponding Run call to return E_ABORT.
Remove Cancels any outstanding Run calls (causing them to return), and removes the package contents.
Run Initiates the streaming operation (request).
