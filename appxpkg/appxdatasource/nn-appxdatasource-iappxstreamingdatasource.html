IAppxStreamingDataSource interface

Provides access to a data source. With this interface, you can create request jobs, and request OOS ranges.

Inheritance

The IAppxStreamingDataSource interface derives from the IUnknown interface.

Methods

The IAppxStreamingDataSource interface has these methods.

 
IAppxStreamingDataSource::CreateRangeRequestJob Creates a range request job to which ranges can be added for streaming.
IAppxStreamingDataSource::GetAttributes Returns attributes information specific to the data source.
IAppxStreamingDataSource::GetRangeData Requests a range of data synchronously, which is handled at the highest priority.
