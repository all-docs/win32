IAppxStreamingDataSource4::Pause method

Terminates the request, causing the corresponding Run call to return E_ABORT. This call leaves the downloaded bytes in a stable state so that the download can be resumed later.

Syntax

HRESULT Pause(
  [in, string] LPCWSTR packageFullName
);

Parameters

packageFullName

The package full name.

Return value

Type: HRESULT

If the function succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
