SignatureStreamFlags enumeration

Defines constants that specify options for the GetSignatureFileStream method.

Syntax

typedef enum __MIDL_IAppxStreamingDataSource4_0003
{
  SignatureStreamFlagsNone = 0,
  SignatureStreamFlagsAddP7xHeader = 0x1,
} SignatureStreamFlags;

Constants

 
SignatureStreamFlagsNone Specifies no options.
SignatureStreamFlagsAddP7xHeader Add a p7x header (defined in the P7xEditor class) to the stream.
