MoveOptions enumeration

Defines constants that specify options for the Move method.

Syntax

typedef enum __MIDL_IAppxStreamingDataSource4_0002
{
  BeginMove = 0,
  EndMove = 1,
  RollbackMove = 2,
  MoveOptionsInvalid = 3
} MoveOptions;

Constants

 
BeginMove Begin the move operation from the source to the target volume.
EndMove End the move operation from the source to the target volume (for example, delete source files).
RollbackMove Abort the move operation, and restore files to the source volume.
MoveOptionsInvalid Specfies invalid move options.
