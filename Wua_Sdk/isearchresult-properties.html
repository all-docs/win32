ISearchResult Properties

The ISearchResult interface defines the following properties.

Property Description
ResultCode Gets an OperationResultCode enumeration that specifies the result of a search.
RootCategories Gets an interface collection of the root categories that are currently available on the computer.
Updates Gets an interface collection of the updates that result from a search.
Warnings Gets a collection of the warnings that result from a search.
