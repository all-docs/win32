IDownloadJob Properties

The IDownloadJob interface defines the following properties.

Property Description
AsyncState Gets the caller-specific state object that is passed to the IUpdateDownloader.BeginDownload method.
IsCompleted Gets the setting that indicates whether the call to IUpdateDownloader.BeginDownload was processed completely.
Updates Gets an interface that contains a read-only collection of the updates that are specified in a download.
