IUpdateInstaller Properties

The IUpdateInstaller interface defines the following properties.

Property Description
AllowSourcePrompts Gets and sets a Boolean value that indicates whether to show source prompts to the user when installing the updates.
ClientApplicationID Gets and sets the current client application.
IsBusy Gets a Boolean value that indicates whether an installation or uninstallation is in progress on a computer at a specific time.
IsForced Gets or sets a Boolean value that indicates whether to forcibly install or uninstall an update.
ParentHwnd Gets and sets a handle to the parent window that can contain a dialog box.
ParentWindow Gets and sets the interface that represents the parent window that can contain a dialog box.
RebootRequiredBeforeInstallation Gets a Boolean value that indicates whether a system restart is required before installing or uninstalling updates.
Updates Gets and sets an interface that contains a read-only collection of the updates that are specified for installation or uninstallation.
