IInstallationBehavior Properties

The IInstallationBehavior interface defines the following properties.

Property Description
CanRequestUserInput Gets a Boolean value that indicates whether the installation or uninstallation of an update can prompt for user input.
Impact Gets an InstallationImpact enumeration that indicates how the installation or uninstallation of the update affects the computer.
RebootBehavior Gets an InstallationRebootBehavior enumeration that specifies the restart behavior that occurs when you install or uninstall the update.
RequiresNetworkConnectivity Gets a Boolean value that indicates whether the installation or uninstallation of an update requires network connectivity.
