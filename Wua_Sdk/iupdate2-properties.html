IUpdate2 Properties

The IUpdate2 interface defines the following properties.

Property Description
CveIDs Gets a collection of common vulnerabilities and exposures (CVE) IDs that are associated with the update.
IsPresent Gets a Boolean value that indicates whether an update is present on a computer.
RebootRequired Gets a Boolean value that indicates whether a system restart is required on a computer to complete the installation or the uninstallation of an update.
