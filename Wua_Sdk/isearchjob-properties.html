ISearchJob Properties

The ISearchJob interface defines the following properties.

Property Description
AsyncState Gets the caller-specific state object that is passed to the IUpdateSearch.BeginSearch method.
IsCompleted Gets a Boolean value that indicates whether the call to the IUpdateSearch.BeginSearch method is completely processed.
