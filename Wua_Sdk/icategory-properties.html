ICategory Properties

The ICategory interface defines the following properties.

Property Description
CategoryID Gets the identifier of the category.
Children Gets an interface collection that contains the child categories of this category.
Description Gets the description of the category.
Image Gets an interface that contains information about the image that is associated with the category.
Name Gets the localized name of the category.
Order Gets the recommended display order of this category among its sibling categories.
Parent Gets an interface that describes the parent category of this category.
Type Gets the type of the category.
Updates Gets an interface that contains a collection of updates that immediately belong to the category.
