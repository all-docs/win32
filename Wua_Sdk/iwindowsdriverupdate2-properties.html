IWindowsDriverUpdate2 Properties

The IWindowsDriverUpdate2 interface defines the following properties.

Property Description
CveIDs Contains a collection of the Common Vulnerabilities and Exposures (CVE) identifiers that are associated with an update.
IsPresent Gets a Boolean value that indicates whether an update is installed on the computer.
RebootRequired Gets a Boolean value that indicates whether the computer must be restarted after you install or uninstall an update.
