IWindowsDriverUpdate5 Properties

The IWindowsDriverUpdate5 interface defines the following properties.

Property Description
AutoDownload Gets an AutoDownloadMode value that indicates the automatic download mode of update.
AutoSelection Gets an AutoSelectionMode value indicating the automatic selection mode of an update in the Control Panel of Windows Update.
