IWindowsDriverUpdate4 Properties

The IWindowsDriverUpdate4 interface defines the following properties.

Property Description
PerUser Gets a Boolean value that indicates whether an update is a per-user update.
WindowsDriverUpdateEntries Gets the driver update entries that are applicable for the update.
