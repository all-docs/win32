IUpdateSession Properties

The IUpdateSession interface defines the following properties.

Property Description
ClientApplicationID Gets and sets the current client application.
ReadOnly Gets a Boolean value that indicates whether the session object is read-only.
WebProxy Gets and sets the proxy settings that are used to access the server.
