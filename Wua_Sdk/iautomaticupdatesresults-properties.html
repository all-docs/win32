IAutomaticUpdatesResults Properties

The IAutomaticUpdatesResults interface defines the following properties.

Property Description
LastInstallationSuccessDate Gets the last time and Coordinated Universal Time (UTC) date when Automatic Updates successfully installed any updates, even if some failures occurred.
LastSearchSuccessDate Gets the last time and Coordinated Universal Time (UTC) date when AutomaticUpdates successfully searched for updates.
