IUpdateServiceCollection Properties

The IUpdateServiceCollection interface defines the following properties.

Property Description
_NewEnum Gets an IEnumVARIANT interface that is used to enumerate the collection.
Count Gets the number of elements in the collection.
Item Gets and sets an IUpdateService interface in the collection.
