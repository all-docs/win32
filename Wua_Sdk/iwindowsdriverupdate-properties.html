IWindowsDriverUpdate Properties

The IWindowsDriverUpdate interface defines the following properties.

Property Description
DeviceProblemNumberAddress Gets the problem number of the device that matches for the Windows driver update.
DeviceStatus Gets the status of the device that matches for the Windows driver update.
DriverClass Gets the class of the Windows driver update.
DriverHardwareID Gets the hardware ID or compatible ID that the Windows driver update must match to be installable.
DriverManufacturer Gets the language-invariant name of the manufacturer of the Windows driver update.
DriverModel Gets the language-invariant model name of the device for which the Windows driver update is intended.
DriverProvider Gets the language-invariant name of the provider of the Windows driver update.
DriverVerDate Gets the driver version date of the Windows driver update.
