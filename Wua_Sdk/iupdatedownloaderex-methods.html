IUpdateDownloaderEx Methods

The IUpdateDownloaderEx interface defines the following methods.

Method Description
BeginDownload2 Starts an asynchronous download of the content files that are associated with the updates.
Download2 Starts a synchronous download of the content files that are associated with the updates.
