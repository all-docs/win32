IUpdateExceptionCollection Properties

The IUpdateExceptionCollection interface defines the following properties.

Property Description
_NewEnum Gets an IEnumVARIANT interface that is used to enumerate the collection.
Count Gets the number of elements in the collection.
Item Gets an IUpdateException interface in the collection.
