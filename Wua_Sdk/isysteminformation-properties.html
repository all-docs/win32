ISystemInformation Properties

The ISystemInformation interface defines the following properties.

Property Description
OemHardwareSupportLink Gets a hyperlink to technical support information for OEM hardware.
RebootRequired Gets a Boolean value that indicates whether a system restart is required to complete the installation or uninstallation of one or more updates.
