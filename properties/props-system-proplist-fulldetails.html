System.PropList.FullDetails

The list of all the properties to show in the details page. Property groups can be included in this list in order to more easily organize the UI. Register under the regvalue of "FullDetails".

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
