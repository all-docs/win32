System.PropList.QuickTip

The list of properties to show in the infotip when the item is on a slow network. Properties with empty values will not be displayed. Register under the regvalue of "QuickTip".

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
