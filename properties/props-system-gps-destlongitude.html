System.GPS.DestLongitude

Indicates the latitude of the destination point. This is an array of three values, as follows: index 0 is the degrees, index 1 is the minutes, index 2 is the seconds. Each is calculated from the values in PKEY_GPS_DestLongitudeNumerator and PKEY_GPS_DestLongitudeDenominator.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
