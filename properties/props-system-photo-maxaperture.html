System.Photo.MaxAperture

The maximum aperture of the lens as recorded by the camera, taken from the Exchangeable Image File (EXIF) information. This property is calculated from System.Photo.MaxApertureNumerator and System.Photo.MaxApertureDenominator.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
