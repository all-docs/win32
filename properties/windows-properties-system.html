Windows Property System

Purpose

The Windows Property System is an extensible read/write system of data definitions that provides a uniform way of expressing metadata about Shell items. The Windows Property system in WindowsÂ Vista and later enables you to store and retrieve metadata for Shell items. A Shell item is any single piece of content, such as a file, folder, email, or contact. A property is an individual piece of metadata associated with a Shell item.

Developer audience

Before you start reading the Windows Property System SDK documentation, you should have a fundamental understanding of the following:

  • Component Object Model (COM)
  • Shell Namespace programming

For an introduction to COM, see COM Fundamentals. For an introduction to Shell Namespace programming, see Getting Started with the Shell Namespace.

For uses of the Windows Property System, see Property System Overview: Development Scenarios.
