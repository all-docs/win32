System.Photo.DigitalZoom

The digital zoom ratio when the image was shot. Read from the camera in the file's Exchangeable Image File (EXIF) information. This property is calculated from System.Photo.DigitalZoomNumerator and System.Photo.DigitalZoomDenominator. If the numerator of the recorded value is 0, a digital zoom was not used.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
