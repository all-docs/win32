System.Photo.MakerNoteOffset

The offset for the maker note specified in System.Photo.MakerNote. See the Exchangeable Image File (EXIF) 2.2 specification for more information. This property is not intended to be displayed in the Shell, but it is available programmatically to applications.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
