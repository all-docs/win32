System.PropList.NonPersonal

The list of properties considered 'non-personal'. The system will leave these properties untouched when directed to remove all non-personal properties from a given file. Register under the regvalue of "NonPersonal".

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
