System.FileFRN

The unique file ID, also known as the File Reference Number. For a given file, this is the same value as the FileId member of the FILE_ID_BOTH_DIR_INFO structure, which is obtained by a call to GetFileInformationByHandleEx.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
