System.Photo.Brightness

The brightness value of the image, in APEX units, usually in the range of -99.99 to 99.99.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

Remarks

PKEY values are defined in Propkey.h.

See the EXIF 2.2 specification, Annex C, for a comparison of System.Photo.Brightness numbers and Foot-Lambert measurements. This property is calculated from System.Photo.BrightnessNumerator and System.Photo.BrightnessDenominator. If the numerator of the recorded value is FFFFFFFF.H, "Unknown" should be indicated.
