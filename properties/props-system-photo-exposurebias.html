System.Photo.ExposureBias

The amount of exposure bias used in the photo, as read from the camera. This property is calculated from System.Photo.ExposureBiasNumerator and System.Photo.ExposureBiasDenominator. The unit is the APEX value. Ordinarily it ranges from -99.99 through 99.99.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
