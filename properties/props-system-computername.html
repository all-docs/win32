System.ComputerName

The name of the computer where the item or file is located. This property is automatically populated by Windows and should not be used for anything other than its intended purpose.

Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista
