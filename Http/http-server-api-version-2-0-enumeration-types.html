HTTP Server API Version 2.0 Enumeration Types

The following are the HTTP version 2.0 enumeration types.

Enumeration Description
HTTP_DELEGATE_REQUEST_PROPERTY_ID Defines constants that specify a type of property information for a delegate request.
HTTP_FEATURE_ID Defines constants that specify an identifier for an HTTP feature.
