System Restore API

The System Restore API enables an application to manage restore points. There is also a scriptable interface available through Windows Management Instrumentation (WMI).

For more information about the System Restore API, see the following topics:

  • Calling SRSetRestorePoint
  • Restore Point Description Text
  • Nested Calls to SRSetRestorePoint

For more information about the System Restore WMI Classes, see Configuring System Restore.
