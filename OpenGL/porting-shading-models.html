Porting Shading Models

Like IRIS GL, OpenGL lets you switch between smooth (Gouraud) shading and flat shading. The following table lists the IRIS GL shading and dithering functions and their equivalent OpenGL functions.

IRIS GL function OpenGL function Meaning
shademodel(FLAT) glShadeModel ( GL_FLAT ) Does flat shading.
shademodel(GOURAUD) glShadeModel( GL_SMOOTH ) Does smooth shading.
getsm glGet ( GL_SHADE_MODEL ) Returns current shade model.
dither(DT_ON) dither(DT_OFF) glEnable ( GL_DITHER )glDisable( GL_DITHER ) Turns dithering on/off.
