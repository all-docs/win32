File Management Constants

The following groups of constants are used in file management.

In this section

Constant Description
File Access Rights Constants Lists the access rights that are specific to files and directories.
File Attribute Constants File attributes are metadata values stored by the file system on disk and are used by the system and are available to developers via various file I/O APIs.
