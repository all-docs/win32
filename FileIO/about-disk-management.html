About Disk Management

The file system provides an abstraction of the physical characteristics of storage devices so that applications can simply write to and read from files. However, storage in an enterprise relies heavily on the concept of disks.

In this section

Topic Description
Disk Devices and Partitions Describes a hard disk, partitioning, and the Master Boot Record.
Basic and Dynamic Disks Describes two disk storage types and discusses partition styles.
Defining an MS-DOS Device Name An MS-DOS device name is a junction that points to the path of an MS-DOS device. These junctions make up the MS-DOS device namespace.
Managing Disk Quotas Administrators can control the amount of data that each user can store on an NTFS file system volume.
