Using Directory Management

The following topics demonstrate the use of the directory management functions.

In this section

Topic Description
Changing the Current Directory An application can change the current directory by calling the SetCurrentDirectory function.
Listing the Files in a Directory To list files in a specified directory, call FindFirstFile, FindNextFile, and FindClose.
Moving Directories To move a directory to another location, along with the files and subdirectories contained within it, call the MoveFileEx, MoveFileWithProgress, or MoveFileTransacted function.
Obtaining Directory Change Notifications An application can monitor the contents of a directory and its subdirectories by using change notifications.
