How to access System Management BIOS (SMBIOS) information from a Universal Windows app.

Access SMBIOS information from a Universal Windows Platform App

Starting with WindowsÂ 10, version 1803, Universal Windows apps can use GetSystemFirmwareTable and EnumSystemFirmwareTables to access SMBIOS information by declaring the smbios restricted capability in the app manifest.

[!IMPORTANT] Only access to raw SMBIOS (RSMB) firmware tables is supported from a Universal Windows app. ACCESS_DENIED will be returned if you try to access other firmware table types from a Universal Windows app.

Â 

To declare the smbios restricted capability in the app manifest, add the rescap namespace and smbios capability as follows:

<Package
  ...
  xmlns:rescap="http://schemas.microsoft.com/appx/manifest/foundation/windows10/restrictedcapabilities"
  IgnorableNamespaces="uap mp rescap">  
  ...
  <Capabilities>
    <rescap:Capability Name="smbios"/>
  </Capabilities>
</Package>
