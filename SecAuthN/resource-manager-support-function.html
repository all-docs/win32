Resource Manager Support Function

The following function frees memory allocated through the use of the SCARD_AUTOALLOCATE length designator, simplifying the use of the other resource manager functions.

Topic Description
SCardFreeMemory Release memory returned through the use of SCARD_AUTOALLOCATE.
