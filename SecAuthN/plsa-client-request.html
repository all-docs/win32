PLSA_CLIENT_REQUEST

The PLSA_CLIENT_REQUEST data type is an opaque data type.

The Local Security Authority (LSA) uses it internally to maintain client information related to individual requests.

#include <windows.h>

typedef PVOID *PLSA_CLIENT_REQUEST;
