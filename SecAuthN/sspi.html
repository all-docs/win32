SSPI

This section presents information about Security Support Provider Interface (SSPI). SSPI allows an application to use various security models available on a computer or network without changing the interface to the security system. Topics include the following:

  • SSPI Model
  • SSPI Options for Distributed Applications
  • SSPI Context Semantics
  • SSP Packages Provided by Microsoft
  • Custom Security Packages
  • Extended Error Information
