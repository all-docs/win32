The Status method gets the current status of the smart card or reader.

Parameters

pStatus [out]

Pointer to an SCARD_STATES enumeration value. On return, contains the current status/state.

pProtocols [out]

Pointer to an SCARD_PROTOCOLS enumeration value. On return, contains the current protocol.

Return value

The method returns one of the following possible values:

Return code Description
S_OK Operation completed successfully.
E_INVALIDARG Invalid parameter.
E_POINTER A bad pointer was passed in.
E_OUTOFMEMORY Out of memory.
