Using Message Support

After a handshake has been completed and a secure connection has been established, an application can use the MakeSignature, EncryptMessage (General), VerifySignature, and DecryptMessage (General) functions to exchange signed or encrypted application data with the remote computer.

Examples that use these functions after a security context has been established are demonstrated in the following sections:

  • Signing a Message
  • Verifying a Message
  • Encrypting a Message
  • Decrypting a Message
