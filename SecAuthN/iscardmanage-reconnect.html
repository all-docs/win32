ISCardManage::Reconnect method

[The Reconnect method is available for use in the operating systems specified in the Requirements section. The Reconnect method allows an application to reconnect to a smart card or reader without having to issue a Detach call followed by an AttachByHandle or AttachByIFD call respectively.

Syntax

HRESULT Reconnect();

Parameters

This method has no parameters.

Return value

The method returns one of the following possible values:

Return code Description
S_OK Operation completed successfully.
E_OUTOFMEMORY Out of memory.

Â 

Remarks

To attach a smart card call AttachByHandle or AttachByIFD.

To detach a smart card, call Detach.

For a list of all the methods defined by this interface, see ISCardManage.

In addition to the COM error codes listed above, this interface may return a smart card error code if a smart card function was called to complete the request. For more information, see Smart Card Return Values.
