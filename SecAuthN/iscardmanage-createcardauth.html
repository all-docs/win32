The CreateCardAuth method creates an ISCardAuth interface.

Parameters

ppISCardAuth [out]

Returned pointer to the created interface.

Return value

The method returns one of the following possible values:

Return code Description
S_OK Operation completed successfully.
E_INVALIDARG The ppISCardAuth parameter is not valid.
E_POINTER A bad pointer was passed in ppISCardAuth.
E_OUTOFMEMORY Out of memory.
