ResourceLocator.ClearSelectors method

Removes all of the selectors from a ResourceLocator object. You can provide a ResourceLocator object instead of specifying a resource URI in Session object operations such as Session.Get, Session.Put, or Session.Enumerate.

Syntax

ResourceLocator.ClearSelectors()

Parameters

This method has no parameters.

Remarks

IWSManResourceLocator::ClearSelectors is the corresponding C++ method.
