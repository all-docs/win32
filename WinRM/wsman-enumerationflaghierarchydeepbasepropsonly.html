WSMan.EnumerationFlagHierarchyDeepBasePropsOnly method

The WSMan.EnumerationFlagHierarchyDeepBasePropsOnly method returns the value of the enumeration flag EnumerationFlagHierarchyDeepBasePropsOnly for use in the flags parameter of Session.Enumerate. This method provides a more efficient syntax for using the constant so that scripts are not required to set a constant value. For more information about how to call this method, see Session Constants.

EnumerationFlagHierarchyDeepBasePropsOnly is a constant in the _WSManEnumFlags enumeration and is described in Enumeration Constants.

Syntax

WSMan.EnumerationFlagHierarchyDeepBasePropsOnly( _
  ByRef flags _
)

Parameters

flags [out]

The value of the constant.

Return value

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
