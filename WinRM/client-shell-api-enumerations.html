Client Shell API Enumerations

The following table provides an overview of the enumerations in the Windows Remote Management (WinRM) Client Shell application programming interface (API).

Enumerations Description
WSManAuthenticationFlags Defines the authentication scheme to use for both server and proxy authentication.
WSManCallbackFlags Defines a set of flags used by all callback functions.
WSManDataType Specifies the current data type of the union in the WSMAN_DATA structure.
WSManProxyAccessType Defines the proxy access type.
WSManSessionOption Defines a set of extended options for the session.
