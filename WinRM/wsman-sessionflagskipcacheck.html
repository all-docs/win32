WSMan.SessionFlagSkipCACheck method

The WSMan.SessionFlagSkipCACheck method returns the value of the WSManFlagSkipCACheck authentication flag for use in the flags parameter of WSMan.CreateSession. This method provides a more efficient syntax for using the constant so that scripts are not required to set a constant value. For more information about how to call this method and code examples, see Session Constants.

WSManFlagSkipCACheck is a constant in the __WSManSessionFlags enumeration. For more information, see Authentication Constants.

Syntax

WSMan.SessionFlagSkipCACheck( _
  ByRef flags _
)

Parameters

flags [out]

The value of the constant.

Return value

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
