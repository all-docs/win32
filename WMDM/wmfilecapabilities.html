WMFILECAPABILITIES structure

The WMFILECAPABILITIES structure describes the MIME type.

Syntax

typedef struct _tagWMFILECAPABILITIES {
  LPWSTR pwszMimeType;
  DWORD  dwReserved;
} WMFILECAPABILITIES;

Members

pwszMimeType

MIME type.

dwReserved

DWORD reserved for future use. Set to zero (0).
