IResultVerb::Icon property

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

This property returns a pointer to handle of the optional icon associated with the verb.

This property is read-only.

Syntax

HRESULT get_Icon(
  [out, retval] SHANDLE_PTR *hicon
);

Property value

hicon is a pointer to the handle of the optional icon associated with the verb.

Requirement Value
