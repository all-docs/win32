Agent Control Events

[Microsoft Agent is deprecated as of Windows 7, and may be unavailable in subsequent versions of Windows.]

The Microsoft Agent control provides several events that enable your client application to track the state of the server:

  • ActivateInput
  • ActiveClientChange
  • AgentPropertyChange
  • BalloonHide
  • BalloonShow
  • Bookmark
  • Click
  • Command
  • DblClick
  • DeactivateInput
  • DefaultCharacterChange
  • DragComplete
  • DragStart
  • HelpComplete
  • Hide
  • IdleComplete
  • IdleStart
  • ListenComplete
  • ListenStart
  • Move
  • RequestComplete
  • RequestStart
  • Show
  • Size

The server no longer sends these Restart and Shutdown events.
