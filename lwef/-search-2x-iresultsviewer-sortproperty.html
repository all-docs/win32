IResultsViewer::SortProperty property

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

This property will set or return the IndexColumn of the property to sort results by.

This property is read/write.

Syntax

HRESULT put_SortProperty(
  [in]          BSTR column
);

HRESULT get_SortProperty(
  [out, retval] BSTR *column
);

Property value

Sets the IndexColumn property.
