DLP_POSTOP_STATUS structure

Specifies status information about an endpoint DLP operation.

Syntax

typedef struct _DLP_POSTOP_STATUS {
    
    DWORD Version;    
    BOOL OperationSuccess;  

} DLP_POSTOP_STATUS, *PDLP_POSTOP_STATUS;

Members

Version [in]

A DWORD specifying the API version. This value should always be DLP_POSTOP_STATUS_V_LATEST. This constant is defined in the endpointdlp.h sample header file listing in the article Endpoint data loss prevention

OperationSuccess [in]

A BOOL indicating whether the open operation was successful.
