Endpoint data loss prevention functions

The following endpoint data loss prevention APIs are deployed as part of Windows Defender. The Data Loss Prevention (DLP) feature requires a Windows Enterprise license to use.

Office interaction

This set of functions used by the endpoint DLP feature are intended to offer integration with Microsoft Office products.

Functions Description
DlpNotifyCloseDocument Provides the system with information about a document before the document close operation is initiated.
DlpNotifyCloseDocumentFile Provides the system with information about a document before the document close operation is initiated.
DlpNotifyEnterDropTarget Provides the system with information about a document when a drop target is entered.
DlpGetEnforcementApiVersion Retrieves the version of the endpoint Data Loss Prevention (DLP) API on the current device.
DlpInitializeEnforcementMode Notifies the system of the desired enforcement modes for a set of endpoint Data Loss Prevention (DLP) operations.
DlpMustPasteFromSystemClipboard Determines whether the app must pull the data from the system clipboard rather than taking it from its internal cache.
DlpNotifyLeaveDropTarget Provides the system with information about a document when a drop target is exited.
DlpNotifyPostCopyToClipboard Provides the system with information about a document after a copy to clipboard operation is completed.
DlpNotifyPostDragDrop Provides the system with information about a document after a drag drop operation is completed.
DlpNotifyPostOpenDocument Provides the system with information about a document after the open operation is completed.
DlpNotifyPostOpenDocumentFile Provides the system with information about a document after the open operation is completed.
DlpNotifyPostPasteFromClipboard Provides the system with information about a document after a paste from clipboard operation has completed.
DlpNotifyPostPrint Provides the system with information about a document after a print operation has completed.
DlpNotifyPostSaveAsDocument Provides the system with information about a document after the save as operation is completed.
DlpNotifyPostStartPrint Provides the system with information about a document after an print operation has started.
DlpNotifyPostStashClipboard Provides the system with status information after a stash clipboard operation is completed.
DlpNotifyPreCopyToClipboard Provides the system with information about a document before a copy to clipboard operation is initiated.
DlpNotifyPreDragDrop Provides the system with information about a document before a drag drop operation is initiated.
DlpNotifyPreOpenDocument Provides the system with information about a document before the open operation is initiated.
DlpNotifyPreOpenDocumentFile Provides the system with information about a document before the open operation is initiated.
DlpNotifyPrePasteFromClipboard Provides the system with information about a document before a paste from clipboard operation is initiated.
DlpNotifyPrePrint Provides the system with information about a document before a print operation is initiated.
DlpNotifyPreSaveDocument Provides the system with information about a document before a save as operation is initiated.
DlpNotifyPreStashClipboard Notifies the system before a stash clipboard operation is initiated.

Edge interaction

This set of functions used by the endpoint DLP feature are intended to offer integration with Microsoft Edge or other web browsers.

Functions Description
DlpInitialize Initializes the internal data structure for endpoint Data Loss Prevention (DLP) functions.
DlpGetFileCloudApplicationPolicy Returns the enforcement level of a file uploaded to a cloud application.
DlpGetFileCloudApplicationPolicyEx Returns the enforcement level of a file uploaded to a cloud application. This function is designed to be called from external sources such as Microsoft Edge, that do not have innate acess to the User SID. The SID will be detected from the currect process where endpointDlp is located.
DlpGetFileApplicationAccessEx Returns a list of DlpAction objects that contain the enforcement level of each operation in a specified file.
DlpAuditFileAccessEvent Deprecated - Generates an Enforcement event for the specified file.
DlpAuditFileAccessEventEx Deprecated - Generates an ETW event for the specified file, and allows a business justification to be provided for that enforcement event.
DlpAuditOperationEnforcementEvent Generates an ETW event for the specified file.
DlpAuditOperationEnforcementEventEx2 Generates an ETW event for the specified file, with additional parameter options.
DlpGetFileLocation Returns whether a specified file is local, on a network share, or on removeable media.
DlpDelegateEnforcement Allows DLP enforcement to be delegated to Windows Defender or to DLP-enlightened applications, as needed.
AuditBrowserOperationEvent Generates an ETW event for a browser operation.
AuditBrowserFileOperationEvent Generates an ETW event for a browser file operations.
GetBrowserExtensionConfiguration Returns whether DLP/IRM is enabled upon the browser.
DlpIsWebsitePolicyConfigured Returns whether any website rules have been configured for DLP.
DlpGetWebSiteAccess Returns whether any website rules have been configured for DLP.
DlpFreeArchiveFileTraceInfo Frees the members of a DlpTraceInfoEx data structure.
DlpGetNotificationSettings Returns the notification settings and strings for a given DLP rule and website action type.
DlpGetWebSiteNotificationSettings Returns the notification settings and strings for a given DLP rule and action type.
AuditBrowserWebSiteOperation Generates an ETW event for a browser website operation.
DlpIsSensitiveStream Checks to see is an input buffer pasted to a browser is sensitive.
