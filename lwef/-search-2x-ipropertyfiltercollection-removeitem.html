IPropertyFilterCollection::RemoveItem property

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

Removes a specific filter to the collection.

This property is write-only.

Syntax

HRESULT put_RemoveItem(
  [in] long index
);

Property value

Accepts a pointer to the index for the filter to be removed.
