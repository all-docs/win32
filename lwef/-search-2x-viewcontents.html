ViewContents enumeration

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

Used by IResultsViewer::Contents to indicate or set how the current query return set is displayed.

Syntax

typedef enum ViewContentsEnum { 
  ResultsDisplayed     = 0,
  ShellViewDisplayed   = 1,
  WebBrowserDisplayed  = 2
} ViewContents;

Constants

ResultsDisplayed

Indicates the type of contents being displayed in the results view.

ShellViewDisplayed

Indicates the type of contents being displayed in the results view is currently set to the Shell view.

WebBrowserDisplayed

Indicates the type of contents being displayed in the results view is currently set to the browser view.
