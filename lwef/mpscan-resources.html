MPSCAN_RESOURCES structure

Resource information passed during a scan operation.

Syntax

typedef struct tagMPSCAN_RESOURCES {
  DWORD            dwResourceCount;
  PMPRESOURCE_INFO pResourceList;
} MPSCAN_RESOURCES, *PMPSCAN_RESOURCES;

Members

dwResourceCount

Type: DWORD

Count of resources.

pResourceList

Type: PMPRESOURCE_INFO

Array of resources. See MPRESOURCE_INFO.

Requirements

Requirement Value
Minimum supported client Windows 8 [desktop apps only]
Minimum supported server Windows Server 2012 [desktop apps only]
