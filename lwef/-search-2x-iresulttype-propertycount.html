IResultType::PropertyCount property

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

This property contains the number of properties exposed by the type.

This property is read-only.

Syntax

HRESULT get_PropertyCount(
  [out, retval] long *count
);

Property value

returns the address of the number of properties exposed.
