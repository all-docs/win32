IResultVerb::HelpText property

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

This property returns a pointer to the localized help string for the verb.

This property is read-only.

Syntax

HRESULT get_HelpText(
  [out, retval] BSTR *text
);

Property value

The value of this property is a pointer to the localized help string for this verb.

Requirement Value
