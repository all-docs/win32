MPSCAN_RESULT structure

The results of a scan.

Syntax

typedef struct tagMPSCAN_RESULT {
  MPSCAN_TYPE      ScanType;
  MPSOURCE         Source;
  GUID             ScanGuid;
  ULARGE_INTEGER   StartTime;
  ULARGE_INTEGER   EndTime;
  MPTHREAT_STATS   ThreatStats;
  MPRESOURCE_STATS ResourceStats;
  ULONGLONG        SignatureVersion;
} MPSCAN_RESULT, *PMPSCAN_RESULT;

Members

ScanType

Type: MPSCAN_TYPE

Scan type. See MPSCAN_TYPE.

Source

Type: MPSOURCE

Scan source, such as user or system initiated. See MPSOURCE.

ScanGuid

Type: GUID

Scan identifier.

StartTime

Type: ULARGE_INTEGER

Scan start time.

EndTime

Type: ULARGE_INTEGER

Scan end time.

ThreatStats

Type: MPTHREAT_STATS

Threat-related statistics. See MPTHREAT_STATS.

ResourceStats

Type: MPRESOURCE_STATS

Resource-related statistics. See MPRESOURCE_STATS.

SignatureVersion

Type: ULONGLONG

Version of signature used for scan.
