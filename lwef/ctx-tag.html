Ctx Tag

[Microsoft Agent is deprecated as of Windows 7, and may be unavailable in subsequent versions of Windows.]

Description

Sets the context of the output text.

Syntax

\Ctx=string\

Part Description
string A string specifying the context of the text that follows, which determines how symbols or abbreviations are spoken. "Address" Addresses and/or phone numbers. "E-mail" Electronic mail. "Unknown" (Default) Context is unknown.

Remarks

This tag is supported only for TTS-generated output. The range of values for the parameter may vary depending on the installed TTS engine.
