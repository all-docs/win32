IResultVerb::Name property

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

This property returns a pointer to the cononical name for the verb such as print, open, etc.

This property is read-only.

Syntax

HRESULT get_Name(
  [out, retval] BSTR *name
);

Property value

name is a pointer to the cononical name for the verb.
