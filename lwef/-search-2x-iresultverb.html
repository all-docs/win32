IResultVerb interface

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

Exposes verb properties.

Members

The IResultVerb interface inherits from the IUnknown interface. IResultVerb also has these types of members:

  • Properties

Properties

The IResultVerb interface has these properties.

Property Access type Description
DisplayName Read-only This property returns a pointer to the localized display name for the verb.
DoIt Write-only Executes the verb.
Enabled Read-only This property returns TRUE if the verb is enabled.
HelpText Read-only This property returns a pointer to the localized help string for the verb.
Icon Read-only This property returns a pointer to handle of the optional icon associated with the verb.
Name Read-only This property returns a pointer to the canonical name for the verb such as print, open, and so forth.

Remarks

These methods expose properties and actions applicable to verbs.
