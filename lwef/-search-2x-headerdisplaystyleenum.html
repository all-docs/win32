Used by IResultsViewer::HeaderStyle to set or return the current header style.

Syntax

typedef enum HeaderDisplayStyleEnum { 
  FullHeader     = 0,
  CompactHeader  = 1,
  NoHeader       = 2
} HeaderDisplayStyle;

Constants

FullHeader

Indicates or sets the use of full headers.

CompactHeader

Indicates or sets the use of compact headers.

NoHeader

Indicates or sets the use of no headers.
