IResultType interface

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

Exposes property type information.

Members

The IResultType interface inherits from the IUnknown interface. IResultType also has these types of members:

  • Properties

Properties

The IResultType interface has these properties.

Property Access type Description
DisplayName Read-only Localized display name of the type:
GetProperty Read/write This property contains specified property information.
PreceivedType Read-only This property contains the string used to identify the type in the index.
PropertyCount Read-only This property contains the number of properties exposed by the type.
UID Read-only This property contains the unique identifier for the type.

Remarks

These methods expose the types of information returned.
