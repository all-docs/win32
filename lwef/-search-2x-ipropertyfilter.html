IPropertyFilter interface

[!NOTE] Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for WindowsÂ XP and Windows Server 2003. On later releases, use the Windows Search API instead.

Exposes properties used to filter the query.

Members

The IPropertyFilter interface inherits from the IUnknown interface. IPropertyFilter also has these types of members:

  • Properties

Properties

The IPropertyFilter interface has these properties.

Property Access type Description
IPropertyFilter::IndexColumn Read/write Indexed column name of the property to filter by.
IPropertyFilter::LogicOperator Read/write Logic Operator to use when applying filter.
IPropertyFilter::NestingDepth Read/write Filters depth within a nested set of parentheses.
IPropertyFilter::Text Read/write Text of the filter.
IPropertyFilter::UID Read/write UID fo the property to filter by.

Remarks

These properties are used to filter the query.
