IEnumWiaItem2::Reset method

Resets the enumeration reference to the first IWiaItem2 object.

Syntax

Parameters

This method has no parameters.

Return value

Type: HRESULT

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
