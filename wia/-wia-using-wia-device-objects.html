Using WIA Device Objects

An imaging device is represented in Windows Image Acquisition (WIA) as a hierarchical tree of WIA item objects (IWiaItem interfaces). Typically, the root of this item tree represents the device itself, while the other items on the tree represent images, for a camera, or scanning regions, for a scanner.

Applications use the WIA device manager to create and enumerate WIA devices. The following sections explain how to create and use WIA devices:

  • Selecting a Device
  • WIA Camera Devices
  • WIA Scanner Devices
