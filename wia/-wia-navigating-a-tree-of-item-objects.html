Navigating a Tree of Item Objects

[!Note]
This scripting system has been replaced with Windows Image Acquisition (WIA) Automation Layer. See Windows Image Acquisition Automation Layer.

Â 

An application or script navigates a Windows Image Acquisition (WIA) device's item tree to find and select images on that device. Using this technique, an application selects and acquires an image without the use of a dialog box.

A WIA device is represented as the root item in a tree of Item objects. The child items in the tree represent folders and images for a camera or scan beds for a scanner.

After connecting to a device, call the Children property of the Item object that represents the device (the root item of the tree) to obtain a collection of the child items (the images or scan beds) of the device.

Enumerate this collection (recursively, if necessary) to navigate item tree.
