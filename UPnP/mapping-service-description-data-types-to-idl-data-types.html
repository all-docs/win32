Mapping Service Description Data Types to IDL Data Types

The following table shows the mapping of XML data types specified in a service description to the corresponding data types used in IDL.

bin.base64 SAFEARRAY
bin.hex SAFEARRAY
boolean VARIANT_BOOL
char wchar_t
date DATE
dateTime DATE
dateTime.tz DATE
fixed.14.4 CY
float float
i1 char
i2 short
i4 long
int long
number BSTR
r4 float
r8 double
string BSTR
time DATE
time.tz DATE
ui1 unsigned char
ui2 unsigned short
ui4 unsigned long
uri BSTR
uuid BSTR
