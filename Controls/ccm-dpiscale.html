CCM_DPISCALE message

Enables automatic high dots per inch (dpi) scaling in Tree-View controls, List-View controls, ComboBoxEx controls, Header controls, Buttons, Toolbar controls, Animation controls, and Image Lists.

Parameters

wParam

Set to TRUE.

lParam

Must be zero.

Return value

The return value is not used.

Remarks

Quick Launch and Taskbar should not specify a dpi scaling, because the images are already scaled.

Any control that uses an image list created with the SmallIcon metric should not scale its icons.

[!Note]
To use this API, you must provide a manifest that specifies Comclt32.dll version 6.0. For more information on manifests, see Enabling Visual Styles.
