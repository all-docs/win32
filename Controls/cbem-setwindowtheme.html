CBEM_SETWINDOWTHEME message

Sets the visual style of a ComboBoxEx control.

Parameters

wParam

lParam

A pointer to a Unicode string that contains the control visual style to set.

Return value

The return value is not used.

Remarks

[!Note]
To use this message, you must provide a manifest specifying Comclt32 version 6.0. For more information on manifests, see Enabling Visual Styles.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
