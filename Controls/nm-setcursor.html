NM_SETCURSOR notification code

Notifies a control's parent window that the control is setting the cursor in response to a WM_SETCURSOR message. This notification code is sent in the form of a WM_NOTIFY message.

NM_SETCURSOR

    lpnmm = (LPNMMOUSE) lParam;

Parameters

lParam

A pointer to an NMMOUSE structure that contains additional information about this notification.

Return value

Return zero to enable the control to set the cursor or nonzero to prevent the control from setting the cursor.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
