HDN_TRACK notification code

Notifies a header control's parent window that the user is dragging a divider in the header control. This notification code is sent in the form of a WM_NOTIFY message.

HDN_TRACK

    pNMHeader = (LPNMHEADER) lParam;

Parameters

lParam

A pointer to an NMHEADER structure that contains information about the header control and the item whose divider is being dragged.

Return value

Returns FALSE to continue tracking the divider, or TRUE to end tracking.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
