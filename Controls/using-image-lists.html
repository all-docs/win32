Using Image Lists

This section contains example code and information about how to create and use image lists.

In this section

Topic Description
How to Create an Image List This topic demonstrates how to use the ImageList_Create function to create an image list.
How to Draw an Image This topic demonstrates how to use the ImageList_Draw function to draw an image.
How to Drag an Image This topic demonstrates how to drag an image on the screen. The dragging functions move an image smoothly, in color, and without any flashing of the cursor. Both masked and unmasked images can be dragged.
