PSM_INDEXTOID message

Takes the index of a property sheet page and returns its resource ID. You can send this message explicitly or use the PropSheet_IndexToId macro.

Parameters

wParam

Zero-based index of the page.

lParam

Must be zero.

Return value

Returns the resource ID of the property sheet page specified by wParam if successful. Otherwise, it returns zero.

Requirements

Requirement Value

Minimum supported server Windows Server 2003 [desktop apps only]
