DTM_SETMCSTYLE message

Sets the style of a date and time picker (DTP) control. Send this message explicitly or by using the DateTime_SetMonthCalStyle macro.

Parameters

wParam

Must be zero.

lParam [in]

Return value

Returns the value of the previous style.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2008 [desktop apps only]
