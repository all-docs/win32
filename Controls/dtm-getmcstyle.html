DTM_GETMCSTYLE message

Gets the style of a date and time picker (DTP) control. Send this message explicitly or by using the DateTime_GetMonthCalStyle macro.

Parameters

wParam

Must be zero.

lParam

Return value

Returns the style value of the control. For more information see Month Calendar Control Styles.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2008 [desktop apps only]
