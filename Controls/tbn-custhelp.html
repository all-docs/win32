TBN_CUSTHELP notification code

Notifies a toolbar's parent window that the user has chosen the Help button in the Customize Toolbar dialog box. This notification code is sent in the form of a WM_NOTIFY message.

TBN_CUSTHELP 

    lpnmhdr = (LPNMHDR) lParam;

Parameters

lParam

Pointer to an NMHDR structure that contains information about the notification code.

Return value

No return value.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
