TDM_SET_PROGRESS_BAR_STATE message

Sets the state of the progress bar in a task dialog.

Parameters

wParam [in]

An int that specifies the state of the progress bar. This parameter can be one of the following values.

Value Meaning
PBST_NORMAL Sets the progress bar to the normal state.
PBST_PAUSED Sets the progress bar to the paused state.
PBST_ERROR Set the progress bar to the error state.

lParam [in]

Must be zero.

Return value

If the function succeeds, the return value is non zero.

If the function fails, the return value is zero. To get extended error information call GetLastError.
