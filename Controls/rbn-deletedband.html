RBN_DELETEDBAND notification code

Sent by a rebar control after a band has been deleted. This notification code is sent in the form of a WM_NOTIFY message.

RBN_DELETEDBAND

    lpnmrb = (LPNMREBAR) lParam;

Parameters

lParam

Pointer to an NMREBAR structure that contains information about the notification code.

Return value

The return value for this notification is not used.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
