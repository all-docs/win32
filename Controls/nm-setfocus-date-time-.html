NM_SETFOCUS (date time) notification code

Notifies a date and time picker control's parent window that the control has received the input focus. This notification code is sent in the form of a WM_NOTIFY message.

NM_SETFOCUS 

    lpnmh = (LPNMHDR) lParam;

Parameters

lParam

A pointer to an NMHDR structure that contains additional information about this notification.

Return value

The return value is ignored.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
