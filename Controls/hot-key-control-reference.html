Hot Key

This section contains information about the programming elements used with hot key controls.

Overviews

Topic Contents
Hot Key Controls A hot key control is a window that enables the user to enter a combination of keystrokes to be used as a hot key.

Messages

Topic Contents
HKM_GETHOTKEY Gets the virtual key code and modifier flags of a hot key from a hot key control.
HKM_SETHOTKEY Sets the hot key combination for a hot key control.
HKM_SETRULES Defines the invalid combinations and the default modifier combination for a hot key control.
