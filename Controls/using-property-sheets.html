Using Property Sheets

This section contains topics that demonstrate how to create and use property sheets.

In this section

Topic Description
How to Create a Property Sheet The example in this section creates a property sheet that contains two pages one for setting the font properties of a cell in a spreadsheet, and another for setting the border properties of the cell.
How to Process Notification Messages A property sheet sends WM_NOTIFY messages to retrieve information from the pages and to notify the pages of user actions.
How to Create Wizards A wizard is a type of property sheet that provides a simple and powerful way to guide users through a procedure.
