NM_RELEASEDCAPTURE (toolbar) notification code

Notifies a toolbar control's parent window that the control is releasing mouse capture. This notification code is sent in the form of a WM_NOTIFY message.

NM_RELEASEDCAPTURE

    lpnmh = (LPNMHDR) lParam;

Parameters

lParam

Pointer to an NMHDR structure that contains additional information about this notification.

Return value

The control ignores the return value from this notification code.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
