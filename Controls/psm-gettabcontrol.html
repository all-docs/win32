PSM_GETTABCONTROL message

Retrieves the handle to the tab control of a property sheet. You can send this message explicitly or by using the PropSheet_GetTabControl macro.

Parameters

wParam

Must be zero.

lParam

Must be zero.

Return value

Returns the handle to the tab control.

Remarks

[!Note]
This message is not supported when using the Aero wizard style (PSH_AEROWIZARD).

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
