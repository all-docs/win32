MCM_GETCALID message

Gets the calendar ID for the given calendar control. You can send this message explicitly or by using the MonthCal_GetCALID macro.

Parameters

wParam

Must be zero.

lParam

Must be zero.

Return value

ID of the current calendar. One of the Calendar Identifiers constants.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2008 [desktop apps only]
