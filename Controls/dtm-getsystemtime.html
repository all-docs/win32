DTM_GETSYSTEMTIME message

Gets the currently selected time from a date and time picker (DTP) control and places it in a specified SYSTEMTIME structure. You can send this message explicitly or use the DateTime_GetSystemtime macro.

Parameters

wParam

lParam

A pointer to a SYSTEMTIME structure. If DTM_GETSYSTEMTIME returns GDT_VALID, this structure will contain the currently selected time. Otherwise, it will not contain valid information. This parameter must be a valid pointer; it cannot be NULL.

Return value

Returns GDT_VALID if the time information was successfully placed in lParam. Returns GDT_NONE if the control was set to the DTS_SHOWNONE style and the control check box was not selected. Returns GDT_ERROR if an error occurs.
