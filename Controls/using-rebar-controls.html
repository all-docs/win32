Using Rebar Controls

This section contains topics that demonstrate how to create and use rebar controls.

In this section

Topic Description
How to Create Rebar Controls An application creates a rebar control by calling the CreateWindowEx function, specifying REBARCLASSNAME as the window class. The application must first register the window class by calling the InitCommonControlsEx function, specifying the ICC_COOL_CLASSES bit in the accompanying INITCOMMONCONTROLSEX structure.
