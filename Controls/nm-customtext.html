NM_CUSTOMTEXT notification code

Notifies a control's parent window about custom text operations. This notification code is sent in the form of a WM_NOTIFY message.

NM_CUSTOMTEXT

    lpnmct = (NMCUSTOMTEXT) lParam;

Parameters

lParam

A pointer to an NMCUSTOMTEXT structure that contains additional information about this notification.

Return value

The return value is ignored by the control.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2008 [desktop apps only]
