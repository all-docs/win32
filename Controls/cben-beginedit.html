CBEN_BEGINEDIT notification code

Sent when the user activates the drop-down list or clicks in the control's edit box. This notification code is sent in the form of a WM_NOTIFY message.

CBEN_BEGINEDIT

    lpnmhdr = (LPNMHDR) lParam;

Parameters

lParam

A pointer to an NMHDR structure that contains information about the notification code.

Return value

The application processing this notification code must return zero.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
