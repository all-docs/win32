NM_CLICK (tab) notification code

Notifies the parent window of a tab control that the user has clicked the left mouse button within the control. This notification code is sent in the form of a WM_NOTIFY message.

NM_CLICK
        
    lpnmh = (LPNMHDR) lParam;

Parameters

lParam

Pointer to an NMHDR structure that contains additional information about this notification.

Return value

The return value is ignored by the tab control.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
