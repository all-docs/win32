WTNCA Values

Specifies flags that modify window visual style attributes. Use one, or a bitwise combination of the following values.

Constant/value Description
WTNCA_NODRAWCAPTION 0x00000001 Prevents the window caption from being drawn.
WTNCA_NODRAWICON 0x00000002 Prevents the system icon from being drawn.
WTNCA_NOSYSMENU 0x00000004 Prevents the system icon menu from appearing.
WTNCA_NOMIRRORHELP 0x00000008 Prevents mirroring of the question mark, even in right-to-left (RTL) layout.
WTNCA_VALIDBITS A mask that contains all the valid bits.
