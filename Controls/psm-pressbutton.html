PSM_PRESSBUTTON message

Simulates the selection of a property sheet button. You can send this message explicitly or by using the PropSheet_PressButton macro.

Parameters

wParam

Index of the button to select. This parameter can be one of the following values.

Value Meaning
PSBTN_APPLYNOW Selects the Apply button. This value is not valid when using the Aero wizard style (PSH_AEROWIZARD).
PSBTN_BACK Selects the Back button.
PSBTN_CANCEL Selects the Cancel button.
PSBTN_FINISH Selects the Finish button.
PSBTN_HELP Selects the Help button. This value is not valid when using the Aero wizard style (PSH_AEROWIZARD).
PSBTN_NEXT Selects the Next button.
PSBTN_OK Selects the OK button. This value is not valid when using the Aero wizard style (PSH_AEROWIZARD).

lParam

Must be zero.

Return value

No return value.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
