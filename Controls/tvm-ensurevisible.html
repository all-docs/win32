TVM_ENSUREVISIBLE message

Ensures that a tree-view item is visible, expanding the parent item or scrolling the tree-view control, if necessary. You can send this message explicitly or by using the TreeView_EnsureVisible macro.

Parameters

wParam

lParam

Handle to the item.

Return value

Returns nonzero if the system scrolled the items in the tree-view control and no items were expanded. Otherwise, the message returns zero.

Remarks

If the TVM_ENSUREVISIBLE message expands the parent item, the parent window receives the TVN_ITEMEXPANDING and TVN_ITEMEXPANDED notification codes.
