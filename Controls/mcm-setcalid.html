MCM_SETCALID message

Sets the calendar ID for the given calendar control. You can send this message explicitly or by using the MonthCal_SetCALID macro.

Parameters

wParam

The calendar ID. One of the Calendar Identifiers constants.

lParam

Must be zero.

Return value

Unused.

Requirement Value

Minimum supported server Windows Server 2008 [desktop apps only]
