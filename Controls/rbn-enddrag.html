RBN_ENDDRAG notification code

Sent by a rebar control when the user stops dragging a band. This notification code is sent in the form of a WM_NOTIFY message.

RBN_ENDDRAG

    lpnmrb = (LPNMREBAR) lParam;

Parameters

lParam

Pointer to an NMREBAR structure that contains information about the notification code.

Return value

The return value for this notification code is not used.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
