DTM_GETIDEALSIZE message

Gets the size needed to display the control without clipping. Send this message explicitly or by using the DateTime_GetIdealSize macro.

Parameters

wParam

Not used. Must be zero.

lParam

A pointer to a SIZE structure to receive the ideal size. The calling application is responsible for allocating this structure.

Return value

Returns TRUE.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2008 [desktop apps only]
