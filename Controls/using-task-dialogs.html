Using Task Dialogs

This section contains implementation details for using task dialog controls in your applications.

In this section

Topic Description
How to Create Task Dialogs A task dialog is created and shown by using either the TaskDialog function or the TaskDialogIndirect function.
How to Get User Input from a Task Dialog To complete a task, users submit the task details to the application by configuring the controls within the task dialog, and then clicking a command button (usually OK).
