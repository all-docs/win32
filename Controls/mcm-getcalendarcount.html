MCM_GETCALENDARCOUNT message

Gets the number of calendars currently displayed in the calendar control. You can send this message explicitly or by using the MonthCal_GetCalendarCount macro.

Parameters

wParam

Must be zero.

lParam

Must be zero.

Return value

Number of calendars currently displayed in the calendar control. The maximum number of allowed calendars is 12.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2008 [desktop apps only]
