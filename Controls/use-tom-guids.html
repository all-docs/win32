How to Use TOM GUIDs

Text Object Model (TOM)Â GUIDs are given in Tom.h inside the MIDL_INTERFACE statements. To use the associated interfaces, you must first declare the interface by using the GUID.

Instructions

Use a TOM GUID

The following example code demonstrates how to use the ITextDocument interface.

#define DEFINE_GUIDXXX(name, l, w1, w2, b1, b2, b3, b4, b5, b6, b7, b8) \
        EXTERN_C const GUID CDECL name \
                = { l, w1, w2, { b1, b2,  b3,  b4,  b5,  b6,  b7,  b8 } }

DEFINE_GUIDXXX(IID_ITextDocument,0x8CC497C0,0xA1DF,0x11CE,0x80,0x98,
                0x00,0xAA,0x00,0x47,0xBE,0x5D);
