RB_GETBANDMARGINS message

Retrieves the margins of a band.

Parameters

wParam

lParam

Pointer to a MARGINS structure that receives the retrieved margins.

Return value

The return value is not used.

Remarks

[!Note]
To use this message, you must provide a manifest specifying Comclt32.dll version 6.0. For more information on manifests, see Enabling Visual Styles.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2003 [desktop apps only]
