Using Windowless Rich Edit Controls

This section provides example code and information about how to enable window functionality in a windowless rich edit control.

In this section

Topic Description
How to Create Windowless Rich Edit Controls Window functionality includes the ability to receive messages and the ownership of a device context into which to draw. Windowless rich edit controls receive this functionality by way of a pair of interfaces: ITextServices and ITextHost.
