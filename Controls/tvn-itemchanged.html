TVN_ITEMCHANGED notification code

Notifies a tree-view control's parent window that item attributes have changed. This notification code is sent in the form of a WM_NOTIFY message.

TVN_ITEMCHANGED
        
    pnm = (NMTVITEMCHANGE *) lParam;

Parameters

lParam

Pointer to a NMTVITEMCHANGE structure describing the item that changed. The uChanged member is set to TVIF_STATE.

Return value

Returns FALSE to accept the change, or TRUE to prevent the change.

Requirements

Requirement Value
Minimum supported client Windows Vista [desktop apps only]
Minimum supported server Windows Server 2008 [desktop apps only]
Header Commctrl.h
Unicode and ANSI names TVN_ITEMCHANGEDW (Unicode) and TVN_ITEMCHANGEDA (ANSI)
