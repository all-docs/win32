Using SysLink Controls

This section contains example code and information about how to create and use SysLink controls.

In this section

Topic Description
How to Create SysLink Controls You implement the SysLink control's hyperlinks through markup in the control's initialization string, or by sending it a LM_SETITEM message.
How to Use SysLink Notifications There are two notification messages associated with the SysLink control one for the mouse (NM_CLICK (syslink)), and one for the keyboard (NM_RETURN).
