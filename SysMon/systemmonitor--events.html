SystemMonitor Events

The SystemMonitor class has the following events:

  • OnCounterAdded
  • OnCounterDeleted
  • OnCounterSelected
  • OnDblClick
  • OnSampleCollected

[!Note]
You must return from the event handler before the update interval expires; otherwise, SYSMON displays a message box indicating to the user that it was unable to sample counter values for the previous update interval.
