SystemMonitor::BrowseCounters method

Displays the Add Counters dialog box.

Syntax

Sub BrowseCounters()

Parameters

This method has no parameters.

Return value

This method does not return a value.

Remarks

This dialog box lets the user select local or remote performance counters from a list of performance objects. The selected counters are added to the Counters collection and displayed in the graph or report.

To receive notification when a counter is added, implement the OnCounterAdded event.
