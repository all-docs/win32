What's New in System Monitor

The following table identifies what is new for each release of System Monitor (SYSMON).

Version Description of features
Version 3.7 This version adds new graph types, the ability to select multiple counters, retrieve counter values from a point on the graph, save graphed counter values to a log file, and the option to have a line graph continuously scroll in the graph window instead of wrap-around on itself.Included in Windows Server 2008 and Windows Vista.

Version 3.7

New properties and methods that were added to CounterItem.

  • GetDataAt
  • Selected
  • Visible
  • Range of valid values changed for CounterItem.Width

New properties and methods that were added to SystemMonitor.

  • BatchingLock
  • ChartScroll
  • ClearData
  • DataPointCount
  • EnableDigitGrouping
  • EnableToolTips
  • GetLogViewRange
  • LoadSettings
  • LogSourceStartTime
  • LogSourceStopTime
  • Relog
  • SaveAs
  • ScaleToFit
  • SetLogViewRange
  • ShowTimeAxisLables

New enumerations that were added.

  • SysmonDataType
  • SysmonFileType
  • New display type values were added to DisplayTypeConstants
