SystemMonitor.OnCounterAdded event

Notifies you when a counter is added to the Counters collection.

Syntax

SystemMonitor.OnCounterAdded( _
  ByVal index As Long _
)

Parameters

index [in]

Index of the counter added to the Counters collection object.

Return value

This event does not return a value.
