SystemMonitor.GridColor property

Retrieves or sets the color of the grid lines used in the graph.

This property is read-only.

Syntax

Property GridColor As stdole.OLE_COLOR

Property value

Color of the horizontal and vertical grid lines used in a graph if the ShowHorizontalGrid or ShowVerticalGrid properties are selected.

Requirements

Requirement Value
Minimum supported client Windows 2000 Professional [desktop apps only]
Minimum supported server Windows 2000 Server [desktop apps only]
