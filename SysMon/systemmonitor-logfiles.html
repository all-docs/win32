SystemMonitor.LogFiles property

Collection of one or more log files to use as the source of counter values displayed in the System Monitor.

Syntax

Property LogFiles As ILogFiles

Property value

Collection of LogFileItem objects that specify the log files used as the data source for the graph.

Remarks

To add or remove a log file from the log file collection, the value of SystemMonitor.DataSourceType must not be set to sysmonLogFiles.

Requirements

Requirement Value

Minimum supported server Windows 2000 Server [desktop apps only]
