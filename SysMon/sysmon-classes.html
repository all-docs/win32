SYSMON Classes

You can use the following classes to configure the System Monitor (SYSMON) control.

Class name Description
CounterItem Provides counter item property information.
Counters Provides the methods for managing a collection of CounterItem objects.
LogFileItem Provides a property for retrieving the path of a single log file accessed by the System Monitor control.
LogFiles Provides the methods and properties for managing a collection of LogFileItem objects.
SystemMonitor Provides the methods and properties used to configure the System Monitor control.
